## 二、用户角色体系（双视图核心）

小程序根据登录用户的角色自动切换首页和 TabBar 布局：

| 层级 | 角色名 | 说明 | 首页视图 |
|------|--------|------|---------|
| L0 | 观察员 Observer | 体验入口，9次/天AI配额（对话3+拍照3+语音3） | 学习者视图 |
| L1 | 成长者 Grower | 完成基础学习，开始微行动打卡 | 学习者视图 |
| L2 | 分享者 Sharer | 激励终点，带动周围用户 | 学习者视图 |
| L3 | 初级教练 Coach | 可管理学员，进入工作台 | 教练工作台视图 |
| L4 | 中级教练 / 高级 | 更多权限 | 教练工作台视图 |
| L5 | 督导 Supervisor | 最高权限，审核晋级 | 教练工作台视图 |

## 四、功能模块分类与功能说明

### 4.1 认证与授权模块
**路径：** `src/pages/auth/`

| 页面文件 | 功能说明 |
|---------|---------|
| `login.vue` | 账号密码登录 + 微信一键登录（wx.login获取code → 后端换取JWT） |
| `register.vue` | 新用户注册，支持邀请码绑定推荐关系 |

**API接口：** `POST /api/v1/auth/login`、`POST /api/v1/mp/wx-login`、`POST /api/v1/auth/refresh`

---

### 4.2 首页（双视图自适应）
**路径：** `src/pages/home/index.vue`

根据 `isCoach`（L3+）自动切换：
- **学习者视图**：今日打卡状态、推荐课程、AI对话入口、积分概览
- **教练工作台视图**：待处理任务数、学员动态、推送审批队列入口、数据概览

**TabBar 双模式：**
- 学习者：首页 / 学习 / AI对话 / 消息 / 我的
- 教练端：工作台 / 学员 / 推送队列 / 消息 / 我的

---

### 4.3 学习中心模块
**路径：** `src/pages/learning/`（分包）

| 页面文件 | 功能说明 |
|---------|---------|
| `index.vue` | 学习中心首页，按学习路径展示课程进度 |
| `catalog.vue` | 课程目录，支持分类筛选；等级门控（等级不足显示锁定遮罩） |
| `course-detail.vue` | 课程详情，展示章节列表、课程简介、已学进度 |
| `course-chapter.vue` | 章节详情，支持章节顺序锁定（完成前章才解锁后章） |
| `content-detail.vue` | 图文内容详情，支持TTS一键朗读（edge-tts后端已实现） |
| `video-player.vue` | 视频播放（**核心**：30s心跳断点续播 + 倍速 + 弹幕 + 进度≥90%弹出随堂测验） |
| `audio-player.vue` | 音频播放页 |
| `quiz.vue` | 随堂测验，完成视频后触发，题目来自课程题库 |
| `quiz-result.vue` | 随堂测验结果，显示得分与解析 |
| `my-learning.vue` | 我的学习记录，已学/在学/待学分类 |
| `credits.vue` | 学分明细，展示三维积分（行动积分/学习积分/影响积分） |

**关键API：** 
- `GET /api/v1/content/:id` 获取内容
- `POST /api/v1/learning/heartbeat` 30s心跳断点（视频播放器核心）
- `GET /api/v1/credits/summary` 三维积分汇总

### 4.4 考试系统模块
**路径：** `src/pages/exam/`（分包）

| 页面文件 | 功能说明 |
|---------|---------|
| `index.vue` | 考试列表，显示可参加的认证考试 |
| `intro.vue` | 考试说明页，展示题目数量/时间限制/考试规则 |
| `session.vue` | **核心考试作答页**：单选+多选+案例分享题；每题选完立即保存（防丢失）；倒计时自动提交 |
| `result.vue` | 考试结果页，显示得分/通过状态/知识点分析 |
| `history.vue` | 历史考试记录 |

**题型支持：**
- 单选题（A/B/C/D）
- 多选题（多选确认机制，防误点）
- 案例分享题（图文+视频形式）

**防丢失策略：** 每题答完立即调用 `POST /api/v1/exam/sessions/:id/answer`，断网/切后台不丢答案

### 4.9 AI 功能模块

AI功能分布在多个页面中，不独立为一个分包。

| 功能 | 技术实现 | 所在页面 |
|------|---------|---------|
| AI对话入口 | SSE流式输出（打字机效果），`/api/v1/agent/chat` | 首页浮窗 + 底部Tab |

| 语音输入 | `POST /api/v1/audio/transcribe`（webm/m4a格式） | AI对话页 |
| AI消息建议 | 教练点击获取AI建议文案，一键填入消息编辑框 | 学员详情-消息Tab |
| AI评估建议 | 审核处方时展示AI生成建议，教练可采纳/修改 | 评估审核页 |
| TTS朗读 | 图文内容一键转语音，`/api/v1/content/:id` + content_audio表 | 图文内容详情页 |
| AI使用配额展示 | 观察员9次/天（对话3+拍照3+语音3），超限引导注册 | AI对话页 |

**注意：** 微信模板消息推送后端未实现，App内通知依赖轮询机制
