### 核心框架

NLP（神经语言程序学，Neuro-Linguistic Programming）由美国班德勒和格林德尔于1976年创立，是**对人类主观经验的研究**——研究大脑如何工作，以及如何提升其运作效率，使人生更成功快乐。

李中莹将NLP浓缩为**十二条前提假设**，这不是NLP要你盲目接受的"真理"，而是**一套处事心态**——假定它们成立，凭结果来决定是否继续采用。香港NLP之父徐志忠神父称之为"NLP的免疫系统"。

**十二条前提假设（BHP应用版）**：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【关于人与人的关系】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① 没有两个人是一样的
   No two persons are the same.
   → BHP含义：每个慢性病患者的"失败原因"都不同，不能套用统一处方

② 一个人不能控制另外一个人
   One person cannot change another person.
   → BHP含义：CoachAgent不能"教"用户健康，只能"引导"用户自我发现

③ 沟通的意义决定于对方的回应
   The meaning of communication is the response one gets.
   → BHP含义：如果用户没有改变，问题在干预方式，而非用户"不配合"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【关于行动与改变】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

④ 有效用比有道理更重要
   Usefulness is more important than being right.
   → BHP含义：讲道理没有用，找到对这个用户"有效"的干预才重要

⑤ 重复旧的做法，只会得到旧的结果
   Repeating the same behavior will repeat the same result.
   → BHP含义：行为改变的第一步是意识到"换一种做法"

⑥ 凡事必有至少三个解决方法
   There are at least three solutions to every situation.
   → BHP含义：当用户说"没有办法"时，帮助他们找出第三条路

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【关于人的本质与潜能】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⑦ 只有由感官经验塑造出来的世界，没有绝对的真实世界
   The map is not the territory.
   → BHP含义：用户对疾病的"感知地图"比疾病本身更影响他们的行为

⑧ 每一个人都选择给自己最佳利益的行为
   Every one chooses the best behavior at the moment.
   → BHP含义：所有"不健康行为"背后都有正面动机——理解它，才能改变它

⑨ 每个人都已经具备使自己成功快乐的资源
   Every one already possesses all the resources needed.
   → BHP含义：用户不缺资源，缺的是发现和调用内在资源的路径

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【关于系统与反馈】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⑩ 在任何一个系统里，最灵活的部分便是最能影响大局的部分
   In any system, the most flexible person has the control.
   → BHP含义：CoachAgent应成为系统中最灵活的那个——不固执于某种方法

⑪ 没有挫败，只有回应信息
   There is no failure, only feedback.
   → BHP含义：健康管理的"失败"是改变策略的信号，不是定性结论

⑫ 动机和情绪总不会错，只是行为没有效果而已
   Intentions and emotions are never wrong, only the behavior has not been effective.
   → BHP含义：接纳用户的动机和情绪，帮助他找到"有效果的新行为"
```

**NLP的核心精神**：NLP不追求"真理"或"真相"，追求效果——在"我好、你好、世界好"（三赢）的基础上的效果。

**技术升华**：NLP最终超越技巧，上升为一种**心态**——对生活、对人生、对他人的每一件事都有效的心态。这与BHP的S6目标（活出生命意义的主人）高度呼应。

### 平台应用价值

NLP十二假设为BHP提供了**底层世界观框架**：

- 解释为何慢病用户的"问题行为"不是道德问题，而是资源/信念/系统问题
- 为CoachAgent提供"不评判、不说教、只引导"的行为准则
- 第⑧条（行为背后有正面动机）直接对应BHP动机性访谈（MI）的精神
- 第⑨条（人人有内在资源）对接BHP三层价值架构的"成长层"

## KI-IDENT-NLP-001 信念/价值观/规条三层结构：身份重塑的底层图谱

<!-- 领域:身份认同 | 来源:李中莹《重塑心灵》第二章·信念系统 | 证据:T2 | 来源等级:C -->
<!-- 适用阶段:S3,S4,S5 | 成长维度:discover_self,realize_self -->
<!-- 文化有效性:High | 反思提问:5个 | 身份连接:有 | CoachAgent触发:信念解构 -->

### 核心框架

李中莹将维持一个人在世界中生活的"内在法则"总结为**信念系统（Belief System）**，由三个层次构成：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第一层：信念（Beliefs）
  定义："事情应该是怎样的" / "事情就是这样的"
  性质：对这个人来说等同于"真理"，存于潜意识，绝大多数无法被意识到
  作用：解释和支持行动/不行动的理由，世界运作的主观逻辑定律

  慢性病场景示例：
  ✗ 限制性信念："血糖高是遗传的，我管不了"
  ✗ 限制性信念："年纪大了，健康就是往下走，这是正常的"
  ✗ 限制性信念："改变健康习惯太痛苦了，不值得"
  ✓ 成长信念："我的行为选择每天都在影响我的健康"
  ✓ 成长信念："每一次的'失败'都是我学到东西的证明"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第二层：价值观（Values）
  定义：一个人对某件事"在乎什么"的排序体系
  性质：是行为的真正驱动力；有正面（想追求的）和负面（想逃避的）两类
  关键洞见：意识层的价值排序≠潜意识层的价值排序——这是"知道但做不到"的根本原因

  示例：
  某用户意识上说"我最在乎健康"，
  潜意识里却把"被家人认可"和"不失去社交乐趣"放在更高位置。
  → 于是，每当健康行为与社交冲突时，社交胜出，健康失败。

  价值观的三种操作方式：
  ① 创造价值：把枯燥的健康行为变成有新奇感、挑战感的体验
  ② 增大价值：帮用户看到健康行为对他最在乎的事（家人/事业/自由）的贡献
  ③ 转移价值：把"为了避免医生批评"改为"为了让自己有精力陪孩子长大"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第三层：规条（Rules）
  定义：事情的安排方式和做法；规条完全是为了取得价值、实现信念而存在
  核心悖论：当规条无效时，人们应该坚持信念价值、改变规条——
            但现实是，人们往往坚持无效的规条，而背离了信念价值

  示例：
  信念："我要健康"（信念）→ 因为健康能给我"自由"（价值）
  规条A："每天跑5公里"（无效：膝盖有伤，坚持跑→受伤更重→放弃）
  规条B："每天走8000步"（有效：用不同的规条实现同样的价值）

  → 很多"意志力不足"的问题，其实是"规条固执"的问题：
    不是不想改变，而是只认为某一种做法才是"正确"的

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
三层结构的关系图：

    信念（为什么这件事应该这样）
         ↓
    价值观（这件事对我的意义/我从中得到什么）
         ↓
    规条（具体怎么做）

    当规条无效时 → 换规条（不是换信念和价值）
    当价值观模糊时 → 澄清价值观（才能找到真正的驱动力）
    当信念限制时 → 松动信念（为成长腾出空间）
```

**NLP改变信念的关键技巧——"价值定位法"核心步骤**：

1. 找出这件事对你的正面价值（"做到了会给你带来什么？"）
2. 找出这件事的负面价值（"不做会让你失去什么？"）
3. 澄清正负价值的轻重排序
4. 发现意识价值与潜意识价值的差距
5. 用"创造/增大/转移价值"方式重建驱动力

### 平台应用价值

这个框架直接解释了BHP"知道但做不到"的核心机制——不是知识不够，而是**潜意识价值排序**与意识目标相矛盾。BHP的行为处方设计应在以下层次发力：

- **规条层**：提供灵活多样的行为选项（而非单一"标准做法"）
- **价值层**：帮用户发现健康行为与他最在乎的事的连接
- **信念层**：S4-S5阶段的深层身份工作

## KI-GROWTH-NLP-002 自我价值三要素：自信/自爱/自尊的建立路径

<!-- 领域:成长超越 | 来源:李中莹《重塑心灵》第三章·自我价值 | 证据:T2 | 来源等级:C -->

<!-- 文化有效性:High | 反思提问:4个 | 身份连接:有 -->
<!-- 注意：铁律7——自我价值内容在S0-S1急性期不作主线 -->

### 核心框架

李中莹将**自我价值（Self-Worth）**定义为支撑一个人"身份能量"的核心素质，包含三个维度：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
自我价值三要素
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

自信（Self-Confidence）
  定义：相信自己有能力处理所面临的挑战
  与慢病的关系：
    • 自信不足 → "我控制不了自己的身体/习惯"
    • 自信过度（自负）→ "我知道这些，不用改" → 也是自我价值不足的表现
  建立路径：通过小目标成功积累经验，每次成功都是自信的基础
  
自爱（Self-Love）
  定义：接受并善待自己，包括自己的不完美和过去的错误
  与慢病的关系：
    • 自爱不足 → 忽视身体信号/自我惩罚式放弃/不值得被健康对待
    • 关键洞见：很多慢病患者潜意识里有"我不值得健康"的隐藏信念
  建立路径："接受自己法"——与过去的自己和解（见KI-GROWTH-NLP-004）

自尊（Self-Esteem）
  定义：认为自己作为一个人有内在价值，不因外在表现而改变
  与慢病的关系：
    • 自尊低 → 把疾病定义为"我的失败/我的羞耻"
    • 自尊低 → 医患关系中习惯性顺从，不敢表达真实需求
  建立路径：区分"我这个人"与"我的行为/状态"——健康状况不等于人的价值

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
自我价值不足的行为模式（BHP常见场景）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 对自己的健康状况感到羞耻，不愿意告诉家人
• 在家庭/工作中总是先满足别人，把自己的健康需求放最后
• 医生说什么就做什么，没有自己的判断（他律而非自律）
• 用不健康行为"惩罚"自己（暴食/熬夜/不运动）
• "反正我就是这样的人，改不了"式的自我放弃

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
身份与角色的关系（李中莹"钻石模型"）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一个人在人生里只有一个"身份"（整颗钻石），
但有很多个"角色"（钻石的每个切面）：
• 作为父母的角色 / 作为子女的角色
• 作为患者的角色 / 作为健康管理者的角色
• 作为员工的角色 / 作为自我的角色

关键洞见：
很多人把"慢病患者"这个角色扩展成了整个身份——
"我是一个糖尿病人"变成了"我这个人就是坏掉的"。
BHP的身份工作是帮助用户区分：
  "我有糖尿病" ≠ "我是一个糖尿病人"
  "我目前健康状态不好" ≠ "我这个人没有价值"
```

**最快建立自我价值的方法**（李中莹总结）：

1. **获得充分的肯定**：被他人看见和认可是建立自我价值最快的路径（BHP社群功能的底层逻辑）
2. **发现自己的能力**：帮助用户看见他们已经做到的事，而不只聚焦于失败
3. **接受自己的不完美**：完美主义是自我价值建设的最大敌人

## KI-GROWTH-NLP-003 迪尔茨六层理解层次：行为改变的精准分层框架

<!-- 领域:成长超越 | 来源:李中莹《重塑心灵》第四章·系统与理解层次 | 证据:T2 | 来源等级:C -->
<!-- 原创者：Robert Dilts（1991）；李中莹华人化应用 | 证据:T2 -->
<!-- 适用阶段:S2,S3,S4,S5,S6 | 成长维度:discover_self,realize_self,transcend_self -->
<!-- 文化有效性:High | 反思提问:5个 | 身份连接:有 | 可视化工具:六层金字塔图 -->
<!-- 这是BHP平台架构层面最核心的NLP工具，与TTM六阶段完美对应 -->

### 核心框架

**Robert Dilts（罗伯特·迪尔茨）**于1991年整理发表的"理解层次"（Logical Levels / Neurological Levels）模型，是NLP对行为改变机制最重要的贡献之一。

大脑处理事情时有六个不同的逻辑层次，**从低到高**：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
              ┌──────────────────┐
              │  ⑥ 系统/使命层  │  超越自我的存在意义
              └────────┬─────────┘
              ┌────────┴─────────┐
              │  ⑤ 身份层        │  我是谁？
              └────────┬─────────┘
              ┌────────┴─────────┐
              │  ④ 信念/价值层  │  我为什么这样做？什么重要？
              └────────┬─────────┘
              ┌────────┴─────────┐
              │  ③ 能力层        │  我能做什么？我有哪些选择？
              └────────┬─────────┘
              ┌────────┴─────────┐
              │  ② 行为层        │  我具体在做什么？
              └────────┬─────────┘
              ┌────────┴─────────┐
              │  ① 环境层        │  何时、何地、与谁？
              └──────────────────┘
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
规律：高层的改变会自然带动低层的改变
      低层的干预对高层不起作用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**六层与BHP干预策略的精准对应**：

```
① 环境层（Environment）
   问题："什么时候/在哪里/和谁在一起时，你更容易做到健康行为？"
   干预：改变物理环境（冰箱里放健康食物/找到步行伙伴）
   BHP工具：行为设计·环境支持模块
   TTM对应：S0-S2（外部触发）

② 行为层（Behavior）
   问题："你现在具体在做什么？哪些行为是你想改变的？"
   干预：行为处方、具体行动计划
   BHP工具：行为处方引擎（behavior_rx）
   TTM对应：S2-S3（行动期）

③ 能力层（Capability）
   问题："你有哪些不同的选择？你还没有尝试过哪些方法？"
   干预：技能训练、提供多样化策略
   BHP工具：BCTTv1行为改变技术包
   TTM对应：S3-S4（培养能力期）

④ 信念/价值层（Beliefs & Values）
   问题："你为什么认为健康改变是可能的/不可能的？健康对你意味着什么？"
   干预：信念松动、价值澄清（见KI-IDENT-NLP-001）
   BHP工具：CoachAgent深层对话、MI访谈
   TTM对应：S4（内在动力转化期）

⑤ 身份层（Identity）
   问题："你是一个怎样的人？你想成为一个怎样的人？"
   干预：身份重构（"我是一个照顾自己的人"而非"我是一个慢病患者"）
   BHP工具：大五人格×身份工作×萨提亚家庭重塑
   TTM对应：S5（身份认同整合期）

⑥ 系统/使命层（System/Spirit）
   问题："你存在的意义是什么？你对这个世界可以有什么贡献？"
   干预：使命探索、意义连接
   BHP工具：弗兰克尔意义治疗、ACT价值澄清
   TTM对应：S6（超越自我，活出使命）
```

**关键应用原则**：

> 李中莹强调：从低层挣扎的时候，如果能走到上层检讨自己的行为，便会带来有效的突破。
>
> 意思是：当一个用户在**行为层**（比如"我就是戒不了甜食"）卡住的时候，真正有效的干预不是再给他一个"戒甜食方案"（还在行为层），而是**升一层**，去问信念层的问题（"你相信自己能改变吗？"）。

**"理解层次贯通法"**（Neurological Alignment Technique）：六个纸片排成一条线，逐层站上去与潜意识沟通——这是NLP实操中调动深层力量、整合六层的核心体验性技术，适合S4-S5深度工作坊使用。

### 平台应用价值

迪尔茨六层模型是**BHP平台架构的底层逻辑**：

- 直接对应BHP三层价值架构（痛苦解决=环境行为层；适应成长=能力信念层；超越=身份系统层）
- 解释为什么只在行为层干预效果有限——BHP需要能识别用户所在层次、匹配干预层次
- S0-S2在①②层 / S3-S4在②③④层 / S5在⑤层 / S6在⑥层


## KI-IDENT-NLP-002 情绪的七大意义与钟摆效应：慢病情绪管理的深层框架

<!-- 领域:身份认同/情绪 | 来源:李中莹《重塑心灵》第九章·情绪 | 证据:T2 | 来源等级:C -->
<!-- 适用阶段:S2,S3,S4,S5 | 成长维度:discover_self,realize_self -->
<!-- 文化有效性:High | 反思提问:4个 | 身份连接:有 -->
<!-- 注意：情绪困扰严重时（S0-S1急性期）不作主线，先稳定 -->

### 核心框架

李中莹认为，情绪是人类历史上**最被忽视、最少被研究**的领域，同时也是最重要的。他对情绪提出了七大重新诠释：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
情绪的七大意义（李中莹版本）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

意义① 情绪是生命不可分割的一部分
  → 没有情绪的人，其实是有缺憾、不完整的人
  → 慢病场景：不要试图"消灭"情绪，试图消灭情绪会让人更痛苦

意义② 情绪绝对诚实可靠和正确
  → 情绪是潜意识对现实的真实反映
  → 慢病场景：用户说"我不知道为什么就是不想动"——那个"不想动"的感觉在说什么？

意义③ 情绪是内心需要的信使
  → 负面情绪是一封信，告诉我们有某个需要未被满足
  → 慢病场景：愤怒→"某个边界被侵犯了"；焦虑→"某个重要的事感到不确定"；悲伤→"失去了什么重要的东西"

意义④ 情绪提供行动的能量
  → 愤怒的意义是"提供力量去改变一个不能接受的情况"
  → 慢病场景：很多患者的"怒"是宝贵的能量——引导得好，可以成为改变的动力

意义⑤ 情绪帮助我们学习成长
  → 当我们从情绪中学到了东西，情绪便会消失；没有学到，同样的事情还会再来
  → 慢病场景：用户反复在同一情境下暴食——那个情绪在试图教他什么？

意义⑥ 情绪连接我们与他人
  → 能够感受和表达情绪，是建立真实关系的基础
  → 慢病场景：很多慢病患者的孤独感来自"我不能让家人知道我有多害怕"

意义⑦ 情绪强化记忆与学习
  → 带有情绪的经验，记得最深，学得最快
  → 慢病场景：体验式、带有情感共鸣的健康教育比纯信息输入有效得多

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"钟摆效应"（The Pendulum Effect）——警示理论
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

某些人因为压力太大，学会了"感觉麻木"——
用各种方式减少自己对负面情绪的敏感度。

但李中莹发现了一个规律：
当一个人在某一种情绪上削减了反应的强度，
在所有其他的情绪上也会有同样的削减。

就像钟摆——
如果摆向负面的幅度减少了，摆向正面的幅度也同样减少。

结果是：不好的事不会那么伤害他，但好的事也不会使他欢欣、喜悦、满意。

慢病场景的应用：
很多长期慢病管理者，因为过度"克制情绪""保持理性"，
反而失去了感受生活乐趣的能力，
变得"什么都无所谓"——
这不是超脱，而是钟摆效应造成的情感麻痹。

矫正方向：不是消灭负面情绪，而是扩大情绪感应幅度——
  充分体验正面情绪（感恩、喜悦、满足）
  有效处理负面情绪（而非压制）
  两者并存，才是健康的情绪生命

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
情绪处理的"驯狗隐喻"（李中莹经典故事）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
遇到一条凶恶的狗（负面情绪），经历了：
  抗拒/害怕（无力感）→
  接受它的存在（初步接纳）→
  给它喂食（建立关系）→
  抚摸它（理解它）→
  与它同行（运用它）→
  让它为你服务（掌控它）

处理自己负面情绪的过程也是如此：
  认识它 → 接受它 → 理解它 → 欣赏它 → 运用它 → 掌控它
```

**常见负面情绪的正面意义（BHP版）**：

| 情绪 | 正面意义 | BHP应用 |
|------|---------|---------|
| 愤怒 | 提供力量去改变不能接受的现状 | 用户对疾病的愤怒是改变的能量来源 |
| 恐惧 | 保护机制，提醒某个危险 | 对健康恶化的恐惧可以转化为行动驱动 |
| 悲伤 | 处理失去，让内心松动 | 确诊后的悲伤是正常的，允许它存在 |
| 焦虑 | 提醒某件重要的事尚未处理 | 对未来不确定的焦虑→澄清可控的行动 |
| 内疚 | 提醒某个价值观被违背 | 内疚感背后有真实的价值在乎着什么 |
| 孤独 | 提醒对连接和被看见的渴望 | 孤独感→寻求社群支持的入口 |

### 平台应用价值

慢病人群情绪负担重，但传统医疗体系对情绪处理几乎为零。这个框架让BHP的CoachAgent能够：
- 不回避用户的情绪，而是把情绪当作信息来解读
- 防止"情感麻木"的钟摆效应——慢病管理不应以牺牲情感丰富度为代价
- 把负面情绪转化为行动能量，而非压制

### 核心框架

李中莹在1999年发展出**"接受自己法"**（灵感来自Jane Price的《From Inside Out》），是NLP心理辅导实践中处理自我价值不足最有效的工具之一。

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
核心洞见：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

没有充分成长的人，往往是小时候有很多不愉快的经历——
无论是因为当时自己做得不好，还是受到别人欺负——
都造成"不接受自己"的情况。

那些不愉快的经历造成了限制性信念，
导致这些人把注意力放在错误的价值和无效的规条上，
经常与自己过不去，否定自己。

关键发现：
不接受自己的人，几乎总是与父母的关系不好，
也几乎全部都有不能接受父母的问题。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"接受自己法"的核心机制：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 找回过去那个"不被接受的自己"
   → 通常是童年/青少年时期，在某个困难处境中的自己
   → 那个小孩独自面对很多他当时没有能力处理的事

2. 理解那个时期的自己
   → "他只可以凭当时拥有的知识和能力去处理每一件事"
   → "他不懂得怎样可以做得更好，也没有人教他"
   → "事实上，他已经做得很好了——看看你现在的情况，便是证明"

3. 理解当时伤害过你的人
   → "他们没有学过怎样做好他们当时的角色"
   → "他们的行为背后有正面动机，只是不知道更好的做法"
   → 这不是为施害者开脱，而是为自己松绑

4. 内在和解的对话
   → 与那个小时候的自己沟通，给予他当时未曾得到的：
     • 肯定（"你已经做得很好了"）
     • 保护（"现在的我来保护你"）
     • 爱（"你值得被爱"）

5. 整合——将过去的自己与现在的自己融合
   → "我们以后再也不会分开，一同快快乐乐在人生中前进"
   → 从内在分裂走向内在整合，自我价值自然提升

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
自我审查七问（日常版内在整合工具）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 在此事此时里，你是谁？（身份定位）
2. 你想这件事情有怎样的结果？（目标澄清）
3. 你正在做什么？（行为审视）
4. 你所做的跟你想有的结果配合吗？（行为-目标对齐）
5. 你正在用的方法持续了多久？有效果吗？（方法评估）
6. 有其他方法的可能吗？（选择扩展）
7. 那个结果背后的价值是什么？你的深层需要是什么？（价值定位）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"洒金粉法"——力量添增（日常支持工具）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

很多不良行为（暴食、吸烟、怒发冲冠）其实是因为：
当事人的力量不足，需要这些行为来填补内心的空缺。

治本之道：
帮助用户找到新的力量来源（信念/身份/使命感），
而非只是要求他们"停止"不良行为。

李中莹的关键发现：
宗教团体戒烟戒毒成功率远高于传统心理治疗（再发率低），
原因正是宗教提供了一个更大的力量来源（信仰/归属/使命）
来替代成瘾行为给予的"填补感"。

BHP应用：
帮助用户建立一个"比不良行为更大的力量来源"——
家庭关系、人生意义、价值观连接——才是行为持久改变的根本。
```
