## KI-TCM-CONST-001｜体质定义与形神两元框架

```yaml

title: 中医体质的定义与形神两元结构
life_dimension: TCM_CONSTITUTION
applicable_stages: [S0, S1, S2, S3]
evidence_tier: T3
tags: [体质定义, 形神, 先天禀赋, 后天养成]
retrieval_queries:
  - 中医说的体质是什么意思？
  - 为什么每个人体质不一样？
  - 体质可以改变吗？
```

**定义**：体质是在人的生命过程中，在先天禀赋和后天获得的基础上，逐渐形成的**形态结构、生理功能、物质代谢和性格心理**方面综合的、固有的特质。

**形神两元**：
- **形**（有形之体）= 肌肉、骨骼、五脏、皮肤、毛发、血脉——看得见摸得着的部分
- **神**（功能之体）= 消化、呼吸、睡眠、情绪、性格、精神活动——操控健康的"隐形指挥官"

**关键洞见**：神是体质最重要的组成部分。许多慢性病的根源是神失调（长期精神压力/情绪紊乱），而非单纯形体问题。

**可变性原则**：体质具有**相对稳定性**（先天决定主线），但也有**动态可调性**（后天修为可改善偏颇）。

| 体质 | 核心特征 | 代谢易感疾病 | 易引发慢病 |
|------|---------|------------|---------|
| **平和** | 脏腑和谐，七情适度，不易生病 | 低风险 | 极少 |
| **阳虚** | 火力不足，畏寒怕冷，手冷过肘 | 血糖代谢迟缓 | 痛证/骨质疏松/痛经/性功能减退 |
| **气虚** | 气力不足，疲劳乏力，易外感 | 低血压/低血糖 | 内脏下垂/慢性炎症/贫血 |
| **阴虚** | 五心烦热，口干咽燥，形体消瘦 | 消耗增加 | 失眠/便秘/高血压/糖尿病 |

| **湿热** | 内环境如"桑拿天"，口臭体臭 | 炎症代谢负担 | 皮肤感染/胆道疾病/痤疮 |
| **瘀血** | 面色晦暗，痛处固定，易生肿物 | 微循环障碍 | 心脑血管病/肿瘤/痛证 |
| **气郁** | 气机不顺，情绪郁闷，胁肋胀痛 | 激素轴失调 | 甲状腺/乳腺/抑郁/功能性胃肠 |

> **临床重要提示**：实际患者多为**复合体质**（2-3种叠加），纯单一体质少见。中老年以后基本无单纯体质。

## KI-TCM-CONST-003｜体质与慢病的关联逻辑

```yaml
knowledge_id: KI-TCM-CONST-003
title: 体质偏颇与慢性病易感性的关联机制
life_dimension: TCM_CONSTITUTION
applicable_stages: [S1, S2, S3]

tags: [体质易感性, 慢病风险, 痰湿, 代谢综合征]
retrieval_queries:
  - 为什么痰湿体质容易得糖尿病？
  - 体质和慢性病有什么关系？
  - 中医体质能预测疾病风险吗？
```

**核心机制**：体质不遗传，但体质决定**对特定疾病的易感性**。

**高代谢风险体质组合**（慢病管理重点关注）：

```
痰湿体质 ──────→ 脾运化不足 → 水谷精微代谢障碍
                              ↓
          肥胖（中心性）/ 高血脂 / 高血糖 / 脂肪肝
          
阳虚体质 ──────→ 阳气不足 → 血脉失温 → 微循环减慢
                              ↓
          心脑血管病风险 / 甲状腺功能减退 / 胰岛素分泌减少

气虚体质 ──────→ 气不摄血 / 脏器失于固托
                              ↓
          贫血 / 内脏下垂 / 免疫力低下
```

**教练应用原则**：体质评估是慢病行为干预的**前置定向工具**，决定干预方向（补虚/泄实/温阳/滋阴），而非替代医疗诊断。

## KI-TCM-CONST-004｜体质形成的六大后天影响因素

```yaml

  - 为什么现代人痰湿体质这么多？
  - 饮食怎么影响体质？
```

**六大后天因素**（均可成为行为干预切入点）：

| 因素 | 促生体质偏颇的典型行为 |
|------|---------------------|
| **年龄** | 中老年阳气渐衰→阳虚；青壮年情志困扰→气郁/阴虚 |
| **饮食** | 过食肥甘厚腻→痰湿/湿热；长期寒凉→阳虚/瘀血；过咸→痰湿/瘀血 |
| **起居** | 熬夜→伤阳耗气；久坐→瘀血/痰湿；过度依赖空调→阳虚 |
| **情志** | 长期紧张焦虑→气郁/阴虚；忧思过度→气虚/血虚 |
| **疾病** | 久病及肾→阳虚；慢性失血→血虚/气虚 |
| **药物** | 长期抗生素/清热药→气虚/阳虚；滥补→湿热/阴虚 |

**现代高频偏颇原因**（与平台用户群高度相关）：
- 痰湿↑：久坐少动 + 高热量饮食 + 空调减少排汗
- 气郁↑：职场压力 + 社交媒体 + 人际关系复杂
- 阴虚↑：熬夜 + 过度用脑 + 辛辣饮食

## KI-TCM-CONST-005｜体质判断：望闻问切四法简版

```yaml
knowledge_id: KI-TCM-CONST-005
title: 体质判断的望闻问切实用方法
life_dimension: TCM_CONSTITUTION
applicable_stages: [S1, S2, S3]
evidence_tier: T3
tags: [望闻问切, 体质判断, 舌象, 面色, 寒热]
retrieval_queries:
  - 怎么自己判断体质？
  - 看什么可以判断体质？
  - 舌头胖大是什么体质？
operational_anchors: [舌象图解, 手脚温度, 大便形态]
```

**望（看外形）**：
- 体型肥胖沉重 → **痰湿**（"胖人多痰湿"）
- 体型偏瘦，面色偏暗 → **阴虚或瘀血**
- 面色苍白无光泽 → **阳虚或气虚**
- 面色口唇晦暗，眼圈发黑 → **瘀血**

**看舌象**（最可靠体质指征）：
| 舌象 | 体质提示 |
|------|---------|
| 舌体胖大，色淡嫩，有齿痕 | 阳虚/气虚 |
| 舌体偏小，色红，苔少 | 阴虚/内热 |
| 舌色紫暗，或有瘀斑 | 瘀血 |
| 舌苔厚腻不退 | 痰湿/湿热 |
| 舌苔黄厚腻 | 湿热 |

**问（问症状）**：
- 常年手冷过肘足冷过膝，冬季尤甚 → **阳虚**
- 容易疲倦，讲话有气无力 → **气虚**
- 手心脚心发热，口干 → **阴虚**
- 情绪经常郁闷，容易叹气 → **气郁**
- 大便长期烂而不成形 → **气虚或阳虚**
- 大便干结臭、小便黄 → **湿热**

## KI-TCM-CONST-006｜养生四大基础原则

```yaml

retrieval_queries:
  - 中医养生的核心是什么？
  - 春夏养阳秋冬养阴怎么理解？
  - 养生从哪里开始？
```

**四大基础**（缺一则养生无根基）：

**① 养心修德**（首要原则）
- 神是体质的操控者，养神>养形
- 七情适度：不过悲/过喜/过怒/过思/过恐
- 操作化建议：每天5分钟"不看手机/不处理事务"的静坐（初级止念训练）

**② 春夏养阳，秋冬养阴**
- 春夏：衣着宽松，多户外，确保出汗，少吃冰凉（"冬吃萝卜夏吃姜"）
- 秋冬："春捂秋冻"渐进适应，早睡晚起，减少大汗运动，适当温补
- 核心逻辑：顺应阳气升降节律，不对抗自然

**③ 简单顺势**
- 大道至简：养生就是"不折腾"
- 禁忌：暴饮暴食、过度治疗、长期熬夜、滥用保健品
- 干祖望总结："猴行（多运动）、龟欲（少贪婪）、蚁食（七成饱）、童心"

**④ 保护环境**
- 自然环境=体质养育的外部土壤
- 实践层面：多接触自然光/空气/食物，少加工食品

## KI-TCM-CONST-007｜阳虚体质：核心特征与禁忌

```yaml
knowledge_id: KI-TCM-CONST-007
title: 阳虚体质核心表现、易感疾病与养生禁忌
life_dimension: TCM_CONSTITUTION
applicable_stages: [S2, S3, S4, S5]
evidence_tier: T3
tags: [阳虚, 畏寒, 夜尿, 火力不足, 上热下寒]
retrieval_queries:
  - 阳虚体质有什么表现？
  - 阳虚的人怎么养生？
  - 上热下寒是什么意思？
  - 阳虚可以吃什么补？
```

**核心诊断特征**：
- 四肢发凉（手冷过肘、足冷过膝为真阳虚）
- 夜尿多（≥2次/夜），小便清白
- 大便偏烂，五更腹泻（清晨4-5点腹泻）
- 性欲减退/痛经/月经延后
- 舌体胖大嫩白，脉沉细

**特殊变体——上热下寒**（临床最易误诊）：
- 表现：下身寒（夜尿/便溏/腰腿冷）+ 上部热（口臭/痤疮/失眠/烦躁）
- 本质：下焦阳虚，虚阳上浮，**下寒是真，上热是假**
- 危险误区：见"上热"症状即用清热解毒药 → 越清越寒，加重阳虚
- 正确处理：温下焦（艾灸神阙/关元/中极）

**绝对禁忌（铁律）**：
1. 禁食冰镇饮料、冰激凌、生冷蔬果（西瓜/苦瓜/黄瓜等）
2. 禁止长期处于强冷空调环境
3. 禁止凉水冲凉/游泳后不擦干
4. 禁止以上热为由乱用抗生素/清热解毒中药

## IT-TCM-CONST-001｜阳虚体质行为干预处方

```yaml
knowledge_id: IT-TCM-CONST-001
title: 阳虚体质生活方式行为干预处方
life_dimension: TCM_CONSTITUTION
applicable_stages: [S3, S4, S5]
evidence_tier: T3
applicable_psych_levels: [L2, L3, L4]
applicable_bpt6: [environment, action]
spi_minimum: 20
target_domain: tcm_constitution
```

**饮食干预**（可操作化清单）：

✅ 优先推荐：
- 温热性食物：羊肉、牛肉、韭菜、核桃、板栗、桂圆、荔枝
- 烹调方式：炖/蒸/焖/煮，避免凉拌生食
- 具体指令：**"炒菜时加3-5片生姜，炖肉时加桂皮/茴香"**（每日操作，无需计量）

❌ 限制/回避：
- 寒凉类：冰镇饮料（全年）、西瓜/梨子/苦瓜（夏日可少量熟食）
- 高盐食物（促生痰湿/水肿，加重阳虚）

**起居干预**（微行为锚定）：
```
早晨链: 起床 → 喝一杯温白开水（不低于40°C）→ 晒太阳10-20分钟 → 早餐
睡前链: 泡脚（热水至踝关节以上，水温42°C左右，15分钟）→ 护腰（背部保暖）→ 入睡
```

**运动指导**：
- 推荐："动能生阳"——快走/广场舞/太极（以微微出汗为度）
- 时间：上午9-11时（阳气最旺）> 傍晚 > 清晨（冬季避免清晨大汗运动）
- 禁忌：冬季清晨剧烈运动、游泳后寒风直吹

**情志干预**：
- 阳虚者易沉静→抑郁，应增加户外活动、日照、轻快音乐
- 避免：久坐不动、阴雨天闭户、过度悲忧

**经络养生（简版）**：
- 艾灸取穴：神阙（肚脐）、关元（肚脐下3寸）→ 每次15-20分钟，每周2-3次
- 泡脚+按揉涌泉穴（脚掌心前1/3处）→ 每晚睡前5分钟

  手心脚心发热，尤其晚上 → +阴虚

  大便经常偏烂、不成形 → +气虚/阳虚
  大便干结、臭、小便黄 → +湿热/阴虚

Q3 身体重量感：
  身体沉重、上楼梯腿如铅灌 → +痰湿
  身体总感觉疲倦无力、气短 → +气虚

Q4 情绪状态：
  经常郁闷、想叹气、胸口憋 → +气郁
  容易烦躁、焦虑、睡眠差 → +阴虚/湿热

Q5 舌头（对着镜子伸舌头）：
  舌体胖大/边缘有牙印 → +气虚/阳虚/痰湿
  舌苔黄厚腻 → +湿热
  舌色暗红或紫 → +瘀血
  舌苔很少/几乎无苔 → +阴虚

【结果解读】：
  单一体质（1种明显）→ 针对性干预
  2种叠加（常见）→ 以最突出的为主，兼顾次要体质
  3种以上（多见于中老年）→ 建议中医专业评估后干预
