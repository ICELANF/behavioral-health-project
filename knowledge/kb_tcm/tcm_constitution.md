## 三、3分钟行为扫描 · 12题题库（可直接入库）

评分规则：A=1分 B=2分 C=3分 D=4分。各题得分乘以权重后加总，各维度满分100分（已归一化）。Q1-Q4→代谢负荷 Q5-Q6→压力激活 Q7-Q8→睡眠修复 Q9-Q10→行为稳定 Q11-Q12→心理掌控


| # | 题目 | 选项（A低风险 → D高风险） | 维度 | 评分权重 |
|---|---|---|---|---|
| Q1 | 你最近多久会出现"脑子断电"、注意力涣散的情况？ | A / 几乎没有 / B / 每周1-2次 / C / 几乎每天下午 / D / 全天都这样 | 代谢负荷 | W=0.25 |
| Q2 | 你的腹部/腰围这一年有明显增加吗？ | A / 没有 / B / 略微增加 / C / 明显增加3cm+ / D / 增加很多，扣子紧了 | 代谢负荷 | W=0.25 |
| Q3 | 餐后你的状态通常是？ | A / 精力充沛 / B / 略有困意 / C / 必须躺一会 / D / 困到眼睛睁不开 | 代谢负荷 | W=0.20 |
| Q4 | 你每天久坐时间大约是？ | A / <4小时 / B / 4-6小时 / C / 6-8小时 / D / >8小时 | 代谢负荷 | W=0.15 |
| Q5 | 你最近的压力感受是？ | A / 平稳 / B / 有些紧张但可控 / C / 持续焦虑/烦躁 / D / 已经喘不过气 | 压力激活 | W=0.30 |
| Q6 | 你的情绪在一天中波动情况？ | A / 比较平稳 / B / 有起伏但可控 / C / 容易因小事崩溃 / D / 情绪持续低落/麻木 | 压力激活 | W=0.25 |
| Q7 | 你的入睡情况？ | A / 躺下15分钟内入睡 / B / 30分钟内 / C / 超过1小时 / D / 明明很累却怎么也睡不着 | 睡眠修复 | W=0.30 |
| Q8 | 早上醒来你的感受通常是？ | A / 精神充沛 / B / 还好 / C / 像没睡过 / D / 比昨晚更累 | 睡眠修复 | W=0.25 |
| Q9 | 你上次坚持一个健康习惯超过14天是什么时候？ | A / 现在正在坚持 / B / 最近3个月内 / C / 半年以上前 / D / 从来没有 | 行为稳定 | W=0.30 |
| Q10 | 当你"计划改变"时，通常发生什么？ | A / 能持续执行 / B / 坚持一周左右 / C / 3天热度就过了 / D / 从计划到放弃不超过1天 | 行为稳定 | W=0.25 |
| Q11 | 你觉得自己能改变目前的身体状态吗？ | A / 完全可以，已经在做 / B / 应该能，但不知道怎么做 / C / 不确定，之前失败太多次 / D / 感觉很难，可能是天生的 | 心理掌控 | W=0.30 |

## 五、survey_api 工程配置草案

完全复用现有 survey_api + assessment_pipeline_api，无需新建任何后端模块。建议同时创建A/B/C三版短码，2周后用完成率数据决定主版本。


| 配置字段 | 建议值 | 说明 / 工程备注 |
|---|---|---|
| survey.title | "代谢压力生态体质扫描"（版本A：焦虑触发型） | 建议同时创建B版（好奇探索）和C版（中医体质入口）做A/B测试 |
| survey.short_code | "mptest"（版本A）/ "body"（B）/ "tcm"（C） | 通过 POST /api/v1/surveys 创建，短码用于分渠道追踪转化率 |
| survey.allow_anonymous | true | 支持匿名填写，降低入口阻力；可选手机号换取完整报告 |
| survey.questions[1-4] | Q1-Q4（代谢负荷维度） / 选项A=1分 B=2分 C=3分 D=4分 | 题目见题库表格；分值越高代谢负荷越重 |
| survey.questions[5-6] | Q5-Q6（压力激活维度） |  |
| survey.questions[7-8] | Q7-Q8（睡眠修复维度） |  |
| survey.questions[9-10] | Q9-Q10（行为稳定维度） |  |
| survey.questions[11-12] | Q11-Q12（心理掌控维度） |  |
| 分型算法位置 | POST /api/v1/assessment (assessment_pipeline_api) | 提交survey_response后，前端调用assessment接口传入五维原始分，后端计算标签 |
| 输出结构 | { label: "高压代谢型", tcm: "TCM-3/TCM-5", radar: {代谢:78,压力:82,睡眠:45,稳定:52,掌控:55}, action: ["餐后步行","腹式呼吸"], next: "7天微实验" } | 前端渲染雷达图 + 生态标签 + 首选微行动 |
| A/B统计 | GET /api/v1/surveys/{id}/stats | 对比三个短码的完成率、跳出点、后续注册率；2周出结论 |

| 体质 | 行为核心障碍 | ⛔ Agent禁忌策略 | ✅ 推荐策略 | 常见TTM阶段 | 核心话术原则 |

| 阳虚质 | 能量低/宿命感 | ❌夜间运动 / ❌生冷食物 / ❌给强目标 | 先查TSH / 温和激发 | S0-S1 | 先排除甲减，再谈行为改变 |
