### 3.1 第一步：营养评估（Nutrition Assessment）

```
营养评估的五大信息类别（IDNT标准）：

类别A：食物/营养相关历史（Food/Nutrition-Related History）
  ↓ 评估内容：
  • 饮食摄入量（24小时回顾法/食物频率问卷）
  • 饮食模式（进餐时间/次数/份量）
  • 液体摄入（水/饮料种类和量）
  • 营养补充剂使用（维生素/矿物质/保健品）
  • 食物过敏/不耐受
  • 饮食限制（宗教/文化/个人偏好）
  • 食物获取/经济因素
  
  BHP快速收集版（CoachAgent问询模板）：
  Q1："您通常每天几点吃早/午/晚饭？有没有不吃的情况？"
  Q2："主食通常是什么？一顿大概吃多少？"
  Q3："每天会喝什么饮料？白开水大概喝多少？"
  Q4："有没有什么不能吃的食物？"
  Q5："有没有在吃什么营养补充剂或保健品？"

类别B：人体测量数据（Anthropometric Measurements）
  ↓ 评估内容：
  • 身高/体重/BMI
  • 腰围/臀围/腰臀比
  • 体成分（体脂率/肌肉量）——有条件时
  • 近期体重变化（3个月/6个月内）
  
  BHP标准收集：
  BMI = 体重(kg) ÷ 身高(m)²
  中国标准：超重≥24，肥胖≥28，向心性肥胖：男腰围≥90，女≥85

类别C：生化检查数据（Biochemical Data）
  ↓ 慢性病相关核心指标：
  
  代谢指标：
  • 空腹血糖/HbA1c（T2DM评估）
  • 血脂四项（TC/LDL-C/HDL-C/TG）（CVD评估）
  • 尿酸（痛风/代谢综合征）
  
  营养状态指标：
  • 血清白蛋白/前白蛋白（整体营养状态）
  • 血红蛋白/铁蛋白（缺铁性贫血）
  • 25-羟维生素D（维生素D状态）
  • 维生素B12（尤其素食者/老年人）
  • 肾功能（eGFR/肌酐）——蛋白质管理前必查
  
  BHP标准提示：
  → 如无近6个月检查报告，引导用户先完善基础检查再进行精准干预

类别D：体格检查发现（Nutrition-Focused Physical Findings）
  ↓ 营养相关体征：
  • 肌肉萎缩/消耗迹象
  • 皮肤/头发/指甲异常（可能提示营养素缺乏）
  • 水肿（心脏/肾脏/营养不良）
  • 黏膜/眼部变化（维生素A/B缺乏）
  
  BHP实用版：
  → 重点关注：外貌可见的肌肉量和精力状态描述

类别E：个人信息（Client History）
  ↓ 评估内容：
  • 个人/家族疾病史
  • 用药情况（影响营养吸收的药物）
  • 生活方式（运动/睡眠/工作）
  • 社会支持/文化背景
  
  BHP关键问题：
  Q："您目前在服用什么药物（降压/降糖/降脂/其他）？"
  Q："您的工作主要是坐着还是体力劳动？"
  Q："家里谁负责做饭？主要做什么菜式？"
```

### 3.2 第二步：营养诊断（Nutrition Diagnosis）

```
营养诊断的PES陈述格式（核心工具）：
  
  P（Problem）：营养问题/诊断标签
  E（Etiology）：病因/相关因素
  S（Signs & Symptoms）：体征/症状/评估依据
  
  格式："P与E相关，如S所示"
  
  示例PES陈述：

  案例1：高血压合并高钠摄入
  P：钠摄入过多
  E：与加工食品摄入频繁（每天吃火腿肠/泡面）相关
  S：如24小时饮食回顾估计钠摄入≥3500mg/天，
     血压持续>140/90所示
  
  完整PES："钠摄入过多，与加工食品摄入频繁相关，
            如估计每日钠摄入≥3500mg、
            血压持续>140/90mmHg所示"

  案例2：T2DM血糖控制不佳
  P：碳水化合物摄入类型和量不一致
  E：与营养知识不足（不了解GI/GL概念）及
     环境因素（工作餐主要为白米饭+馒头）相关
  S：如HbA1c=8.2%（目标<7%）、
     主食主要为白粥/白米饭所示
  
  案例3：超重伴含糖饮料过量摄入
  P：含糖饮料摄入过量（空热量摄入）
  E：与工作场合饮料机取用习惯/口渴时饮料替代水相关
  S：如每日摄入≥2罐饮料（约70g糖/天）、
     BMI=27.8所示
```

```
常用营养诊断标签（BHP平台慢性病相关）：

  摄入量相关：
  → 钠摄入过多（NI-5.10.2）
  → 碳水化合物摄入类型不当（NI-5.8.3）
  → 膳食纤维摄入不足（NI-5.8.5）
  → 含糖食物/饮料摄入过多（NI-5.8.6）
  → 不适宜的脂肪摄入（NI-5.6.1/5.6.2）
  → 液体摄入不足（NI-3.1）
  
  知识/信念相关：
  → 与食物/营养相关知识不足（NB-1.1）
  → 不准备进行饮食改变（NB-1.3）
  → 不良饮食模式（NB-1.5）
  → 活动量不足（NB-2.1）
  
  临床相关：
  → 与营养相关的超重/肥胖（NC-3.3）
  → 食物/药物相互作用（NC-2.3）
```

### 3.3 第三步：营养干预（Nutrition Intervention）

```
营养干预的四大类别：

类别1：食物/营养素提供（Food/Nutrient Delivery）
  → 个性化饮食计划（制定具体膳食方案）
  → 营养素补充（需要时）
  → 调整食物质地/成分（特殊需求）
  
  BHP实现：
  → 依据营养诊断生成M-Action行为处方
  → 参照KI-NUTR-CVD-001/KI-NUTR-DM-001/KI-NUTR-OBE-001

类别2：营养教育（Nutrition Education）
  → 内容教育（知识传递）
  → 应用指导（技能培训）
  
  BHP实现：
  → CoachAgent话术变体库（各KI模块8）
  → 平台教育内容推送（基于TTM阶段匹配）

类别3：营养咨询（Nutrition Counseling）
  → 动机性访谈（Motivational Interviewing）
  → 认知行为改变策略
  → 目标设定
  
  BHP实现：
  → 行为改变技术（BCTTv1：KI-COACH-76M-001）
  → XZB行为激活模块

类别4：跨学科协调（Coordination of Nutrition Care）
  → 转介其他专科（医生/心理咨询师）
  → 与医疗团队协作
  → 营养支持团队（营养师+医生+护士+药剂师）
  
  BHP实现：
  → 转介触发阈值（各KI铁律D）
  → 平台内多专业协作功能
```

### 3.4 第四步：营养监测与评价（Nutrition Monitoring & Evaluation）

```
监测的三类指标：

类型1：食物/营养摄入量指标
  监测：钠摄入/碳水化合物摄入/饱和脂肪摄入
  工具：24小时饮食回顾/食物频率问卷（随访时重复）
  频率：每次随访（推荐4-6周/次）

类型2：营养相关体测指标
  监测：体重/BMI/腰围
  频率：
    减重期：每周体重
    达标维持期：每月腰围+每季度BMI

类型3：营养相关生化指标
  监测：HbA1c/血脂四项/血压
  频率：
    血压：每日（高血压患者）
    血糖：每月自我监测（T2DM）
    HbA1c：每3个月（血糖未达标）/每6个月（达标）
    血脂：每3-6个月
  
评价标准（判断干预是否有效）：
  改善：指标向目标值方向移动
  达标：指标达到预设目标值
  无变化：需分析原因（依从性/方案调整）
  恶化：升级干预/医疗转介
```

## 模块4：M-Action清单（NCP流程执行版）

```
对BHP健康管理师/教练的NCP执行步骤：

第一次接触用户时：
  ✅ 完成A类：简要饮食回顾（5个问题版，见3.1）
  ✅ 记录B类：身高/体重/腰围
  ✅ 确认E类：慢性病史和用药
  ✅ 收集C类：提示用户提供近期检查报告（如有）
  
首次营养诊断（第一次接触后）：
  ✅ 写出至少1个PES陈述
  ✅ 优先级排序（最影响当前健康问题的营养诊断优先）
  
制定干预方案：
  ✅ 依据PES的P（问题）选择对应KI：
     钠摄入过多 → KI-NUTR-CVD-001
     碳水摄入不当 → KI-NUTR-DM-001
     超重/能量失衡 → KI-NUTR-OBE-001
  ✅ 从对应KI的模块4生成用户专属M-Action清单

随访时：
  ✅ 重新评估营养诊断（PES）是否改变
  ✅ 记录监测指标变化
  ✅ 根据评价结果调整干预方案
```

## 模块6：铁律边界

```
铁律A NCP实施主体铁律：
  营养诊断和营养处方的制定需具备资质人员执行。
  BHP平台作为辅助工具，不替代注册营养师的专业判断。
  AI/CoachAgent可进行初步筛查和教育支持，
  复杂营养问题必须转介专业营养师。

铁律B PES陈述完整性铁律：
  营养干预必须基于清晰的营养诊断（PES陈述）。
  禁止：在没有明确营养诊断的情况下给出饮食建议
       （避免"万金油式"建议）。
  例：不是"你要多吃蔬菜"，而是
     "因为你的蔬菜摄入不足（评估显示每天<200g），
      导致膳食纤维摄入不足，所以我们增加蔬菜到500g/天"。

铁律C 医疗信息收集权限铁律：
  生化指标的解读（血液检测结果）必须结合医疗诊断。
  BHP平台收集用户生化数据仅用于营养评估参考，
  不对检验结果进行医疗诊断。

铁律D 医疗权限铁律（铁律4）：
  NCP营养诊断≠医学诊断。
  "营养诊断：钠摄入过多" ≠ "高血压诊断"
  营养干预不替代药物治疗。
```

## 模块8：话术变体库

### 引导用户配合营养评估（S0-S1）
1. "我想先了解一下你平时的饮食习惯——不是要批评你吃什么，而是想找到最适合你个人情况的建议。这比泛泛的'少吃多运动'要有用得多。你愿意花3分钟告诉我你昨天吃了什么吗？"
2. "我需要先做一个简单的营养评估——就像医生要先检查才能开药方一样，我需要先了解你现在的饮食状态，才能给你有用的建议。"

### 解释营养诊断结果（S1-S2）
1. "从你告诉我的饮食情况来看，我们找到了一个关键问题：你每天的钠摄入可能比推荐量高了一倍左右——主要来自你常吃的加工食品。这一个问题解决了，对你的血压改善会非常明显。"
2. "你的主食结构是一个主要突破口：目前以白米饭/白粥为主，血糖负荷比较高。如果把早餐白粥换成燕麦，这一步就能改善你餐后血糖的波动。"

### 随访时回顾（S3-S4）
1. "上次我们说要尝试先吃蔬菜再吃主食——你这周试了几次？有没有感觉到不同？"
2. "你上次说要把含糖饮料戒掉，体重有没有变化？——就算只是第一步，已经很了不起了。我们来看一下下一步是什么。"
