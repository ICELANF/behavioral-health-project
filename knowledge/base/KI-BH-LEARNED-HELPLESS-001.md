## 核心知识内容

### 1. 理论定位

**习得性无助（Learned Helplessness）** 由塞利格曼（Martin Seligman）基于动物实验发展而来，是理解**为什么慢病患者"知道该改变但就是不动"**的最重要心理学框架之一。

核心机制：当一个人反复经历**无论自己做什么，结果都不会改变**的情境时，会习得"努力是无效的"这一认知，并将其泛化到新情境，即使在新情境中努力本可以改变结果。

---

### 2. 习得性无助三维症状

习得性无助不只是"懒"或"没动力"，而是一个包含三个相互交织维度的心理综合征：

#### 2.1 动机症状（Motivational Deficit）
- 表现：**启动行为的意愿减少**，不尝试、不开始、拖延
- 机制：预期努力无效→行为启动代价高于预期收益→不行动
- 慢病表现：
  - "试过很多次了，都没用"
  - "我就是改变不了"
  - 对健康教育内容漠然，不问问题，不参与

#### 2.2 认知症状（Cognitive Deficit）
- 表现：**即使在新情境中，也难以学习"努力→结果"的有效关联**
- 机制：已习得"控制是不可能的"→无法识别新的控制机会
- 慢病表现：
  - 即使医生给出了有效方案，也认为"对我不会有用"
  - 难以将过去小的成功经验迁移到新行为
  - 无法从成功案例中产生"我也可以"的认知

#### 2.3 情绪症状（Emotional Deficit）
- 表现：**抑郁样情绪状态**，无望感、无助感、自尊低落
- 机制：持续无控制体验→抑郁→进一步降低行为启动阈值（恶性循环）
- 慢病表现：
  - 对未来没有期待（"反正都会恶化"）
  - 自我否定（"我就是这种命"）
  - 可能伴随真实的抑郁症状（需关注）

---

### 3. 归因模式与无助的关系（Abramson, Seligman & Teasdale，1978重构版）

现代归因-无助模型认为，无助的**泛化程度和严重程度**取决于归因维度：

| 归因维度 | 高无助风险归因 | 低无助风险归因 |
|---------|-------------|-------------|
| **内/外部性** | 内部（"是我的问题"） | 外部（"是情境的问题"） |
| **稳定/不稳定** | 稳定（"这种情况不会改变"） | 不稳定（"这次特殊"） |
| **全局/特定** | 全局（"我什么都做不好"） | 特定（"只是这件事"） |

**最危险的归因组合（慢病场景）**：
- "我就是没有自制力（内部+稳定+全局）"→最深度的习得性无助
- "我这种体质天生就是容易胖（内部+稳定+全局）"→同上

**保护性归因**：
- "这次计划没坚持下来，是因为那段时间工作太忙（外部+不稳定+特定）"
- "这个运动方案不适合我，换一个可能更好（外部+特定）"

---

### 4. 习得性无助与TTM前意向期的对应关系

```
TTM S0（前意向期）的两个核心子类型：

类型A：无知型S0
原因：信息不足，未意识到改变的必要性
干预：信息提供、意识唤醒

类型B：无助型S0  ← 习得性无助的典型表现
原因：有认知（知道需要改变）但预期无效（"改了也没用"）
干预：习得性无助干预，不是更多信息

⚠️ 错误诊断后果：
对"无助型S0"用户进行信息轰炸（健康教育）会适得其反——
他们已经知道信息，需要的是对控制感的重建，而非更多的"你应该怎么做"
```

---

### 5. 归因治疗四策略（针对习得性无助）

| 策略 | 操作描述 | 实施要点 |
|------|---------|---------|
| **重归因训练** | 帮助用户将失败从内部稳定全局（能力）重新归因到外部不稳定特定（情境/方法） | "不是你不行，是方案不适合你的生活节奏" |
| **情境化归因** | 指出外部/情境因素在失败中的真实作用 | "食物环境、压力、睡眠都会影响——这些不是借口，是真实的影响因素" |
| **成功体验归因强化** | 成功时将原因归于内部可控因素 | "你今天完成了，是因为你…（具体可控行为）" |
| **未来导向重构** | 从"我为什么总是失败"转向"下次我可以做什么不同" | "过去的结果不代表未来，让我们找一个对你来说更可行的方法" |

---

### 6. 可控性训练（Controllability Training）

突破习得性无助的核心是**重建"我的行动可以改变结果"的体验**：

**"小赢"策略（Small Wins Strategy）**：
1. **极小步骤**：设计几乎不可能失败的第一步（如"每天走100步"而非"每天走1万步"）
2. **立即反馈**：让用户立即看到行动→结果的连接（血糖/体重/能量的即时变化）
3. **可控归因**：将成功明确归因于用户的行动（"你做到了"而非"运气好"）
4. **渐进挑战**：成功体验积累后，逐渐增加挑战，扩展控制感范围

```
"小赢"序列示例（针对无助型S0用户）：
Day 1-3: 每天记录三餐（无需改变饮食，只是记录）
Day 4-7: 每天餐后步行5分钟
Week 2: 替换一个零食选择
（每步都配有即时正反馈）
```

## BHP平台应用映射

### 无助型S0用户识别协议

```
识别信号：
- 评估问卷中出现"我以前试过很多次""感觉没用""不相信能改变"
- 行为历史：多次开始/放弃的循环
- 语言模式："总是""从来""永远"（稳定全局归因词汇）

触发特殊干预流程：
→ 跳过标准健康教育模块
→ 激活"重建控制感"专项序列
→ 分配有小赢设计经验的教练
```

### 教练会话禁忌与替代话术

```
❌ 强化无助的话术：
   "你知道你应该怎么做的，就是没有做到。"
   "你要更有意志力。"
   "其他用户都能做到，你也可以。"

✅ 重建控制感的话术：
   "告诉我，有没有哪怕一次，你做到了某个健康的小事？
    哪怕很小的事情也算。"
   "我们不从之前那些大计划开始，我们只找一件
    这周你有90%把握能做到的事情。"
   "这次没做到，我们来看看是哪个环节出了问题，
    不是你的问题，是计划的问题。"
```

**主题句**：
习得性无助是慢病行为改变中最常见的隐性障碍，表现为动机（不启动）、认知（无法学习新控制关系）、情绪（无望感）三维缺失，由内部×稳定×全局的归因模式维持，针对"无助型前意向期"用户的干预核心不是更多健康教育而是通过极小可控步骤重建"行动→结果"的有效连接体验，配合归因治疗四策略系统重塑其对改变可能性的认知。
