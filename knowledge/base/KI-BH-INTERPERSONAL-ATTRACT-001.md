## 核心知识内容

### 1. 理论定位

社会关系是健康行为改变最被低估的力量之一。元分析研究表明，**社会支持是健康行为长期维持的最强预测变量之一**，甚至强于许多基于个体认知的干预。理解人际吸引和社会影响机制，是设计有效BHP社群功能和教练关系的理论基础。

---

### 2. 人际吸引的三大机制

#### 2.1 强化理论（Reinforcement Theory of Attraction）
- **核心**：我们被那些给我们奖励（正向情感、自尊提升、实用帮助）的人所吸引
- **慢病应用**：
  - 教练与用户的关系质量 = 教练能提供多少有价值的正强化
  - 同伴关系的黏性 = 互动能带来的愉悦感、被理解感、实用信息价值
  - **设计警示**：仅靠批评/督促（负强化）的教练关系不可持续

#### 2.2 得失理论（Gain-Loss Theory，Aronson & Linder）
- **核心**：得到一个原本不喜欢我们的人的认可（得），比始终被喜欢的人（恒定正评）吸引力更强；反之，失去一个一直喜欢我们的人的认可（失）伤害更大
- **慢病应用**：
  - 教练对用户进步的认可，在经历挫折之后更有激励效果
  - 始终给予无差别积极反馈（"做得很好！"）的教练会边际效用递减
  - 教练关系的深度建立在真实观察和有时的建设性质疑上

#### 2.3 相似性吸引（Similarity Attraction）
- **核心**：人们被与自己相似的人吸引（态度、价值观、背景、处境）
- **最重要的相似性维度**：态度相似 > 人格相似 > 外表相似

**在健康行为改变中的应用**：
```
同病相怜效应（Shared Struggle Effect）：
患有相同慢性病的同伴 > 专业的"健康人"教练
经历过相同改变困难的同伴 > 从未有过此困难的专家

→ 同伴导师（Peer Mentor）设计原则：
  匹配标准应包括：相同疾病类型、相似生活处境、相似改变阶段（领先1-2个TTM阶段）
```

### 3. 自我暴露（Self-Disclosure）与关系深化

**奥尔特曼-泰勒社会渗透理论（Social Penetration Theory）**：
- 关系通过渐进的、互惠的自我暴露而深化（洋葱模型：从表层→深层信息）
- 暴露的**深度**（个人程度）比**广度**（话题数量）更影响亲密感
- **互惠性**至关重要：单方面深度暴露会导致关系失衡和回避

**在健康行为改变中的双重角色**：

作为**收益**：
- 用户对教练的深度自我暴露（讲述真实的生活困境、情绪挣扎）是获得精准支持的前提
- 同伴间的真实分享创造真正的连接感

作为**障碍**：
- 健康信息（体重、血糖、饮食记录）具有高度私密性
- 在信任建立之前要求过多自我暴露 → 用户流失
- **设计原则**：渐进式信息收集，先建立信任再要求深度暴露

### 4. 人际影响三机制（在健康社群中的应用）

#### 4.1 遵从（Compliance）
- 基于外部压力/奖惩的行为改变
- 特点：仅在监督/奖惩存在时维持，内化程度低
- **健康场景**：医嘱遵从（依从性差的主要原因是纯遵从而非内化）

#### 4.2 认同（Identification）
- 基于对他人的认同和希望被认可
- 特点：行为改变与关系维持直接相关
- **健康场景**：用户改变行为因为希望被教练/群体认可

#### 4.3 内化（Internalization）
- 基于对信念/价值的真实认同
- 特点：最稳定，不依赖外部关系
- **健康场景**：用户真正认同健康生活方式的价值

> **SDT对应**：遵从=外部调节，认同=认同调节，内化=整合调节/内在动机——社会影响与SDT动机内化在此汇合。

### 5. 班杜拉模仿三功能（社会学习在健康社群中）

班杜拉（Bandura）的社会学习理论认为，模仿/观察学习具有三种功能：

| 功能 | 描述 | BHP社群应用 |
|------|------|-----------|
| **习得新行为** | 通过观察学习以前不会的行为 | 观看同伴的健康烹饪视频→习得新食谱技能 |
| **抑制/去抑制现有行为** | 看到模仿对象受惩罚→抑制；受奖励→去抑制 | 看到同伴坚持锻炼的成果→激活自己已知但未执行的行为 |
| **促进/引发已有行为** | 提示线索激活潜在行为 | 打卡功能、群体晨练打卡→激发自己的同步行为 |

**最有效的健康行为模型特征**：
- 与用户相似（非遥不可及的"完美健康人"）
- 展示困难和真实挣扎的过程（而非只展示结果）
- 与用户处于相近的改变阶段（领先约1-2阶段效果最佳）

---

### 6. 社会交换理论与冲突管理

**社会交换理论（Social Exchange Theory）**：
- 人际关系 = 收益（Rewards）- 成本（Costs）
- 当前关系满意度 = 收益/成本 vs 比较水平（过去的经验）
- 关系维持决策 = 收益/成本 vs 替代水平（有没有更好的选择）

**健康社群设计应用**：
```
用户留在BHP社群的条件：
收益（帮助、连接感、认可、知识）> 成本（时间、隐私暴露、情感代价）
AND 没有比BHP更好的替代社群（独特价值定位）

提升收益：精准匹配、有价值的内容、真实情感连接
降低成本：减少摩擦、保护隐私、减少无效互动
```

## BHP平台应用映射

### 同伴匹配算法设计原则

```
匹配变量优先级：
1. 疾病类型/健康目标（相似性吸引核心维度）
2. TTM阶段（领先1-2阶段的同伴导师效果最佳）
3. 生活背景（年龄段、家庭状况、职业类型）
4. 改变挑战类型（情绪性进食者 vs 运动障碍者 vs 依从性差者）
5. 性格兼容性（外向/内向，指导型/支持型）

避免：
- 跨度过大的匹配（S1用户与S5用户）→ 相似性太低，产生自卑
- 仅基于疾病类型匹配→ 忽略生活背景差异
```

### 社群内容设计原则

```
✅ 高吸引力内容类型：
- 真实挣扎+如何克服的过程叙述
- 具体可复制的行为改变技巧
- 阶段性成果展示（非外貌/体重，而是功能/感受改善）
- 日常情境中的小智慧分享

❌ 低吸引力/有害内容类型：
- 竞争性排行（谁减重最多）→ 产生羞愧和比较
- 完美健康形象展示→ 不相似，产生距离感
- 专家说教性内容（无互动）→ 遵从而非认同
```

### 教练关系建立序列（社会渗透原则）

```
Week 1-2: 表层信任建立
- 了解生活背景、健康历史（广度为主）
- 教练适度自我暴露（营造互惠氛围）

Week 3-4: 深度关系建立
- 探索真实的改变障碍、情绪经历（深度）
- 共同确定有意义的健康目标

Month 2+: 真实关系维持
- 在用户挫折时给予真实支持（非纯粹正面反馈）
- 建立"失败也可以说"的安全对话空间
```
