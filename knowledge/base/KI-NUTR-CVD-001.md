## 模块2：干预核心逻辑

### 心血管疾病营养治疗的作用机制
```
饮食 → 影响路径 → CVD风险指标
  ↓
钠摄入 → 体液潴留 → 血压升高
饱和脂肪 → LDL氧化 → 动脉粥样硬化
反式脂肪 → LDL升高+HDL降低 → 双重CVD风险
钾/钙/镁 → 血管舒张 → 血压降低
膳食纤维 → 胆固醇吸收减少 → LDL降低
欧米伽-3 → 抗炎/降甘油三酯 → 综合CVD保护
体重降低 → 胰岛素敏感性改善 → 血压/血脂双改善
```

**最高证据级别的干预效果**：
- DASH饮食可降低收缩压8-14 mmHg（与高血压药物效果相当）
- 减重5-10%可降低收缩压5-20 mmHg
- 每日钠摄入减少1000mg → 降低收缩压2-8 mmHg
- 有氧运动30分钟/天 → 降低血压4-9 mmHg

### 3.1 高血压营养治疗（DASH饮食）

#### DASH饮食核心原则
**证据基础**：DASH（Dietary Approaches to Stop Hypertension）试验——迄今最具证据力的饮食降压研究

```
DASH饮食目标营养素：
  ↑ 增加：钾 / 钙 / 镁 / 膳食纤维 / 植物蛋白
  ↓ 减少：饱和脂肪 / 钠 / 添加糖 / 酒精

DASH饮食结构（以2000千卡/天为基准）：
  蔬菜：4-5份/天
  水果：4-5份/天
  低脂/脱脂乳制品：2-3份/天（钙的主要来源）
  全谷物：7-8份/天
  瘦肉/禽类/鱼：≤6份/天
  坚果/种子/豆类：4-5份/周
  脂肪/油脂：2-3份/天（以不饱和脂肪为主）
  糖果/添加糖：≤5份/周
```

#### DASH饮食操作化版本（BHP行为处方）

**第一步——减盐（优先级最高）**：
```
目标：每日钠摄入 ≤2300mg（约6g食盐）
  理想目标：≤1500mg/天（适合已确诊高血压者）

实际行为转化：
  ✅ 烹调减盐法：
    - 起锅时放盐（而非炒菜过程中），用量减半口感差异小
    - 用醋/柠檬汁/香草替代部分食盐（提升风味感知）
    - 选择低钠版酱油（减盐50%以上）

  ✅ 识别隐形盐来源：
    - 高钠食物（每100g含钠>600mg）：
      腌制食品（榨菜/泡菜/咸菜）/ 火腿肠/腊肉/腊肠
      方便面/膨化食品/饼干
      酱类（豆瓣酱/蚝油/豆腐乳）
    - 判断方法：查看食品标签"钠"含量

  ✅ 今天就能做的第一步：
    - 把餐桌上的盐瓶移走
    - 外出就餐告诉服务员"少盐"
    - 下次买调味品时选低钠版本
```

**第二步——增加钾/钙/镁**：
```
钾（目标：3500-4700mg/天）：
  高钾食物：香蕉/土豆/甘薯/番茄/菠菜/橙子/牛油果
  行为化：每天1根香蕉+1个番茄（已提供约600mg钾）

  ⚠️ 注意：肾功能不全用户需遵医嘱，
         不可自行大量补钾

钙（目标：1000-1200mg/天）：
  优选：低脂牛奶/酸奶（每杯约300mg钙）
  替代：豆腐（石膏豆腐）/深绿色蔬菜（芥蓝/小白菜）
  行为化：每天1-2杯低脂牛奶或酸奶

镁（目标：310-420mg/天）：
  高镁食物：全谷物/坚果/绿叶蔬菜/豆类
  行为化：每天一小把坚果（约20g，混合坚果）
```

**第三步——限制酒精**：
```
目标：男性 ≤2标准饮酒单位/天，女性 ≤1单位/天
    （1单位 ≈ 350ml啤酒/150ml红酒/45ml白酒）

机制：酒精直接升高血压，且增加心律失常风险
行为化：
  - 把饮酒场合明确区分（习惯性饮酒 vs 社交饮酒）
  - 社交场合：提前告知不饮酒/选无醇啤酒
  - 用茶/气泡水/柠檬水替代
```

#### 高血压用户营养诊断速查
```
常见营养诊断：
  1. 钠摄入过多（>2300mg/天）→ 减盐处方
  2. 钾/钙/镁摄入不足 → 增加对应食物
  3. 超重/肥胖（BMI≥24）→ 体重管理处方
  4. 身体活动不足 → 运动处方（每周≥150分钟中等强度）
  5. 酒精摄入过多 → 限酒处方
  6. 营养知识缺乏 → 教育干预
```

### 3.2 血脂异常营养治疗

#### 血脂指标解读（NCEP ATP III标准）
```
总胆固醇（TC）：
  理想：<200 mg/dL（<5.2 mmol/L）
  临界高：200-239 mg/dL
  高：≥240 mg/dL

LDL-C（低密度脂蛋白，"坏"胆固醇）：
  优选：<100 mg/dL（<2.6 mmol/L）
  接近最优：100-129 mg/dL
  临界高：130-159 mg/dL
  高：160-189 mg/dL
  极高：≥190 mg/dL

HDL-C（高密度脂蛋白，"好"胆固醇）：
  低（风险因素）：<40 mg/dL（男），<50 mg/dL（女）
  高（保护因素）：≥60 mg/dL

甘油三酯（TG）：
  正常：<150 mg/dL（<1.7 mmol/L）
  临界高：150-199 mg/dL
  高：200-499 mg/dL
  极高：≥500 mg/dL（急性胰腺炎风险）

中国常用单位换算：
  TC/LDL/HDL：mg/dL ÷ 38.7 ≈ mmol/L
  TG：mg/dL ÷ 88.6 ≈ mmol/L
```

#### 血脂异常饮食干预矩阵

**针对高LDL（首要目标）**：
```
最有效的三个饮食干预：

① 减少饱和脂肪（最强证据）
  目标：<7%总热量（约<15g/天，以1800千卡为例）
  主要来源（需减少）：
    - 肥肉/猪油/牛油/羊油
    - 全脂奶/黄油/奶油/硬质奶酪
    - 椰子油/棕榈油
  替代方案：
    - 烹调油 → 橄榄油/茶籽油/菜籽油（富含单不饱和脂肪）
    - 全脂奶 → 低脂/脱脂奶
    - 五花肉 → 鸡胸肉/鱼/豆腐

② 消除反式脂肪
  目标：尽量接近零摄入
  主要来源（严格避免）：
    - 部分氢化植物油（人造黄油/起酥油）
    - 含"氢化/部分氢化"配料的加工食品
    - 炸鸡/薯条/油炸食品（多次用油加工）
    - 市售奶油夹心饼干/植脂末（奶精）
  识别方法：
    看食品配料表是否含"氢化植物油/精炼植物油（部分氢化）"

③ 增加可溶性膳食纤维
  目标：10-25g可溶性纤维/天
  最有效食物：
    - 燕麦（β-葡聚糖）：每天50g燕麦≈3g可溶性纤维
    - 大麦/糙米
    - 豆类（黄豆/黑豆/绿豆）：每份约3-6g可溶性纤维
    - 苹果/梨/柑橘类（果胶）
    - 车前子壳（psyllium）：可添加到酸奶/粥中
  行为化：
    - 早餐改为燕麦粥（替代白粥或白面包）
    - 每周至少3次豆类菜肴（替代部分肉类）
```

**针对低HDL（需升高）**：
```
有效策略：
  ✅ 增加身体活动（最有效）
     有氧运动30分钟/天×5天 → HDL升高3-9%

  ✅ 适量不饱和脂肪（代替饱和脂肪）
     坚果（核桃/巴旦木）/ 橄榄油 / 牛油果
     每天一小把坚果（25-30g）

  ✅ 戒烟
     吸烟直接降低HDL

  ❌ 避免大量精制碳水化合物（白饭/白面/糖）
     升高甘油三酯，间接降低HDL
```

**针对高甘油三酯**：
```
最有效干预：

① 减少精制碳水化合物和添加糖
  关键：果糖是甘油三酯升高的首要饮食原因
  需严格控制：
    - 含糖饮料（奶茶/果汁/碳酸饮料）
    - 添加糖的食品（糕点/糖果/蜂蜜/白砂糖）
    - 白米饭/白面包/白面条（精制淀粉）
  
② 增加欧米伽-3脂肪酸
  目标：EPA+DHA ≥1g/天（高甘油三酯时4g/天）
  食物来源：
    - 深海鱼：三文鱼/沙丁鱼/鲭鱼/金枪鱼（每周2-3次，每次100-150g）
    - 亚麻籽/奇亚籽/核桃（α-亚麻酸，转化率低）
  行为化：每周"鱼2天"替代猪肉/鸡肉

③ 限制酒精
  酒精是甘油三酯升高的重要原因
  甘油三酯>500 mg/dL时→严格禁酒

④ 体重管理
  减重5-10% → 甘油三酯降低20-30%
```

### 3.3 动脉粥样硬化综合营养治疗

#### 冠心病/动脉粥样硬化营养处方（综合版）

```
"心脏保护性饮食"五要素：

要素1：脂肪质量（最重要）
  ✅ 目标：
     单不饱和脂肪（橄榄油/茶籽油）
     多不饱和脂肪（欧米伽-3/6，深海鱼/坚果）
  ❌ 避免：
     饱和脂肪（动物油脂/全脂奶）<7%总热量
     反式脂肪≈0

要素2：鱼类摄入（优先推荐）
  目标：每周≥2次深海鱼
  机制：EPA/DHA降低甘油三酯/抑制血小板聚集/抗心律失常

要素3：坚果
  目标：每天25-30g混合坚果（1小把）
  机制：单不饱和脂肪/植物甾醇/L-精氨酸改善血管内皮功能

要素4：植物甾醇/固醇（phytosterols）
  目标：2g/天（可显著降低LDL 7-10%）
  来源：植物甾醇强化食品（部分人造黄油/橙汁）/天然食物含量较低

要素5：叶酸
  机制：降低同型半胱氨酸（高同型半胱氨酸是CVD独立风险因素）
  食物来源：深绿色蔬菜/豆类/全谷物/肝脏（少量）
```

#### 代谢综合征诊断标准（WHO/IDF）
```
符合以下3项或以上：
  腰围：男性>90cm（中国标准），女性>80cm
  甘油三酯：≥150 mg/dL（1.7 mmol/L）
  HDL-C：男性<40 mg/dL，女性<50 mg/dL
  血压：收缩压≥130 或 舒张压≥85 mmHg
  空腹血糖：≥100 mg/dL（5.6 mmol/L）

营养治疗优先级：
  ① 体重管理（中心性肥胖→首要目标）
  ② 减少精制碳水化合物（改善胰岛素抵抗）
  ③ 增加膳食纤维（血糖/血脂双调）
  ④ 减少饱和脂肪（改善血脂）
  ⑤ 增加身体活动（改善胰岛素敏感性+HDL）
```

### 3.4 心力衰竭营养治疗

```
心力衰竭用户的特殊营养管理：

钠限制（核心）：
  目标：<2000mg/天（严重心力衰竭<1500mg）
  原因：钠潴留→液体潴留→心脏负担加重→水肿/呼吸困难

液体管理：
  目标：通常限制在1500-2000ml/天
  需监测：每日称重（晨起空腹）
  警示信号：2-3天内体重增加≥2kg → 告知医生

营养风险（心脏恶病质）：
  心力衰竭患者易出现营养不良/体重过低
  定期监测：体重/BMI/肌肉量
  与超重管理区分：心力衰竭患者的减重需谨慎

药食相互作用：
  利尿剂（呋塞米等）→ 钾流失 → 需监测血钾，
                              可能需增加高钾食物或补钾
  ACEI/ARB类降压药 → 钾潴留风险 → 避免过量高钾食物

CoachAgent话术：
  "你现在每天测体重吗？——心力衰竭患者每天早上起床后、
  上完厕所、吃早饭前称体重，是非常重要的自我监测。
  如果3天内重了2斤以上，要及时告诉医生。"
```

## 模块4：M-Action清单

```
P1 - DASH饮食第一步：减盐
  今天：把餐桌食盐移走 / 外出就餐要求少盐
  本周：学会看食品标签，识别高钠食品
  本月：购置低钠酱油，替换常用调味品

P1 - 血脂干预第一步：消除反式脂肪
  今天：检查家中是否有含"氢化植物油"的食品（植脂末/酥皮点心）
  本周：停止购买含反式脂肪的食品
  替代：橄榄油/茶籽油用于烹调

P2 - 增加可溶性纤维
  今天：早餐改燕麦粥（替代白粥/白面包）
  本周：每周3次豆类菜肴
  操作：即食燕麦+热水/牛奶5分钟即可（无需额外烹饪）

P2 - 每周"鱼2天"
  目标：用深海鱼替代猪肉/鸡肉2次/周
  简单做法：清蒸/水煮（保留欧米伽-3，避免油炸破坏）

P3 - 体重管理联动
  CVD风险用户BMI≥24：启动体重管理处方
  每减重1kg：收缩压降低约1-2mmHg

P4 - 每日坚果习惯
  目标：每天1小把混合坚果（25-30g）
  替代：把坚果放在桌上作为零食替代糖果/饼干
```

## 模块5：监测重点

| 指标 | 频率 | 目标值 | 关注阈值 | 行动触发 |
|------|------|--------|---------|---------|
| 血压 | 每天（高血压患者）| <130/80 mmHg | ≥140/90 或 变化>10mmHg | 复查+调整方案 |
| TC/LDL/HDL/TG | 每3-6月 | LDL<2.6 mmol/L，TG<1.7 | LDL>3.4 或 TG>5.6 | 医疗评估+强化饮食 |
| 体重/腰围 | 每月 | 腰围男<90/女<85cm | 体重增加>2kg/月 | 饮食回顾+运动增加 |
| 钠摄入估算 | 每周回顾 | <2300mg/天 | 持续高钠摄入 | 识别隐形盐来源 |
| 心力衰竭体重 | 每天（心衰用户）| 稳定 | 3天内+2kg | 立即告知医生 |

**代谢安全红线（铁律1）**：
- 血压 >180/110 mmHg → 停止运动建议，要求医疗评估
- 甘油三酯 >500 mg/dL → 严格禁酒，建议医疗评估（急性胰腺炎风险）

## 模块6：铁律边界

```
铁律A 肾功能检查前限钾铁律：
  推荐高钾饮食（香蕉/土豆等）前，必须确认肾功能正常。
  肾功能不全用户（eGFR<60）：禁止推荐高钾饮食。
  原因：肾功能不全无法排钾→高钾血症→心律失常风险。

铁律B 利尿剂用药交叉铁律：
  服用利尿剂（呋塞米/氢氯噻嗪等）用户：
  需与医生确认钾补充方案，营养建议不可自行增减。
  原因：不同利尿剂对钾的影响方向相反（排钾/保钾）。

铁律C 减重速度安全铁律：
  心力衰竭/心功能不全用户的减重建议必须保守。
  禁止：快速减重处方（>0.5kg/周）用于心力衰竭患者。
  原因：心脏恶病质风险，过度减重损害心功能。

铁律D 医疗权限铁律（铁律4）：
  营养治疗不替代降压药/他汀类/抗血小板药物。
  用户询问是否可以"靠饮食替代药物"：
  回答："饮食改变非常重要，但药物由医生决定——
       好的消息是，如果饮食改善很好，医生可能会
       主动调整你的用药。"
```

## 模块7：叠加情境提示

```
高频叠加组合：
  组合1：高血压 + 糖尿病 = 最高CVD风险
    → 同时执行DASH减盐 + 低血糖负荷饮食
    → 体重管理作为最高优先级干预（同时改善血压和血糖）

  组合2：血脂异常 + 超重 = 代谢综合征核心
    → 先减少精制碳水化合物（同时改善血脂和体重）
    → 增加运动（提升HDL的最有效单一干预）

  组合3：高血压 + 肾功能不全 = 特殊限制需求
    → 限钠同时需限钾/限磷/限蛋白质（三重限制）
    → 必须转介肾科营养专项评估

  组合4：CVD营养 + 五运六气（KI-TCM-WUYUN）
    → 冬季（终之气）：血压管理强化期
    → 春季（初之气）：情志管理+血压波动监测

情境处理优先级：
  代谢红线（>180/110或TG>500）> 医学评估
  > DASH/降脂处方 > 体重管理 > 行为习惯优化
```

## 模块8：话术变体库

### 解释高血压与盐的关系（S1）
1. "血压高，盐是一个很重要的因素——不是因为盐本身有毒，而是钠会让身体留住水分，血管里的血液量增多，压力自然就高了。我们来看看你每天吃多少钠。"
2. "减盐听起来很无聊，但科学告诉我们：认真减盐3个月，血压可以降8-14mmHg——和一片降压药的效果差不多。你愿意试试吗？"

### 血脂高用户建立信心（S2）
1. "血脂高不是你的错——饮食确实有影响，但遗传和代谢也有关系。好消息是：你能控制的那些因素，每一个都是有证据的。我们来一个一个来？"
2. "降低LDL最有效的饮食改变，不是不吃肉，而是换油——把猪油/黄油换成橄榄油，这一步的效果非常显著，而且做起来比你想象的容易。"

### DASH饮食引入（S2-S3）
1. "DASH饮食不是减肥饮食，是'降压饮食'——它的研究显示比许多降压药的效果还好，而且没有副作用。最重要的是：它的核心就是'多吃蔬菜水果+少吃盐'，你已经知道了，现在就是做到。"
2. "你现在每天能吃几份蔬菜？——DASH饮食的目标是4-5份/天。听起来多，但一份只是半碗炒青菜那么多。你早饭有没有可能加一份水果？"
