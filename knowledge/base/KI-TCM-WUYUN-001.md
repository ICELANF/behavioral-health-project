## 模块2：五运六气核心框架

### 2.1 五运六气的本质（BHP操作化理解）

五运六气是TCM的"自然节律×人体适应"框架：
```
五运：金/木/水/火/土 × 每年的运势强弱（太过/不及/平气）
六气：风/寒/暑/湿/燥/火 × 一年六个节气段的主导气候特征

核心逻辑：
  自然气候特征 → 影响人体脏腑状态 → 需要相应的行为/饮食/情志调整

BHP应用价值：
  提供"为什么这个季节要这样做"的理论解释
  增强用户对季节性建议的接受度（文化共鸣）
  为行为处方增加"顺势而为"的自然感，减少"为了血糖"的工具感
```

### 2.2 六气节段划分（一年六个阶段）

```
初之气：大寒→春分（约1月20日→3月20日）
        主气：厥阴风木（风气主导）

二之气：春分→小满（约3月20日→5月20日）
        主气：少阴君火（温热上升）

三之气：小满→大暑（约5月20日→7月22日）
        主气：少阳相火（暑热最盛）

四之气：大暑→秋分（约7月22日→9月22日）
        主气：太阴湿土（湿气主导）

五之气：秋分→小雪（约9月22日→11月22日）
        主气：阳明燥金（燥气主导）

终之气：小雪→大寒（约11月22日→1月20日）
        主气：太阳寒水（寒气最盛）
```

## 模块3：六气节段行为健康干预矩阵

---

### 初之气（大寒→春分，风木当令）
**气候特征**：风气主导，气温回升但不稳定，忽冷忽热

**易发问题**：
- 风邪入侵：头痛、颈肩僵硬、关节不适
- 肝气升发：情绪波动、易怒、睡眠浅
- 旧疾复发：高血压患者血压波动

**行为健康处方**：
```
饮食：
  ✅ 疏肝理气食物：芹菜、韭菜、葱、生姜、陈皮茶
  ✅ 酸味适量（入肝）：山楂水、少量醋
  ❌ 避免大量生冷（寒热交替期，脾胃仍需保护）

运动：
  ✅ 户外散步、太极拳（顺应阳气升发）
  ✅ 早起时间可逐渐提前（随日照延长调整）
  ❌ 避免剧烈运动（风邪期关节保护）

情志：
  ✅ 疏导情绪，不压抑（顺应肝气升发特性）
  ✅ 可开始新计划（春生之气，适合启动）
  ❌ 避免过度烦躁（肝火已旺，不宜再激）

睡眠：
  ✅ "夜卧早起"（春季睡眠模式调整）
  目标：比冬季晚睡时间推后30分钟，比冬季早起15分钟
```

**CoachAgent话术（S3）**：
"现在是初春，风气比较重，这段时间血压波动是正常的——TCM说'风善行而数变'。我们可以把这段时间的重点放在：保暖（尤其脖子和腰）、情绪疏导、不要急着换春装。你最近血压有没有起伏？"

### 二之气（春分→小满，君火温热）
**气候特征**：温度明显升高，雨水增多，湿热开始积聚

**易发问题**：
- 湿热上犯：皮肤问题、大便不成形
- 心火偏旺：心烦、口腔溃疡
- 过敏季：鼻炎、皮肤过敏

**行为健康处方**：
```
饮食：
  ✅ 清热利湿：薏苡仁粥、荷叶茶、冬瓜汤、绿豆汤
  ✅ 补心食物：莲子、百合（养心安神）
  ❌ 避免过多辛辣（火上加火）
  ❌ 避免高脂油腻（湿热已盛，不宜再助湿）

运动：
  ✅ 增加运动量（最适合户外运动的节段）
  ✅ 游泳、骑行（有利于排汗祛湿）
  注意：避免在最热时段（10-16点）剧烈运动

睡眠：
  ✅ 午睡20-30分钟（顺应二之气特点）
  目标：21:30-22:30入睡，6:00-6:30起床
```

---

### 三之气（小满→大暑，相火暑热最盛）
**气候特征**：全年最热阶段，暑热夹湿

**易发问题**：
- 中暑风险
- 消化功能下降（暑气耗气）
- 血糖不稳定（出汗多，进食减少，运动量可能下降）
- 心脑血管风险增加

**行为健康处方**：
```
饮食（暑热期最关键）：
  ✅ 补水优先：每天1500-2000ml，分次少量
  ✅ 消暑食物：冬瓜、苦瓜、黄瓜、西红柿
  ✅ 益气：黄芪水（防暑虚）、大枣
  ❌ 严格控制冷饮（寒凉直接伤脾胃，影响血糖吸收曲线）
  ❌ 避免过于油腻（暑湿困脾）

运动：
  ✅ 调整到清晨（6-8点）或傍晚（18点后）
  ✅ 室内运动（空调环境下强度可维持）
  ❌ 避免11-16点户外运动（中暑+血糖波动风险）

血糖管理特别提示（糖尿病用户）：
  ⚠️ 出汗增多导致血糖假性升高
  ⚠️ 食欲下降导致低血糖风险增加
  ⚠️ 每日额外监测一次血糖（进食不规律时）
```

---

### 四之气（大暑→秋分，湿土主导）
**气候特征**：暑热消退，湿气最重（南方尤明显）

**易发问题**：
- 痰湿体质：消化迟钝、困倦、水肿
- 胃肠功能弱：腹泻、消化不良
- 情绪：长夏湿气压人，易感疲惫抑郁

**行为健康处方**：
```
饮食（健脾祛湿是核心）：
  ✅ 健脾食物：薏苡仁/山药/茯苓/扁豆
  ✅ 芳香化湿：陈皮茶、紫苏、生姜
  ❌ 避免甜腻/冷饮/生冷水果（寒湿伤脾）
  ❌ 避免过量水果（多数水果助湿）

运动：
  ✅ 保持适度出汗（有利于祛湿）
  ✅ 餐后散步30分钟（健脾助消化）
  ❌ 过度汗出（耗气）

居家环境：
  ✅ 保持通风，开除湿设备
  ✅ 早晨开窗（趁露水前）
```

---

### 五之气（秋分→小雪，燥金主导）
**气候特征**：秋燥明显，温度下降，昼夜温差大

**易发问题**：
- 肺燥：咳嗽、皮肤干燥、口干
- 情志："悲秋"情绪，血压波动
- 旧病复发：呼吸道疾病

**行为健康处方**：
```
饮食（润燥为主）：
  ✅ 润肺食物：梨、百合、银耳莲子羹、蜂蜜（少量）
  ✅ 补肺气：山药、枸杞
  ✅ 增加水分摄入（早晨空腹一杯温水）
  ❌ 避免辛辣（燥上加燥）

运动：
  ✅ 登山、快走（秋高气爽，最适合户外运动）
  ✅ 深呼吸练习、八段锦
  注意：增加保暖意识（温差大）

情志调节：
  ✅ 主动增加社交活动（应对悲秋情绪）
  ✅ 记录秋收的成果（行为改变的里程碑回顾）
  CoachAgent话术："秋天是总结的季节——我们来盘点一下，
    从春天到现在，你做到了哪些让你骄傲的改变？"
```

---

### 终之气（小雪→大寒，寒水主导）
**气候特征**：全年最冷，收藏封藏季节

**易发问题**：
- 阳虚：怕冷、尿频、腰膝酸软
- 心脑血管高危期（寒冷→血管收缩→血压升高）
- 抑郁倾向（日照减少）

**行为健康处方**：
```
饮食（补肾温阳为主）：
  ✅ 温热食物：羊肉汤、核桃、黑芝麻、枸杞
  ✅ 补肾食物：黑豆、黑米（冬入肾）
  ✅ 晨起喝温热粥（固护脾胃阳气）
  ❌ 避免大量生冷（冬季寒气重，更伤脾肾阳）

运动：
  ✅ 调晚起床，等太阳出来后运动（冬季养藏，不宜过早出汗）
  ✅ 室内运动为主（太极、八段锦、拉伸）
  ❌ 避免在寒冷大风天进行户外剧烈运动（心脑血管风险）

血压管理特别提示（高血压用户）：
  ⚠️ 冬季是血压最难控制的季节
  ⚠️ 晨起血压监测尤其重要（起床后30分钟内测量）
  ⚠️ 温差适应：室内外温差>15℃时，出门前做5分钟热身

睡眠：
  ✅ "早卧晚起，以待日光"（冬季最合理的睡眠模式）
  目标：21:00-21:30入睡，7:00-7:30起床
```

## 模块4：M-Action清单

```
P1 - 当季饮食调整（每个节气段一个饮食重点）
  操作：每季节提供1-2个具体的食材替换建议
  示例：
    初春 → 每天一杯陈皮茶（疏肝理气）
    盛夏 → 把甜饮料换成荷叶茶（消暑祛湿）
    秋季 → 每周吃一次银耳莲子羹（润肺）
    冬季 → 早餐从冷牛奶换成热粥（温阳护胃）

P2 - 运动时段调整（顺应季节光照/温度变化）
  夏季：早晨6-8点或傍晚18点后
  冬季：上午9-11点（太阳出来后）
  春秋：全天均可，推荐下午15-17点

P3 - 季节转折点特别监测
  节气转换前后2周：血压/血糖监测频率+1次/天
  原因：气候转换期身体适应负担最重

P4 - 情志顺势调整
  春：允许自己情绪流动，不压抑
  夏：保持愉快轻松，避免大怒
  秋：主动创造"收获感"，对抗悲秋
  冬：接受内向休养，不要求自己"表现"
```

## 模块6：铁律边界

```
铁律A 文化框架限制铁律：
  五运六气为TCM传统框架，不可作为临床预测或诊断依据。
  表达方式："TCM认为这个季节..."而非"你这个季节一定会..."
  原因：五运六气是群体规律描述，个体差异巨大。

铁律B 体质优先铁律：
  季节性建议必须与用户个人体质结合，不可统一给出。
  示例：阳虚体质的人在夏天也不适合大量喝冷饮；
       阴虚体质的人在冬天也不宜过度进补温热食物。

铁律C 代谢安全红线不因季节改变（铁律1）：
  血糖<2.8或>16.7 mmol/L、血压>180/110 mmHg
  → 无论处于哪个节气，立即触发CrisisAgent，停止一切行为处方

铁律D 不可替代西医管理铁律：
  五运六气框架只作为行为处方的文化增值，不可建议用户
  根据运气调整西医用药（降糖药/降压药/降脂药）。
```

1. "中医有个说法，叫'天人合一'——就是说人的身体状态和季节节律是同步的。现在是[节气]，这个时候[特点]，所以我们的饮食和作息可以相应调整一下。"
2. "你有没有发现，每年这个季节你的血糖/血压会有一点波动？中医对这个是有解释的——我们来聊聊怎么顺着季节调整。"
3. "中医养生不是复杂的事——其实就是在对的时间，用对的食物，做对的运动。我给你说说这个月最简单的调整是什么。"

1. "冬天是血压最难控制的季节——这是全世界心血管科医生都知道的规律，中医叫'寒主收引'。我们这段时间要特别注意：早上起床不要太急，出门前先在室内活动5分钟。"
