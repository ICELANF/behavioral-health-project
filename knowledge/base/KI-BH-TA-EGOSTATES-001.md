## 核心知识内容

### 1. 理论定位

**沟通分析（Transactional Analysis, TA）** 由埃里克·伯恩（Eric Berne）创立，是同时作为人格理论、沟通理论、儿童发展理论和心理治疗方法的综合体系。其哲学基础：**人是好的，每个人都有思考能力，人可以改变自己的决定**——这与BHP三层价值架构的人本主义基础高度契合。

**自我状态三模型（PAC模型）** 是TA的核心结构，也是教练理解对话模式、识别阻抗来源、精准干预的实用地图。

---

### 2. 三种自我状态定义

**自我状态（Ego State）**：一组相互关联的行为、想法和感受，是一个人在特定时间对自身某部分人格的表达。

| 自我状态 | 英文 | 本质 | 来源 | 特征 |
|---------|------|------|------|------|
| **父母自我**（P） | Parent ego-state | 模仿父母或重要他人 | 外部内化 | 规则、评判、保护、照顾 |
| **成人自我**（A） | Adult ego-state | 对当下现实做出直接反应 | 当下处理 | 理性、客观、问题解决 |
| **儿童自我**（C） | Child ego-state | 重现童年的行为、想法和感受 | 内部历史 | 感受、直觉、创造、反应 |

> 注意：父母/成人/儿童（大写）指自我状态；真实的父母/成人/儿童（小写）指现实中的人。

---

### 3. 功能性细分（第一层次功能模型）

```
父母自我（P）
  ├── 控制型父母（CP）：评判、规则、要求、批评
  └── 照顾型父母（NP）：支持、保护、照顾、共情

成人自我（A）：数据收集、现实检验、中性评估

儿童自我（C）
  ├── 自由型儿童（FC）：自发、创造、好奇、愉悦
  └── 适应型儿童（AC）：服从、取悦、顺从或反抗
```

**健康人格的特征**：三种自我状态都能自如调用，能够在不同情境中灵活切换。

---

### 4. 自我状态识别方法（教练诊断工具）

伯恩提出四种识别维度：

**行为识别（最直接）**：
```
父母自我信号：手指指向、摇头、点头认可、叹气、语气评判
成人自我信号：直视、身体前倾、语调平稳、提问清晰
儿童自我信号：低头、手摸嘴、嗓音变高/变细、眼泪、笑声放开
```

**语言识别**：
```
父母自我（控制型）："你应该…""你必须…""你总是…""你不应该…"
父母自我（照顾型）："我来帮你…""没关系的…""你已经做得很好了"
成人自我："那我们来看看数据…""有哪些可能性？""下一步是什么？"
儿童自我（自由型）："太棒了！""我想试试这个！""好有趣！"
儿童自我（适应型）："我不知道…""我试了但是…""您看着办吧"
```

**社交互动识别（观察触发反应）**：
- 面对权威时：成人自我→保持稳定；儿童自我→服从/反叛
- 面对弱者时：父母自我激活→照顾或批评

### 5. 病态自我状态结构

#### 5.1 污染（Contamination）
成人自我被父母自我或儿童自我的内容"污染"，用来支持偏见或幻想：
- **父母污染**：偏见被当作事实（"糖尿病患者就是注定不能逆转的"）
- **儿童污染**：幻想被当作事实（"我只要想想美好未来，问题就会消失"）
- **双重污染**：同时存在两种污染

#### 5.2 排除（Exclusion）
习惯性地只使用一种或两种自我状态：
- **排除儿童自我**：过度理性，缺乏情绪感知和直觉
- **排除父母自我**：无规则感，难以为他人建立界限
- **排除成人自我**：混淆过去和现在，难以客观评估当下

## BHP平台应用映射

### 6. 三种自我状态在慢病用户中的典型表现

| 自我状态 | 健康行为改变中的典型表现 | 识别关键词 |
|---------|---------------------|---------|
| **控制型父母（CP）** | 对自己苛刻："我真没用，又吃了垃圾食品" | "我应该""我太差了""必须" |
| **照顾型父母（NP）** | 过度照顾家人忽视自己："等孩子大了再管自己" | "他们需要我""我不能先顾自己" |
| **成人自我（A）** | 客观评估、制定计划、查阅资料 | "我来看看这周的数据…" |
| **自由型儿童（FC）** | 对新行为充满好奇热情、享受运动乐趣 | "好！我想试试这个！" |
| **适应型儿童-顺从（ACC）** | 表面顺从但内心抵抗，很快放弃 | "好的好的"（但不执行） |
| **适应型儿童-反抗（ACR）** | 对医嘱/教练明显抵触 | "你们这些专家都一样""我不信" |

---

### 7. 教练自我状态管理

**教练的最优状态**：主要在**成人自我**运作，调用**照顾型父母**提供支持，允许**自由型儿童**的真诚好奇——避免**控制型父母**评判和**适应型儿童**讨好。

```
❌ 控制型父母模式（常见陷阱）：
   "你上周没有完成运动目标，这样不行的。"
   → 触发用户儿童自我（顺从或反抗），破坏自主性

❌ 适应型儿童模式（讨好陷阱）：
   "您做得真的太好了！（即使并没有）"
   → 虚假安抚，失去信任

✅ 成人自我+照顾型父母模式：
   "我注意到这周的数据和上周不太一样，你愿意聊聊发生了什么吗？"
   → 邀请用户成人自我参与，同时传递关心
```

---

### 8. 交流分析（三种交流模式）

**互补交流**（畅通）：
```
教练（成人自我）→"你觉得这周什么情况影响了你的运动？"
用户（成人自我）→"主要是周三的加班，我没来得及去健身房"
→ 平行交流，对话流畅推进
```

**交错交流**（中断）：
```
教练（成人自我）→"我们来看看你的血糖趋势"
用户（儿童自我）→"你说的这些我都做了，还是不行，我真的没救了"
→ 对话被打断，需要先响应情绪（切换到照顾型父母或成人自我共情）
```

**暧昧交流**（双重信息）：
```
表面（成人→成人）："您看这个方案怎么样？"
隐层（儿童自我）：等待被认可，害怕被否定
→ 识别隐含信息，直接响应深层需要
```

### 9. 自我状态在BHP角色体系中的映射

| BHP角色 | 主导自我状态 | 风险自我状态 |
|--------|-----------|-----------|
| 健康教练 | 成人自我+照顾型父母 | 控制型父母（评判）、适应型儿童（讨好） |
| Observer用户 | 适应型儿童或父母自我（自批评） | 缺乏成人自我客观评估 |
| Sharer用户 | 自由型儿童（分享热情） | 容易进入照顾型父母（过度帮助他人忽视自己） |
| Coach角色 | 成人自我+照顾型父母 | 控制型父母权威化 |

## 向量化注记

**主题句**：
沟通分析（TA）的PAC自我状态模型将人格分为父母自我（规则/评判/照顾）、成人自我（当下现实处理）、儿童自我（历史感受/直觉）三个系统，健康教练应主要在成人自我+照顾型父母模式下运作，识别并响应用户儿童自我的情绪信号和适应型儿童的表面顺从，通过互补交流（而非交错或暧昧交流）推动用户调用成人自我参与行为改变的客观评估与自主决策。
