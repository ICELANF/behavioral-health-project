## 模块1：适用情境与触发条件

### 核心适用场景
用户在慢性病管理过程中**对中药/食疗产生询问，或已有中医就诊基础，需要将中医干预整合进行为处方**时，提供TCMAgent参考框架。

### 不适用情境
- ❌ 用户无中医基础知识，直接推荐单味中药（风险：自行用药）
- ❌ 急性症状期直接给出中药建议（必须先建议就医）
- ❌ 替代西医用药（本KI仅作辅助参考，不作为停药/减药依据）

### TTM阶段映射
- **S1-S2**：介绍TCM食材/药食同源品，引发兴趣（非用药建议）
- **S3**：将药食同源食材整合进日常饮食行为处方
- **S4**：作为中医体质养生的长期生活方式要素

### 触发关键词
`中药` `能不能吃中药` `食疗` `药膳` `体质调理` `中医能治糖尿病吗` `中医减肥` `降压的中药` `调理血糖`

## 模块2：干预核心逻辑

### BHP中医整合理念
TCM在行为健康平台的定位：**文化接受度高的补充性行为资源**，而非替代治疗路径。

```
中药干预价值层：
  第一层：症状缓解（降糖/降压/降脂的辅助功效）→ 增强用户对平台的信任
  第二层：食物行为化（将药食同源品整合进日常饮食行为）→ 增加依从性
  第三层：体质认同（"我是痰湿体质，这些食物适合我"）→ 身份认同强化
```

### 核心原则
- **药食同源优先**：强调既是食物又是中药的品种（山药/茯苓/薏苡仁/荷叶等）
- **安全性优先**：只推荐无明显毒性、日常可食用的品种
- **配合体质**：与TCMAgent的九种体质模型结合使用

## 模块4：M-Action清单（TCM整合行为处方）

```
P1 - 药食同源优先策略：
  在日常饮食处方中优先推荐"既是食物又是中药"的品种
  目标：增加用户依从性，降低使用门槛
  具体品种：山药/茯苓/薏苡仁/荷叶/山楂/陈皮/生姜/大枣/枸杞/百合/莲子

P2 - 体质匹配查表：
  先确认用户中医体质类型（通过TCMAgent九种体质评估）
  再推荐对应品种
  禁止：不了解体质就推荐寒凉/温热性中药

P3 - 行为整合路径：
  Step 1：替代含糖饮料（荷叶茶/决明子茶/山楂水代替奶茶/果汁）
  Step 2：烹饪添加（山药/薏苡仁/茯苓加入日常主食）
  Step 3：零食替代（枸杞/大枣代替糖果/饼干）

P4 - 安全提示标准话术：
  必须在每次推荐后加上："以上建议仅供参考，具体用药请在中医师指导下进行。"
```

## 模块5：代谢监测重点

| 整合行为 | 监测指标 | 参考阈值 | 行动触发 |
|---------|---------|---------|---------|
| 荷叶茶替代含糖饮料 | 每日液体摄入品质评分 | 含糖饮料→0杯/天 | 血糖改善后记录 |
| 山药/薏苡仁入主食 | 餐后2h血糖 | <7.8 mmol/L | 超过10.0 → 检讨摄入量 |
| 情志类食材（百合/莲子）| 主观睡眠质量 / 压力评分 | 睡眠≥6h，压力≤6/10 | 持续恶化 → 心理评估 |

## 模块6：铁律边界

```
铁律A 医疗权限铁律（铁律4）：
  TCMAgent/CoachAgent不得给出具体中药配方或用药剂量建议。
  原因：中药配伍须辨证论治，错误配伍有毒副作用风险。

铁律B 替代医疗禁止铁律：
  禁止暗示或明示中药可以替代西医用药（降糖药/降压药）。
  原因：擅自停药/减药会导致急性代谢危机。

铁律C 药食同源边界铁律：
  只有在"药食同源"品种目录内的才可作为日常食材推荐。
  非药食同源品种（如黄连/黄芩/泽泻/半夏/丹参等）必须标注
  "须在中医师处方和指导下使用"。

铁律D 体质确认铁律：
  推荐寒性中药（荷叶/决明子/黄连等）前，必须确认用户体质非
  阳虚/寒凉体质，否则可能加重症状。
```

## 模块8：话术变体库

### 用户询问"中药能治糖尿病吗"（S1-S2）
1. "中药在辅助血糖管理方面有一些研究支持，但目前的医学共识是：中药作为辅助治疗，不能替代正规的降糖药物。我们可以聊聊哪些药食同源的食材，能帮你改善饮食习惯——你感兴趣吗？"
2. "中医的视角和西医不太一样——它更关注整体体质调理，而不只是血糖这个数字。你知道自己是什么中医体质吗？"
3. "有一些常见食材，比如山药、荷叶、薏苡仁，既是食物也有中药功效——把它们加进日常饮食是一个很自然的开始，不需要'吃中药'的感觉。"

### 用户说"我在吃中药了"（S2-S3）
1. "很好！中西医结合是很多人的选择。你的中医大夫有没有给你饮食建议？我们可以在他的建议基础上，帮你把行为层面做得更完整。"
2. "你现在服用的中药是调理哪方面的？这样我们可以避免在饮食建议里有冲突的地方。"

### 推荐药食同源食材（S2-S3）
1. "你现在喝的饮料是什么？如果用荷叶茶或山楂水代替，一天省掉的糖分是相当可观的——要不要试试？"
2. "炒菜的时候加点陈皮、煮粥时放些山药，这些改变很小，但坚持下来是真的有效的。"
3. "薏苡仁和大米一起煮粥，口感好，又适合你的痰湿体质——这是你可以今天就开始的事情。"
