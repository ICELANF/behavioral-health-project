## 核心知识内容

### 1. 人际关系的基本结构

**人际关系定义**：人与人在相互交往过程中所形成的**心理关系**，包含三种成分：
- **认知成分**：对他人的了解和评价（基础）
- **情感成分**：对他人的情绪体验（核心）
- **行为成分**：与他人交往的行为倾向

**人际关系的动态轨迹**：
```
良好人际关系发展：注意 → 接触 → 融合
人际关系恶化过程：冷漠 → 疏远 → 终止
```

> **BHP应用**：用户与教练/平台关系的发展阶段，直接影响干预效果。冷漠阶段是干预最危险时刻，需要及时识别并干预。

### 2. 沟通网络理论（莱维特×戴维斯）

#### 莱维特正式群体沟通模式：

| 模式 | 特征 | 适用场景 |
|------|------|---------|
| **轮式**（中心辐射） | 核心人物控制信息流 | 专家—患者传统模式（效率高，但扼杀创造性） |
| **圆形**（平等网络） | 各成员等距，无核心 | 同伴支持社群（创造性强，速度慢） |
| **链式**（串行传递） | 信息逐人传递 | 信息容易失真 |
| **Y式** | 混合型 | 组织层级信息流 |

#### 戴维斯非正式群体模式：

| 模式 | 特征 | BHP社群风险 |
|------|------|-----------|
| **流言型** | 一人主动向所有接触者传播 | 负面情绪快速扩散 |
| **集中型** | 有选择地告知亲密者 | 信息孤岛，部分用户被排除 |
| **偶然型** | 偶然机会传播 | 信息覆盖不均 |
| **单线型** | 链式传递到终端 | 信息失真风险 |

> **BHP社群设计原则**：引导"圆形+集中型"混合模式——平等参与+鼓励小圈子正向传播。避免流言型负面情绪扩散机制。

### 3. 态度理论与改变

**态度的三要素**（认知-情感-行为，CAB模型）：
- **认知成分**（基础）：对健康行为的知识和信念
- **情感成分**（核心）：对健康行为的喜恶、价值判断
- **行为成分**：行为倾向和实际行为

> **关键洞见**：情感成分是态度的核心——改变知识（认知）容易，改变情感关联才能真正改变行为倾向。这解释了为什么健康教育"只改变认知，不改变情感"的干预效果有限。

**态度形成理论**：
- **学习理论**：通过强化和观察学习形成态度
- **平衡理论**（Heider）：人们追求认知元素之间的心理平衡
- **认知失调理论**（Festinger）：失调产生改变压力

**态度改变方法**：

| 方法 | 机制 | BHP应用 |
|------|------|---------|
| **参与活动法** | 让对象亲身参与，行为先于态度改变 | 先做小行动，后生成认同感 |
| **登门拜访法** | 先提小要求→再提大要求（脚在门里效应） | 先要求一个微行为打卡，逐步升级 |
| **团体规定法** | 利用群体规范影响成员态度 | 社群共识和公开承诺 |
| **全面宣传法** | 提供证据+消除疑虑 | 平衡性健康科普（不只强调好处） |

**诱导服从的四个前提**：自愿选择 + 承诺 + 责任感 + 预期的消极后果

### 4. 归因理论在健康行为中的应用

#### 凯利三维归因模型（Kelley's Covariation Model）：

判断行为是内部原因还是外部原因，需要分析三个维度：

| 维度 | 含义 | 健康行为示例 |
|------|------|-----------|
| **特异性（Distinctiveness）** | 该行为是否只在特定情境发生 | 只在压力大时暴食？还是一直暴食？ |
| **一致性（Consensus）** | 其他人面对同样情境是否有相同反应 | 大多数人在这种情况下也会这样吗？ |
| **一贯性（Consistency）** | 该人在同样情境中是否一贯如此 | 每次压力都暴食？还是偶发？ |

#### 归因偏见的识别：

**行为中心偏见**（Actor-Observer Bias）：
- 观察者过度将行为归因于行为者的内在特质（"他就是没有意志力"）
- 行为者倾向于归因于外部情境（"是因为环境太难了"）

> **教练应用**：避免对用户行为做"内在特质"式的稳定归因（"你就是缺乏坚持力"），而是共同分析行为发生的情境条件。

**自我服务偏见**（Self-Serving Bias）：
- 成功时归因于自己（内部、稳定）
- 失败时归因于环境（外部、不稳定）

> **应用价值**：适度自我服务偏见有助于维护自尊；但过度时导致无法从失败中学习。

**归因治疗四策略**（对习得性无助的干预）：
1. 改变对结果的评价（"失败不那么严重"）
2. 改变认知目的指向（"我们是在找线索，不是判罪"）
3. 从不可控期望→可控期望（"这件事我能做什么？"）
4. **从内部/整体/稳定归因→外部/特殊/不稳定归因**（"这次是因为……，不是因为我这个人不行"）

### 5. 印象形成与自我表现

**印象形成的四个效应**：

| 效应 | 含义 | 健康教练应用 |
|------|------|-----------|
| **优先效应（首因效应）** | 第一印象影响最大 | 第一次会话决定整个关系质量 |
| **新近效应** | 最近信息影响大（长期关系中） | 每次会话结束时留下积极印象 |
| **晕轮效应** | 一个特质扩散影响整体评价 | 教练的一次真诚有效帮助→整体信任感建立 |
| **自我实现预言** | 对他人的预期影响其实际行为 | 教练对用户的期望信念影响用户表现 |

**自我实现预言的正向应用**：
```
教练语言示例（积极期望激活）：
"基于我对你的了解，我认为你是一个一旦真正决定就能坚持的人。"
"你今天做到的这件事，说明你有这个能力。"
→ 高期望→用户行为向期望靠近（罗森塔尔效应的健康版本）
```

### 6. 自我导向：卡罗利四阶段模型

卡罗利（Karoly）自我调节变化的四阶段，与TTM高度对应：

| 卡罗利阶段 | 核心任务 | TTM阶段对应 |
|-----------|---------|-----------|
| **1. 问题识别** | 认识到问题的存在和需要改变 | S1前意向→S2意向 |
| **2. 承担义务** | 做出改变的承诺，承担责任 | S3准备 |
| **3. 扩展自我管理** | 执行具体的行为改变计划 | S4行动 |
| **4. 习惯形成** | 新行为自动化，内化为习惯 | S5维持 |

**有效强化的三个必要条件**：
1. 必须能加强个体的某种特定行为
2. 强化方式必须切实可行（方便且可立即使用）
3. 强化方式必须足够有力

> **BHP应用**：每个TTM阶段对应不同的主要干预任务——不在错误阶段提供错误干预（如对S1用户提供行动方案是无效的）。

### 7. 利他行为与助人动机

**利他决策的五步骤模型**（Latané & Darley）：
1. 注意到紧急事态
2. 正确解释情景（需要帮助？）
3. 决定对个人责任（该我出手？）
4. 决定恰当介入方式
5. 执行助人行为

**旁观者效应**（Bystander Effect）：群体越大，个人承担责任感越低→越不容易采取行动。

**影响利他行为的个体因素**：
- 助人者心境（正性情绪增加助人行为）
- 同情心和内疚感
- 责任感和承诺

> **BHP Sharer角色设计**：需要明确个人责任感（而非依赖群体扩散的责任稀释），设计"一对一关怀"机制——让助人行为具有明确的个人责任锚定。

## BHP平台应用映射

### 教练-用户关系发展阶段管理

```
关系发展路径图：
注意（建立第一印象）
  → 接触（第一次实质互动）
    → 融合（信任+开放度充分建立）
      ↓ 如果中断或负面体验：
冷漠（互动减少，情感疏离）
  → 疏远（主动回避）
    → 终止（流失）

教练关键行动：
注意阶段：管理好首次印象（首因效应）
接触阶段：每次互动后留下正面情绪（新近效应+晕轮效应）
融合阶段：维持真实性，避免虚假安抚侵蚀信任
冷漠预警：用户沉默/延迟回应→立即主动触达
```

### 态度改变的BHP实践路径

```
认知→情感→行为 的正确干预顺序：

❌ 错误路径（常见）：
   提供大量健康知识（改变认知）
   → 期望产生行为改变
   → 通常失败

✅ 正确路径（BHP）：
   先激活情感关联（"这个和你最在意的什么有关系？"）
   → 在情感激活的基础上提供认知支持
   → 设计微行为践行（行为先于态度强化）
   → 行为成功→正面情感体验→态度巩固
```
