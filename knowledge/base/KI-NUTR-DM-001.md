## 模块1：适用情境与触发条件

### 核心适用场景
1. T2DM确诊用户的饮食教育与医学营养治疗（MNT）
2. 糖尿病前期（空腹血糖受损/糖耐量减低）的逆转干预
3. 血糖控制不达标用户的饮食调整
4. 胰岛素或口服降糖药用药用户的碳水化合物管理

## 模块2：干预核心逻辑

### 糖尿病营养治疗的作用机制
```
饮食摄入 → 血糖调节链 → HbA1c
  ↓
碳水化合物 → 葡萄糖吸收速度 → 餐后血糖峰值
膳食纤维 → 减缓葡萄糖吸收 → 平缓血糖曲线
脂肪种类 → 胰岛素敏感性 → 整体血糖控制
体重减轻 → 胰岛素抵抗改善 → 内生血糖降低
饮食规律性 → 血糖波动幅度 → 并发症风险
```

**最高证据级别的干预效果**：
- 低GI饮食替代高GI：HbA1c降低约0.4%
- 每减重1kg：空腹血糖降低约0.1-0.2 mmol/L
- 低碳水化合物饮食（<130g/天）：短期内有效降低餐后血糖和HbA1c
- 地中海饮食：可使T2DM发病风险降低30%，对已确诊者改善血糖控制

### 关键概念速查（用于用户教育）
```
血糖相关核心指标：
  空腹血糖（FPG）：
    正常：3.9-6.1 mmol/L（<100 mg/dL）
    糖尿病前期：6.1-7.0 mmol/L
    糖尿病诊断：≥7.0 mmol/L（两次）

  餐后2小时血糖（2h PG）：
    正常：<7.8 mmol/L
    糖尿病前期：7.8-11.1 mmol/L
    糖尿病诊断：≥11.1 mmol/L

  糖化血红蛋白（HbA1c）——最重要！
    正常：<5.7%
    糖尿病前期：5.7-6.4%
    糖尿病控制目标：<7.0%（一般成人）
    HbA1c代表近2-3个月平均血糖，不受单日波动影响
```

## 模块3：核心干预方案

### 3.1 碳水化合物管理（最核心干预）

#### 每日碳水化合物目标
```
个体化推荐范围（基于ADA MNT指南）：

  常规管理：总热量的45-60%来自碳水化合物
    以1800千卡/天为例：200-270g碳水/天

  低碳水化合物方案（需医生/营养师监督）：
    低碳：100-130g/天
    极低碳/生酮：<50g/天（通常仅短期/监督下使用）

  ⚠️ 使用胰岛素或磺脲类药物的用户：
     碳水摄入减少必须同步调整药物剂量（防止低血糖）
     必须与医生协调再执行
```

#### 碳水化合物计数（Carbohydrate Counting）
```
方法：追踪每餐摄入的碳水化合物克数

简化版"手掌法"：
  1份主食 = 1个拳头大小的米饭/面条/面包
           = 约15g碳水化合物
  1份水果 = 1个中等苹果/橙子/100g葡萄
           = 约15g碳水化合物
  1份牛奶 = 250ml全脂/低脂牛奶
           = 约12g碳水化合物

典型餐次碳水目标（以180g碳水/天为例）：
  早餐：45g（3份）
  午餐：60g（4份）
  晚餐：60g（4份）
  加餐：15g（1份）×如有
```

#### 高GI vs 低GI食物对照表
```
血糖生成指数（GI）分级：
  低GI（<55）→ 选择
  中GI（55-70）→ 适量
  高GI（>70）→ 限制/替换

常见食物GI对照：

  主食类：
  高GI → 白米饭(72) / 白面包(75) / 白馒头(65) / 粥/稀饭(88!)
  低GI → 糙米(55) / 全麦面包(51) / 燕麦(55) / 意大利面(42)
  ★行为化：把白粥改为燕麦，是最高性价比的单一改变

  蔬菜类（大部分低GI）：
  低GI → 西兰花/菠菜/番茄/黄瓜/芹菜（GI<15）
  高GI → 土豆(78，但冷却后降至56) / 胡萝卜(71) / 南瓜(75)

  水果类：
  低GI → 苹果(38) / 梨(38) / 柑橘(43) / 草莓(40) / 蓝莓(53)
  高GI → 西瓜(72) / 荔枝(79) / 枣子(103!)

  豆类（优秀低GI食物）：
  低GI → 黄豆(18) / 绿豆(38) / 黑豆(30) / 扁豆(29)
  
  乳制品：
  低GI → 牛奶(27) / 酸奶(36-51)
```

### 3.2 糖尿病前期逆转策略

```
糖尿病前期的窗口期价值（铁律8——第三层价值）：
  糖尿病前期是完全可逆的状态！
  干预目标：5-7%体重减轻 + 每周150分钟中等强度运动
  证据：DPP（糖尿病预防计划）研究——生活方式干预使T2DM
        发病率降低58%（比二甲双胍的31%效果更强）

具体执行：

  ① 体重管理（首要目标）
     目标：减轻体重5-7%
     以70kg为例：目标减重3.5-5kg
     机制：内脏脂肪减少→肝脏/肌肉胰岛素敏感性恢复

  ② 减少精制碳水化合物
     立即行动：用低GI替代高GI（见3.1）
     戒含糖饮料（每罐可乐含35g糖=2份碳水）

  ③ 每周150分钟运动
     快走/游泳/骑车（中等强度）
     抗阻训练每周2次（增加肌肉=增加葡萄糖消耗库）
     行为化：每天30分钟快走，不需要连续

  ④ 膳食纤维增加
     目标：25-38g/天
     首选：燕麦/豆类/蔬菜（每天500g蔬菜）
```

### 3.3 饮食模式选择

#### 循证支持的T2DM友好饮食模式
```
模式1：地中海饮食（最强长期证据）
  核心：橄榄油/蔬菜/豆类/全谷物/鱼/坚果为主
       适量红酒（可选）/少量红肉/低乳制品
  优势：心血管保护+血糖控制双重获益
       适合中国用户的改良版：茶籽油替代橄榄油
  血糖效果：HbA1c降低0.3-0.5%

模式2：低碳水化合物饮食（短期降糖最快）
  碳水：100-130g/天
  优势：餐后血糖快速改善，部分研究显示药物减量
  注意：
    长期依从性挑战
    必须在医疗监督下进行（药物调整）
    极低碳水（<50g）不建议长期自行执行

模式3：DASH饮食（CVD高风险合并型最优）
  适合：T2DM合并高血压用户（最常见组合）
  参见：KI-NUTR-CVD-001 DASH部分

模式4：间歇性禁食（IF）——新兴证据
  16:8模式（8小时进食窗口）
  当前证据：短期血糖改善，长期证据不足
  禁忌：使用胰岛素/磺脲类药物用户（低血糖风险！）
```

### 3.4 特殊营养素管理

**蛋白质**：
```
T2DM用户的蛋白质摄入：
  常规目标：1.0-1.5g/kg体重/天
  （高于一般人群推荐，因肌肉保护和饱腹感价值）

  优选蛋白质来源：
    ✅ 鱼类（尤其深海鱼）/ 禽类去皮 / 豆类/豆腐
    ✅ 低脂乳制品（酸奶）
    ❌ 限制：红肉/加工肉制品（增加CVD风险）

  ⚠️ 肾功能受损（糖尿病肾病）：
     蛋白质必须限制至0.8g/kg/天以下
     必须转介肾科营养专项评估
```

**脂肪**：
```
T2DM脂肪管理（同CVD，因高度重叠）：
  饱和脂肪：<7%总热量
  反式脂肪：≈0（严格避免）
  欧米伽-3：每周2次深海鱼
  
  ★关键：脂肪种类比总量更重要
```

**膳食纤维**：
```
目标：25-38g/天（大多数T2DM用户严重不足）

最重要的纤维来源：
  可溶性纤维（直接改善血糖）：
    燕麦(β-葡聚糖) / 豆类 / 苹果 / 车前子
  不可溶性纤维（增加饱腹感/肠道健康）：
    全谷物 / 蔬菜茎叶 / 麸皮

行为化：
  每餐先吃蔬菜（延缓碳水吸收，压低餐后血糖峰值）
  "先菜后饭"——这一个行为改变就可以降低餐后血糖约10-20%
```

### 3.5 血糖监测联动行为

```
自我血糖监测（SMBG）与饮食行为联动：

  餐后2小时血糖目标：<10.0 mmol/L（一般），<8.0（严格）

  记录方式：
    饮食记录 + 对应血糖值 → 识别个人"血糖炸弹食物"
    （同样食物不同人血糖反应差异显著）

  常见"血糖陷阱"需检测验证：
    ✗ 白粥/米粥（GI=88，比白米饭更高！）
    ✗ 果汁（去掉纤维的水果，液态糖）
    ✗ 蜂蜜/红糖（与白糖血糖影响相似）
    ✗ 无糖食品（可能含大量淀粉/麦芽糖醇，仍升血糖）

CoachAgent话术：
  "你有没有试过：吃完东西2小时后测一次血糖？
  不是要你每次都测，而是选几样你经常吃的食物，
  亲眼看看它们对你血糖的影响——
  这比任何饮食课都更有说服力。"
```

## 模块5：监测重点

| 指标 | 频率 | 目标值 | 关注阈值 | 行动触发 |
|------|------|--------|---------|---------|
| 空腹血糖 | 每天（血糖未达标时）| 4.4-7.0 mmol/L | >10或<3.9 | 立即就医/调整方案 |

**低血糖安全红线（铁律1）**：
- 血糖 <3.9 mmol/L = 低血糖发作
- 立即处理：15g快速碳水（半杯果汁/3颗糖/一勺蜂蜜）
- 15分钟后复测，若仍低 → 再摄入15g
- 使用胰岛素/磺脲类用户：增加运动/减少饮食前，必须与医生协调

## 模块6：铁律边界

```
铁律A 低血糖优先铁律：
  任何血糖降低措施（饮食/运动）在使用胰岛素或磺脲类
  药物的用户中，必须同步与医生协调药物剂量。
  禁止：推荐低碳水饮食而不提醒低血糖风险。
  话术："这个饮食改变很好，但因为你在用降糖药，
        请你先告诉医生你打算这样调整——
        他可能需要相应地调整你的药量。"

铁律B 肾病分期铁律：
  T2DM患者发生肾病（eGFR<60或尿蛋白+）：
  蛋白质摄入必须限制（禁止推荐高蛋白饮食）。
  钾/磷管理规则可能反转（高钾食物可能变禁忌）。
  必须转介肾科营养专项评估。

铁律C 无糖食品误导铁律：
  禁止将"无糖食品"等同于"糖尿病患者可以随意吃"。
  无糖≠无碳水≠不升血糖。
  需解释：无糖食品中的淀粉/麦芽糖醇同样升血糖。

铁律D 医疗权限铁律（铁律4）：
  营养治疗是降糖药物/胰岛素的辅助，不是替代。
  T2DM逆转的可能性：特定患者（病程<10年，β细胞功能残存）
  在强化生活方式干预下可能实现缓解（参考DiRECT研究）。
  话术："饮食+运动改善血糖的效果是真实的，
        但请和你的医生一起讨论是否可以减药——
        这个决定必须在医疗监督下做。"
```

## 模块8：话术变体库

### 破除"完全戒糖"误区（S1）
1. "糖尿病不是要完全不吃糖，而是要管理吃糖的方式和量——特别是主食（米饭、面条）里的淀粉，它们进入血液的速度，其实和白糖差不多。"
2. "不是甜的食物就不好，苦瓜、西兰花其实对血糖非常友好；白粥虽然没有甜味，但升血糖的速度是白米饭的1.5倍。关键是食物的GI，不是甜不甜。"

### 引导执行先菜后饭（S2-S3）
1. "有一个简单的技巧：每餐先把蔬菜吃完，再吃主食和肉。研究显示这一个改变就能降低餐后血糖10-20%。你下次吃饭的时候可以试试吗？"
2. "不需要计算，不需要秤——先吃菜，把半碗蔬菜吃完之后再吃其他的，就这一步。"

### 面对"吃得少但血糖还是高"的用户（S3）
1. "血糖高不一定是吃得多——很可能是吃的食物'升血糖太快'。我们来看一下，你通常早饭吃什么？——有一个特别隐蔽的陷阱就是白粥。"
2. "血糖是个很诚实的指标——它不会骗你。如果你愿意做一个实验：今天的早饭换成燕麦，两小时后测一次血糖，跟你平时的早饭比一比。你愿意试吗？"

### 糖尿病前期用户的希望建立（S1-S2）
1. "这个诊断其实是个好消息——不是因为它没什么，而是因为你现在知道了，还有机会改变。58%的人通过饮食和运动，让血糖回到了正常范围。你是想成为那58%的人吗？"
