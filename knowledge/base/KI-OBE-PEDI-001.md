## 模块1：适用情境与触发条件

### 核心适用场景
1. **家长询问孩子体重问题**："我孩子是不是太胖了？"
2. **VisionGuard模块用户**：青少年视力档案同步评估体重状态
3. **家庭慢病管理**：父母自身有慢性病，孩子体重需一并关注
4. **学校体检后咨询**：体检报告显示"超重"或"肥胖"后的行为指导

### BMI计算方法
```
BMI = 体重(kg) ÷ 身高(m)²

示例：12岁男孩，体重50kg，身高1.55m
BMI = 50 ÷ (1.55)² = 50 ÷ 2.4025 ≈ 20.8
对照男生12.0岁超重界值20.7：超重（接近肥胖临界）
```

## 模块3：行为干预触发决策树

```
【节点1】用户输入孩子体重/身高数据
      ↓
计算BMI，对照标准表

【节点2A】BMI < 超重界值
  → 正常体重范围
  话术："你孩子的体重在正常范围内。保持均衡饮食和每天60分钟运动
       就很好。需要我给你一些具体建议吗？"
  行动：提供预防性饮食行为建议（非干预级）

【节点2B】BMI ≥ 超重界值 且 < 肥胖界值
  → 超重范围
  话术："按中国卫生标准，你孩子的体重属于超重。这是一个需要开始
       关注的信号，但不需要紧张——主要通过饮食行为和运动调整就可以。"
  行动：进入家庭行为处方（见节点3）

【节点2C】BMI ≥ 肥胖界值
  → 肥胖范围
  话术："按标准，你孩子的体重达到了肥胖级别。我强烈建议先去医院
       做一次评估，排除内分泌等原因——同时我们也可以开始行为层面的调整。"
  行动：
    - 建议儿科/内分泌科评估（必要）
    - 进入家庭行为处方（同步进行）
    - 评估是否有相关合并问题（高血压/血脂异常/睡眠问题）

【节点3】家庭行为处方（超重/肥胖通用）
  ├── 饮食调整（首要）
  │   ├── 减少含糖饮料（奶茶/果汁/碳酸饮料 → 白水/淡茶）
  │   ├── 减少超加工食品（薯片/饼干/方便面）
  │   ├── 增加蔬菜比例（每餐半盘）
  │   └── 规律三餐，不跳餐（尤其不跳早餐）
  │
  ├── 运动增加（第二要素）
  │   ├── 目标：每天至少60分钟中等强度活动
  │   ├── 起点：从现有基础每天增加10分钟
  │   └── 形式：学校体育课 + 放学后活动 + 家庭周末运动
  │
  ├── 屏幕时间控制
  │   ├── ≤2小时/天（娱乐屏幕）
  │   └── 每坐40分钟起身活动5-10分钟
  │
  └── 睡眠保障
      ├── 6-12岁：9-11小时
      └── 13-18岁：8-10小时

【节点4】家长行为支持
  家长是孩子行为改变的核心环境变量：
  ├── 不在孩子面前使用"胖""丑"等评价性语言
  ├── 家庭共同行动（家长同步减少含糖饮料/增加运动）
  ├── 不用食物作为奖励/惩罚
  └── 以身作则：家长先改变
```

## 模块4：M-Action清单

```
P1 - BMI计算与分级评估（今天就可以做）
  工具：体重(kg) ÷ 身高(m)² → 对照标准表 → 明确分级

P1 - 含糖饮料替代（最高优先级饮食行为）
  具体：奶茶/果汁/碳酸饮料 → 白水/无糖茶/气泡水（无糖）
  原因：液体糖分是儿童超重的首要可控因素

P2 - 每日60分钟活动目标分解
  具体：学校体育课30分钟 + 放学后步行/骑车回家20分钟 + 饭后散步10分钟

P2 - 屏幕时间记录与限制
  工具：记录一周屏幕时间基线 → 每周减少15分钟

P3 - 家庭餐盘改造
  具体：每顿午晚餐蔬菜占餐盘1/2，主食1/4，蛋白质1/4
  
P4 - 睡眠时间保障
  具体：睡前1小时关闭所有屏幕，建立入睡仪式
```

## 模块5：监测重点

| 指标 | 评估频率 | 正常/良好 | 关注阈值 | 行动触发 |
|------|---------|---------|---------|---------|
| BMI变化 | 每月一次 | 保持/缓慢下降 | 3个月无改善 | 重新评估方案+医学评估 |
| 含糖饮料摄入 | 每周回顾 | 0-1次/周 | ≥5次/周 | 强化替代策略 |
| 每日活动时间 | 每天记录 | ≥60分钟 | ≤30分钟 | 分析阻力，调整方案 |
| 屏幕时间 | 每周回顾 | ≤2小时娱乐 | ≥4小时 | 环境设计干预 |
| 血压（10岁以上超重）| 每季度 | 正常范围 | 收缩压≥130/舒张压≥80 | 建议儿科评估 |

## 模块6：铁律边界

```
铁律A 不评判儿童体貌铁律：
  绝对禁止使用"胖""难看""肥"等描述性语言评价儿童外貌。
  原因：体象羞耻是青少年饮食障碍的重要诱因，损害长期心理健康。

铁律B 家长系统优先铁律：
  儿童行为改变的核心干预对象是家长，而非儿童本身。
  禁止：将全部压力和指令直接施压于儿童。
  原因：儿童饮食环境由家长决定，家长不改变，儿童难以改变。

铁律C 医学评估触发铁律：
  肥胖级别（BMI≥肥胖界值）必须建议医学评估，不可只做行为干预。
  原因：需排除Cushing综合征、甲状腺功能减退等内分泌疾病，
       以及评估是否合并高血压/胰岛素抵抗。

铁律D 减重速度控制铁律：
  儿童减重目标是"控制增重速度/维持体重"，而非急速减重。
  禁止：推荐儿童节食、极低热量饮食或任何形式的断食。
  原因：生长发育阶段的营养不足影响身高发育和认知发展。
```

## 模块7：叠加情境提示

```
高频叠加组合：
  组合1：VisionGuard模块 + 本KI = 青少年全面健康档案
    → 视力问题和体重问题高度共存（久坐屏幕时间是共同风险因素）
    → 两个问题可以用同一个行为干预（减少屏幕时间）同步解决

  组合2：家长慢病管理 + 本KI = 全家健康行动
    → 父母有高血压/糖尿病的家庭，孩子超重风险更高
    → "全家一起改变"的框架接受度更高，依从性更好

  组合3：营养行为处方 + 本KI = 儿童友好版饮食设计
    → 本KI确定问题级别，营养模块设计具体饮食方案

情境处理优先级：
  肥胖级别且合并症状（疲劳/高血压/睡眠问题）> 医学评估优先
  超重级别无合并症状 > 家庭行为干预优先
```

## 模块8：话术变体库

### 告知超重结果（家长S1）
1. "根据你提供的数据，按中国卫生部门的标准，你孩子的BMI是X，属于超重范围。这不是你和孩子的错——这个阶段干预，效果是最好的。"
2. "超重不代表不健康，但它是一个信号——告诉我们需要在饮食和运动上做一些调整了。你现在最想先从哪里开始？"

### 家长问"怎么让孩子减肥"（S2）
1. "比起'减肥'，我更建议说'让孩子更健康'——这样的框架对孩子更好，也更可持续。"
2. "孩子的饮食习惯很大程度上是家庭环境决定的。如果家里的食物环境变了，孩子的体重会自然跟上——我们从家里的购物清单开始说说？"
3. "最有效的第一步，通常不是让孩子运动更多，而是先把含糖饮料戒掉——你家孩子现在喝多少奶茶/果汁？"

### 家长担心孩子心理受影响（S2）
1. "你的担心非常对——在孩子面前关注体重要非常谨慎。我们的目标是让孩子建立'健康行为'的习惯，而不是让他意识到'我需要减肥'。"
2. "如果孩子问为什么要改变饮食，你可以说：'我们全家都在更健康地吃东西'——把他变成全家行动的参与者，而不是被改变的对象。"
