## 模块1：适用情境与触发条件

### 核心适用场景
教练面对用户需要**激活行为改变动力、设计改变路径、处理阻力与挑战、促进深度成长**时的全场景工具矩阵。

### TTM阶段映射
- **S0（前意向期）**：好奇心缺口工具、认知失调触发、Lewin解冻模型
- **S1（意向期）**：Maslow需求分析、ARCS动机设计、Rogers促进行为
- **S2（准备期）**：GROW目标设定、Egan高明助人者模型、IF-THEN实施意图
- **S3（行动期）**：Bandura角色示范、内化游戏（Inner Game）、神经可塑性强化
- **S4（维持期）**：认知负荷管理、CLEAR模型、情商提升工具
- **S5（整合期）**：神经逻辑层次（Bateson）、转型学习、STEPPPA成长模型
- **S6（超越期）**：变革型领导框架、贡献导向工具、团队发展模型

### 触发关键词
`为什么改变` `怎么开始` `我做不到` `坚持不住` `我是谁` `有什么意义` `目标设定` `解决问题` `找不到方向` `情绪管理`

### 工具一：GROW模型（Whitmore）
**适用阶段：S2-S4 | 核心功能：目标—现实—选择—行动四步引导**

```
G - Goal（确立目标）
  引导问题：
  "你今天最想实现什么？"
  "这个目标和你更长远的生活目标有什么联系？"
  "你什么时候会知道自己达到了？"

R - Reality（审视现实）
  引导问题：
  "你现在的状态是什么？"
  "到目前为止，你已经尝试过什么？"
  "是什么让这件事变得困难？"

O - Options（探索选择）
  引导问题：
  "如果什么都不是障碍，你会怎么做？"
  "还有什么其他办法？"
  "哪个选择让你感觉最可行？"

W - Will（树立意志）
  引导问题：
  "你下一步准备做什么？"
  "你什么时候开始？"
  "我怎么能支持你？"
```

**BHP应用话术（S2用户，目标模糊）**：
- "你说想'健康一些'——如果我们把这个变成一个具体的目标，你最想先改变什么？"
- "想象三个月后，你为自己感到骄傲的那一刻——那时候你在做什么？"
- "从1到10，你现在对这个改变有多大信心？让这个数字提高1分，需要什么？"

### 工具二：Egan高明的心理助人者模型（三阶段）
**适用阶段：S1-S4 | 核心功能：探索现状→挑战盲点→转化行动**

```
阶段一：探索（建立安全关系，澄清现状）
  技术：
  - 开放性提问（不带评判地倾听）
  - 用自己的话复述（确认理解）
  - 聚焦于对方正在说的，而不是你打算说的

阶段二：挑战（从不同角度看问题）
  技术：
  - "从你朋友的角度看，这个情况是什么样的？"
  - "如果换一个视角，你会看到什么？"
  - "你内心真正想要的是什么？"

阶段三：行动（将意图转为具体步骤）
  技术：
  - 让对方想出尽可能多的策略
  - 评估可行性并锁定第一步
  - 约定跟进时间
```

**关键原则**：若仍有阻力，退回阶段一重新建立关系，不急于推进。

**BHP应用话术**：
- 探索："你说戒不掉宵夜，能跟我说说，通常是什么情况下会发生？"（阶段一）
- 挑战："如果三个月后的你回来看今天的自己，会说什么？"（阶段二）
- 行动："今晚9点之前，你愿意做一件事来迈出第一步，那会是什么？"（阶段三）

### 工具三：Maslow需求层次理论
**适用阶段：S0-S2 | 核心功能：识别用户当前真实需求层次**

```
需求五层（从底到顶）：
  生理需求   → 身体舒适、基本生存
  安全需求   → 免受伤害、心理安全
  归属需求   → 被接受、被关心
  尊重需求   → 成就感、被认可
  自我实现   → 充分发挥潜力、高峰体验
```

**BHP需求诊断工具**：
```
用户信号               → 当前主导需求层次    → CoachAgent策略
"我很痛苦/太累了"      → 生理/安全层        → 先做症状缓解，不谈目标
"家人不理解我"         → 归属层             → 先建立关系认同，再谈行为
"我想证明自己能做到"   → 尊重层             → 强化成就记录，设置里程碑
"我想活出自己的样子"   → 自我实现层         → 进入S5成长框架对话
```

**注意**：用户停留在低层需求时，禁止直接跳入成长框架（铁律7）。

### 工具四：Rogers促进行为三要素
**适用阶段：全阶段 | 核心功能：构建有效教练关系的三个基础**

```
要素一：真诚一致（Congruence）
  含义：教练的言行与内心体验保持一致，不表演
  应用话术：
  - "我听到这里，感觉你其实很累了——是这样吗？"（如实反映感受，不修饰）

要素二：无条件积极关注（Unconditional Positive Regard）
  含义：接受用户的全部，包括挣扎和失败，不评判
  应用话术：
  - "你昨晚又吃宵夜了——谢谢你告诉我，这本身就需要勇气。"

要素三：共情理解（Empathic Understanding）
  含义：从用户的视角精准理解其体验
  应用话术：
  - "你说'又没做到'——我感觉这里面有一点自责，是吗？"
  - "你在说这些的时候，我感受到了你的疲惫。"
```

**铁律**：三要素同时缺失 = 关系破裂。在任何干预前，优先检查这三个要素是否到位。

### 工具五：认知失调（Festinger）
**适用阶段：S0-S1 | 核心功能：通过暴露认知矛盾触发改变动力**

```
原理：当人们的行为与自我认知/价值观不一致时，产生心理不适
     → 驱动他们改变行为或修正认知来消除不适

BHP应用三步：
  Step 1：确认用户的自我认同
    "你说你是一个很爱家人的人——"
  Step 2：温和呈现行为矛盾
    "——但你说你从来没有给孩子做过一顿饭，因为'太忙了'。"
  Step 3：开放式空间，让用户自己解决矛盾
    "你怎么看这两件事？"（不要替用户下结论）
```

**警告**：
- ❌ 不可用于羞辱（"你说爱家人，但你连血糖都不管"）
- ❌ 不可连续使用，会产生防御
- ✅ 配合Rogers三要素使用效果最佳

### 工具六：Lewin行为重塑模型（解冻—改变—再冻结）
**适用阶段：S1-S3 | 核心功能：系统性行为改变的三阶段框架**

```
阶段一：解冻（打破现有平衡）
  驱动力：增加"改变的理由"，减少"维持现状的理由"
  BHP应用：
  - "你最近有没有感觉到，原来的方式开始失效了？"
  - 分析驱动力（健康恐惧、家人期望）和阻力（习惯、环境）

阶段二：改变（引入新行为）
  关键：提供具体、可操作的新行为（微行为优先）
  BHP应用：
  - "这周就做一件事：每天饭后走10分钟——只要这一件。"

阶段三：再冻结（巩固新状态）
  关键：将新行为嵌入身份认同和社会系统
  BHP应用：
  - "你现在说'我是每天饭后会散步的人'——这个身份你觉得如何？"
  - 建立社会支持（告诉家人）、物理锚点（运动鞋放门口）
```

### 工具七：内化游戏——Inner Game（Gallwey）
**适用阶段：S3-S5 | 核心功能：处理内在批评声音，释放自然表现力**

```
核心公式：
  表现 = 潜力 - 干扰
  干扰 = 自我1（批判、焦虑的声音）的过度控制

两个自我：
  自我1：理性批判者，"你又做错了""你不行"
  自我2：自然感知者，身体的智慧，"就这样做"

消除干扰的三个技术：
  技术1：觉察而不评判
    "你现在注意到什么？"（描述，不评价）
  技术2：关注感官细节（转移注意力给自我2）
    "今天吃了什么，你吃第一口的时候，感觉如何？"
  技术3：信任自然过程
    "如果你放手让身体来做这件事，会发生什么？"
```

**BHP核心应用**：
- 用户自我批评过度（"我太没有意志力了"）时的干预：
  - "我注意到你用了'没有意志力'这个词——这是自我1在说话。你的自我2，那个真正体验过运动之后舒适感的你，他会怎么说？"

### 工具八：情商五维度（Goleman）
**适用阶段：S2-S5 | 核心功能：提升情绪管理与人际效能**

```
五个维度：
  1. 自我觉察（Self-Awareness）
     识别自己的情绪状态和对行为的影响
     → CoachAgent：帮助用户识别情绪触发→行为模式

  2. 自我管理（Self-Management）
     调节情绪、延迟满足、适应变化
     → CoachAgent：应对诱惑时的情绪调节策略

  3. 社会觉察（Social Awareness）
     理解他人的情绪需求（共情）
     → CoachAgent：改善家庭饮食环境协作

  4. 关系管理（Relationship Management）
     影响、激励和引导他人
     → CoachAgent：建立社会支持系统

  5. 动机（Motivation）
     内在驱动、成就取向、乐观
     → CoachAgent：识别内在动机，连接到身份认同
```

**情绪—行为链工具**：
```
[情绪] → [冲动] → [行为] → [后果]
愤怒/孤独/压力 → 想吃东西 → 宵夜 → 血糖升高/内疚

介入点：
  在[情绪]层：觉察是什么情绪（自我觉察）
  在[冲动]层：暂停6秒，用替代行为（自我管理）
  在[环境]层：减少诱惑物，建立支持（关系管理）
```

### 工具九：Bateson神经逻辑层次
**适用阶段：S4-S6 | 核心功能：身份认同层面的深度改变**

```
六个层次（从底到顶）：
  环境    → 我在哪里/什么时候？
  行为    → 我做了什么？
  能力    → 我能做什么？
  价值观  → 为什么这对我重要？
  身份认同 → 我是谁？
  使命    → 这是为了什么更大的目的？
```

**关键洞见**：大多数行为干预停留在"行为层"（你做了什么），但持久改变需要在"身份认同层"发生。

**BHP S4→S5转折话术**：
```
行为层（旧）："你已经坚持了三个月。"
      ↓
身份层（新）："三个月的行动告诉了你关于自己的什么？"
      ↓
价值层（更深）："你说'健康'对你重要——它让你能做什么？
                失去它，你最怕失去的是什么？"
      ↓
使命层（最深）："这样的生命，你想留给孩子/这个世界什么？"
```

**铁律**：S0-S2用户处于急性痛苦期，不可直接跳入身份/使命层对话（铁律7）。

### 工具十：Mezirow转型学习
**适用阶段：S4-S6 | 核心功能：批判性反思触发深度视角转变**

```
转型学习的关键时刻：
  "迷失方向的困境"（Disorienting Dilemma）
  = 当旧有的框架无法解释新的体验时

BHP触发时刻：
  - "我以为控制血糖只是为了不死，但现在我发现我变成了
    一个完全不同的人——这是我没想到的。"
  - 疾病诊断后的世界观重构
  - 指标达标后的"然后呢？"危机

转型学习四步：
  Step 1：暴露困境（不要快速安抚）
    "你说'不知道接下来要为什么坚持'——这是个很重要的问题。"
  Step 2：批判性反思（检视原有假设）
    "你原来以为健康是为了什么？现在你怎么看？"
  Step 3：理性对话（探索新可能）
    "如果'为了血糖'不够用，那还有什么理由值得你继续？"
  Step 4：行动整合（在新框架下行动）
    "带着这个新的理解，你明天会做什么不一样的事？"
```

### 工具十一：Bandura角色示范与自我效能
**适用阶段：S2-S4 | 核心功能：通过示范和替代性体验建立自我效能**

```
自我效能的四个来源（重要性从高到低）：
  1. 成功体验（最强）：自己真正做到过
  2. 替代性体验：看到与自己相似的人做到了
  3. 语言说服：被他人鼓励"你能行"
  4. 生理状态：身体感觉良好时，信心更高

BHP自我效能建设策略：
  成功体验路径：
    - 设置极小胜利（第一天：今天比昨天少一口糖）
    - 记录并回顾每一次做到
    - "你想起来了——三周前你第一次走了10分钟——那时候你的感觉是？"

  替代性体验路径：
    - 分享平台用户成功故事（相似背景优先）
    - "有一位和你情况相似的阿姨，也是刚确诊高血压，她三个月后..."

  角色示范话术：
    - "你已经做到了一件很多人觉得不可能的事——你愿意告诉我一件让你骄傲的事吗？"
```

### 工具十二：正念基础（Kabat-Zinn）
**适用阶段：S3-S5 | 核心功能：培养当下觉察，减少自动化反应**

```
核心定义：
  以非评判、当下时刻的方式对自己的体验保持觉察

BHP三个应用场景：
  场景1：情绪性进食干预
    "在你准备拿零食之前，停1分钟——注意你身体里有什么感觉？
     是饥饿感，还是别的什么？"

  场景2：运动过程中的正念强化
    "下次散步时，试试全程只注意脚踩地的感觉——不需要想别的。"

  场景3：睡前血糖管理
    "睡前5分钟，把手放在肚子上，感受呼吸——这也是照顾身体的一种方式。"

正念微练习（针对忙碌用户）：
  - 吃第一口食物前：深呼吸3次，注意食物的颜色/气味
  - 每小时：闭眼10秒，扫描一遍身体感受
  - 入睡前：默数10次呼吸，不数对了就重新开始
```

## 模块5：监测重点

### S0-S4 用户行为指标
| 指标 | 良好信号 | 关注信号 |
|------|---------|---------|
| GROW对话完成度 | 每次都能完成W步（行动意愿）| 停留在G/R步，无法进入O/W |
| 自我效能评分 | 从首次≥5/10，逐步提升 | 连续3次<4/10 |
| 情绪—行为链觉察 | 能命名触发情绪 | "不知道为什么就吃了" |
| Rogers关系质量 | 用户主动分享挣扎细节 | 只报喜不报忧，回答简短 |

### S4-S6 成长指标
| 指标 | 良好信号 | 关注信号 |
|------|---------|---------|
| 身份叙事层次 | 从行为层到身份层（"我是..." | 停留在"我做了..."层 |
| 意义感 | 能说出超越血糖的改变理由 | "就是为了血糖" |
| 视角转型 | 有"我以前没这样想过"的表达 | 无新洞见产生 |

## 模块6：铁律边界

```
铁律A 关系优先铁律：
  在Rogers三要素（真诚/接纳/共情）建立之前，禁止使用认知失调等
  挑战性工具。关系是一切工具的前提。
  原因：工具在不安全关系中使用 → 防御 → 流失

铁律B 层次匹配铁律：
  不可把神经逻辑层次/转型学习工具用于S0-S2急性痛苦期用户。
  这些用户需要的是"被稳住"，不是"被提升"。
  原因：用户感到"被看穿但不被接住"→直接流失（铁律7）

铁律C 自我批评保护铁律：
  当用户说"我没有意志力/我太差了"时，禁止顺势认可其负面自我叙事。
  必须温和挑战内化游戏中的"自我1"声音。
  原因：强化负面身份 → 习得性无助 → 放弃改变

铁律D GROW工具完整使用铁律：
  禁止只使用G（目标）而跳过R（现实）直接到W（行动意志）。
  脱离现实的目标设定会让用户失败体验感加重。
  原因：未评估现实障碍的计划=注定失败的计划→损伤自我效能
```

## 模块7：叠加情境提示

```
高频叠加组合：
  组合1：GROW + Egan = 目标明确后的深度问题探索
    → 先用GROW设定目标和行动，再用Egan处理情绪阻力

  组合2：Rogers三要素 + 认知失调 = 安全关系下的温和挑战
    → 必须先建立Rogers关系，才能有效使用认知失调

  组合3：情商情绪链 + 内化游戏 = 情绪性进食干预
    → 先识别情绪（情商），再消除内在批评（内化游戏）

  组合4：Bateson层次 + Mezirow转型 = S4→S5转折引导
    → 用Bateson定位身份层，用Mezirow触发视角转变

  组合5：Bandura自我效能 + 正念 = 行动期信心建设

情境优先级：
  急性危机（CrisisAgent触发）> 关系建立（Rogers）>
  动机激活（Maslow/ARCS）> 目标设定（GROW）>
  深度成长（Bateson/Mezirow）
```

## 模块8：话术变体库

### GROW目标设定（S2）
1. "你今天最想完成什么——如果我们今天的对话能帮你推进一件事，那是哪件？"
2. "你说想'好好管理血糖'——能帮我把这个变成一个更具体的画面吗？"
3. "三个月后你回来看今天的自己，你最希望那时候的你能对今天的你说什么？"

### Rogers共情接纳（全阶段）
1. "你昨天又没做到——谢谢你告诉我，能说说当时发生了什么吗？"（无条件接纳）
2. "我听到你说'累了'——你说的这个'累'，是身体上的还是心里的？"（精准共情）
3. "你每次来都很诚实——这对我们的工作真的很重要。"（真诚确认）

### 身份认同转型（S4-S5）
1. "你已经坚持了100天——这100天告诉了你什么关于你自己的事？"
2. "如果用一句话描述'现在的你'和'三个月前的你'的不同，你会怎么说？"
3. "你说以前觉得自己'没有意志力'——你现在还这么看自己吗？"

### 内化游戏干预（S3-S5，自我批评时）
1. "我注意到你评价自己很严苛——如果你的最好的朋友遇到了和你一样的情况，你会对他说什么？"
2. "你的身体其实已经做了很多——它上周走了多少步？它比上个月进步了什么？"
3. "自我批评这个声音说了什么？——现在，把注意力转到身体感觉上，它想要什么？"
