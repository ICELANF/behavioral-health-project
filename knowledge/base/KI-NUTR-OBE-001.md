## 模块1：适用情境与触发条件

### 核心适用场景
1. 超重（BMI 24-27.9）用户的预防性体重管理
2. 肥胖（BMI≥28）用户的医学营养治疗
3. T2DM/高血压/血脂异常合并超重用户的体重优先干预
4. 体重反弹用户的维持策略
5. 减重手术前后的营养支持（教育层面）

### 中国肥胖诊断标准
```
体重相关指标（中国成人标准，与西方人群不同）：
  BMI（体质指数）= 体重(kg) ÷ 身高(m)²
  体重不足：BMI < 18.5
  正常：18.5-23.9
  超重：24.0-27.9     ← 中国标准比西方更低（西方≥25）
  肥胖：≥28.0

  向心性肥胖（腰围）——比BMI更重要的CVD/DM风险指标：
  男性：腰围 ≥90cm（中国标准）
  女性：腰围 ≥85cm（中国标准）

  ⚠️ BMI的局限性：
     BMI不区分肌肉和脂肪
     肌肉多者BMI可能偏高但体脂正常
     老年人/体成分特殊者需结合腰围和体脂率评估
```

## 模块2：干预核心逻辑

### 肥胖营养治疗的作用机制
```
能量平衡方程（不可违背的物理学）：

  体重变化 = 能量摄入 - 能量消耗

  体重减轻的条件：能量缺口（摄入<消耗）
  每减少7700千卡摄入（约1周）≈ 减重1kg体脂

  ⚠️ 重要修正：
     这不是简单线性的——代谢适应会发生
     身体在节食时会降低基础代谢率（"节俭模式"）
     这是减肥平台期和反弹的主要原因
     → 解决方案：配合抗阻训练维持肌肉量

能量缺口的两个来源：
  ① 减少摄入：目标500-750千卡/天缺口
     对应减重速度：0.5-0.75kg/周（安全可持续）
  ② 增加消耗：运动（有氧+抗阻）
     联动参见：KI-EX-NBA-001
```

### 为什么"节食减肥"长期失败的科学解释
```
失败原因一：代谢适应（身体抵抗）
  极低热量摄入 → 基础代谢率下降15-30%
  → 同样食物量产生更少减重效果
  → 停止节食后体重快速反弹（"报复性反弹"）

失败原因二：肌肉流失
  快速减重中：30-50%减掉的重量是肌肉（非脂肪！）
  肌肉减少 → 基础代谢率降低 → 更容易反弹

失败原因三：环境触发（KI-BH-SWITCH-001路径原理）
  食物环境、压力、情绪饮食模式未改变
  → 行动期的意志力消耗 → 难以维持

解决方案框架：
  速度：0.5-1.0kg/周（安全范围）
  结构：减少热量摄入 + 保留蛋白质 + 配合抗阻训练
  行为：改变食物环境和触发行为（而非只靠意志力）
```

### 3.1 低热量饮食方案

```
方案A：中等热量限制（首选，长期可持续）
  能量目标：每天减少500-750千卡
  操作化：
    女性目标热量：1200-1500千卡/天
    男性目标热量：1500-1800千卡/天
  
  如何实现-500千卡/天（不需要计算）：
    ✅ 停喝含糖饮料（-200-400千卡）
    ✅ 每餐少一把主食（-100-150千卡）
    ✅ 减少一份油炸食品（-200-300千卡）
    合计可达：-500千卡/天，无需精确计算

方案B：低热量饮食LCD（800-1200千卡/天）
  适用：BMI≥30，需要快速减重的医疗情况
  注意：
    需在医疗监督下进行
    通常作为短期（12-16周）方案
    需维生素矿物质补充

方案C：极低热量饮食VLCD（<800千卡/天）
  仅在严格医疗监督下使用
  适用：BMI≥35，减重手术前准备，或重大医学需求
  ❌ 不适合自行执行
```

### 3.2 饮食构成优化

#### 蛋白质优先策略
```
减重期蛋白质目标：1.2-1.6g/kg体重/天
（高于维持期，目的：保留肌肉 + 增加饱腹感）

以70kg为例：目标84-112g蛋白质/天

每份约含20-30g蛋白质的食物：
  ✅ 鸡胸肉100g ≈ 30g蛋白质
  ✅ 三文鱼100g ≈ 22g蛋白质
  ✅ 鸡蛋2个 ≈ 12g（辅助）
  ✅ 低脂希腊酸奶150g ≈ 15g
  ✅ 豆腐100g（北豆腐）≈ 8-12g

高蛋白饮食的减重优势：
  - 蛋白质热效应（消化消耗20-30%热量）
  - 饱腹感最强（延迟饥饿感）
  - 保护肌肉量（防止代谢适应）

⚠️ 肾功能损害者：不可执行高蛋白方案
```

#### 碳水化合物管理
```
减重期碳水优化：
  总量：减少精制碳水为核心（不是完全不吃主食）
  替换策略：
    精制 → 全谷物（饱腹感更强，血糖更平稳）
    白米饭 → 糙米/杂粮饭（体积相同，热量相近，饱腹感更强）
    白面包 → 全麦面包（增加纤维+延缓消化）

  优质低热量高填充感食物：
    蔬菜：大量（目标500g/天）
    魔芋（几乎零热量）/白萝卜/冬瓜（低热量高体积）
    豆类（蛋白质+纤维双重饱腹效应）

减重期最需要控制的碳水来源：
  ❌ 含糖饮料（空热量，不产生饱腹感）
  ❌ 精制甜食（蛋糕/饼干/糖果）
  ❌ 白粥（体积大但固体感不足，饥饿回来很快）
```

#### 脂肪管理
```
减重期脂肪策略：
  总量：限制总脂肪（每克脂肪=9千卡，是碳水/蛋白质的2倍）
  但：脂肪质量仍重要（不饱和＞饱和，零反式）

  实际操作：
    烹调用油：减少总量，换用橄榄油/茶籽油
    目标：每天烹调用油≤25g（标准白瓷勺约2.5勺）
    避免：油炸/高油加工食品
```

### 3.3 减重行为策略（行为疗法）

```
循证有效的减重行为技术（BCTTv1映射）：

技术1：自我监测（Self-Monitoring）——最强证据
  记录饮食 + 体重变化
  研究：定期记录的人减重效果比不记录的多2-3倍
  行为化：
    - 每天早上记录体重（标准时间：晨起空腹）
    - 使用手机APP/记事本记录主要食物
    - 不需要精确到克——粗略估算已有效

技术2：食物环境设计（KI-BH-SWITCH-001路径原理）
  "改变路径"而非依赖意志力：
    ✅ 家中不购买/储存高热量零食
    ✅ 将水果/低热量零食放在显眼处（台面/冰箱第一层）
    ✅ 用小盘子盛食（视觉欺骗实验：相同食物，小盘感觉更多）
    ✅ 不边看手机/电视边吃（分心饮食使摄入量多25%）

技术3：结构化饮食计划
  规律进餐时间（降低血糖波动，减少冲动性饮食）
  三餐 vs 少量多餐：证据两者效果相当
  关键：避免长时间空腹（>5-6小时 → 暴食风险）

技术4：刺激控制
  识别触发暴食的环境/情绪信号：
    高风险情境：
      压力/焦虑状态下的"情绪性进食"
      看电视时自动拿零食
      聚会/外出就餐时的社交压力
    应对策略：
      情绪性进食 → 非食物应对（散步/电话朋友）
      外出就餐 → 提前查看菜单，选定健康选项

技术5：目标设定与反馈
  SMART目标：
    不好的目标："我要减肥"
    好的目标："我本周每天快走30分钟，少喝一瓶饮料"
  每周回顾体重变化，调整策略
```

### 3.4 减重维持策略（防止反弹——最难的阶段）

```
为什么体重维持比减重更难：
  生理层面：代谢适应仍在，食欲激素（Ghrelin）分泌增加
  行为层面：目标达成后行为改变松弛
  研究：5年后，减重成功者中只有约20%能维持减重5%+

减重维持的行为科学策略：

策略1：持续自我监测（不可停止）
  维持期仍需每周称体重
  设定"反弹警戒线"：+2kg → 立即启动复位方案

策略2：不停止运动（最关键）
  NWCR（美国国家体重控制登记）研究显示：
  成功维持减重者90%坚持每周60分钟以上中等强度运动
  关键是抗阻训练（防止肌肉流失/维持代谢率）

策略3：维持期"80/20法则"
  80%时间执行健康饮食计划
  20%弹性空间（社交/节日）
  比"100%完美执行"更可持续

策略4：识别"危险信号"并有预案
  危险信号：体重反弹>2kg / 恢复旧的饮食习惯 / 停止锻炼
  预案：回顾减重成功时的关键行为，选1-2个立即恢复
```

## 模块4：M-Action清单

```
P1 - 今天就能做的减重第一步
  ✅ 停喝含糖饮料（-200-400千卡/天，最高性价比单一改变）
  ✅ 把零食从桌面移到不方便拿到的地方

P1 - 本周执行
  ✅ 每餐先吃蔬菜（减少总热量摄入，增加饱腹感）

  ✅ 建立每天早晨称体重习惯

P2 - 系统性改变（2-4周）
  ✅ 增加蛋白质摄入（每餐至少1份优质蛋白）
  ✅ 开始记录饮食（APP/笔记本，不需精确）
  ✅ 每周至少3次30分钟有氧运动
  ✅ 增加每周2次抗阻训练（联动KI-EX-NBA-001）

P3 - 识别个人高风险行为
  ✅ 识别自己的情绪性进食触发点
  ✅ 制定3个非食物应对情绪压力的替代行为
  ✅ 制定外出就餐策略（提前看菜单/提前吃健康食物再出门）
```

## 模块6：铁律边界

```
铁律A 减重速度安全铁律：
  推荐：0.5-1.0kg/周（最大1.5kg/周）
  超过1.5kg/周 → 肌肉流失风险高
  快速减重（>每周1.5kg）仅在医疗监督下执行。
  禁止推荐极低热量饮食（<800千卡）作为自行执行方案。

铁律B 青少年/儿童特殊铁律：
  18岁以下儿童青少年不推荐减重饮食（影响生长发育）。
  干预目标：维持体重，通过身高增长使BMI改善。
  参见：KI-OBE-PEDI-001（儿童青少年BMI标准）

铁律C 进食障碍排查铁律：
  评估是否存在限制性饮食/暴食/补偿行为的进食障碍迹象。
  如有迹象，营养咨询必须结合心理支持，
  禁止单纯推进减重计划。

铁律D 医疗权限铁律（铁律4）：
  减重药物（奥利司他/GLP-1受体激动剂等）：
  必须由医生评估处方，BHP平台仅提供生活方式支持。
  减重手术适应症评估：必须专科转介。
  话术："生活方式干预是所有减重方案的基础，
        如果你考虑用药物辅助，这个决定需要你的医生来做。"
```

    → 通常发生于减重4-8周后

       a) 增加运动量（尤其抗阻训练）
       b) 重新审视"隐形热量"来源
       c) 允许维持1-2周（让身体适应新体重点位）
    → 话术："平台期不是减肥失败——它是身体在
             适应新的状态。这是正常的，
             不需要更极端的节食。"
```

## 模块8：话术变体库

### 破除"管住嘴就能减肥"的误解（S1-S2）
1. "减肥不只是少吃——如果只靠饿肚子，身体会启动'省电模式'，把代谢率降下来，这就是很多人减肥越来越难、一恢复饮食就反弹的原因。我们要做的是更聪明，不是更痛苦。"
2. "你有没有发现：同样的饮食，有些人就是更容易胖？这不是意志力问题，而是肌肉量、代谢率、激素的差异。好消息是：这些都可以通过运动改善。"

### 面对体重平台期（S3维持困难）
1. "平台期是每个人减肥过程中都会经历的——不是你失败了，是身体在适应。这时候最好的策略不是更严格节食，而是增加一点力量训练，给身体一个新的信号。"
2. "你最近两周体重没变化，但这两周你的习惯有没有改变？我们来看一下有没有'隐形热量'悄悄回来了——最常见的是饮料、零食、炒菜的油。"

### 建立长期减重信心（S4维持期）
1. "减肥最难的不是减下来，而是维持住。你现在做到的是前20%的人才能做到的事。维持的关键只有两个：继续运动，继续称体重。就这两件事。"
2. "你减了X公斤——这意味着你的心脏每次跳动少负担了X公斤的重量。这不只是数字，是你每天更轻松、更有精力、更健康的感受。"

## 入库检验记录

```
