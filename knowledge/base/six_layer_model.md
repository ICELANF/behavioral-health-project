# 行健平台六层干预模型 | BHP Six-Layer Intervention Model

> 文档版本：v4.0 | 创建：2026-03-07 | 遵循BHP知识库建设规则v4.0 | 行健平台知识库

---

## 概述

行健平台六层干预模型是平台**所有AI代理、教练介入、内容推送**的统一操作框架。每一层定义了不同深度的干预维度，从最外层的知识传递到最内层的身份重塑，形成由外而内的同心圆结构。

```
Layer 6: 身份层（Identity）          ← 最深，最持久
Layer 5: 信念层（Belief）
Layer 4: 能力层（Capability）
Layer 3: 行为层（Behavior）
Layer 2: 环境层（Environment）
Layer 1: 知识层（Knowledge）          ← 最浅，最易触达
```

*参考框架：Dilts神经逻辑层次 × COM-B行为改变模型 × TTM阶段模型*

---

## Layer 1：知识层（Knowledge）

### 定义
用户对健康知识的**认知状态**。这一层的干预是信息传递：让用户从"不知道"变成"知道"。

### 干预工具
- 知识推送（知识条目KI）
- 教育内容（图文/视频/课程）
- FAQ问答
- XZB行诊智伴知识检索

### 评估指标
- 知识测验得分
- 内容阅读完成率
- 自我效能感（知识维度）

### 局限性
**知识≠行为**。仅靠知识传递无法改变行为，这是"知-行分离"陷阱。Layer 1干预必须与更深层结合。

### 典型场景
> 用户知道"高糖饮食危害血糖"，但仍每天喝奶茶。
> → 需要从Layer 1（知道危害）深入到Layer 3（建立替代行为）。

---

## Layer 2：环境层（Environment）

### 定义
用户所处的**物理环境、社会环境、数字环境**对行为的塑造作用。环境是行为的默认设置。

### 干预工具
- 环境审计（家庭/工作场所/社交圈分析）
- 环境重设建议（移除诱因/添加提示物）
- 社会支持激活（家人、同伴、教练）
- 平台功能推送时机优化（情境触发）

### 关键原理
```
福格行为模型: B = MAP
  Motivation（动机）× Ability（能力）× Prompt（提示）
  → 环境干预主要作用于Ability和Prompt两个维度
```

### 评估指标
- 健康环境评估量表得分
- 社会支持指数
- 行为触发频率

### 典型场景
> 用户想戒夜宵，但冰箱里满是零食，家人也在吃。
> → Layer 2干预：清空冰箱零食（移除诱因）+ 家人共同参与（社会支持）。

---

## Layer 3：行为层（Behavior）

### 定义
用户**实际执行**的健康相关行为，包括饮食行为、运动行为、睡眠行为、压力管理行为。这是平台核心干预层。

### 干预工具
- 行为处方生成（BehaviorRx引擎）
- 习惯堆叠设计（habit stacking）
- 打卡追踪系统
- 行为分析反馈（CGM/步数/睡眠数据）
- BCT技术（BCTTv1 93种技术库）

### 行为改变五步序列
```
1. 目标行为识别（最小可执行单元）
2. 触发条件分析（何时/何地/何种情绪）
3. 替代行为设计（同等满足需求）
4. 实施意图绑定（If-Then计划）
5. 即时反馈闭环（奖励/数据可视化）
```

### 评估指标
- 目标行为执行率（7日/30日）
- 行为稳定性指数
- 打卡连续天数

### 典型场景
> 用户目标：每天饭后散步20分钟。
> Layer 3处方：饭后放餐具（触发）→ 换上运动鞋（习惯堆叠）→ 开始散步→ 
> 记录步数（即时反馈）→ 打卡获积分（奖励）。

---

## Layer 4：能力层（Capability）

### 定义
用户执行健康行为所需的**知识技能、身体能力、自我调节能力**。能力不足是行为改变的隐性障碍。

### 能力维度
```yaml
认知能力:
  - 健康素养（读懂化验单/理解数据）
  - 问题解决技能（应对意外情境）
  - 决策能力（食物选择/风险权衡）

行动能力:
  - 运动技能（动作正确性/安全性）
  - 烹饪技能（健康餐准备）
  - 自我监测技能（CGM使用/血压测量）

调节能力:
  - 情绪调节（应激性进食的管理）
  - 冲动控制（延迟满足）
  - 习惯监控（自我觉察）
```

### 干预工具
- 技能训练课程
- 模拟练习（角色扮演/情境演练）
- 能力自评工具（CAPACITY量表）
- 教练能力辅导

### 评估指标
- CAPACITY量表得分
- 技能测评通过率
- 自我效能感（能力维度）

---

## Layer 5：信念层（Belief）

### 定义
用户对自身健康改变能力的**核心信念**，包括自我效能感、结果预期、健康控制点、身份认同。信念是行为的深层驱动力。

### 关键信念维度

| 信念类型 | 健康信念 | 阻碍信念 |
|---------|---------|---------|
| 自我效能 | "我能做到" | "我天生就是这样的体质" |
| 结果预期 | "改变会带来真实变化" | "我试了很多次都没用" |
| 控制点 | "我的健康我能掌控" | "这是遗传/命运" |
| 身份认同 | "我是一个照顾自己的人" | "我不是运动型的人" |

### 干预工具
- 动机访谈（MI）技术
- 认知重构（CBT框架）
- 成功经验积累设计（掌控感建立）
- 正向榜样激活（同伴故事/案例）
- 优势探寻（Appreciative Inquiry）

### 关键技术
```
"傻瓜相机测试"：
  干预是否足够简单，让用户感到"我一定能做到"？
  → 复杂的干预破坏自我效能感

自我效能四来源（Bandura）：
  1. 掌控经验（最强）: 安排用户体验成功
  2. 替代经验: 展示相似用户的成功案例
  3. 言语说服: 教练/AI的正向反馈
  4. 生理状态管理: 减少焦虑/疲劳
```

## Layer 6：身份层（Identity）

### 定义
用户对"**我是谁**"的最深层认知——不是"我想做什么"，而是"做这件事的人是我"。身份认同改变是最持久的行为改变机制。

### 身份转化轨迹
```
"我在尝试少吃糖"（行为层）
    ↓
"我是一个控制饮食的人"（信念层）
    ↓
"我是一个热爱健康的人"（身份层）
    ↓
"作为一个热爱生命的人，我自然选择滋养身体的食物"（超越层）
```

### 干预工具
- 身份宣言（Identity Statement）书写
- 可能的自我可视化练习（KI-IDENT-POSSIBLE-001）
- 价值观澄清（ACT框架）
- 生命意义探索（弗兰克尔框架）
- 身份一致性强化（社交分享/传授他人）

### 评估指标
- 健康身份认同量表
- 角色承诺度
- 传播意愿指数（S5-S6用户）

### 典型场景
> S5用户："我已经不需要打卡了，因为这就是我的生活方式。"
> S6用户："我想帮助更多人改变，这给我带来意义感。"

## 六层模型的整合应用

### 诊断优先：找到阻力层

用户改变失败，问题往往不在你干预的那一层。诊断流程：

```
失败行为 → 逐层追溯阻力来源
  "知道但不做" → Layer 3行为设计问题 或 Layer 2环境阻力
  "做了但停了" → Layer 4能力不足 或 Layer 5信念阻碍
  "改了又回去" → Layer 6身份未转化
```

### TTM阶段×六层映射

| TTM阶段 | 主要干预层 | 重点工具 |
|---------|----------|---------|
| S0无知无觉 | Layer 1 | 知识唤醒、风险意识 |
| S1矛盾挣扎 | Layer 1+5 | 动机访谈、信念挑战 |
| S2意向萌发 | Layer 2+3 | 环境评估、行为设计 |
| S3准备行动 | Layer 3+4 | 具体计划、技能准备 |
| S4持续行动 | Layer 3+2 | 习惯优化、支持系统 |
| S5巩固习惯 | Layer 5+6 | 身份强化、意义探索 |
| S6维持超越 | Layer 6 | 身份传播、成长领袖 |
