## 模块1：适用情境与触发条件

1. **用户希望系统性提升体能**：已有基础活动习惯，想进阶
2. **超重/肥胖用户的运动进阶**：从步行过渡到结构化训练
3. **中老年慢性病患者的功能性训练**：维持日常生活能力

## 模块2：体能训练准备基础

### 2.1 热身的正确顺序（5-10分钟，不可省略）

```
热身三步骤：
Step 1：一般性热身（3-5分钟）
  目的：提升体温、增加心肺输出
  方法：原地踏步/轻快步行/低强度骑车

Step 2：动态拉伸（3-4分钟）
  目的：激活关节，增加活动范围
  动作组合：
  ① 腿部摆动（前后/侧向）×10次/腿
  ② 手臂绕环×10次/方向
  ③ 髋关节绕环×10次/方向
  ④ 弓步行走×5步/腿
  ⑤ 原地高抬腿×20步

Step 3：专项热身（1-2分钟）
  目的：为当次主要训练内容做准备
  方法：以当次训练动作的30-40%强度做2-3组
```

**BHP应用适配（慢性病用户）**：
- 热身时间延长至10-15分钟（心血管准备更充分）
- 监测：热身结束后心率<100次/分才进入主训练
- 冬季特别：寒冷天气延长热身5分钟，避免冷刺激心血管

### 2.2 放松运动（训练结束必须执行）

```
放松三步骤（5-10分钟）：
Step 1：低强度有氧冷却（3-5分钟）
  方法：慢走、原地轻踏步

Step 2：静态拉伸（4-6分钟）
  每个肌群保持20-30秒
  重点部位：
  ① 股四头肌（大腿前侧）
  ② 腘绳肌（大腿后侧）
  ③ 小腿腓肠肌
  ④ 胸大肌
  ⑤ 颈肩部

Step 3：呼吸调整（1-2分钟）
  深呼吸4-5次，让心率恢复到接近静息水平
```

**安全禁忌**：心脑血管疾病患者禁止训练后立即停止活动（防止直立性低血压）。

## 模块3：五大体能要素训练模块

---

### 要素一：柔韧性（Flexibility）
**重要性**：慢性病患者最需要但最常忽视的体能要素

**科学基础**：
- 拉伸分为静态拉伸和动态拉伸
- 拉伸进入肌肉感受到轻微牵拉感即到位，不应引起疼痛
- 每个动作保持20-30秒，避免弹振式拉伸（容易损伤）

**BHP适配版柔韧性训练（每日10-15分钟）**：
```
早起床后拉伸序列（5分钟）：
  ① 颈部侧屈拉伸 - 坐姿，头向一侧缓慢倾斜，保持20秒/侧
  ② 肩部交叉拉伸 - 手臂横过胸前，另一手辅助，保持20秒/侧
  ③ 脊椎旋转拉伸 - 坐姿，手扶椅背扭转，保持20秒/侧
  ④ 股四头肌拉伸 - 站立，单腿屈膝手持脚踝，保持20秒/腿
  ⑤ 小腿拉伸 - 靠墙站立，后脚跟踩地前弓步，保持30秒/腿

睡前拉伸序列（5分钟）：
  ① 仰卧膝胸抱紧 - 保持20-30秒
  ② 仰卧脊椎扭转 - 双膝倒向一侧，保持30秒/侧
  ③ 蝴蝶式髋关节拉伸 - 坐姿双脚相对，轻压膝盖，保持30秒
```

**慢性病患者特别注意**：
- 骨质疏松患者：避免前屈弯腰动作（脊椎压缩骨折风险）
- 高血压患者：避免头部低于心脏的倒置动作
- 关节炎患者：拉伸范围在无痛范围内进行

### 要素二：肌肉力量与耐力（Strength & Endurance）

**核心观念**：
```
力量训练不是"举铁"，是维持日常生活能力的必要手段：
  上楼梯 = 下肢力量
  提重物 = 上肢/核心力量
  保持坐姿 = 核心力量
  防止跌倒 = 平衡+下肢力量
```

**腹肌/核心训练（最优先，慢性病患者基础）**：
```
Level 1（起始级）：
  ① 平板支撑 - 肘撑，保持姿势，目标20秒×3组
  ② 仰卧腹式呼吸 - 腹部随呼吸起伏，10次×3组

Level 2（进阶）：
  ① 平板支撑 - 延长至45-60秒×3组
  ② 仰卧卷腹 - 肩胛离地即止，15次×3组
  ③ 鸟狗式 - 对侧手脚伸展，保持3秒，10次/侧×3组

Level 3（强化）：
  ① 侧向平板支撑 - 15-30秒×3组/侧
  ② 反向卷腹 - 双腿抬起臀部离地，15次×3组
```

**下肢力量训练（防跌倒/改善代谢必选）**：
```
自重深蹲进阶：
  Level 1：靠椅子坐下站起（座椅辅助深蹲）×10次×3组
  Level 2：标准自重深蹲 - 脚与肩同宽，膝盖不超过脚尖，×15次×3组
  Level 3：单腿支撑练习（站立单腿平衡）×30秒×3组/腿

原则：慢下快上（离心收缩更重要），不锁膝
```

**上肢力量训练**：
```
弹力带推举/俯卧撑进阶：
  Level 1：扶墙俯卧撑 ×15次×3组
  Level 2：标准膝上俯卧撑 ×10次×3组
  Level 3：标准俯卧撑 ×10次×3组
```

### 要素三：爆发力（Power）
**BHP适用人群**：S3-S4，有基础运动能力，希望提升运动效益

**科学基础**：
```
爆发力公式：爆发力 = 力量 × 速度
爆发力是日常生活中最快速流失的能力（40岁后加速）
爆发力训练 = 防跌倒的最有效方式（快速反应能力）
```

**安全前提**：
- ❌ 急性期损伤/关节不稳定 → 禁止
- ❌ 严重骨质疏松 → 禁止跳跃类
- ❌ 未经过基础力量训练（需先完成Level 1-2力量训练）

**BHP适配版爆发力训练（安全优先）**：
```
Level 1（基础功能性爆发力）：
  ① 快速起坐 - 从椅子快速站起，3秒坐下，×10次×3组
  ② 快速原地踏步 - 最快速度踏步20秒，休息40秒，×5组

Level 2（进阶）：
  ① 小跳绳/原地小跑 - 连续30秒，休息30秒，×5组
  ② 侧步移动 - 快速左右迈步，30秒×5组
```

### 要素四：速度与灵活性（Speed & Agility）
**BHP适用场景**：主要针对中青年用户的运动表现提升

**对慢性病患者的价值**：
- 灵活性训练 = 神经系统激活（提升认知功能）
- 变向能力 = 防跌倒关键能力
- 心肺功能强化（高效的有氧代谢刺激）

**日常适用训练**：
```
方向移动练习（室内即可完成）：
  ① T型跑模式（用椅子标记四个点，前/左/右/后快速变向）
     时间：20秒×3组，间歇60秒
  ② 侧滑步 - 两点间横向快速移动
     距离：3-5米，10次往返×3组
  ③ 八字绕圈跑（用两个水瓶标记）
     10圈×3组
```

### 要素五：身体成分（Body Composition）
**核心工具**：皮褶厚度/BMI/腰围——多指标综合评估

```
BHP简化版身体成分评估：
  方法1（最简单）：腰围测量
    男性：腰围<90cm（<85cm更优）
    女性：腰围<85cm（<80cm更优）
    中心性肥胖 = 腹型肥胖 = 代谢风险最高

  方法2：BMI（见KI-OBE-PEDI-001儿童版/成人版）
    成人超重：≥24.0 kg/m²
    成人肥胖：≥28.0 kg/m²

  方法3：体脂率（如有InBody等设备）
    男性：18-24%正常，>25%偏高
    女性：25-30%正常，>32%偏高
```

## 模块4：BHP完整运动计划模板（适用S3用户）

### 入门级（运动习惯建立期，第1-4周）
```
频率：3次/周（隔天）
每次时长：30-40分钟

结构：
  热身（10分钟）→ 柔韧性训练（10分钟）→ 核心训练L1（10分钟）→ 放松（5分钟）

每次选择：
  柔韧性：早起拉伸序列完整版
  核心：平板支撑 + 腹式呼吸
```

### 进阶级（基础巩固期，第5-8周）
```
频率：4次/周（3天力量+1天有氧）
每次时长：45-50分钟

结构（力量日）：
  热身（10分钟）→ 下肢力量Level 2（15分钟）→
  上肢力量Level 2（10分钟）→ 核心L2（10分钟）→ 放松（5分钟）

有氧日：
  持续步行/骑车/游泳 40-50分钟，心率维持在50-70%最大心率
  最大心率估算：220 - 年龄
```

### 维持级（长期维持，S4用户）
```
频率：5次/周（3天力量+2天有氧）
原则：
  - 每4-6周调整一次训练内容（避免平台期）
  - 保持挑战感（重量/次数/时间逐步递增）
  - 运动多样化（防止单一动作伤痛）
```

## 模块5：M-Action清单

```
P1 - 每次训练前热身（绝对不省略）
  原因：热身不充分是慢性病用户运动伤害的首要原因

P1 - 核心训练优先
  原因：核心稳定性是所有其他训练的基础；对腰痛/体态改善效果最快

P2 - 每天拉伸（不计入"正式训练"）
  操作：早起5分钟 + 睡前5分钟拉伸；无需换装，任何时间可执行

P2 - 腰围每月测量一次
  工具：软尺，站立，在最细腰围处测量
  记录：纳入健康档案，作为代谢健康重要指标

P3 - 下肢力量优先（防跌倒/代谢改善）
  操作：靠椅深蹲每天10次起，无需器械

P4 - 爆发力训练（S3以上，有基础后）
  快速起坐×10次：比慢速起坐对肌肉骨骼系统刺激高4倍
```

## 模块6：铁律边界

```
铁律A 热身不可省略铁律：
  慢性病用户（高血压/心脏病/糖尿病）在进行任何中高强度训练前
  必须完成至少10分钟热身。
  原因：冷启动的剧烈运动→心肌缺血/运动损伤风险。

铁律B 运动强度分级铁律：
  新用户必须从Level 1开始，不可跳级。
  连续3次训练感觉"非常轻松"（RPE<12/20）才升级。
  原因：过快进阶→伤痛→放弃运动→比不运动更差的结果。

铁律C 心率监测铁律：
  高血压/冠心病用户：运动中心率不超过(220-年龄)×75%
  运动后5分钟心率未降至100次/分以下 → 下次降低强度

铁律D 痛感红线铁律：
  运动中出现胸痛/胸闷/头晕/关节剧烈疼痛 → 立即停止
  → 触发CrisisAgent或建议就医评估（铁律1延伸）

铁律E 运动替代用药禁止铁律（铁律4）：
  运动处方不可建议用户减少或停止降糖/降压/降脂药物。
  运动是辅助治疗，在医师监督下方可调整用药。
```

```
高频叠加组合：
  组合1：临床运动处方KI + 本KI = 完整运动处方（原则+执行）
```

## 模块8：话术变体库

### 用户从零开始（S2）
1. "你说以前从来不运动——那今天我们从最小的起点开始：每天早起做5分钟拉伸。不换衣服，不出门，就在床边做。你愿意这样开始吗？"
2. "运动不需要一开始就去健身房。我给你设计一个30分钟的家庭版方案，不需要任何器械——你有兴趣看看吗？"

### 用户说"我膝盖不好/腰痛"（S2-S3）
1. "膝盖/腰痛反而更需要运动——但要选对的运动。疼痛的时候最适合做的是：拉伸 + 核心训练 + 游泳/骑车（非冲击性运动）。我给你说说具体怎么做？"
2. "核心力量弱是很多腰痛的根本原因之一——不是因为腰弱，而是因为撑腰的肌肉太弱。我们先从平板支撑开始？"

### 用户不知道如何进阶（S3-S4）
1. "你现在走路30分钟已经很轻松了——说明你的心肺已经适应了。下一步可以加一些力量训练，让肌肉量提升，对血糖/血压的改善更持久。"
2. "你做深蹲已经做了一个月了——我们来评估一下：20个做完你还有余力吗？如果有，可以升级了。"
