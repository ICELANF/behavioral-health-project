# 行健平台代谢安全红线 | BHP Metabolic Safety Redlines

> 文档版本：v4.0 | 创建：2026-03-07 | 遵循BHP知识库建设规则v4.0 | 行健平台知识库
> **本文件为平台全局安全阈值统一标准，所有模块必须遵循，不得覆盖或降低标准**

---

## 概述

代谢安全红线是行健平台在提供行为干预和处方生成时的**不可逾越的安全边界**。任何AI生成的建议、教练干预方案、行为处方，在遇到以下红线情况时，必须**立即暂停常规流程，启动危机转介协议**。

**适用范围**：所有用户角色（Observer/Grower/Sharer/Coach/Supervisor/Admin），所有慢病类型。

## 一、血糖安全红线

### 1.1 低血糖危机红线（最高优先级）

| 指标 | 红线阈值 | 处置动作 |
|------|---------|---------|
| 血糖 < 3.9 mmol/L | ⚠️ 警告级 | 提示立即补充15g快速碳水，15分钟后复测 |
| 血糖 < 3.0 mmol/L | 🚨 危机级 | 停止所有运动/禁食建议，启动低血糖急救流程，建议就医 |
| 血糖 < 2.8 mmol/L | 🆘 急救级 | 立即建议拨打急救电话，通知紧急联系人 |
| 无症状低血糖 | 🚨 危机级 | 所有感知阈值下移，加强CGM监测频率 |

**低血糖急救流程（15-15原则）**：
1. 立即摄入15g快速碳水（葡萄糖片/果汁/糖水）
2. 等待15分钟
3. 复测血糖
4. 若仍<3.9 mmol/L，重复步骤1-3
5. 三次无效→立即就医

### 1.2 高血糖红线

| 指标 | 红线阈值 | 处置动作 |
|------|---------|---------|
| 空腹血糖 > 13.9 mmol/L | 🚨 危机级 | 暂停所有生活方式干预强化，建议就医调整药物 |
| 餐后2h血糖 > 16.7 mmol/L | 🚨 危机级 | 同上 |
| 血糖持续 > 11.1 mmol/L（>24h） | ⚠️ 警告级 | 建议联系医生，暂缓高强度运动处方 |
| 酮体阳性 + 高血糖 | 🆘 急救级 | 立即就医，疑似DKA |

### 1.3 CGM特殊指标红线

| 指标 | 红线阈值 | 说明 |
|------|---------|------|
| TBR（<3.9） | > 4% | 低血糖时间占比过高，需调整处方 |
| TAR（>10.0） | > 25% | 高血糖时间占比过高，需加强干预 |
| 血糖变异系数CV | > 36% | 血糖不稳定，禁止激进饮食干预 |
| 血糖快速下降速率 | > 3 mmol/L/15min | 提示速发低血糖风险 |

## 二、体重与营养安全红线

### 2.1 热量摄入红线

| 情况 | 红线阈值 | 处置动作 |
|------|---------|---------|
| 每日热量摄入 | < 1200 kcal（女）/ < 1500 kcal（男） | 禁止生成此类处方，提示营养不足风险 |
| 极低热量饮食（VLCD） | < 800 kcal/日 | 必须有医疗监督，平台不独立推荐 |
| 连续禁食 | > 24小时（糖尿病用户） | 需医疗监督，平台提示风险 |
| 蛋白质摄入 | < 0.8g/kg体重/日 | 提示肌肉流失风险 |

### 2.2 体重变化红线

| 指标 | 红线阈值 | 处置动作 |
|------|---------|---------|
| 体重快速下降 | > 1 kg/周（持续3周） | 排除病理原因，建议就医 |
| 体重快速增加 | > 2 kg/周 | 排除水肿/用药影响 |
| BMI | < 18.5 | 禁止减重处方，转营养支持方案 |
| 体脂率（男） | < 5% | 停止体脂减少目标 |
| 体脂率（女） | < 12% | 停止体脂减少目标 |

## 三、运动安全红线

### 3.1 运动禁忌红线

以下情况平台**禁止生成中高强度运动处方**：

```yaml
绝对禁忌（必须暂停运动处方）:
  - 血糖 < 5.6 mmol/L 且准备运动 → 先补充碳水
  - 血糖 > 13.9 mmol/L → 禁止激烈运动，可轻度步行
  - 活动性心肌梗死/不稳定心绞痛
  - 急性感染发热（体温 > 38.5°C）
  - 严重低血压（收缩压 < 90 mmHg）
  - 近期（<6个月）眼底出血史
  - 严重肾病（eGFR < 30）

相对禁忌（需医疗确认后方可执行）:
  - 增殖性糖尿病视网膜病变（避免高强度/冲击性运动）
  - 周围神经病变（避免负重运动，推荐水中运动）
  - 自主神经病变（心率反应异常，强度监测需特殊方案）
  - 骨质疏松（避免跌倒风险运动）
```

### 3.2 运动强度红线

| 用户类型 | 建议强度上限 | 说明 |
|---------|------------|------|
| 初始S1-S2用户 | 40-60% HRmax | 从低强度开始 |
| 合并心血管疾病 | 需医疗评估 | 不独立推荐 |
| 65岁以上 | 50-70% HRmax | 侧重平衡/柔韧 |
| 术后恢复期 | 禁止中高强度 | 需医疗许可 |

## 四、心理安全红线

### 4.1 自伤/自杀风险红线

**触发关键词**（一旦识别，立即启动危机协议）：
- 明确表达：不想活了、活着没意义、消失、轻生、自杀
- 间接表达：没有我大家会更好、不想坚持了（结合其他消极信号）
- 行为信号：近期放弃所有健康管理活动 + 告别性语言

**危机协议**：
1. 停止健康话题，切换至情感支持模式
2. 明确询问安全状态（"你现在安全吗？"）
3. 提供危机热线：北京 010-82951332 / 全国 400-161-9995
4. 通知平台督导角色

### 4.2 进食障碍红线

以下行为禁止平台强化：
- 极度限制饮食（< 500 kcal/日且呈现回避特征）
- 暴食后清除行为（催吐/泻药）
- 体象严重扭曲（BMI正常但执着于"太胖"）

**处置**：停止热量/体重相关处方，转心理支持资源。

## 五、药物交互安全红线

### 5.1 行为干预与药物的冲突红线

| 药物类型 | 行为干预禁忌 |
|---------|------------|
| 胰岛素/磺脲类 | 运动前必须调整剂量，禁止剧烈空腹运动 |
| 二甲双胍 | 对比剂检查前48h暂停，恢复肾功能正常后续用 |
| GLP-1受体激动剂 | 合并极低热量饮食需医疗监督（恶心/呕吐风险） |
| SGLT-2抑制剂 | 高强度运动+低碳饮食→euglycemic DKA风险 |
| 降压药（β受体阻滞剂） | 心率不能作为运动强度指标，改用主观疲劳感评分RPE |

## 六、红线触发处理流程

```
检测到红线信号
    ↓
立即停止当前流程
    ↓
按红线级别处置：
  ⚠️ 警告级 → 提示用户 + 记录日志 + 调整处方
  🚨 危机级 → 暂停干预 + 建议就医 + 通知教练
  🆘 急救级 → 终止对话 + 急救指引 + 通知督导+紧急联系人
    ↓
记录事件（crisis_event表）
    ↓
督导24h内回访
```

## 七、免责声明与法律边界

- 行健平台提供**行为改变支持**，不提供医疗诊断或处方
- 所有红线阈值参考国际临床指南（ADA/CDA/中华医学会），但**个体情况以主治医师判断为准**
- 用户在知情同意书中已确认平台的支持性（而非医疗性）定位
- 红线触发事件须在72小时内存档备查
