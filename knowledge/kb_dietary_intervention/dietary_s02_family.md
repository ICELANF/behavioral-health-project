# S02 · 家庭现实冲突 — 被「爱」与「节奏」吞噬

> **典型话语**：「家里做饭重油重盐，不吃怕伤感情，吃了又焦虑」

---

## 适用识别条件

| 维度 | 判断标准 |
|------|---------|
| TTM阶段 | S2–S3（意向期→准备期） |
| 信任分 | building→established（40–65%） |
| BPT6特征 | 环境控制弱 · 社会支持系统阻力高 · 行动自我效能低 |
| 触发关键词 | 家里/长辈/妈妈/做饭/油盐/劝吃/家人/伤感情/没法坚持 |

**代谢风险**：长期高盐→血压升高；重油高热量→内脏脂肪累积；情绪内耗→皮质醇持续升高→胰岛素抵抗

---

## 干预核心逻辑

家庭是最大支持系统，也是干预「触礁」的高风险区。
**策略：嵌入而非对抗——在不惊动「爱」的前提下，悄悄置换伤害身体的习惯。**
代谢核心：在不引发家庭冲突的条件下，改变食物进入身体的质量，而非改变餐桌结构。

---

## 决策树（6节点，按顺序执行）

### 节点1：家庭系统评估
**触发条件**：用户提到家庭饮食阻力
```
Agent话术：
「谁在家里做饭？谁最关注你吃什么？
上次你们因为饮食产生矛盾是什么情况？」
→ 绘制家庭饮食决策地图
```
**教练行动**：识别关键影响人（通常是母亲/配偶）；若关键人「主动阻碍」≥ 3次 → 启动家庭教育协议

---

### 节点2：情绪成本识别
**触发条件**：用户表现出强烈的内疚感
```
Agent话术：
「在餐桌上，拒绝那碗饭，对你来说意味着什么？」
→ 帮助分离「食物的营养属性」和「食物的情感属性」
```
**教练行动**：记录内疚感强度（0–10分）；≥ 7分 → 优先处理情感层面，暂缓饮食量干预

---

### 节点3：嵌入点寻找
**触发条件**：用户愿意尝试但不想引发冲突
```
Agent话术：
「我们不改变菜谱，只找一个小小的嵌入口。
你能在餐桌上准备一小碗温水吗？涮一下再吃，不用解释。」
→ 物理降级策略
```
**教练行动**：追踪涮水行为完成率；7天达成率 ≥ 60% → 推进下一个M-Action

---

### 节点4：语言转译训练
**触发条件**：家人询问或反对时
```
Agent话术（转译模板）：
把「我要减肥不能吃」
→ 换成「最近胃部不好，医生让少油盐，我用水涮一下」
把「不饿/不吃这个」
→ 换成「这道菜真诱人，我先留两块明天带便当，今晚给胃留空间喝汤」
→ 把个人选择转译为医学指令
```
**教练行动**：提供5条「语言转译模板」，让用户选择最适合自家文化的表达

---

### 节点5：主动参与策略
**触发条件**：用户希望在家庭中争取更多空间
```
Agent话术：
「你能不能主动承担『每餐加一道白灼蔬菜』的任务？
不改变主菜，自己增加绿色区。
这既表达了对家庭的参与，又为自己争取到了安全的绿色区。」
```
**教练行动**：追踪绿色区蔬菜摄入频率；连续14天达成 → 升级「家庭健康大使」成就

---

### 节点6：关键人影响
**触发条件**：家庭阻力持续 ≥ 1个月
```
Agent话术：
「我们能不能找个机会，让家人一起看看你的血糖或体检报告？
不是说教，是让数据说话。」
```
**教练行动**：若家庭阻力持续，建议邀请关键人参与一次教练视频会议（需用户同意）

---

## M-Action（按代谢改善优先级）

| 优先级 | 具体动作 | 代谢机制 |
|--------|---------|---------|
| P1 | 餐桌备温水（涮水技术） | 减少油盐摄入约30%，不改变家庭饮食氛围 |
| P2 | 餐具降级（换小碗） | 利用视觉错觉，减少摄入量15–20%，家人几乎不会察觉 |
| P3 | 进食顺序：蔬菜→蛋白质→主食 | 膳食纤维先行，降低餐后血糖峰值，胰岛素分泌平缓化 |
| P4 | 主动增补「白灼蔬菜」一道 | 自主控制一道菜，增加饱腹感，减少重油菜摄入比例 |
| P5 | 饭后15分钟散步 | 降低餐后血糖约15–20%，代替「瘫着消化」的惯性 |

---

## 代谢监测重点

| 指标 | 风险阈值 | 行为健康干预 |
|------|---------|------------|
| 血压 | ≥ 130/80 mmHg | 高盐→血管紧张；涮水技术+增加钾（蔬菜）是一线干预 |
| 内脏脂肪 | CT/超声评估 | 长期重油→内脏脂肪累积；改变进食顺序是低干预高效手段 |
| 餐后血糖 | ≥ 8 mmol/L | 白米饭+重油菜GI极高；蔬菜先行可缓冲升糖速率 |
| 体重趋势 | 每月 > 2kg增加需预警 | 家庭饮食干预效果以月为单位评估，不要求立即见效 |

---

## 铁律边界 ⚠️

| 禁令 | 原因 |
|------|------|
| ❌ 禁止拆散家庭 | 「对抗家人」或「坚持原则」——文化情境下饮食是情感的一部分，对抗必败 |
| ❌ 禁止忽视环境 | 「意志力足够就能克服」——环境支配力远大于个人意志 |
| ❌ 禁止孤立建议 | 「别管家里人，你自己决定」——忽视家庭系统的干预必然失败 |
| ⚕️ 医疗转诊条件 | 血压持续 ≥ 140/90 或餐后血糖持续 ≥ 11.1 → 必须转医疗干预 |
