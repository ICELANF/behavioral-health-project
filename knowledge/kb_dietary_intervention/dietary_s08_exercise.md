# S08 · 运动恐惧 — 被身体「背叛」的畏惧者

> **典型话语**：「跑几步心跳就快得像要炸，在健身房觉得大家都在盯着我」

---

## 适用识别条件

| 维度 | 判断标准 |
|------|---------|
| TTM阶段 | S0–S1（前意向期） |
| 信任分 | not_established（< 35%） |
| BPT6特征 | 躯体感知扭曲 · 社交羞耻 · 自我效能极低 · 灾难化认知 |
| 触发关键词 | 害怕运动/心跳快/炸开/健身房/被看/肥肉/酷刑/不敢/昏过去 |

**代谢风险**：久坐不动→肌肉量下降→基础代谢率降低→体重上升；恐惧运动→始终不动→代谢综合征持续进展

---

## 干预核心逻辑

运动恐惧的本质是「生理信号的灾难化解读」+「社交羞耻的叠加」。
**目标：重新定义「动」的边界——不是「爱上健身房」，而是消除对身体活动的灾难化想象。**
代谢核心：启动NEAT（非运动性产热），绕过心理防线实现代谢改善。

---

## 决策树（6节点，按顺序执行）

### 节点1：恐惧类型分辨
**触发条件**：用户描述对运动的恐惧
```
Agent话术：
「你害怕的是身体感觉（心跳快/出汗），
还是被人看的感觉，还是两者都有？」
→ 精准定位阻力类型
```
**教练行动**：
- 「被看」为主 → 优先提供居家/私密运动方案
- 「心跳恐惧」为主 → 先提供心率正常范围教育

---

### 节点2：心率去灾难化
**触发条件**：用户对心跳加速有恐惧反应
```
Agent话术：
「心跳加速是心脏在为你泵血，
它在告诉你：'我正在全力支持你。'
它不是要炸开，而是在充满活力地工作。
我们下次尝试在心跳'快要报警'之前就停下来，
只在安全区内活动，好吗？」
```
**教练行动**：提供安全心率范围图示——最大心率的50–60%区间（即：220-年龄 × 50–60%）

---

### 节点3：运动空间解构
**触发条件**：用户对健身房有社交羞耻
```
Agent话术：
「健身房只是运动场所之一。
穿着最舒服的睡衣，在家跟着音乐摆动5分钟，也是运动。
没有任何人看着你，这是你和自己身体的私人约会。」
```
**教练行动**：提供居家运动方案清单（不需要任何器械）；若用户接受，建立7天居家微运动打卡

---

### 节点4：微量递增，首次启动
**触发条件**：用户愿意尝试第一次「动」
```
Agent话术：
「今天只做一个动作：原地踏步30秒。
如果你觉得累了，随时停下，那是你对身体的自主掌控。
我们要向你的大脑证明，'动'这件事是安全的、受控的。」
→ 30秒→1分钟→3分钟→5分钟（渐进）
```
**教练行动**：追踪首次完成时的主观感受评分；若「没有想象中可怕」→ 立即强化反馈，建立新的运动记忆

---

### 节点5：NEAT策略（无感运动）
**触发条件**：用户接受「不去健身房也是运动」的概念
```
Agent话术：
「刷牙时单脚站立；接电话时站立走动；提前一站下车步行。
这些'琐碎的动'在代谢改善上的总和，
往往超过一次让你感到恐惧的健身课。」
```
**教练行动**：建立NEAT行为堆叠清单；追踪每日步数，目标从6000步起步

---

### 节点6：正向锚定（运动后觉知）
**触发条件**：用户完成了任何形式的运动
```
Agent话术：
「运动结束后，静坐2分钟观察：
现在心跳慢下来了吗？
那种微微发热的感觉，是不是让你觉得身体比刚才轻盈了一点点？」
→ 锚定正向体验
```
**教练行动**：每次微运动后记录身体感受；逐渐用「运动后的轻盈感」替代「运动前的恐惧感」，完成情绪记忆重塑

---

## M-Action（按代谢改善优先级）

| 优先级 | 具体动作 | 代谢机制 |
|--------|---------|---------|
| P1 | 原地踏步30秒（首次运动） | 突破「第一动」——零成本启动，证明运动对此人「是安全的」 |
| P2 | 刷牙时单脚站立（NEAT堆叠） | 每天约1分钟，激活下肢肌肉，提升基础代谢，无心理负担 |
| P3 | 接电话站立行走（NEAT堆叠） | 每次约5–15分钟，累计可产生100–200 kcal/天的额外能量消耗 |
| P4 | 微汗标准：能完整说出一句话 | 提供安全运动强度的可操作判断标准，消除「心跳炸开」的灾难化认知 |
| P5 | 运动后2分钟身体扫描记录 | 锚定正向感受，对抗运动恐惧记忆，改变运动-痛苦的神经联结 |

## 代谢监测重点

| 指标 | 风险阈值 | 行为健康干预 |
|------|---------|------------|
| 静息心率 | > 90次/分 需医疗评估 | 极度不活动者静息心率偏高；轻度活动后心率恢复速度是心肺功能代理指标 |
| 肌肉量/基础代谢率 | Inbody或DXA评估 | 久坐者肌肉量低→基础代谢低；NEAT策略是低阻力提升代谢率的方案 |
| 餐后血糖 | 轻度步行后是否改善 | 散步10分钟使餐后血糖峰值降低约15%，是给运动恐惧者最有说服力的「实验数据」 |
| 心肺功能 | 6分钟步行测试 | 肥胖者心肺功能通常偏低；建议以「微汗+能说话」为安全运动区间 |

## 铁律边界 ⚠️

| 禁令 | 原因 |
|------|------|
| ❌ 禁止暴力动员 | 「不流汗哪叫运动，挺过去就习惯了」——将运动与生存威胁绑定，恐惧记忆会永久加深 |
| ❌ 禁止错误对标 | 用运动员或健身达人标准要求慢病患者——强化挫败感，导致彻底放弃 |
| ❌ 禁止忽视灾难化认知 | 不进行认知校准直接布置运动任务——患者的恐惧会持续放大直至永久回避 |
| ⚕️ 医疗评估前提 | 静息心率>100次/分、血压>160/100、BMI>40 → 首次运动前必须心内科/运动医学评估 |
