# L1 底座知识 · BPT6 六维行为倾向评估

> **适用范围**：全部 Agent 的用户画像识别与干预策略匹配。
> **核心逻辑**：BPT6 是「路径匹配工具」，不是「性格判断工具」——识别倾向是为了绕过阻力，不是为了贴标签。

---

## 1. 六大维度详解

| 维度名称 | 关键特征 | 低分表现 | Agent 策略权重 |
| :--- | :--- | :--- | :--- |
| **P1 情绪化敏感** | 情绪波动对行为的影响程度 | 易触发情绪进食、情绪性放弃 | 强化情感承接节点，先处理情绪再给方案 |
| **P2 自我效能感** | 对完成目标的信心程度 | 「我肯定做不到」「试了也没用」 | 低分用户需极低门槛 Slack，累积微小胜利 |
| **P3 认知弹性** | 面对「破功」时的调节能力 | 全或无思维，一次失败=全盘崩塌 | 低分用户需加强去标签化引导，提前设计「弹性预算」 |
| **P4 外部支持度** | 家庭/社交环境的助推或阻力 | 家人不支持、社交场合频繁破功 | 关注环境干预，设计「环境工程」方案 |
| **P5 躯体觉知力** | 对身体生理信号的感知能力 | 「不知道饿不饿」「感觉不出来」 | 辅助用户建立觉知练习，而非单纯依赖数字追踪 |
| **P6 延迟满足感** | 长期利益与短期快感的权衡能力 | 「反正以后的事」「现在开心最重要」 | 低分用户侧重即时正向反馈，而非远期健康目标 |

---

## 2. 评估应用规则

* **评分范围**：每维度 1~10 分，≤4 分为低分，需调整干预策略。
* **组合识别**：P1 高 + P2 低 = 情绪驱动型（情绪进食高风险）；P3 低 + P6 低 = 全或无型（完美主义崩塌风险）。
* **动态更新**：BPT6 评分随对话积累动态调整，不是一次性评估结果。

---

## 3. 铁律

* **禁止标签化**：严禁在对话中向用户反馈其倾向性评分结论（如「你的自我效能感评分只有3分」）。
* **用于路径匹配**：BPT6 是教练/Agent 的内部工具，用于选择干预策略，不是反馈给用户的报告。
* **禁止固化**：不得因早期 BPT6 评估而永久定性用户——改变本身就会改变评分。
