# S04 · 绩效焦虑 — 越急越胖的「精密机器」

> **典型话语**：「血糖一波动就崩溃，月减10斤完不成就彻底放弃」

---

## 适用识别条件

| 维度 | 判断标准 |
|------|---------|
| TTM阶段 | S2–S3（意向期→准备期）—— 外驱力驱动型 |
| 信任分 | building（35–55%） |
| BPT6特征 | 目标导向过强 · 完美主义 · 对波动灾难化认知 |
| 触发关键词 | 指标/血糖没达标/没减下去/焦虑/崩溃/不行/全毁了/KPI/又胖回来 |

**代谢风险**：绩效焦虑→高皮质醇→胰岛素抵抗→腹部脂肪堆积；极端补偿→代谢弹性破坏；外驱力枯竭→指标恶化

---

## 干预核心逻辑

**皮质醇的升糖作用会直接抵消患者的所有努力。**
核心转向：从「管理数字」到「管理系统」，建立「过程导向」的代谢健康观。
代谢核心：降低皮质醇基线是改善胰岛素敏感性的前提，比任何饮食方案都重要。

---

## 决策树（6节点，按顺序执行）

### 节点1：焦虑程度评估
**触发条件**：用户提到对数字/指标的强烈情绪反应
```
Agent话术：
「当血糖/体重数字上升时，你会产生什么感觉？
能打个分吗（0–10分）？」
→ 测量绩效焦虑强度
```
**教练行动**：焦虑 ≥ 7分且影响日常功能 → 优先处理心理状态；≥ 9分 → 建议转心理咨询

---

### 节点2：数字去神圣化
**触发条件**：用户把指标波动解读为「失败」
```
Agent话术：
「空腹血糖在一天内自然波动0.5–1 mmol/L是正常的——
这不是你吃错了，是人体的正常节律。
血糖不是你的成绩单，它是你身体的天气预报。」
```
**教练行动**：提供血糖变异性正常参考范围，帮助用户建立「正常波动」vs「异常波动」的认知框架

---

### 节点3：反馈频率降级
**触发条件**：用户存在强迫性检查数据的行为
```
Agent话术：
「我们先把每天称重改成每周一次，
把看血糖的频率减半——
暂时模糊反馈，减少杏仁核激活频率。」
```
**教练行动**：追踪用户「数据检查焦虑次数」；若仍频繁自测 → 探讨是否存在强迫性检查行为

---

### 节点4：过程指标替代
**触发条件**：准备从结果指标转向过程指标
```
Agent话术：
「这周我们不看体重，只关注：
精力水平是不是比上周好？腰围有没有松一点点？
你今天的选择让你的身体感觉怎么样？」
```
**教练行动**：建立「精力-睡眠-情绪」三维日志，代替单一体重/血糖数字追踪

---

### 节点5：违规预算设计
**触发条件**：用户理解弹性框架的价值
```
Agent话术：
「方案里预设每周20%的行为偏离——
这是系统的减压阀，不是失败。
当偏离被包含在计划内，负罪感就失去了滋生的土壤。」
```
**教练行动**：在行为处方中明确写入「弹性空间」，教练不得在审核时删除此条款

---

### 节点6：身份转译
**触发条件**：用户开始接受过程导向
```
Agent话术：
「把『我今天减了多少克』
改成『我今天发现了身体对某种食物的新反应』。
你是代谢实验员，不是减肥考生。
每一次波动是实验数据，不是绩效评分。」
```
**教练行动**：当用户开始使用「实验」「发现」语言时，记录为「身份转译成功信号」，准备阶段升级评估

---

## M-Action（按代谢改善优先级）

| 优先级 | 具体动作 | 代谢机制 |
|--------|---------|---------|
| P1 | 停止每日称重，改为每周固定时间一次 | 消除每日体重波动带来的皮质醇激活，稳定神经-内分泌节律 |
| P2 | 晨起精力评分（1–10分，记录一个数字） | 提供与数字脱钩的健康感知指标，训练「内感觉知」能力 |
| P3 | 「实验性饮食」日记（观察而非评判） | 用「观察」替代「评判」，降低情绪负载，提升行为依从性 |
| P4 | 每周一次「压力步行」20分钟 | 有氧运动降低皮质醇，改善胰岛素敏感性，阻断焦虑→升糖恶性循环 |
| P5 | 腹式呼吸3分钟（看完数据后） | 激活副交感神经，降低因数据波动引发的急性应激反应 |

---

## 代谢监测重点

| 指标 | 风险阈值 | 行为健康干预 |
|------|---------|------------|
| 皮质醇节律 | 晨间唾液皮质醇偏高 | 焦虑型患者皮质醇基线偏高，直接导致空腹血糖升高；须与行为干预同步处理 |
| 腰围/内脏脂肪 | 男>90cm / 女>85cm | 高皮质醇→优先堆积内脏脂肪；体重秤无法反映，腰围更敏感 |
| 胰岛素敏感性 | HOMA-IR > 2.5 | 焦虑→胰岛素抵抗→体重平台期→更焦虑；须打破皮质醇环路 |
| HRV心率变异性 | HRV < 20ms 提示自主神经失调 | 绩效焦虑者HRV普遍偏低；HRV改善是焦虑缓解的生理证据 |

## 铁律边界 ⚠️

| 禁令 | 原因 |
|------|------|
| ❌ 禁止KPI式目标 | 「月减10斤」「血糖必须维持5.6以下」的极端目标会激活HPA轴，皮质醇反噬 |
| ❌ 禁止每日体重汇报 | 每日称重打卡会强化「数字绑架」，是主动制造绩效焦虑 |
| ❌ 禁止补偿性指令 | 「明天断食补偿」「多跑5公里抵消」——极端补偿破坏代谢弹性 |
| ⚕️ 医疗联动条件 | HbA1c持续上升 + 腰围增大 + HRV持续低下 → 联合内分泌科评估 |
