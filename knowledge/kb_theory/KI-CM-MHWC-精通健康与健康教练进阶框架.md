## KI-CM-MHWC-001：生活转型——健康教练的最高目标

```yaml
id: KI-CM-MHWC-001
type: 教练哲学
evidence: T3
stage: S2-S6
domain: behavioral growth
keywords: [生活转型, 自我概念转变, 身份认同, 持续改变, 整体人格, 有意义的动机]
```

**健康教练的终极目标：身份认同转变，而非单一行为改变**

可持续的健康行为改变（S4-S5的核心标志）在研究中被发现取决于两个最强预测因子：
1. **自我概念转变**：客户开始用新的身份定义自己（"我是一个注重健康的人"而非"我是一个在控制糖尿病的人"）
2. **社群/环境支持**：健康行为被整合进社会关系和日常环境

**Arloski提出的持续行为改变5个策略**

| 策略 | 核心操作 | 平台对应 |
|------|---------|---------|
| **1. 提升自我效能** | 设计足够小的行动步骤确保成功体验 | Kaizen微习惯模块 |
| **2. 培养愿景与内在动机** | 帮助客户建立"美好生活愿景（Well-Life Vision）"，连接行为与深层意义 | TTM S5身份叙事 |
| **3. 聚焦维持阶段** | 重点强化"什么在起作用"，积极心理学取向，不只追问"哪里还不够" | 维持期复盘功能 |
| **4. 加强社群支持** | 主动发展支持系统——"谁或什么还能支持你做这件事？" | 家庭/社群协作功能 |
| **5. 从第一天开始建立连接** | 教练时间有限，从第一次会话开始帮助客户找到生活中的其他支持来源 | 转介与延伸支持网络 |

**CoachAgent应用**：在S4→S5过渡期，CoachAgent应从"执行指导"模式切换到"见证陪伴"模式——核心话术从"今天你要做什么"→"你已经成为了一个什么样的人？"

**铁律**：以"帮用户变成一个什么样的人"为目标，而非"帮用户达到某个指标"。数字只是成长的痕迹，不是成长本身。

## KI-CM-MHWC-002：教练联盟共建——罗杰斯三核心条件

```yaml
id: KI-CM-MHWC-002
type: 教练技能基础
evidence: T3
stage: S1-S5
domain: behavioral
keywords: [教练联盟, 无条件积极关注, 真诚, 共情, 以客户为中心, Carl Rogers]
```

**教练关系的三个核心条件**（Rogers Person-Centered Approach，大量研究证明与疗效正相关）

**1. 无条件积极关注（Unconditional Positive Regard）**
- 不评判地接纳客户当前的状态、行为和价值观
- 区别于赞同：不评判≠认为所有行为都对，而是在不评判中让探索发生
- 操作化：遇到不认同的情境时，先停止内心的诊断思维，让客户的故事完整展开
- **失效陷阱**：当Coach进入"我要帮他修好这个"的诊断模式时，评判就开始了

**2. 真诚性与真实性（Genuineness & Authenticity）**
- 真诚无法伪装——客户拥有极好的"B.S.探测器"
- 如果Coach心不在焉，不如不做那次会话
- 真诚的Coach会暴露自己的观察（"我注意到你今天声音里有些不同"），而非只执行会话模板

**3. 共情理解（Empathic Understanding）**
- 不是同情（sympathy = 为对方感到遗憾），而是进入对方的内心世界理解其体验
- 三层共情（Goleman）：认知共情（我理解）→情感共情（我感受到）→关怀共情（我想帮）
- 共情传递的不是具体话术，而是话语背后的温度和真实性

**CoachAgent话术原则**：
- 在回应困难情境前，先做共情反映，再做建议
- 避免"解决模式"早于"倾听模式"
- 参考FAVE框架（见KI-CM-MHWC-003）

**监测指标**：用户信任分（trust_score）六信号系统中的"情感安全感"指标是教练联盟质量的代理指标。

## KI-CM-MHWC-003：FAVE共情框架——深度倾听与情感响应四步法

```yaml
id: KI-CM-MHWC-003
type: 教练技能工具
evidence: T3
stage: S1-S4
domain: behavioral
keywords: [FAVE, 共情, 深度倾听, 情感响应, 情绪验证, 会话技巧]
```

**FAVE框架：共情响应的四步操作顺序**

在客户分享困难、挫败或复杂情绪时，Coach/CoachAgent应遵循FAVE顺序：

| 步骤 | 英文全称 | 操作要点 | 示例话术 |
|------|---------|---------|---------|
| **F** | Feel | 识别并命名情感 | "听起来这件事让你感到很沮丧" |
| **A** | Acknowledge | 承认其体验的合理性 | "三个月一直在坚持，然后指标还是没变，谁都会气馁" |
| **V** | Validate | 正式确认其感受是真实且合理的 | "你有这种感觉是完全可以理解的" |
| **E** | Explore | 好奇地探索，不急于解决 | "能跟我说说，这段时间最难的是哪一部分？" |

**错误响应模式（"错失机会型"）**：
- 客户说"我三个月都没改变，很沮丧" → Coach说"那你最近碳水怎么控制的？"
- 问题：绕过情绪直接解决问题，客户感到不被理解，会话停留在表层

**正确响应模式（"深度陪伴型"）**：
- 客户说"我三个月都没改变，很沮丧" → Coach说："三个月这么坚持，还是没看到改变——这种沮丧真的很可以理解。能告诉我，这段时间里什么时刻最难熬？"
- 结果：客户感到被完全听见，往往会进入更深层的自我探索，揭示真正的障碍

**CoachAgent铁律**：当用户表达挫败感/放弃倾向时，必须先完成F-A-V步骤，才能进入E（探索）或建议模式。禁止收到挫败信号后立即输出行动建议。

## KI-CM-MHWC-004：指令-非指令连续谱与会话结构

```yaml
id: KI-CM-MHWC-004
type: 教练技能框架
evidence: T3
stage: S1-S5
domain: behavioral
keywords: [指令性, 非指令性, 会话结构, 教练谱, 客户中心, 灵活调适]
```

**指令-非指令连续谱**

健康教练实践中，Coach在以下连续谱上选择当前位置：

```
←非指令端                              指令端→
倾听理解 → 反映 → 意译 → 总结 → 提问 → 建议 → 反馈 → 提供信息 → 建议行动
```

**不同客户需要不同位置**（Control/Experience/Motivation/Ability四维矩阵）：
- 高经验+高动机+高能力 → 偏非指令，以探索和发现为主
- 低经验+低动机+低能力（如新诊断慢性病患者） → 偏指令，提供更多结构和信息
- 关键原则：无论在连续谱哪个位置，都要保持以客户为中心（客户始终在驾驶座）

**标准6步会话结构**

| 步骤 | 内容 | 时间比例 |
|------|------|---------|
| **1. 开场确认聚焦** | 检入/共情/确认本次会话焦点 | 10% |
| **2. 上次承诺回顾** | 复查上次行动承诺，庆祝进展 | 15% |
| **3. 深入探索** | 当前议题的深度探索（情绪/障碍/资源） | 30% |
| **4. 前进行动设计** | 与客户共创策略+阶段匹配行动步骤 | 30% |
| **5. 承诺确认** | 明确下一步行动承诺，询问支持来源 | 10% |
| **6. 关闭会话** | 总结/动机连接/确认理解 | 5% |

**CoachAgent的关键操作**：步骤4"行动设计"中必须询问"是什么让这个行动和你真正想要的生活相连？"——确保行动与意义锚定，不只是任务清单。

## KI-CM-MHWC-005：自我决定理论（SDT）在健康教练中的应用

```yaml
id: KI-CM-MHWC-005
type: 行为理论应用
evidence: T2（SDT为T2，教练应用为T3）
stage: S1-S5
domain: behavioral growth
keywords: [SDT, 内在动机, 外在动机, 自主性, 胜任感, 归属感, 动机内化]
```

**动机光谱——从无动机到内在动机**

```
无动机 → 外部调节 → 内摄调节 → 认同调节 → 整合调节 → 内在动机
（冷漠）  （被迫做）  （应该做）  （认为值得）  （完全认同） （热爱做）
```

- **外部调节→内摄调节**：客户参与健康项目是因为医生说要参加、家人施压——完成要求就离开，无持续性
- **认同调节→整合调节**：客户认为健康重要，自己选择参与——坚持度显著提升
- **内在动机**：健康行为本身带来乐趣/成就感——最强可持续性

**三个基本心理需求（Deci & Ryan）**：

| 需求 | 定义 | Coach如何支持 |
|------|------|-------------|
| **胜任感** | 感到自己有能力完成任务 | 设置可成功的小步骤，真实地确认成就 |
| **自主感** | 感到是自己在做选择 | 客户自己设定目标和步骤，Coach不替代决定 |
| **归属感** | 感到被他人关心、有归属 | 教练关系本身的连接，+引导客户建立支持网络 |

**傻瓜相机操作——如何在会话中支持自主感**：
- 避免：直接告诉客户"你应该每天走路30分钟"
- 改为："你觉得在运动这件事上，现在最想尝试的是什么？"
- 当客户给出过于保守的承诺时，温和挑战："这个频率会帮你达到你想要的效果吗？"（而非直接给更高目标）

**外在动机向内在动机的迁移策略**：帮助客户找到行为的内在意义连接→从"不得不"到"我想要"的关键是找到"这和我真正关心的什么相关"。

## KI-CM-MHWC-006：自我效能建设——Bandura社会认知理论的教练应用

```yaml
id: KI-CM-MHWC-006
type: 行为理论应用
evidence: T2（Bandura为T2，教练应用为T3）
stage: S1-S4
domain: behavioral
keywords: [自我效能, Bandura, 社会认知理论, 掌握体验, 替代体验, 言语劝说]
```

**自我效能——行为改变成功的核心预测变量**

自我效能（Self-Efficacy）= 个体对自己在特定领域成功执行行为能力的信念。低自我效能导致：①目标设定过低②遇挫即放弃③预期失败成为自我实现预言。

**四种提升自我效能的方法**（按影响力由强到弱）：

**1. 掌握体验（Mastery Experiences）**——最有效
- 设计可成功的小步骤：让客户经历真实的成功感
- Kaizen原则：行动步骤小到"觉得太容易了"才对
- CoachAgent操作："我们把这个目标拆小——这周，你觉得最小的那一步是什么，一步你觉得可以100%做到的？"

**2. 替代体验（Vicarious Experiences）**——观察榜样成功
- 帮助客户找到与自己条件相似的成功案例
- 关键：榜样需要是客户认为"和我差不多的人"，精英榜样反而降低效能感
- 平台应用：同类用户成功故事匹配推送

**3. 言语劝说（Verbal Persuasion）**——口头鼓励
- 有效的言语劝说需要真实可信："我相信你能做到"需要基于对客户实际能力的了解
- 空洞的"加油"不如"我见过你上次处理那么难的情况都挺过来了，这次也一定能"

**4. 生理与情绪状态（Physiological States）**——身体状态影响信念
- 帮助客户在尝试新行为前进入积极生理状态（充足睡眠、适度活动）
- 负面情绪状态时不适合设定新挑战性目标

**自我效能⬆自我效能建设循环**：
小步骤成功→掌握体验→自我效能提升→设定更大目标→再次成功→正向螺旋

## KI-CM-MHWC-007：深度倾听——教练会话中的五个侦听维度

```yaml
id: KI-CM-MHWC-007
type: 教练技能工具
evidence: T3
stage: S1-S5
domain: behavioral
keywords: [深度倾听, 大五倾听, 焦点, 心态, 资源, 情感, 模式, 觉察, CoachAgent会话]
```

**深度倾听——听的不只是内容，而是整个人**

大多数人以为倾听就是理解对方说的话（内容层）。真正的教练倾听是"听话外之话"——观察客户在内容之下的整个状态。

**"大五"倾听维度（Arloski）**

**1. 焦点（Focus）**
- 倾听客户真正关注什么——当下他们被什么吸引或困扰？
- 注意：客户表述的焦点和真实焦点可能不同（说"想减重"，实际焦点是"想被家人认可"）

**2. 心态/态度（Mindset/Attitude）**
- 倾听客户的思维框架：乐观/悲观？内控/外控？固定型/成长型心态？
- 不要提前下判断，而是让心态在对话中自然显现

**3. 资源与优势（Resources & Strengths）**
- 主动倾听客户描述中透露的能力、优势、过去的成功
- 立即确认并为其命名："你刚才描述的那个处理方式——那是真正的组织能力"

**4. 情感（Affect/Emotions）**
- 倾听情绪信号：声音的变化、节奏的变化、词汇的选择
- 不评判情绪，先反映再探索："我听到你说这话时有一些不一样，能说说那是什么感觉吗？"

**5. 模式（Patterns）**
- 识别跨会话的重复主题和行为模式
- 当模式出现时，可以轻柔地提出："我注意到每次谈到家人的时候，话题就会转向……这是个我们值得一起探索的地方吗？"

**CoachAgent应用**：在每次对话中至少扫描一遍大五维度，并在会话记录中标注本次对话中最突出的1-2个维度，供下次会话继续。

## KI-CM-MHWC-008：欣赏性探询（AI）的4D循环——优势取向教练方法

```yaml
id: KI-CM-MHWC-008
type: 教练方法论
evidence: T3
stage: S2-S6
domain: behavioral growth
keywords: [欣赏性探询, Appreciative Inquiry, 4D循环, 优势取向, 积极探询, 可能性思维]
```

**欣赏性探询（AI）的理论基础**

AI基于建构主义理论：我们的对话和问题本身塑造现实。从"哪里不好"出发的问话创造问题导向的现实；从"什么在起作用"出发的问话创造优势和可能性的现实。

**4D循环**（适用于个体教练会话）

**Discovery（发现）**——寻找生命肯定的故事
- 探询正向核心："告诉我一次你感到真正充满活力/健康的时刻"
- 代表问题："在你的改变旅程中，哪个时刻让你觉得'原来我能做到'？"
- 目标：建立正向核心清单，发现客户的真实能力和优势

**Dream（梦想）**——探索什么是可能的
- 从发现的主题出发，想象"如果这些优势充分发挥，生活会是什么样子"
- 代表问题："如果你完全成为了那个你想成为的健康的自己，你的一天会是什么样子？"
- 与"Well-Life Vision"对接

**Design（设计）**——确定想要什么
- 将愿景转化为具体的行动计划（Who-What-Where-When）
- 结合TTM阶段和COM-B诊断，确定阶段匹配的行动步骤

**Destiny（使命）**——学习、调整、保持活力
- 持续实践，从每次经历中学习，不断迭代
- 不只是"执行计划"，而是保持"成长和发现"的姿态

**CoachAgent应用场景**：在S4→S5过渡阶段，当用户打卡100天以上但开始问"然后呢"时，启动AI 4D探询，帮助用户从执行者身份转向成长者身份。

## KI-CM-MHWC-009：积极心理学在健康教练中的核心应用

```yaml
id: KI-CM-MHWC-009
type: 理论应用
evidence: T2（积极心理学为T2，教练应用为T3）
stage: S2-S6
domain: behavioral growth
keywords: [积极心理学, PERMA, 自我效能, 心流, 成长型心态, 感恩, 优势识别]
```

**积极心理学与健康教练的契合性**

积极心理学通过科学方法验证了人本主义心理学的许多核心主张，为健康教练提供了循证基础。核心发现：

**PERMA幸福模型（Seligman）在教练中的落地**

| PERMA要素 | 英文 | 教练操作 | 平台功能对应 |
|----------|------|---------|------------|
| **积极情绪** | Positive Emotion | 帮客户增加每日积极体验比例 | 日记功能/感恩打卡 |
| **投入** | Engagement | 帮客户找到健康行为中的"心流"时刻 | 行为记录的体验反馈 |
| **关系** | Relationships | 建立支持性人际网络 | 社群功能/家庭协作 |
| **意义** | Meaning | 连接健康行为与更大的人生意义 | Well-Life Vision模块 |
| **成就** | Achievement | 庆祝每个阶段的真实进步 | 徽章/里程碑可视化 |

**积极情绪的"拓展-建构"理论（Fredrickson）**：
- 积极情绪→拓展思维（更多可能性）→建构资源（身体/心理/社会）→螺旋上升
- 教练应用：在会话中主动创造积极情绪体验，不只聚焦在问题解决上
- 操作化：每次会话开始时先庆祝一件进展，再进入探索

**意义和目的感的健康效应**（多项纵向研究）：
- 生命目的感低→HbA1c更高（Mayo Clinic研究）
- 生命目的感与全因死亡率、阿尔茨海默发生率显著负相关
- **教练行动**：将意义探索作为常规会话要素，不只在"遇到困难时"才讨论

**CoachAgent触发条件**：当用户使用以下表述时，启动意义探索对话：
- "我不知道还要坚持多久"→询问"是什么让你一开始开始这段旅程？"
- "血糖达标了，然后呢"→启动PERMA-M探询
- "我这样做到底有什么意义"→进入S5成长超越知识包

**Leonard教练精通五要素**（《精通》George Leonard）

| 要素 | 描述 | 教练应用 |
|------|------|---------|
| **1. 指导（Instruction）** | 持续寻求高质量学习机会 | NBHWC认证培训、督导、ICF持续教育 |
| **2. 练习（Practice）** | 反复练习，享受高原期 | 录制会话并回听，识别模式 |
| **3. 放手（Surrender）** | 向比自己更高水平的人学习，放下已有认知 | 接受督导反馈而不防御 |
| **4. 意向性（Intentionality）** | 每次练习有意识地专注于改进一个方面 | 每次会话前设定一个技能聚焦点 |
| **5. 推边（Pushing the edge）** | 在安全范围内适度挑战自己的舒适区 | 接受更复杂的客户案例，尝试新技术 |

**四阶段能力发展模型**（学习曲线）

| 阶段 | 描述 | 典型表现 |
|------|------|---------|
| **1. 无意识无能** | 不知道自己不会 | "我在给建议，但这是教练啊" |
| **2. 有意识无能** | 知道自己不会，但还做不到 | "我知道应该深度倾听，但做不到" |
| **3. 有意识有能** | 有意识地能做到，需要努力 | "提醒自己才能不急着给答案" |
| **4. 无意识有能** | 自然地做到，无需刻意思考 | "共情和深度倾听变成了我的方式" |

**掌握曲线的关键洞见**：进步不是线性的，而是"短暂的突破+漫长的高原期"的循环。高原期不是倒退，而是新技能在神经层面稳固的过程。学习享受高原期是所有精通的关键。

**平台教练培训应用**：行健平台教练工作站设计应反映上述发展框架，提供：
- 会话录制与AI分析反馈（BCT使用识别）
- 能力自评工具（大五倾听/FAVE/指令谱）
- 同伴督导匹配功能
- 案例复杂度渐进分配（低信任值→高信任值用户的循序承接）
