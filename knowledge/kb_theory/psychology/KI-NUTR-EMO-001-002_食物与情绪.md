### 核心理论框架：食欲=情绪仪表盘

**关键命题**：食欲失控不是道德失败或意志力不足，而是**情绪失衡的生理信号**。

> "你的食欲就像飞机驾驶室的仪表盘——它会在你的精神和情感燃料耗尽时向你发出警告。闪烁着的红灯是饥饿的信号：'我需要更多的平静！'"（Virtue）

**内稳态（Homeostasis）模型**：
```
内心平静 → 正常食欲 → 适量饮食
    ↑                        ↓
情绪失衡 ← 无法靠食物真正解决 ← 情绪性进食（暂时安慰）
（恶性循环）
```

**关键研究支持**（Lyman，加拿大不列颠哥伦比亚大学）：
200名被试研究证实——**不同情绪状态与特定食物偏好之间存在统计显著的一致对应**。

| 情绪状态 | 典型食物渴望 | 生理机制（简述） |
|---------|-----------|----------------|
| 焦虑/紧张 | 脆的、咸的零食（薯片、坚果） | 咀嚼摩擦感→下颌肌肉紧张释放→应激缓解 |
| 抑郁/低落 | 奶制品（冰淇淋、奶酪） | 乳制品含色氨酸→5-HT前体→暂时情绪提升 |
| 无聊/空虚 | 高脂高糖（饼干、蛋糕） | 多巴胺快速激活快乐中枢 |
| 愤怒/挫败 | 花生酱、高脂食物 | 口腔摩擦感+热量密度→宣泄感 |
| 孤独/渴望爱 | 巧克力、甜食 | 苯乙胺（巧克力含有）→类爱情化学物质 |

### 情绪性饥饿的八个特征

以下特征可用于区分**生理性饥饿**与**情绪性饥饿**：

1. **突然发作**：情绪性饥饿突然出现，生理饥饿是逐渐增加的
2. **针对特定食物**：情绪性饥饿只想吃特定食物（如必须是巧克力，不是其他甜食）
3. **紧迫感**：有"现在就要吃"的急迫感
4. **不在胃部**：饥饿感来自口腔、头脑（"嘴馋"），不是胃部空空的感觉
5. **伴随情绪触发**：在特定情绪出现后随即产生（如压力后、争吵后）
6. **不受进食量影响**：吃完后仍有冲动想吃更多
7. **伴随内疚**：生理性饥饿进食后满足；情绪性进食后常常内疚
8. **分心即消减**：专注其他事情后，情绪性饥饿往往消退

### 五种情绪化饮食类型

（Virtue临床分型，对应BPT-6情绪导向型细分）

| 类型 | 核心情绪 | 典型触发 | 常见食物选择 |
|------|---------|---------|------------|
| **焦虑型** | 焦虑/紧张/恐惧 | 工作压力、社交场合 | 脆的零食、咸食 |
| **抑郁型** | 悲伤/低落/无价值感 | 失去、批评、比较 | 奶制品、碳水（安慰食物） |
| **愤怒型** | 愤怒/挫败/委屈 | 冲突、不公平感 | 高脂高盐、花生酱类 |
| **无聊型** | 空虚/无意义感 | 闲暇、缺乏目标 | 高糖、刺激性口感 |
| **爱缺型** | 孤独/渴望连接/不被接受 | 分离、拒绝、忽视 | 巧克力、甜食、软糯食物 |

### BPT-6情绪导向型的饮食干预特异性策略

**情绪导向型（Emotion-Oriented）** 用户的营养干预与其他类型的关键区别：

❌ **错误做法**：只给饮食计划表（信息）→ 情绪进食不是信息匮乏，是情绪调节技能不足

✅ **正确做法**：情绪识别→情绪命名→替代满足→环境设计

**四步干预框架**：
```
步骤1 [识别]：教用户识别情绪性饥饿的信号
  → "在进食冲动出现时，先问自己：我的胃真的空了吗？还是有什么情绪在说话？"

步骤2 [命名]：帮助用户将情绪具体化
  → CoachAgent: "您现在想吃甜食——如果这个甜食冲动会说话，它在说什么情绪？"

步骤3 [替代满足]：提供情绪真正需要的东西
  → 焦虑→4-2-6呼吸 / 抑郁→给朋友发一条消息 / 孤独→给自己写一句肯定语

步骤4 [环境设计]：减少触发物的可及性
  → 家里不存放"触发食物"（见FOGG-002锚点系统）
```

### CoachAgent对话示例

**场景**：S1期用户，体重超标，知道该减肥但每次压力大就暴饮暴食

```
CoachAgent（识别+去羞耻化）:
"您说压力大就吃很多——我想先帮您换一个理解框架。
您的身体很聪明：它知道食物能暂时缓解压力（这在生物学上是真实的），
所以它发出'吃东西'的信号。这不是您意志力差，是您的神经系统在自救。
现在我想问您：当压力来临时，您吃东西之前，有没有感受到一种
——'想要平静下来'的感觉？"
```

```
CoachAgent（步骤2 命名）:
"那个吃甜食的冲动，在您感到[焦虑/孤独/委屈]时特别强——
如果食物可以说话，它在对您说什么？是'我需要被安慰'，
还是'我需要一点奖励'，还是'我需要逃离这个感受'？"
```

🔴 **铁律边界**：
- 情绪性进食评估不等于进食障碍诊断（神经性贪食、暴食症须转介临床心理科）
- 当用户报告定期催吐、大量食物后内疚、进食行为严重影响生活功能时，必须转介
- 平台处理的是**亚临床**情绪性进食，不处理临床级别的进食障碍

### 傻瓜相机验证

**目标用户**：52岁，初中文化，家庭主妇，肥胖，失眠，自述"一烦心就想吃东西，停不下来"

**评估**：情绪导向型 + 焦虑型/爱缺型混合

**可执行干预处方**：
> "王阿姨，您下次想吃东西但不是因为饿的时候，先做一件事：
> 拿出手机，给您的孩子（或朋友）发一条消息，说一件今天发生的事。
> 就这一件事。发完之后，如果还想吃，就吃。
> 但很多时候，发出那条消息，那个饿的感觉就消了。
> 我们试3次，看看结果。"

✅ 这个处方针对"爱缺/孤独"的根本需求（连接），绕过食物直接满足，极简可执行。

### 核心模型：食物改变情绪的三条通路

```
食物
 ├─通路1：神经递质前体供给
 │   （色氨酸→血清素，酪氨酸→多巴胺）
 ├─通路2：血糖波动→能量与情绪振荡
 │   （高GI食物→血糖峰谷→情绪不稳）
 └─通路3：感官刺激→即时神经激活
     （口感/气味/温度→杏仁核→情绪反应）
```

### 血清素（5-HT）：情绪与食欲的核心控制器

**Virtue第8章核心内容**：
血清素是控制**情绪状态、能量水平、食欲调节、睡眠-觉醒**的关键神经递质。

**血清素失调的行为表现**：
- 情绪低落、对事物失去兴趣
- 冲动性进食，尤其是碳水化合物渴望（身体的血清素自我补充机制）
- 入睡困难（血清素是褪黑素的前体）
- 对甜食和高淀粉食物的强烈渴望

**碳水化合物的血清素效应（重要机制）**：
碳水化合物→血糖升高→胰岛素释放→色氨酸相对比例升高→更多色氨酸进入大脑→血清素合成增加

这解释了为什么：
- 压力后想吃面包/米饭/甜食是生理性冲动（自我药疗）
- 严格低碳饮食可能在某些用户中加重情绪低落
- "抑郁型"情绪进食者特别渴望碳水类食物

### 食物×情绪×神经递质对照表（临床应用版）

| 情绪问题 | 相关神经递质缺乏 | 饮食干预方向 | 注意事项 |
|---------|----------------|------------|---------|
| 抑郁/低落 | 5-HT、DA | 增加色氨酸来源（蛋奶豆坚果）+ 复合碳水 | 过度低卡加重抑郁 |
| 焦虑/紧张 | GABA↓、5-HT↓ | 减少咖啡因、增加镁（坚果、深绿蔬菜） | 酒精短暂缓解但长期加重 |
| 疲劳/无动力 | DA↓、NE↓ | 增加蛋白质摄入（早餐优先）、B族维生素 | 高糖快速修复后更疲劳 |
| 情绪不稳 | 血糖波动 | 低GI饮食、规律进食时间、减少精制糖 | 空腹时情绪特别差是血糖信号 |
| 睡眠差 | 5-HT↓（褪黑素前体） | 睡前1-2小时温奶或小份坚果+碳水 | 睡前高蛋白反而影响入睡 |

### TCM食物理论与现代研究的交汇点

Virtue在书中特别指出：
> "古代中医对食物与情绪、身心健康关系的观点与现代科学研究不谋而合！中医认为食物具有寒热温凉四种药性，几个世纪前标记为热性的食物，如今被发现具有收缩血管或刺激情绪的作用。"

**行健平台整合价值**：这为TCM九体质×饮食情绪干预提供了现代神经科学的桥接依据：

| TCM体质 | 常见情绪特征 | 食物-情绪干预方向 |
|---------|-----------|----------------|
| 气郁质 | 抑郁、焦虑、胸闷 | 低咖啡因 + 疏肝理气食物（陈皮、薄荷、玫瑰） + 色氨酸食物 |
| 阳虚质 | 情绪低沉、活力不足 | 温热食物（姜、枣）+ 蛋白质 + 早晨光照 |
| 湿热质 | 急躁易怒 | 清热食物（绿豆、薏仁）+ 减少高脂高糖 + 减少酒精 |
| 阴虚质 | 情绪烦躁、失眠 | 滋阴食物（银耳、枸杞、百合）+ 减少刺激性食物 |

### 食物成瘾的自救框架（Virtue第9章）

**食物成瘾的识别信号**：
- 对特定食物有抵制不了的渴望（强迫性）
- 吃这种食物能暂时缓解负面情绪
- 想停却停不下来
- 用食物处理应激的模式固化，其他应对方式被废弃

**三步脱离食物成瘾框架**：
```
步骤1：识别触发情绪（不是触发食物，而是触发情绪）
步骤2：提供真正满足情绪需求的替代选择
步骤3："精神食粮"——识别在食物之外真正滋养自己的事情
  （创造力/连接/意义感/成就感）
```

**"精神食粮"的行健平台实践**：
用户的真正需求往往是：被看见、被认可、有意义、有控制感。
CoachAgent可以在饮食干预中引入"意义感设计"：
> "您今天的饮食选择，是在照顾未来能陪孙子玩耍的自己——这比今天那块饼干重要吗？"

### CoachAgent对话示例

**场景**：肥胖用户，严格节食两周但情绪越来越差，开始暴食

```
CoachAgent: "您节食期间情绪变差，然后暴食——这不是意志力崩溃，
这是您的身体发出了一个信号：血清素水平在下降。
严格低热量+限制碳水会减少血清素合成。您的大脑在求救。
我们需要调整一下计划——不是让您多吃，
而是把碳水化合物换成慢释放的（糙米、燕麦、红薯），
这样血清素稳定了，那种控制不住要暴食的冲动会减轻很多。"
```

🔴 **铁律边界**：
- 上述饮食建议均为行为健康层面的一般性建议，不构成个体营养处方
- 严重抑郁、双相情感障碍等心理疾病的饮食干预须配合精神科药物治疗，不能替代
- 任何建议改变糖尿病患者碳水化合物摄入的内容须先确认医生方案

### 傻瓜相机验证

**目标用户**：45岁，初中文化，服装厂女工，体重超标，情绪低落，特别爱吃甜的和奶制品

**评估**：抑郁型情绪进食 + 血清素需求信号

**可执行干预处方**：
> "张阿姨，您爱吃甜的和奶制品，这其实是您的身体在寻找一种叫血清素的好情绪物质。
> 这很聪明——奶制品里真的有它的原料（色氨酸）。
> 我们不是要让您戒掉，而是升级一下：
> 每天早餐加一杯温牛奶+一小把坚果（10-15颗），
> 这样您的大脑有了充足的血清素原料，下午那种控制不住要吃甜食的冲动会减少。
> 试两周，看看情绪有没有好一点？"

✅ 机制正确（色氨酸→5-HT），去羞耻化，正向替代而非禁止，门槛极低。
