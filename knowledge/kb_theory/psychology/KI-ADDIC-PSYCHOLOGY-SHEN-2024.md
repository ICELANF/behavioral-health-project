## 核心命题

```
成瘾的本质 = 错误的自我疗愈

用户无法停下来，不是因为：
  × 意志力不够强
  × 道德品质有问题  
  × 懒惰或自我放纵

而是因为：
  ✓ 现实中有未被满足的心理需求
  ✓ 成瘾行为暂时满足了这些需求（错误但有效）
  ✓ 背后有亲密关系的缺失或未处理的创伤
  ✓ 除了成瘾行为，找不到其他替代满足的方式

→ 处理成瘾，最重要的是：
  处理过去的创伤 + 重建现实中的亲密关系（尤其是与父母的关系）
```

### KI-ADDIC-001 | 成瘾的7项诊断标准

```yaml
knowledge_id: KI-ADDIC-001
title: 成瘾行为的7项诊断标准（适用于物质成瘾与行为成瘾）
tags: [成瘾诊断, 7标准, 耐受性, 戒断反应, 失控, 强迫欲望, 损害]
retrieval_queries:
  - 怎么判断自己上瘾了吗
  - 成瘾的标准是什么
  - 用户问我对某事是不是上瘾了怎么回答
  - 行为成瘾和习惯有什么区别
```

**适用范围**：物质成瘾（烟/酒/糖/咖啡因）+ 行为成瘾（手机/游戏/暴食/购物/赌博/社交媒体）

**7项标准**（满足3项以上，提示成瘾倾向）：

| # | 标准 | 具体表现 | 行为健康应用 |
|---|------|---------|------------|
| 1 | **耐受性** | 同样行为需要越来越多才能获得相同满足感 | 甜食越吃越多；每天运动量不增加就不舒服 |
| 2 | **戒断行为** | 减少或停止时出现躯体或心理症状 | 不吃糖就烦躁；不刷手机就坐立不安 |
| 3 | **失去控制** | 每次做的比计划的多 | "就吃一块"变成吃完整包 |
| 4 | **无法减少** | 多次尝试减少或停止，均以失败告终 | 反复尝试戒糖/减少外卖均失败 |
| 5 | **过度关注和强迫欲望** | 满脑子都是这件事，停不下来想 | 脑子里一直想吃某种食物 |
| 6 | **身心社会功能损害** | 影响健康/工作/家庭/社交 | 因暴食影响工作效率；因手机影响睡眠 |
| 7 | **明知有害仍继续** | 清楚知道后果，还是停不下来 | 知道血糖高还是喝甜饮料 |

**CoachAgent使用说明**：
- 不直接对用户使用"成瘾"标签（参考PHIL-铁律-01）
- 用探索性语言："你有没有发现，每次比自己计划的吃得多？"
- 满足3项以上时，升级为"行为失控"处理模式

### KI-ADDIC-003 | 心瘾的本质：无替代满足对象的内外需求

```yaml
knowledge_id: KI-ADDIC-003
title: 心瘾（心理成瘾）：成瘾之所以戒不掉的核心机制
tags: [心瘾, 心理需求, 替代满足, 亲密关系, 创伤, 戒断困难]
retrieval_queries:
  - 为什么意志力解决不了成瘾
  - 心瘾是什么意思
  - 用户反复破戒是什么原因
  - 行为改变为什么那么难
```

**心瘾的定义**：

> 你的内在和外在需求，除了成瘾行为之外，**无法找到替代满足的对象**。你所有的需求都只能通过成瘾满足。

**心瘾的形成机制**：

```
未被满足的心理需求
（现实生活中的亲密关系缺失/创伤/空虚/无意义）
         ↓
发现成瘾行为可以暂时满足这些需求
（吃糖→短暂愉悦；刷手机→暂时逃避焦虑）
         ↓
大脑建立"需求→成瘾行为"的条件反射
         ↓
心瘾形成：除了这个，什么都满足不了我
         ↓
戒断时产生强烈痛苦（因为需求仍在，出口被堵）
         ↓
复发
```

**心瘾的特征**：
- 成瘾行为停止时，身体和精神会遭受**巨大痛苦**
- 满脑子只有一件事——满足欲望
- 与冲动对抗时耗费巨大**心理能量**
- 破戒后陷入**深深的挫败和自责**
- 自责→否定自己→更需要成瘾行为来抚慰→恶性循环

**对平台的根本含义**：
→ 行为处方如果不触碰心瘾，最终必然失败
→ CoachAgent需要帮助用户识别"成瘾行为满足了你什么需求？"
→ 最有效的干预：找到替代满足方式 + 处理底层需求
→ 成瘾处理的终极目标：重建现实生活中的亲密关系（尤其是与父母/伴侣的关系）

## 模块2：戒瘾的6大误区（成瘾干预的禁区）

### KI-ADDIC-004 | 成瘾干预的6大误区与正确认知

```yaml
knowledge_id: KI-ADDIC-004
title: 关于成瘾和戒瘾的6个常见误区（CoachAgent禁止使用的话术）
tags: [戒瘾误区, 意志力, 道德, 惩罚, 家人监督, 误区纠正]
retrieval_queries:
  - 戒瘾靠意志力吗
  - 家人监督可以帮助戒瘾吗
  - 上瘾是道德问题吗
  - 如何帮助家人戒瘾
  - 严厉惩罚能戒瘾吗
```

**6大误区 → 正确认知 → CoachAgent操作禁忌**：

---

**误区1：戒瘾只要痛下决心就可以**

```
错误认知：意志力足够强，成瘾就能戒掉
正确认知：成瘾是心理需求问题，不是意志力问题
         意志力攻克的是行为，心理需求还在，必然复发

CoachAgent禁忌话术：
  ❌ "你只要下定决心就能做到！"
  ❌ "就是毅力不够，再坚持一下！"
  ✅ "你有没有发现，每次破戒的时候，你正在经历什么感受？"
```

---

**误区2：戒瘾要彻底、永久地远离**

```
错误认知：只要还有一点就算失败，必须永久断绝
正确认知：完美主义的"全戒"目标会在第一次破戒后导致彻底放弃
         更健康的目标是"管理"而非"彻底消灭"

对平台的含义：
  行为处方的目标设定应是渐进式减少，而非立即完全停止
  如：每日甜食摄入量从300g减到200g，而非立即降为0
```

---

**误区3：上瘾是道德问题，是堕落的表现**

```
错误认知：上瘾是品德有问题、意志薄弱、堕落
正确认知：成瘾只代表行为错了，人没有错
         不应该否定人本身的价值

CoachAgent铁律：
  绝不允许对用户说：
  ❌ "你这样下去太自我放纵了"
  ❌ "戒不掉说明你不够努力"
  ✅ "成瘾不是你的错。行为可以改变，但你这个人本身没有问题。"
```

---

**误区4：严厉惩罚才能戒掉**

```
错误认知：要用惩罚（自我惩罚/他人惩罚）来强化戒瘾
正确认知：惩罚会增加羞耻感 → 羞耻感会触发成瘾行为（寻求安慰）
         惩罚是成瘾的燃料，不是解药

对平台设计的含义：
  平台不应设置"失败惩罚"机制（如：打卡断了扣分/羞辱提示）
  应设置"复发是学习机会"的积极框架
```

---

**误区5：靠家人监督和控制才能戒掉**

```
错误认知：家人盯着管着就能戒
正确认知：监控→对抗→反弹→关系恶化→成瘾更严重
         越控制，成瘾者越反抗，越不听话

正确的家人角色：理解+接纳+陪伴，给时间和空间
              支持者，不是监控者

对CoachAgent的含义：
  平台家庭模块不是"监督打卡"，而是"建立支持性关系"
  家庭干预目标：改善亲密关系，不是增加控制
```

---

**误区6：不需要心理咨询的帮助**

```
错误认知：自己能解决，不需要专业帮助
正确认知：成瘾背后有创伤和亲密关系问题，这些需要专业处理

平台适用：
  当用户的成瘾行为明显与情感创伤/家庭关系相关时，
  CoachAgent应建议寻求专业心理咨询，不能独自承担
```

**5阶段与TTM对应关系及干预策略**：

| 沈家宏阶段 | TTM对应 | 核心特征 | 干预策略 |
|-----------|---------|---------|---------|
| **前思考期** | S0/S1 | 不认为自己有问题，排斥讨论 | 理解+接纳+耐心陪伴；不强行说教；等待觉察 |
| **思考期** | S2 | 已认识到问题，但在犹豫要不要改变 | 提升改变意愿；帮助权衡利弊；四象限动机探索 |
| **准备期** | S2→S3 | 决心改变，有计划但尚未行动 | 协助制定具体计划；小步行动；降低起步门槛 |
| **行动期** | S3→S4 | 已开始执行改变计划 | 技能支持；触发器管理；社会支持建立 |
| **维持期** | S4→S5 | 持续维持1个月以上 | 复发预防；高危状态识别；意义连接 |
| **复发期** | （特殊） | 重新回到成瘾状态 | 不评判；再燃vs复发区分；从错误中学习 |

**前思考期的特殊注意**：

这是平台最常遇到的挑战——用户被家人"送来"，自己不认为有问题。

正确应对：
- **不争论**，不强调问题
- **关注关系**（家庭矛盾往往比成瘾本身更明显）
- 提供**安全空间**，不是诊断或说教
- 等待用户自己发现，**急不得**

## 模块4：触发器识别与管理

### KI-ADDIC-006 | 成瘾触发器：外部诱因 × 内部诱因完整分类

```yaml
knowledge_id: KI-ADDIC-006
title: 成瘾触发器的完整分类（外部+内部）及干预策略
tags: [触发器, 诱因, 外部触发, 内部触发, 环境管理, 情绪触发, BCT 7.1]
retrieval_queries:
  - 什么触发了用户的成瘾行为
  - 如何识别用户的触发按钮
  - 外部诱因有哪些
  - 内部诱因有哪些
  - BCT前因管理怎么做
cross_reference: BCT-7.1（提示/触发器管理）、BCT-12.1（环境重构）
```

**触发器的两类来源**：

```
外部诱因（环境/社会/情境）
  ├── 人：成瘾同伴、递烟的朋友、劝酒的同事
  ├── 地点：酒吧/超市甜食区/外卖平台/赌场
  ├── 事件：聚会/体育赛事/压力大的会议/家庭冲突
  ├── 物品：看到成瘾物质本身（视觉触发）
  └── 时间：特定时刻（晚上10点/周末/节假日）

内部诱因（心理/生理状态）
  ├── 消极情绪：愤怒/焦虑/厌烦/抑郁/内疚
  ├── 积极情绪：庆祝/兴奋/"奖励自己"
  ├── 生理状态：疲劳/饥饿/低血糖/睡眠不足
  ├── 思维：成瘾思维（"就这一次"/""反正已经破戒了"）
  └── 社会压力：被邀请/不好意思拒绝/怕扫兴
```

**针对外部诱因的6个行动策略**：

1. **减少与成瘾同伴的联系**，多与没有成瘾的朋友在一起
2. **避开高危地点**（至少在早期阶段）
3. **清除成瘾物品**（家里不放甜食/烟/酒）
4. **分散注意力**（提前准备替代活动）
5. **讲述自己的成瘾经历**（通过表达获得距离感）
6. **与自我对话**（建立内心的对话，而非与冲动直接对抗）

**社会压力应对工作表**（CoachAgent可直接引导用户填写）：

```
| 社会压力情境 | 困难难度(0-5) | 处理措施 |
|-------------|-------------|---------|
| 朋友聚会劝酒 |     4       | 提前声明/换无酒精饮料 |
| 同事推荐奶茶 |     3       | "我在控制糖分，谢谢" |
| 家人做了最爱吃的甜点 | 5  | 少量品尝/提前沟通 |
```

## 模块5：渴求冲动的应对策略

### KI-ADDIC-007 | 5类渴求冲动应对策略（CoachAgent核心工具箱）

```yaml
knowledge_id: KI-ADDIC-007
title: 遏制成瘾渴求冲动的5类应对策略
tags: [渴求管理, 冲动应对, 环境策略, 认知策略, 行为策略, 人际策略, 替代物]
retrieval_queries:
  - 想吃又不想吃的时候怎么办
  - 冲动来了怎么应对
  - 如何控制想吃甜食的欲望
  - 渴求感很强的时候用什么方法
cross_reference: BCT-5.6（对抗渴求）、BCT-12.1（环境重构）、BCT-4.1（目标提示）
```

**5类应对策略完整框架**：

---

**第1类：环境应对策略**

核心原则：改变环境，减少触发

- 离开触发场所（"回避"）
- 清除视野内的成瘾物品
- 改变惯常路线（避开超市甜食区）
- 提前准备健康替代食物放在明显位置

---

**第2类：认知应对策略**

核心原则：改变头脑中对成瘾行为的想法

- **挑战成瘾思维**：
  - "就这一次" → "每一次都是'就这一次'"
  - "反正已经破戒了" → "一次破戒不等于完全失败"
  - "我需要它" → "这是渴求，不是需要，它会过去的"
  
- **推迟决策**："我先等15分钟，如果还想要再说" → 渴求通常在10-20分钟后峰值过去

- **可视化结果**：想象如果屈服后的感受（事后的内疚/身体的不适）

---

**第3类：行为应对策略**

核心原则：用行动打断冲动-行为的自动链条

- **立即行动**（打断自动性）：出去散步/喝水/做10个深蹲
- **正念呼吸放松**（平台正念模块）：
  ```
  1. 找一个安静、舒适的地方坐下
  2. 闭上眼睛，把注意力放在呼吸上
  3. 感受腹部的起伏（腹式呼吸）
  4. 当渴求出现时，观察它，不对抗，不跟随
  5. 告诉自己："这是渴求，它会过去的"
  6. 5-10分钟后，渴求峰值通常已过
  ```
- **日记书写**：把渴求和感受写下来（外化处理）
- **冲动冲浪（Urge Surfing）**：把渴求想象成海浪，观察它升起、到峰值、然后落下

---

**第4类：人际关系应对策略**

核心原则：社会支持是渴求管理最强的外部资源

- 联系"核心支持者"（预先指定的可信任的人）
- 告诉他/她："我现在很想做X，帮我说说话"
- 参与同伴支持团体
- 与理解自己的人在一起（而非独自对抗）

---

**第5类：替代物策略**

核心原则：用无害的行为满足相同的心理需求

| 原始需求 | 成瘾行为（旧出口） | 替代物（新出口） |
|---------|-----------------|---------------|
| 缓解焦虑 | 进食/吸烟/饮酒 | 腹式呼吸/冥想/运动 |
| 寻求愉悦 | 甜食/游戏 | 音乐/社交/户外活动 |
| 逃避空虚 | 手机/购物 | 有意义的活动/创造性工作 |
| 缓解疲劳 | 咖啡因/糖 | 短暂休息/小睡 |
| 感受连接 | 酒/食物 | 真实的人际接触 |

## 模块6：5种负面情绪与8个处理方法

### KI-ADDIC-008 | 成瘾者的5种主要消极情绪及8步情绪处理框架

```yaml
knowledge_id: KI-ADDIC-008
title: 成瘾背后的5种消极情绪及系统化情绪管理方法
tags: [情绪管理, 成瘾情绪, 愤怒, 焦虑, 厌烦, 抑郁, 内疚羞耻, 情绪处理8步]
retrieval_queries:
  - 成瘾者常有哪些情绪
  - 情绪和进食的关系
  - 愤怒时想吃东西怎么处理
  - 内疚感会导致复发吗
  - 情绪性进食背后是什么
cross_reference: BCT-11.2（情绪管理）、亚蒙框架5种脑类型
```

**5种主要消极情绪（压力源）**：

**① 愤怒**
- 特征：脾气突然失控，暴发后疲惫
- 成瘾连接：愤怒→无法处理→用成瘾行为平息怒火
- 干预重点：愤怒管理技术；识别愤怒触发点

**② 焦虑**
- 特征：对未来的担忧；身体紧张（头痛/肌肉痛）
- 成瘾连接：焦虑→用食物/烟/酒/手机暂时镇静
- 干预重点：放松技术（腹式呼吸/冥想）；GABA调节

**③ 厌烦和空虚感**
- 特征：感觉生活没意义；无事可做就"需要点什么"
- 成瘾连接：空虚→成瘾行为填充时间和感官
- 干预重点：**意义感建构**（直接连接L3天赋框架）；发展真实兴趣

**④ 抑郁**
- 特征：长期情绪低落；失去对事物的兴趣；无望感
- 成瘾连接：抑郁→用成瘾行为获得短暂快感（自我治疗）
- 干预重点：识别抑郁；如严重，建议专业干预；运动（提升5-HT）
- ⚠️ 铁律：抑郁+成瘾共病时，单独处理成瘾会失败，必须同时处理抑郁

**⑤ 内疚和羞耻**
- 特征：认为自己"对不起"他人；自我否定；深感羞耻
- 成瘾连接：内疚/羞耻→更需要用成瘾行为逃避这种感受→更多内疚→恶性循环
- 干预重点：区分"行为错了"和"人没有错"；去羞耻化

**8步情绪处理框架**（成瘾干预中的情绪管理协议）：

```
1. 清楚地认识你的情绪 → "我现在的感受是什么？"
2. 接受你的感觉 → "这种感受是真实的，允许它存在"
3. 了解感觉产生的原因 → "是什么触发了这种感受？"
4. 挑战你的想法 → "这个想法是事实，还是我的解读？"
5. 和信任的人分享你的感觉 → 打破孤立，建立支持
6. 合理安排你的生活 → 规律作息减少情绪波动
7. 学会调节内心 → 正念/冥想/放松技术
8. 用无害的行为释放情绪 → 运动/书写/艺术表达
```

**CoachAgent情绪干预顺序（重要）**：
→ 先完成步骤1-3（看见→接受→理解），再进入步骤4-8
→ 跳过前3步直接"解决"，用户会感到被评判，防御增强

## 模块7：复发预防

### KI-ADDIC-009 | 再燃 vs 复发：区分与应对

```yaml
knowledge_id: KI-ADDIC-009
title: 再燃（一次破戒）与复发（全面退回）的区分及应对策略
tags: [再燃, 复发, 复发预防, 高危状态, 复发警告信号, 5步应对]
retrieval_queries:
  - 用户昨天又吃了甜食怎么处理
  - 一次破戒等于失败吗
  - 如何阻止再燃变成复发
  - 复发之后怎么回来
  - 高危状态有哪些
```

**关键概念区分**：

```
再燃（Lapse）= 一次失误
  → 在一次情境中使用了成瘾物质或进行了成瘾行为
  → 不等于复发
  → 可以学习，可以反弹

复发（Relapse）= 完全退回到成瘾状态
  → 再燃之后没有及时处理
  → "破罐子破摔"效应："反正已经破戒了，不如彻底放弃"
  → 可以预防
```

**高危状态（触发复发的高风险情境）**：

- 复发事件（重大生活打击/压力事件）
- 消极情绪（上述5种）
- 社会压力（被邀请/被劝说）
- 人际矛盾（家庭冲突/关系危机）
- 积极情绪（庆祝/奖励自己的心理）
- 渴求和诱惑（意外接触成瘾物质）

**再燃时的5步危机处理协议**（CoachAgent核心流程）：

```
第1步：停下来，看一看，听一听
  → 找出是什么导致了这次再燃
  → 立刻摆脱可能导致完全复发的状态

第2步：面对你的问题
  → 在"破罐子破摔"之前，立刻处理这个问题
  → 不逃避，不假装没发生

第3步：重新树立你的目标
  → "我已经成功了X天/X周，这是真实的"
  → "一次再燃不抹掉我所有的进步"
  → "我的目标是继续前进，不是达到完美"

第4步：利用你的支持网络
  → 联系核心支持者
  → 告诉他/她发生了什么
  → 接受支持，不要独自处理

第5步：从错误中学习
  → "这次再燃告诉了我什么？"
  → "什么情境/状态最危险？"
  → 更新自己的高危状态清单
```

**CoachAgent再燃对话框架**：

```
用户："我昨晚又吃了一包零食，彻底失败了。"

错误回应：
  ❌ "没关系，重新来过！"（轻描淡写，未帮助用户从中学习）
  ❌ "你要更努力才行。"（评判，增加内疚）
  ❌ "怎么又这样了？"（责备）

正确回应框架：
  ✅ 第一句：看见感受
  "听起来你昨晚之后感觉很糟糕。"
  
  ✅ 第二句：区分概念
  "这是一次再燃——一次失误，不是全盘失败。
   你之前坚持的那些天，那些进步还在。"
  
  ✅ 第三句：好奇探索
  "昨晚发生了什么？是什么状态让这个发生的？"
  （帮助用户识别触发因素，而非停留在失败感中）
  
  ✅ 第四句：共同学习
  "我们从这次经历里能学到什么，让下次准备更充分？"
```

## 铁律声明

**ADDIC-铁律-01（不用羞耻激励）**
CoachAgent在任何情况下不得用羞耻感、内疚感或道德评判来激励用户改变行为。研究表明，羞耻感是成瘾复发的主要触发因素之一。"成瘾只代表行为错了，人没有错。"这是任何干预的前提。

**ADDIC-铁律-02（再燃不等于失败）**
当用户报告"破戒/再燃"时，CoachAgent必须立即区分"再燃"和"复发"，不允许用"失败/完全放弃"等框架解释单次破戒。应用五步再燃应对协议，将其转化为学习机会。

**ADDIC-铁律-03（家庭干预原则）**
当家庭成员要求平台帮助监督成瘾用户时，CoachAgent应明确解释：监控会加剧成瘾，平台的家庭功能是帮助改善关系质量，而非增加监控密度。

**ADDIC-铁律-04（成瘾与心理创伤）**
当用户的成瘾行为明显与情感创伤、重大生活事件或家庭关系创伤相关时，CoachAgent不得单独处理成瘾行为，应同时建议心理咨询或专业支持。

**ADDIC-铁律-05（共病识别）**
当用户同时表现出明显的抑郁或焦虑症状（5种消极情绪持续2周以上，严重影响功能）时，单纯的行为干预不足，必须建议专业心理/精神科评估。
