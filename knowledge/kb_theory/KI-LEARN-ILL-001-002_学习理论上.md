### 核心模型：学习三角

Illeris构建的学习三角模型揭示：任何真实的学习过程都由**两个过程**和**三个维度**组成。

```
              内容维度
         （知识/理解/技能）
              ▲
             / \
            /   \
           /     \
动机维度 ←————————→ 互动维度
（动力/情绪/意志）  （活动/对话/合作）
          ↑
    获得过程（个体内部）
          ↑
    互动过程（个体与环境）
```

**两个基本过程**：
1. **获得过程**（Acquisition）：完全发生于个体内部，新冲动与已有认知结构/情绪记忆相互作用
2. **互动过程**（Interaction）：个体与社会性、物质性环境之间的交流

关键原则：**这两个过程总是同时发生，无法单独存在。**

### 三个维度详解

| 维度 | 核心要素 | 发展目标 | 在行为改变中的角色 |
|------|---------|---------|-----------------|
| **内容维度** | 知识、理解、技能 | 构建意义，掌握知识技能，提升功能性 | 用户"知不知道"健康知识，"能不能做"健康行为 |
| **动机维度** | 动力、情绪、意志 | 维持心智与身体平衡，发展敏感性 | 用户"想不想"改变，情绪状态，改变动力来源 |
| **互动维度** | 活动、对话、合作 | 实现社会整合，发展社会性 | 用户"有没有"社会支持，环境是否支持改变 |

> **Illeris核心洞见**：传统教育和健康指导几乎只关注内容维度（"告诉你应该怎么做"），
> 严重忽视动机维度（"你为什么不做"）和互动维度（"环境是否让你能做"）。
> 这正是知识充分但行为不改变的根本原因。

### 三维度诊断框架：行为改变失败的根因分析

当用户的行为改变目标未能实现时，优先诊断哪个维度出了问题：

**内容维度障碍** → 用户不知道、不理解，或理解错误
- 症状：问"我该怎么做？"/"我不知道吃什么"
- 干预：信息提供、循证教育、操作演示

**动机维度障碍** → 用户不想、情绪阻力、意志耗竭
- 症状："我知道但就是做不到"/"我没动力"/"坚持不下去"
- 干预：动机性访谈、情绪识别、价值观联结、降低认知负荷

**互动维度障碍** → 用户没有环境支持、社会压力反向、缺乏合作伙伴
- 症状："家人不支持"/"没人陪我一起"/"工作太忙没条件"
- 干预：家庭系统介入、环境设计、社群支持激活

**混合障碍**（最常见）：
大多数慢性病患者的行为改变失败是三个维度共同作用的结果——医生只告诉了内容（知识），没有处理动机（想不想）和互动（环境支持）。

### CoachAgent三维度诊断对话框架

**第一步：快速维度定位**

```
CoachAgent: "您之前尝试过改变饮食习惯——可以告诉我，
那次主要是在哪个环节遇到了困难？
是不确定该怎么做（知识/方法问题），
还是知道该怎么做但就是很难坚持（动力/情绪问题），
还是外部条件不支持，比如家人、工作等（环境/支持问题）？"
```

**第二步：精准干预匹配**
- 内容障碍 → 提供行为处方（具体可操作的微习惯）
- 动机障碍 → 价值观探索 + 情绪支持 + Fogg庆祝机制
- 互动障碍 → 家庭会谈邀请 + 环境重新设计 + 社群匹配

### 傻瓜相机验证

**目标用户**：52岁，初中文化，肥胖，知道要减肥，3次失败，自述"就是做不到"

**三维度快速诊断**：
- 内容维度：基本知识有（知道该少吃）→ 不是主要问题
- 动机维度：3次失败后动力耗尽，有习得性无助倾向 → **主要障碍**
- 互动维度：未询问家庭支持情况 → 需进一步评估

**针对动机维度的处方**：
> "您3次都坚持了一段时间——我们先不说失败的部分。
> 每次坚持的阶段，是什么让您撑下来的？
> 我们今天的目标，就是找回那个让您动起来的理由。"

✅ 绕过内容讲解（已知），直接处理动机维度的核心问题。

### 四种学习类型概述

Illeris从皮亚杰的同化/顺应理论发展出四种学习类型，按照深度和能量消耗从低到高排列：

```
累积学习 → 同化学习 → 顺应学习 → 转换学习
（最浅）                              （最深）
（能量最少）                          （能量最多）
（改变最小）                          （改变最大）
```

> **关键原则**：更深的学习类型≠"更好"的学习。在特定情境下最适合的才是最好的。

### 类型1：累积学习（Cumulation）

**特征**：在没有已有图式的空白处，建立第一个元素。机械记忆型。
**典型场景**：死记国王年表、背单词表、记急救口诀
**首要特征**：刻板性——只能在与原始学习情境完全相同的环境下激活

**行为健康应用**：
- 用于建立全新知识模块的第一个元素（如用户第一次接触血糖概念）
- 不适合作为主要学习方式——慢性病管理需要的不是死记规则，而是灵活应用能力
- 风险：只用累积学习培训出的用户，换一个情境就不会用（如背诵了"不能吃白饭"但面对炒饭时不知道怎么处理）

### 类型2：同化学习（Assimilation）

**特征**：新信息进入已有心智图式，扩展和差异化已有知识。日常最普遍的学习形式。
**核心机制**：已有图式 + 新信息 → 扩展后的图式（结构不变，内容增加）
**学习成果**：知识和技能可在一系列具有相同特征的情境中被激活

**行为健康应用**：
- 适合S1-S3期用户的主要干预方式
- 例：用户已有"多运动对健康有好处"的图式 → 同化学习添加"每天步行30分钟可降低血糖5-10%"
- 局限：在同化框架内，用户的根本信念和生活方式图式不改变——适合改善，不适合颠覆

### 类型3：顺应学习（Accommodation）

**特征**：已有图式被突破和重构，建立新图式。需要更多心理能量。
**核心机制**：旧图式不再适用 → 解构旧图式 → 重建新图式
**学习成果**：更持久、更具适应力，可在不可预期的新情境中灵活应用

**行为健康应用**（关键：这是慢性病逆转的核心学习机制）：
- 适合S2-S4期，用户准备做根本性改变
- 例：用户从"血糖高了就吃药"的图式，转变为"血糖是生活方式的信号，我可以通过行为管理它"
- 这正是从依赖医疗管理到自我赋权的认知转变
- **需要比同化更多的支持和能量补给**：教练需在顺应过程中提供更强的情感支持

**顺应学习的代价**：
顺应学习要求比同化学习更多的能量——如果课题对个体没有特殊兴趣，往往会倾向于回避（退回到同化）。

**CoachAgent操作含义**：
> 当用户准备进行顺应性改变时，教练必须先**确认情感安全**（"这个改变会让您感到有些颠覆，这很正常"），
> 再**提供充足的价值驱动**（"这个改变会让您……"），
> 然后**分阶段执行**，避免一次性要求过多重构。

### 类型4：转换学习（Transformation）

**特征**：多个图式同时被重组，涉及学习的全部三个维度，包括自我认同的根本性变化。
**核心机制**：意义视角（参照框架）被质疑 → 深度反思 → 视角转换 → 全新自我理解
**来源**：梅齐洛（Mezirow）转换学习理论
**特征**：常常是"一种非常紧张的、带有威胁感的情感体验"

**行为健康应用**（最深、最持久的改变，也最难触发）：
- 对应S4-S5期，从行为改变到价值观整合
- 例："我是一个管理自己健康的人"这一身份认同的形成
- 转换学习的触发点往往是：
  - 确诊重大疾病（"现在不改变可能活不过60岁"）
  - 家庭角色触发（"我要陪孙子长大"）
  - 重大生活事件（退休/亲人去世）

**梅齐洛转换学习的触发条件**：
```
1. 触发事件/两难困境（人生转折点）
2. 自我检视（对已有假设的反思）
3. 批判性评估（质疑参照框架）
4. 认同寻找（与有相似经历者连接）
5. 探索新角色（实验新的自我理解）
6. 行动计划
7. 知识技能积累
8. 临时性地尝试新角色
9. 能力建设
10. 融入新视角，重新整合生活
```

**行健平台与转换学习的连接点**：
- XZB行诊智伴的深度对话设计可以支持第2-4步
- 用户故事/社群中的真实案例触发第4步"认同寻找"
- BAPS行为评估+TTM评估支持用户的自我检视（第2步）

### 四种类型×TTM阶段对应关系

| 学习类型 | 适配TTM阶段 | 教练主要工作 |
|---------|-----------|-----------|
| 累积学习 | S0-S1 | 提供基础健康信息，建立概念框架 |
| 同化学习 | S1-S3 | 扩展已有健康知识，添加新的操作技能 |
| 顺应学习 | S2-S4 | 支持认知框架重构，帮助突破核心信念 |
| 转换学习 | S4-S5 | 深度身份认同工作，价值观整合 |

### 傻瓜相机验证

**目标用户**：58岁，高中文化，长期烟民，刚被确诊为冠心病，医生建议戒烟

**四种学习类型在戒烟过程中的角色**：
- 累积：知道"吸烟有害心脏"（已有，不是主要障碍）
- 同化：了解"每少吸一支烟，心脏病风险降低多少"（可扩展动机）
- 顺应：打破"我就是个烟民，戒不了"的心智图式 → **当前需要的关键学习**
- 转换：从"吸烟是我的习惯/身份"转变为"我是一个关爱自己心脏的人"（长期目标）

**针对顺应学习的处方对话**：
> "您吸了30年烟，'我是个烟民'这个想法已经变成了您对自己的一部分描述。
> 但您刚确诊的那一刻，有没有一个声音在说：
> '我不想这样下去了'？那个声音是真的您。
> 那个烟民的身份，其实只是一个30年的习惯——习惯是可以改变的，身份也是。"
