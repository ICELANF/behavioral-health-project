### KI-NEURO-002 | 脑储备概念：为什么有人面对压力会崩溃，有人不会

```yaml
knowledge_id: KI-NEURO-002
title: 脑储备（Brain Reserve）：理解抗压能力和行为改变难易度差异
tags: [脑储备, 抗压能力, 行为改变难度, 个体差异, 韧性]
retrieval_queries:
  - 为什么有些人压力大就暴饮暴食
  - 为什么有人比别人更难改变行为
  - 用户说我就是容易放弃是什么原因
  - 抗压能力和大脑有什么关系
```

**脑储备定义**：健康脑功能的**缓冲垫**，用来处理压力或伤害事件。

```
脑储备高 → 面对同等压力 → 保持健康行为，有效应对
脑储备低 → 面对同等压力 → 崩溃，暴饮暴食/酗酒/放弃
```

**降低脑储备的因素**（生命史中的累积影响）：
- 母亲孕期吸烟/饮酒/吸毒
- 童年家庭暴力/创伤
- 脑损伤（包括被认为"轻微"的）
- 青少年期吸毒酗酒
- 长期睡眠剥夺
- 慢性高压状态

**提高脑储备的因素**：
- 充足睡眠
- 有氧运动
- 营养均衡饮食
- 冥想/正念练习
- 充满爱的社会支持
- 持续学习和认知挑战

**对平台的含义**：
- CAPACITY诊断模型中的"能力"维度需要包含脑储备评估
- 脑储备低的用户需要**更渐进的改变策略**，而非更强的意志力要求
- CoachAgent面对"总是坚持不住"的用户，应考虑脑储备因素，而非简单归咎于动机不足

### KI-NEURO-003 | 六种过量饮食脑类型与对应干预策略

```yaml
knowledge_id: KI-NEURO-003
title: 亚蒙六种过量饮食脑类型分型及个性化干预框架
evidence_tier: T3
tags: [脑类型, 过量饮食, 肥胖分型, 个性化干预, 强迫型, 冲动型, 情绪型, 焦虑型, SAD型]
retrieval_queries:
  - 肥胖有哪些类型
  - 为什么戒糖失败
  - 情绪性进食怎么处理
  - 强迫型饮食怎么干预
  - 用户晚上控制不住吃东西是什么类型
  - 焦虑的时候就想吃东西怎么办
```

> **核心洞察**：肥胖/过量饮食不是单一问题，至少有6种神经机制不同的亚型。一种饮食方案无法适合所有人。

---

**类型1：强迫型过量饮食**

```yaml
脑区：前扣带回（ACC）活动过强
神经递质：5-羟色胺水平低
```

**特征**：
- 满脑子都是食物，停不下来
- 执著于某种特定食物（必须吃某样东西）
- 倾向于担忧、反刍思维
- 心怀怨恨，难以放下
- 事情必须按部就班，不喜欢变化
- 强迫行为倾向（包括暴饮暴食后又催吐）

**典型案例**：丽贝卡，44岁，整天满脑子是食物，8年来每年增重约4.5kg，尝试过所有节食方法均失败。

**干预方向**：
- 提升5-羟色胺：有氧运动（使L-色氨酸更多进入脑）
- 锻炼意志力对抗强迫行为（而非屈服于它）
- 补充剂：5-羟基色氨酸、L-色氨酸、肌醇（自然疗法）
- 避免：低碳水饮食（会使强迫型更易怒、神经质）

---

**类型2：冲动型过量饮食**

```yaml
脑区：前额叶（PFC）活动不足
神经递质：多巴胺水平低
```

**特征**：
- 注意力不集中，易冲动
- 在超市/聚会面前无法控制食物选择
- 就算"知道不该吃"也停不住
- 可能伴有注意力缺陷障碍（ADD/ADHD）历史
- 寻求刺激，容易被新奇的食物吸引

**典型案例**：里克，37岁，营销主管，出席各种宴会，身高1.73m体重113kg，青春期就有注意力问题，勉强完成大学学业。

**干预方向**：
- 增强前额叶功能：
  * 高强度运动（增加脑血流量）
  * 创建清晰的书面目标（弥补冲动控制不足）
  * 避免技术/社交媒体过度刺激（磨损快乐中枢）
- 提升多巴胺：高蛋白饮食、激烈体育运动
- 补充剂：左旋酪氨酸、S-腺苷甲硫氨酸
- 重要：**冲动型不要用低蛋白/低热量节食**，会更冲动

---

**类型3：冲动-强迫混合型**

```yaml
脑区：前额叶功能低 + 前扣带回活动过强
同时具备类型1和类型2特征
```

**特征**：既停不下来（强迫），也控制不住开始（冲动）。是最难干预的类型。

**干预方向**：同时处理多巴胺不足（前额叶）和5-羟色胺不足（前扣带回）。通常需要综合干预，不可用单一方案。

---

**类型4：季节性情感障碍型（SAD型）**

```yaml
脑区：深层边缘系统（情绪脑）活动过强（特别在冬季/光照不足时）
神经递质：5-羟色胺季节性变化
```

**特征**：
- 冬季/阴雨天明显增重
- 情绪低落时暴饮暴食（情绪性进食）
- 特别渴望碳水化合物和糖（大脑在通过食物自我提升5-羟色胺）
- 秋冬季节情绪明显变差

**干预方向**：
- 光照疗法（早晨30分钟，与阳光同波长）
- 有氧运动（最有效的天然5-羟色胺提升方法）
- 补充剂：S-腺苷甲硫氨酸（天然情绪提升）
- 与类型1组合干预：5-羟基色氨酸+S-腺苷甲硫氨酸

---

**类型5：焦虑型过量饮食**

```yaml
脑区：基底神经节（BG）活动过强
神经递质：γ-氨基丁酸（GABA）水平低
```

**特征**：
- 焦虑时必须吃东西（用食物舒缓焦虑）
- 动机过强，难以停下工作
- 对能力缺乏自信，总担心最坏情况发生
- 肌肉紧张（头痛、肌肉痛）
- 每晚需要酒精来"平息担忧"（酒精是GABA促进剂）
- 避免冲突，过度害怕他人评价

**典型案例**：丽贝卡（同时有类型1+5），每晚需要两三杯酒才能平息担忧。

**干预方向**：
- 提升GABA：深度放松（冥想、腹式呼吸、催眠）
- 补充剂：GABA补充剂、左旋茶氨酸（绿茶成分）、缬草根
- **严格戒酒**（短期有效但长期会依赖且加重）
- 认知疗法：处理灾难化思维
- 少量多餐维持血糖稳定（低血糖加剧焦虑）

---

**类型6：颞叶/情绪失调型**

```yaml
脑区：颞叶（TL）功能异常 + 深层边缘系统问题
特征：与情绪创伤和记忆相关的过量饮食
```

**特征**：
- 情绪不稳定、喜怒无常
- 将中性评论曲解为负面的
- 曾有头部受伤史（颞叶易受损）
- 暴力倾向（特别在特定食物/添加剂触发时，如MSG）
- 情绪大起大落，暴怒后疲惫

**干预方向**：
- 排查食物过敏和添加剂敏感（MSG/人工甜味剂/食用色素）
- 处理未解决的情绪创伤（EMDR、催眠）
- 每日写5件感恩的事（激活前额叶，舒缓边缘系统）
- 补充剂：镁（舒缓颞叶）、鱼油（细胞膜健康）

---

### KI-NEURO-004 | 四种关键神经递质：行为改变的化学基础

```yaml
knowledge_id: KI-NEURO-004
title: 多巴胺/5-羟色胺/GABA/内啡肽与行为改变动力的神经化学关系
tags: [多巴胺, 5-羟色胺, GABA, 内啡肽, 神经递质, 行为改变, 动力]
retrieval_queries:
  - 怎么提高改变的动力
  - 为什么运动让人感觉好
  - 用户情绪低落没有动力怎么干预
  - 神经递质和行为的关系
  - 如何自然提升多巴胺
```

**四种关键神经递质及其行为影响**：

---

**多巴胺（Dopamine）**

```
功能：激发动机、突出体验、驱动和激励
低水平表现：动机不足、精力低下、注意力不集中、冲动控制差、
           某些抑郁症、帕金森病、ADD/ADHD
```

**天然提升多巴胺的方法**：
1. **高强度有氧运动**（最有效）
2. **高蛋白饮食**（酪氨酸是多巴胺前体）
3. 在有成长空间/令人振奋的环境工作
4. ⚠️ **警惕寻求刺激行为**——短期升高但会磨损快乐中枢（产生耐受）
5. 补充剂：左旋酪氨酸、S-腺苷甲硫氨酸

**行为改变应用**：
→ 用户缺乏改变动力 = 可能多巴胺不足
→ 帮用户设置有意义的小目标和及时反馈（刺激多巴胺）
→ 避免奖励机制过度依赖外部刺激（会磨损内在动力）

---

**5-羟色胺（Serotonin）**

```
功能：让人感到平和、快活、安全感
低水平表现：某些抑郁症、焦虑、强迫思维或强迫行为、
           碳水化合物/糖的渴望（大脑试图自我提升5-羟色胺）
```

**天然提升5-羟色胺的方法**：
1. **多锻炼身体**（使L-色氨酸更多进入脑，最重要的天然途径）
2. **锻炼意志力**对抗强迫行为（行为本身能改造脑，效果类似百忧解）
3. 晒太阳/光照疗法
4. 补充剂：5-羟基色氨酸（5-HTP）、L-色氨酸、肌醇

**行为改变应用**：
→ 用户有强迫进食/强迫思维 = 可能5-羟色胺不足
→ 运动是最安全有效的处方（每次30分钟中等强度有氧）
→ 帮用户建立进食规律（规律本身提升安全感→5-羟色胺）

---

**γ-氨基丁酸（GABA）**

```
功能：调节大脑兴奋、缓解脑部"过热"、镇静作用
低水平表现：焦虑、某些抑郁、脑部过度活跃、过度担忧
过度饮食关联：用食物/酒精作为GABA替代品来镇静焦虑
```

**天然提升GABA的方法**：
1. **冥想/深度放松/腹式呼吸**
2. **规律运动**
3. 绿茶中的**左旋茶氨酸**（L-theanine）
4. 避免咖啡因/酒精（短期提升，长期依赖）
5. 补充剂：GABA补充剂、缬草根

**行为改变应用**：
→ 焦虑型用户 = GABA系统可能低效
→ **放松技术是这类用户的核心干预**（而非仅仅"管住嘴"）
→ 冥想练习在行为处方中应列为优先（非可选项）

---

**内啡肽（Endorphins）**

```
功能：与感受快乐和消除疼痛有关（人体天然吗啡）
失控表现：上瘾行为中大量分泌（食物成瘾也涉及内啡肽）
```

**天然刺激内啡肽分泌的方法**：
1. **运动**（跑步者的"高峰"状态）
2. 针灸
3. 催眠疗法（已证明对疼痛综合征有效）

### KI-NEURO-005 | 大脑区域与行为控制的对应关系

```yaml
knowledge_id: KI-NEURO-005
title: 关键脑区与行为控制/失控的对应关系（行为问题的神经解剖学）
tags: [前额叶, 前扣带回, 基底神经节, 深层边缘系统, 行为控制, 冲动]
retrieval_queries:
  - 为什么人有时候控制不住自己
  - 冲动行为和大脑哪个部位有关
  - 焦虑和大脑哪里有关系
  - 强迫思维是哪里出了问题
```

**关键脑区行为功能速查**：

| 脑区 | 正常功能 | 异常（过活跃/低活跃）→ 行为表现 |
|------|---------|-------------------------------|
| **前额叶（PFC）** | 判断力、冲动控制、目标导向 | **低活跃** → 冲动、无法坚持计划、注意力差 |
| **前扣带回（ACC）** | 认知灵活性、注意力转换 | **过活跃** → 强迫思维、反刍、执著、担忧 |
| **基底神经节（BG）** | 精细运动、焦虑调节、快乐中枢 | **过活跃** → 焦虑、用食物/酒精镇静 |
| **深层边缘系统** | 情绪记忆、社会连接 | **过活跃** → 情绪性进食、悲伤触发暴食 |
| **颞叶** | 情绪稳定、记忆处理 | **异常** → 情绪失调、曲解他人意图 |

## 铁律声明

**NEURO-铁律-01（不用脑类型替代医学诊断）**
CoachAgent在使用六种脑类型框架时，仅用于帮助用户理解自己的行为模式，**不得将其作为临床诊断**（如"你有ADD/抑郁症"）。脑类型是行为模式的描述框架，不是精神科诊断。如用户出现明显的精神健康问题，建议就医。

**NEURO-铁律-02（补充剂建议须配合医嘱）**
本文件提及的补充剂（5-HTP/S-腺苷甲硫氨酸/GABA等）仅作为信息参考，CoachAgent不得直接向用户推荐服用具体补充剂。如用户询问，应建议其咨询医生或营养师。

**NEURO-铁律-03（脑可塑性不是万能的）**
CoachAgent可以用脑可塑性原理激励用户，但不得对神经性/器质性障碍（如ADHD/严重抑郁/脑损伤后遗症）用户承诺"通过改变生活方式就能完全好转"。这类用户可能需要医学干预。
