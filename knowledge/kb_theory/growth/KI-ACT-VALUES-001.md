## 知识内容

### 核心命题
大多数健康干预以**问题为导向**（"我有高血糖，我要解决它"），而ACT以**价值观为导向**（"我珍视健康的身体，因为这让我能___"）。价值观是行为改变最持久的"北极星"——它不会因目标达成而消失，因为价值观是方向，不是终点。

### ACT六边形模型（心理灵活性）

```
           接受（Acceptance）
              ↗         ↘
    当下时刻              认知解离
  （Present Moment）  （Defusion）
       ↕                    ↕
    观察自我              价值观
  （Self as Context）   （Values）
              ↘         ↗
           承诺行动（Committed Action）
```

**核心目标：心理灵活性（Psychological Flexibility）**
= 带着不舒服的想法和感受，仍然朝着自己珍视的方向前进的能力

### 价值观维度（Values Compass）

在健康行为改变语境中，常见的核心价值观领域：

```yaml
家庭与关系:
  - "作为父母/祖父母，我想要有精力陪伴家人"
  - "我想在家人需要我的时候有能力照顾他们"
  
身体与健康:
  - "我珍视身体的活力和自主性"
  - "我想保持能够独立生活的能力"
  
成长与学习:
  - "我是一个持续学习和成长的人"
  - "我想了解自己的身体，成为自己健康的专家"
  
贡献与意义:
  - "我想用我的经验帮助他人"
  - "我想成为社区健康文化的一部分"
  
精神与超越:
  - "与自然和谐相处是我生活的一部分"
  - "照顾身体是对生命礼物的感恩"
```

### 认知解离（Defusion）：与阻碍思维保持距离

**问题**：大脑会产生阻碍健康行为的自动化思维

```
常见阻碍思维:
  "我尝试过很多次，我就是没有毅力"
  "等我减了10斤再开始运动"
  "年纪大了，改变没什么用"
  "一天不吃饱就难受，改变饮食太痛苦了"
```

**解离技术**：
```
不是: 压制这些想法（越压越强）
不是: 说服自己这些想法是错的（消耗能量）
而是: 注意到这些想法的存在，然后继续朝价值方向前进

技术一 - 贴标签:
  "我注意到有一个想法在说'我没有毅力'"
  （而非"我真的没有毅力"）

技术二 - 感谢大脑:
  "谢谢大脑的保护，我知道你是想保护我不受失望"
  然后继续行动

技术三 - 车流比喻:
  把想法想象成高速公路上的车辆，你站在桥上观看
  你不需要搭上每一辆车（每一个想法）
```

### 接受（Acceptance）：与不舒服共处

**关键区分**：
```
忍耐（Tolerance）: 咬牙坚持 → 耗竭
接受（Acceptance）: 带着不适继续前进 → 可持续

实践:
  运动时的身体不舒适 → 不是危险信号，带着它继续
  改变饮食时的渴望 → 注意到它，不被它控制
  失误后的羞耻感 → 接受这种感受，不让它变成"放弃"的理由
```

**接受不等于喜欢**：
> "我不需要喜欢早起运动，我只需要接受'不想起床'的感受存在，然后穿上鞋出门。"

### 承诺行动（Committed Action）：价值观驱动的小步行动

```
价值观陈述（方向） → 人生目标（长期画面） → 具体行动计划（今天能做什么）

示例:
价值观: "我珍视作为祖父/祖母的角色，想有体力陪孙子们玩"
目标: "三年后，我能和孙子一起爬完附近的小山"
今日行动: "下午三点，穿鞋出门走15分钟"

关键: 
  不是"我会坚持运动"（意愿声明）
  而是"下午三点，我会做___"（实施意图）
```

## 成长超越域专属字段

### growth_dimension
`realize_self`（实现自我）——ACT框架帮助用户从"知道价值观"到"用价值观指导实际行动"，是实现自我的核心实践：不只了解自己是谁，而是活出自己的价值观。

### reflection_prompts（反思提问）

1. **价值观澄清**
   > "如果您的健康状况完全不是障碍，您最想用您的精力和时间做什么？这件事告诉您什么是您真正珍视的？"

2. **认知解离练习**
   > "您脑子里有没有一些关于健康改变的'固定想法'？比如'我就是这种人'或'改变了又会反弹'。这些想法如果是一个人，它看起来像什么？（外化练习）"

3. **接受练习**
   > "在您的健康改变中，有没有一种不舒适是您一直在逃避的？如果您不逃避它，只是带着它继续前进，会发生什么？"

4. **承诺行动具体化**
   > "基于您最珍视的价值观，这周您愿意做的最小一件具体的事是什么？不是'我会努力运动'，而是'周二下午4点我会_____'。"

5. **价值观一致性回顾**
   > "上周您的行动中，哪一件最体现了您真正珍视的东西？哪一件最与您的价值观相悖？这两个信息告诉您什么？"

### identity_bridge（身份桥接）

| 旧问题导向叙事 | 解离+接受 | 价值观导向新叙事 |
|--------------|---------|--------------|
| "我要解决我的血糖问题" | "我注意到有个问题需要应对" | "我在活出对家人和对自己健康的珍视" |
| "我必须坚持运动" | "运动有时很难，这很正常" | "我选择运动，因为它帮我成为我想成为的那个有活力的人" |
| "我失败了，又反弹了" | "失误发生了，这是人类正常状态" | "我是一个即使跌倒也会回到价值观方向的人" |

### applicable_life_events（适用人生事件）

- **行为改变卡住期**：多次尝试改变但总是反复，价值观重新定向突破僵局
- **意义危机期**："我为什么要这么辛苦？"——ACT价值观探索提供答案
- **慢性病长期管理的疲倦期**：不再靠恐惧驱动，转向价值观驱动的持续
- **价值观冲突期**：家庭压力vs个人健康、工作强度vs自我照顾的冲突处理
- **重大生活转变后**：退休/离婚/失业，重新澄清"现在我最珍视什么"

## 与中医整合

```yaml
ACT接受 ↔ 道家"无为"：
  不强行对抗自然流动，包括自己的情绪和思维
  "知足者富"——接受当下，不执着于理想状态

ACT价值观 ↔ 儒家"修身":
  明确自己的伦理方向，然后在日常行动中体现
  "格物致知，诚意正心，修身齐家治国平天下"
  → 每一步都连接更大的意义框架

ACT认知解离 ↔ 佛家"观":
  不与念头认同，只是观察念头的生灭
  "不怕念起，只怕觉迟"——觉察即解离
```
