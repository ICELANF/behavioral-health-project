# L2 成长超越知识 · 可能的自我（Possible Selves）理论与身份叙事工具

<!-- ki_id: KI-IDENT-POSSIBLE-001 | layer: L2 | domain: growth | scope: domain | evidence: T2 | ttm_stages: S4,S5,S6 | growth_dimension: discover_self | document_version: v1.0 | ingest_date: 2026-03-07 -->

> **来源**：Hazel Markus & Paula Nurius, "Possible Selves" (1986); Daphna Oyserman, Identity-Based Motivation (2007)
> **证据等级**：T2（可能的自我与行为动机的关联有大量实证研究支持，包括健康行为领域）
> **平台应用**：S5身份叙事工具，帮助用户从"我是一个有糖尿病的人"转向"我是一个重视健康的人"

---

## 1. 核心理论

### 1.1 什么是"可能的自我"

"可能的自我"是一个人对未来自己的具体想象，包括：

| 类型 | 定义 | 慢病语境示例 |
|------|------|------------|
| **希望的自我** | 我想成为的样子 | "一个精力充沛、能爬山的60岁人" |
| **预期的自我** | 我觉得自己可能变成的样子 | "一个把健康习惯当成日常的人" |
| **恐惧的自我** | 我害怕变成的样子 | "一个靠轮椅和胰岛素过日子的人" |

### 1.2 可能的自我如何驱动行为

```
希望的自我（拉力）
  "我想成为那种活到80岁还能自己买菜做饭的人"
  → 为日常行为提供方向和持续动力

恐惧的自我（推力）
  "我不想变成需要别人照顾的负担"
  → 在犹豫时提供紧迫感

两者必须同时存在才最有效：
  只有希望 → 容易空想不行动
  只有恐惧 → 容易焦虑和逃避
  希望 + 恐惧 → 行动的双引擎
```

### 1.3 关键发现：身份-行为一致性

> 当人们把某个行为看作"像我这样的人会做的事"时，行为坚持率提升300%。
> ——Oyserman, 2007

**平台启示**：
- 不说"你应该运动"（外部要求）
- 说"你现在是那种会主动选择楼梯的人了"（身份确认）

## 2. 身份叙事工具

### 2.1 "三个我"练习

**CoachAgent引导版**（对话中使用，每次聚焦一个）：

**过去的我**：
> "一年前的你，是什么样的？你怎么描述那个时候的自己？"

**现在的我**：
> "今天的你，和一年前相比，最大的不同是什么？不只是身体——是整个人。"

**未来的我**：
> "一年后，你希望自己是什么样的？用三个词描述一下。"

### 2.2 身份桥梁对话

当用户完成"三个我"后，CoachAgent搭建桥梁：

> "你说一年前你是'迷茫的/害怕的/不知道怎么办的'，
> 现在你说自己是'有方向的/平静的/知道该怎么做的'。
> 你有没有想过——从那个你走到这个你，是什么让你走过来的？"

*预期效果*：用户开始用"我选择了""我坚持了""我没有放弃"来回答——这些就是身份证据。

### 2.3 身份证据收集

**CoachAgent持续跟踪**——每当用户做出与"希望的自我"一致的行为时，标注为身份证据：

| 用户行为 | 身份证据话术 |
|---------|------------|
| 主动拒绝不健康饮食 | "你刚才做的选择——这就是你说的'重视健康的人'会做的事。" |
| 在困难时没有放弃 | "很多人在这种时候会放弃。你没有——这说明了什么？" |
| 帮助他人改变 | "你开始影响身边的人了。这是你一年前能想到的吗？" |
| 主动学习健康知识 | "你现在主动去了解这些——这不是任务，是你真的在乎。" |

## 3. 恐惧的自我：安全使用规则

### 3.1 何时使用

- **可以使用**：S4用户在犹豫是否要坚持时，轻触恐惧的自我作为推力
- **慎重使用**：S0-S2用户已经在恐惧中，不需要加码
- **禁止使用**：作为威胁或恐吓手段

### 3.2 安全话术

> "你之前提到过不想变成需要别人照顾的人。你现在做的每一件事，都在让那个可能性变小。"

**禁止话术**：
- ~~"如果你不坚持，以后就要坐轮椅了"~~ （恐吓）
- ~~"你不怕以后变成那样吗？"~~ （激活焦虑）

## 4. 成长超越必填四字段

```yaml
growth_dimension: discover_self
  # 可能的自我帮助用户发现自我——"我可以成为什么样的人"

  "从一年前的你到今天的你——这条路是你自己走出来的。你觉得是什么力量让你走到了这里？"

applicable_life_events:
  - 慢病管理满一年的回顾时刻
  - 即将退休/角色转换时
  - 孩子成家/自己成为祖父母时
  - 被他人问起"你是怎么做到的"时
```

## 5. 阶段适用性

| 阶段 | 使用方式 |
|------|---------|
| S0-S1 | **不使用**。用户尚未建立行为基础，谈"可能的自我"过早。 |
| S2-S3 | 轻触"希望的自我"作为远景——"你希望半年后的自己是什么样的？" |
| S4 | "三个我"练习入门——帮用户看见从过去到现在的变化 |
| **S5** | **完整使用**——三个我 + 身份桥梁 + 身份证据收集 |
| S6 | 帮助用户引导他人做"三个我"练习，成为导师 |

---

## 6. 禁止行为

- **禁止**替用户定义他们的"可能的自我"（"你应该成为一个健康达人"）
- **禁止**否定用户的恐惧的自我（"你不会变成那样的"——这会让用户觉得被否定）
- **禁止**在没有足够行为证据时说"你已经变了"——空洞的鼓励让人感到被敷衍
- **禁止**将"可能的自我"变成目标设定工具——它是身份探索，不是绩效管理
