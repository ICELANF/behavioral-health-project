## 知识内容

### 核心命题

人同时生活在三个自我之间：
- **现在的自我**（Now Self）：我现在是谁、我现在的状态
- **可能的自我**（Possible Selves）：我可能成为谁——包括希望成为的和恐惧成为的
- **应该的自我**（Ought Self）：他人认为我应该成为的样子

**可能的自我理论的核心发现**：
> **清晰、生动、可信的正向可能自我，是自我调节和行为改变最强大的内在驱动力之一。**

它不是"目标"（外部成就），而是**身份**（我会成为什么样的人）。

### 可能的自我的三种类型

**1. 希望的可能自我（Hoped-for Possible Selves）**
```yaml
定义: 我希望成为的样子——激励、吸引、令人兴奋
健康领域示例:
  - "一个每天精力充沛、能爬山的70岁老人"
  - "一个不需要吃任何药就能控制血糖的人"
  - "一个孙子辈的健康榜样"
  - "一个了解自己身体的人"
功能: 提供正向吸引力，激活接近动机（approach motivation）
```

**2. 恐惧的可能自我（Feared Possible Selves）**
```yaml
定义: 我害怕成为的样子——推动、警示、令人不安
健康领域示例:
  - "像父亲一样卧床不起、失去自主的晚年"
  - "截肢后依赖他人的自己"
  - "成为家人负担的自己"
  - "在该享受生活时却生病缠身的自己"
功能: 提供负向推力，激活回避动机（avoidance motivation）
重要注意: 单独使用恐惧可能自我→高焦虑→回避行为
         必须与希望可能自我配对使用→平衡的动机系统
```

**3. 预期的可能自我（Expected Possible Selves）**
```yaml
定义: 我认为自己很可能会成为的样子（不论希望与否）
健康领域示例:
  - "我很可能会慢慢变胖"（消极预期）
  - "我相信我能逐步改变"（积极预期）
功能: 反映自我效能感和结果预期
干预点: 修改消极的预期可能自我→提升自我效能感
```

### 可能的自我的力量机制

```
机制一：身份一致性驱动
  "作为一个热爱运动的人，不运动感觉不对" 
  → 行为与身份一致的内在驱动（不需要意志力）

机制二：自我差距激活
  现在的自我 ←→ 希望的可能自我
         差距 = 改变动力
  （差距太大→绝望；差距适中→激励；差距太小→无动机）

机制三：未来预演功能
  清晰可视化的未来自我
  → 大脑将其处理为"真实存在的未来记忆"
  → 增强改变的心理可信度
  → "我已经'看见'过那个健康的自己了"

机制四：叙事连贯性
  "我是一个正在成为健康版自己的人"
  → 将改变过程纳入自我叙事
  → 即使有失误，叙事仍然连贯
  → 防止"全或无"认知模式
```

### 可能的自我可视化练习

**标准版（20分钟深度练习）**：

```
步骤1：放松引导（2分钟）
  闭眼，深呼吸，让身体放松

步骤2：时间旅行（5分钟）
  想象自己来到5年后的某一天。
  提示：
  "您在哪里？早晨醒来，您感觉怎样？"
  "您的身体感觉怎样？"
  "这一天您会做什么？"
  "您身边有谁？他们如何与您互动？"

步骤3：细节建构（8分钟）
  "这个5年后的您，看起来怎样？"
  "他/她的日常生活是什么样的？"
  "他/她对自己的健康是什么态度？"
  "他/她在做的最有意义的事是什么？"

步骤4：桥梁建构（5分钟）
  "今天的您，和那个未来的您之间，有什么连接？"
  "那个未来的您，对今天的您有什么想说的？"
  "您今天能做的最小一件事，是朝那个未来靠近的？"
```

**日常迷你版（2分钟）**：
> 在开始某个健康行为前，花30秒想象那个"5年后的你"在做同样的事。
> 然后以"那个人"的身份行动。

### 可能的自我与身份标签

```yaml
旧的标签设置（医学系统）:
  "您是一个糖尿病患者"
  "您是一个高血压患者"
  → 将疾病固化为核心身份→消极可能自我主导

行健平台的身份重设:
  "您是一个正在逆转代谢状态的人"
  "您是一个了解自己身体的人"
  "您是一个实践健康生活方式的人"
  → 将可能的自我嵌入当下叙事→接近动机激活

渐进式身份陈述序列（与TTM对应）:
  S2: "我是一个开始考虑改变的人"
  S3: "我是一个正在准备改变的人"
  S4: "我是一个正在实践健康生活的人"
  S5: "我是一个已经建立健康习惯的人"
  S6: "我是一个热爱并分享健康生活的人"
```

### 恐惧可能自我的安全处理

```
恐惧可能自我是强大的动力，但使用不当会造成焦虑和回避：

错误使用:
  "如果你不改变，你会截肢/失明/死亡"
  → 引发防御性反应、情绪性回避
  → 短期内可能有效，长期破坏自我效能

正确使用（恐惧-希望配对）:
  步骤1: 允许接触恐惧可能自我（简短，不超度）
  步骤2: 立即连接希望可能自我
  步骤3: 将行动定位为"从恐惧走向希望"的桥梁

话术示例:
  "是的，如果不改变，可能会___（恐惧版本）"
  "AND——如果您改变，您会成为___（希望版本）"
  "今天的___行动，让您朝哪个方向走？"
```

## 成长超越域专属字段

### growth_dimension
`discover_self, realize_self`（发现自我+实现自我）——可能的自我框架既帮助用户探索"我可以成为谁"（发现），又将这个探索落实为"我正在成为那个人"的具体行动（实现）。

### reflection_prompts（反思提问）

1. **希望的可能自我探索**
   > "闭上眼睛，想象五年后健康状态完全实现的您。那个人的一天是怎样开始的？他/她的身体感觉怎样？最让您感到兴奋的那个画面是什么？"

2. **恐惧的可能自我面对**
   > "如果您完全不做任何改变，五年后的您可能是什么样子？允许自己真实地看看那个画面——不是为了恐惧，而是为了了解您真正想要避免什么。"

3. **身份连接**
   > "那个五年后健康的您，现在正在做一件您今天还没做到的事。是什么？他/她是怎么开始做到的？"

4. **当下行动的身份锚定**
   > "当您做出一个健康选择时（比如选择走路而不是打车），哪个'您'做了这个决定——现在的您，还是那个希望中的未来的您？"

5. **叙事完整性**
   > "您正在经历的健康改变，如果写成一本书，这本书叫什么名字？您现在在第几章？下一章会写什么？"

### identity_bridge（身份桥接）

| 现在的身份叙事 | 可能的自我桥接 | 正在成为的身份叙事 |
|--------------|---------|--------------|
| "我是一个不爱运动的人" | "五年后那个我喜欢运动——他是怎么开始的？" | "我是一个正在发现适合自己运动方式的人" |
| "我是一个控制不了饮食的人" | "五年后那个我与食物有什么关系？" | "我是一个正在学习倾听身体信号的人" |
| "我是一个生病的人" | "五年后那个管理好健康的我，如何看待这段经历？" | "我是一个用智慧和行动改写自己健康故事的人" |

### applicable_life_events（适用人生事件）

- **慢性病确诊初期**：确诊打碎旧身份，可能的自我框架帮助重建新的健康身份叙事
- **中年危机/人生转折**：40-55岁重新审视"我还能成为什么样的人"的窗口期
- **身份真空期**：退休/子女离家/职业转换后，需要重建自我定义
- **行为改变启动困难**："我就是这种人"的固化信念阻碍启动，可能的自我打破固化
- **减重/疾病逆转成功后**：新的身体需要新的身份叙事，防止"回到旧的自我"的反弹

## 中国文化适配

```yaml
"可能的自我"在中国传统文化中的根基:

儒家"志于道": 
  立志——对未来可能自我的承诺
  "三十而立"——在某个年龄明确自己的存在方向

道家"真人":
  不被外在角色定义，回归本真自我
  可能的自我探索是找回"真我"的过程

中医体质与可能的自我:
  体质可以通过调养改变（非固定论）
  "阳虚体质的人可以通过___成为阳气充沛的人"
  → 体质改变 = 自我转化的具体化路径

传统蒙养文化:
  "近朱者赤，近墨者黑"——环境塑造可能的自我
  家族中的"正向可能自我榜样"：长辈的健康老年形象
```
