## KI 知识条目（Knowledge Items）

### KI-BH-001 | 行为与健康的关系：行为决定健康的60%

```yaml
knowledge_id: KI-BH-001
title: 生活方式与行为模式决定健康的60%
life_dimension: 综合行为维度
applicable_stages: [S0, S1, S2, S3]
evidence_tier: T4
tags: [行为健康, 行为决定论, 慢性病, 生活方式]
retrieval_queries:
  - 行为和健康的关系是什么
  - 生活方式对健康有多大影响
  - 为什么行为习惯决定慢性病
```

**核心知识**

生活方式和行为模式决定一个人健康的60%。行为作为"临门一脚"或最后一步，是生活方式性疾病是否发生与转归的枢纽环节，居于承前启后、左右未来的关键地位。

**操作锚点**（傻瓜相机原则）

> CoachAgent表达模板：
> "您知道吗？您的生活习惯和行为方式，决定了您健康状态的60%。药物只能控制症状，而改变您每天的行为才能从根本上影响疾病的走向。"

**行为两分法**（用于S0-S1阶段意识唤醒）

| 类型 | 举例 | 对慢性病的影响 |
|------|------|--------------|
| 有益健康的行为 | 均衡饮食、定期锻炼、良好睡眠、戒烟限酒 | 预防疾病、维持健康 |
| 有害健康的行为 | 不良饮食、缺乏运动、吸烟酗酒、长期久坐、压力管理不善 | 导致肥胖、心血管病、糖尿病等 |

### KI-BH-002 | 慢性病的两大核心特征

```yaml
knowledge_id: KI-BH-002
title: 慢性病的窗口期漫长与危险因素可控两大特征
life_dimension: 疾病认知
applicable_stages: [S0, S1, S2]
evidence_tier: T4
tags: [慢性病, 窗口期, 危险因素, 可逆性]
retrieval_queries:
  - 慢性病为什么可以逆转
  - 慢性病的特点是什么
  - 什么时候干预慢性病最有效
```

**特征一：窗口期漫长**
- 从健康到疾病的转变需要多年甚至几十年
- 路径：低风险→高风险→早期变化→临床症状
- **干预机会**：在确诊前进行干预，可有效阻断、延缓甚至逆转疾病发展

**特征二：危险因素可控**
- 可改变的危险因素：高血压、高血脂、超重/肥胖、缺乏运动、蔬果摄入不足、吸烟
- 管理得当，甚至可以**逆转**慢性病的病理状态

**操作锚点**（意识唤醒用）

> 给S0-S1用户的觉醒脚本：
> "慢性病不是突然来的，是几十年积累的结果。好消息是——那些导致您生病的因素，大多数是可以改变的。现在开始干预，真的不晚。"

### KI-BH-003 | 知信行模式（KAP）的五大局限性

```yaml
knowledge_id: KI-BH-003
title: 健康教育知信行模式的局限及行为健康的必要性
life_dimension: 教育理论
applicable_stages: [S1, S2, S3]
evidence_tier: T4
tags: [KAP模式, 知信行, 健康教育局限, 行为改变理论]
retrieval_queries:
  - 为什么知道健康知识还是做不到
  - 健康教育效果差的原因
  - 知识不能直接转化为行动的原因
```

**KAP模式定义**：通过增加个体对健康知识的了解（知）→改变其态度（信）→最终影响其行为（行）。

**五大局限性**（平台设计的理论依据）

1. **知识与行为不一定直接相关**：即使知道某种行为对健康有益，仍可能因习惯、懒惰不采取行动
2. **忽视情感和动机的影响**：行为改变不仅取决于知识和态度，还受情感、社会支持、环境因素影响
3. **对社会文化因素重视不够**：不同文化背景对知识理解、态度形成和行为实施存在差异
4. **行为改变可持续性不足**：短期因知识增加而改变行为，长期维持仍是挑战
5. **个人责任的过度强调**：忽视了社会、经济和环境因素，忽略了集体和制度层面的"土壤"作用

**平台意义**：这五大局限性是行健平台超越单纯知识传授、构建评估→处方→陪练系统的理论依据。

### KI-BH-004 | 行为健康的定义与核心目标

```yaml
knowledge_id: KI-BH-004
title: 行为健康的整合性定义与四大具体目标
life_dimension: 平台基础理论
applicable_stages: [S0, S1, S2, S3, S4, S5, S6]
evidence_tier: T4
tags: [行为健康定义, 平台理念, 主动健康, 逆转慢病]
retrieval_queries:
  - 行为健康是什么意思
  - 行健平台的理念是什么
  - 主动健康的含义
```

**行为健康定义**

从行为养成到习惯成自然的个性化、动态性、复杂性、系统性和社会交互性特点出发，研究各类行为与身心健康之间的交互作用，通过养成有益于健康的行为、矫正不利于身心健康的行为，以实现整体健康状态的改善。

**平台理念（四字诀）**：健康与成长 · 主动健康 · 行为养成 · 逆转慢病

**四大具体目标**

| 目标 | 说明 |
|------|------|
| 预防疾病 | 通过健康行为培养，降低慢性病发病率 |
| 促进康复 | 帮助已患病个体通过行为改变加速康复 |
| 提高生活质量 | 通过全面行为管理提升整体健康水平 |
| 延长健康寿命 | 延缓衰老过程，保持长期身心健康 |

### KI-BH-005 | 慢性病防治的系统性要素

```yaml
knowledge_id: KI-BH-005
title: 慢性病防治需要系统而长期过程的六个关键要素
life_dimension: 综合管理
applicable_stages: [S2, S3, S4, S5]
evidence_tier: T4
tags: [慢性病管理, 个性化, 持续支持, 行为改变]
retrieval_queries:
  - 慢性病管理需要做什么
  - 怎么长期维持健康行为
  - 行为改变为什么需要系统支持
```

**六个关键要素**

1. **个性化健康管理**：每人生活方式、基因、环境、心理状态各异，需个性化方案
2. **持续的教育与支持**：专业指导帮助个体理解疾病本质，掌握具体实践方法
3. **多方协作与支持**：家庭（情感支持）+ 社区（支持性环境）+ 医疗系统（连续监测指导）
4. **循序渐进的行为改变**：逐步设定小目标，正向反馈和持续鼓励维持动力
5. **环境的积极影响**：创建有利于健康行为的环境（健康饮食选择、便利运动设施、无烟环境）
6. **长效动力机制**：行为自我监控 + 激励措施 + 社会支持网络建设

**平台映射**：以上六要素对应平台的 BAPS评估→处方→CoachPushQueue→陪练→社群系统 全链路设计。

### KI-BH-007 | 从业人员需要完成的三个转换

```yaml
knowledge_id: KI-BH-007
title: 行为健康实践对从业者要求的三大转换
life_dimension: 专业能力发展
applicable_stages: [S4, S5, S6]
evidence_tier: T4
tags: [健康教练, 从业者培训, 三转换, 能力要求]
retrieval_queries:
  - 健康教练需要具备什么能力
  - 医生如何转型做行为健康
  - 行为健康从业者的核心技能
```

**三大转换**

| 转换维度 | 从 | 到 |
|---------|----|----|
| 理念转换 | 重视治疗 | 治防并重；以疾病为中心→以健康为中心 |
| 知识结构转换 | 以治疗疾病为主的专科知识 | 影响健康的主要风险因素防范相关的生活方式、行为模式知识 |
| 能力结构转换 | 单一疾病治疗技能 | 指导公众获取与维持健康生活的技能 |

**五步技能路径**（行为健康从业者核心技能）
1. 点对点的基本沟通技能
2. 点对面表达传播技能
3. 群体活动的主持技能
4. 面向个体特定行为的指导和教练技能
5. 个体成长的导师素养

**CoachAgent使用场景**：Coach评估用户时，自身也需要具备这五步技能逻辑；平台Coach培训模块设计依据。

## IT 干预模板（Intervention Templates）

### IT-BH-001 | 行为唤醒处方（S0→S1阶段）

```yaml
template_id: IT-BH-001
name: 行为意识唤醒处方
applicable_stages: [S0, S1]
intervention_type: 认知唤醒
duration: 1-2周
matching_dimensions:
  stage: [S0, S1]
  psychological_level: any
  bpt6_type: any
  spi_min: 0
  domain: behavioral_health
```

**Phase 1（第1-3天）：事实镜子**
- 推送一个与用户当前状态相关的健康数据（如"您这个年龄段的人，X%已经出现了Y风险"）
- 不做评判，只提供镜像
- 傻瓜操作：用户只需点击"了解"或"跳过"

**Phase 2（第4-7天）：行为画像**
- 引导用户完成3分钟行为快照问卷（3题）
- 呈现个性化行为画像（非评分，非排名）
- 结尾提问：「如果继续保持现在的方式，您觉得3年后会有什么变化？」

**Phase 3（第8-14天）：激活动机**
- 呈现1个与用户相似的成功改变案例（匿名）
- 提供一个"我想了解如何改变"的按钮（非强制）
- 进入S1→S2过渡流程

**复发/退出处理**：连续3天无响应 → 暂停，30天后重新触达。不追问。
