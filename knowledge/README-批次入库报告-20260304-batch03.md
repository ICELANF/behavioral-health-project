1. **一线管理的科学背书**：KAS 1明确将生活方式干预定位为T2D一线管理，这是"平台核心主张"的T1级循证依据。当用户、医生或投资人质疑"生活方式能逆转糖尿病？"时，直接引用。

2. **健康教练角色背书**：HWC被明确列为核心干预资源，且给出了精确剂量（≥12次/6个月产生有意义改变）。直接支撑平台Coach角色的定价和服务设计。

3. **6大支柱 = 平台行为处方6维度**：营养/运动/压力/睡眠/社会连接/戒毒物，与平台行为处方维度完全吻合，可作为结构设计依据。

4. **关键数字库**（CoachAgent可直接引用）：

   - 仅24.1%的美国T2D患者达到推荐运动量
   - 每增加10g/天纤维→T2D风险降9%
   - ≥12次健康教练会话/6个月：39% vs 20%（标准护理）达到目标血糖

**铁律提示**：
- 药物减量须由处方医生决定，平台建议用户"主动与医生沟通"，而非自行操作
- WFPB饮食需中国化适配（糙米/豆制品/杂粮替代）
- "12次/6个月"的教练剂量概念应体现在平台服务设计中
