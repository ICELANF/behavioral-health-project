## KI 知识条目（Knowledge Items）

### KI-GUT-001 | 肠道菌群与代谢健康的核心关联

```yaml
knowledge_id: KI-GUT-001
title: 肠道菌群失调与肥胖、糖尿病、代谢综合征的关系
life_dimension: 代谢健康
applicable_stages: [S1, S2, S3, S4]
evidence_tier: T3
tags: [肠道菌群, 代谢健康, 短链脂肪酸, SCFA, 胰岛素敏感性, 肥胖, 糖尿病]
retrieval_queries:
  - 肠道细菌和糖尿病有什么关系
  - 为什么肠道不好会影响血糖
  - 短链脂肪酸是什么
  - 肠道菌群失调的危害
```

**核心知识**

肠道微生物群（肠道菌群）在消化、营养吸收和能量调节等代谢功能中起关键作用。**菌群失调**（微生物群失衡）与以下疾病密切相关：
- 肥胖
- 2型糖尿病
- 代谢综合征

**关键机制：短链脂肪酸（SCFA）**

| 来源 | 产生过程 | 健康效应 |
|------|---------|---------|
| 膳食纤维 | 有益肠道细菌发酵纤维 | 产生SCFA（乙酸、丙酸、丁酸） |
| SCFA | 作用于胰岛素受体和脂肪细胞 | 改善胰岛素敏感性、调节脂肪储存 |

**操作锚点**（傻瓜相机原则）

> CoachAgent对话模板（S2-S3用户）：
> "您的肠道里住着数以亿计的细菌，它们帮您'消化'吃进去的食物。当您多吃蔬菜和粗粮时，这些细菌会产生一种叫'短链脂肪酸'的物质——这就像给身体的'血糖调节器'加了润滑油。"

**行为处方映射**：
- 每日膳食纤维目标：25-30g（参照T1级营养指南）
- 傻瓜执行：每餐半盘蔬菜 + 粗粮替代精米精面1/4

### KI-GUT-002 | 肠脑轴：肠道健康影响情绪与心理

```yaml
knowledge_id: KI-GUT-002
title: 肠脑轴——肠道菌群影响焦虑、抑郁和认知功能
life_dimension: 心理健康
applicable_stages: [S2, S3, S4]
evidence_tier: T3
tags: [肠脑轴, 心理健康, 焦虑, 抑郁, 神经递质, 情绪管理]
retrieval_queries:
  - 肠道和大脑有什么关系
  - 肠道不好会影响情绪吗
  - 为什么压力大会影响肠胃
  - 肠道菌群和抑郁的关系
```

**核心知识**

**肠脑轴**是指肠道微生物群与大脑之间的双向信息传递网络。研究表明：
- 肠道菌群影响大脑功能和情绪状态
- 菌群失调可能与**焦虑、抑郁**相关
- 神经退行性疾病（阿尔茨海默症、帕金森病）与肠道菌群变化存在关联
- 自闭症谱系障碍（ASD）与特定菌群组成相关

**双向影响示意**

```
压力/焦虑/抑郁  ←——→  肠道菌群失调
       ↑                      ↓
  大脑功能变化  ←——→  神经递质（血清素、多巴胺）产生减少
```

> 注：约**95%的血清素**（"快乐激素"）在肠道中产生。

**平台整合意义**

本条目与平台五层心理模型（生命基调→情绪基调→心态→情绪→行为）高度相关：肠道健康状态属于"生命基调"层，影响整个心理-行为链路的底层。

**操作锚点（傻瓜相机）**

> "您知道吗？肠道被称为'第二大脑'。当您总感到焦虑或情绪低落时，有时根源不只在心里，也可能在肠道里。改善饮食结构，帮助肠道菌群恢复平衡，是调节情绪的一条鲜为人知但很有效的路径。"

**安全提示**：如用户表现出持续情绪低落、焦虑症状，首先评估心理健康风险（按CrisisAgent流程），不可单纯以肠道健康替代心理干预。

### KI-GUT-003 | 饮食对肠道菌群的五类影响

```yaml
knowledge_id: KI-GUT-003
title: 不同饮食成分对肠道菌群的正向和负向影响
life_dimension: 饮食营养
applicable_stages: [S2, S3, S4, S5]
evidence_tier: T3
tags: [饮食, 膳食纤维, 发酵食品, 多酚, 高脂高糖饮食, 肠道菌群]
retrieval_queries:
  - 吃什么对肠道好
  - 什么食物改善肠道菌群
  - 高脂饮食对肠道的影响
  - 发酵食品有什么好处
```

**菌群友好食物 vs 菌群破坏食物**

| 食物类型 | 对菌群的影响 | 操作建议 |
|---------|------------|---------|
| **富含纤维的食物**（蔬菜、杂粮、豆类） | 支持有益细菌生长 ✅ | 每餐半盘蔬菜，主食1/3换粗粮 |
| **发酵食品**（酸奶、泡菜、纳豆、醋） | 直接补充有益菌 ✅ | 每日1份低糖原味酸奶 |
| **多酚类食物**（绿茶、蓝莓、黑巧克力、橄榄油） | 调节菌群组成 ✅ | 每日1杯绿茶/少量深色浆果 |
| **高脂高糖饮食**（快餐、甜食、含糖饮料） | 促进有害细菌生长 ❌ | 每周不超过1次 |
| **过量酒精** | 破坏肠道屏障完整性 ❌ | 严格限制（参照T1级指南） |

**TCM关联**（与中医体质对应）

| 中医体质 | 常见肠道问题 | 菌群角度解释 |
|---------|------------|------------|
| 痰湿质 | 大便粘腻、腹胀 | 产甲烷菌过多、丁酸菌减少 |
| 湿热质 | 大便臭秽、口苦 | 革兰阴性菌过度增殖 |
| 阳虚质 | 大便溏稀、怕冷 | 肠道蠕动相关菌群功能低下 |
| 气郁质 | 肠易激综合征倾向 | 肠脑轴紊乱、SCFA产生不足 |

### KI-GUT-004 | 生活方式四因素对肠道菌群的影响

```yaml
knowledge_id: KI-GUT-004
title: 压力、运动、睡眠、抗生素对肠道菌群的影响机制
life_dimension: 综合生活方式
applicable_stages: [S2, S3, S4, S5]
evidence_tier: T3
tags: [运动, 睡眠, 压力, 抗生素, 肠道菌群, 生活方式]
retrieval_queries:
  - 运动对肠道有什么好处
  - 睡眠不好影响肠道吗
  - 压力大肠道会怎样
  - 抗生素吃了对肠道的影响
```

**四大生活方式因素**

| 因素 | 对菌群的影响 | 可操作干预 |
|------|------------|----------|
| **压力过大** | 通过皮质醇破坏肠道屏障，改变菌群组成 | 正念冥想5分钟/天；腹式呼吸 |
| **规律运动** | 增加菌群多样性，提高短链脂肪酸产生 | 每天30分钟快走（步速>100步/分钟） |
| **睡眠质量** | 睡眠剥夺与菌群失调互为因果 | 固定入睡时间±30分钟，睡眠7-8小时 |
| **抗生素** | 严重破坏肠道菌群，影响持续数月至数年 | 非必要不使用；使用后补充益生菌 |

**行为链设计示例（运动→肠道）**

```
晚饭后 → 换运动鞋 → 出门快走30分钟 → 
          ↓（菌群获益）
  次日大便通畅 → 正向反馈 → 强化行为
```

### KI-GUT-005 | 益生菌与益生元的使用边界

```yaml
knowledge_id: KI-GUT-005
title: 益生菌和益生元的作用、适用人群与使用限制
life_dimension: 营养补充
applicable_stages: [S3, S4, S5]
evidence_tier: T3
tags: [益生菌, 益生元, 膳食补充剂, 临床证据, 适用人群]
retrieval_queries:
  - 益生菌有什么用
  - 什么时候应该吃益生菌
  - 益生元和益生菌的区别
  - 益生菌对糖尿病有帮助吗
```

**定义区分**

- **益生菌**：活的有益细菌（如乳酸菌、双歧杆菌），直接补充到肠道
- **益生元**：促进有益细菌生长的物质（如菊糖、低聚果糖、抗性淀粉），是益生菌的"食物"

**有效证据较充分的场景**（T2-T3级证据）
- 抗生素相关腹泻的预防和治疗
- 艰难梭菌感染后肠道恢复
- 部分肠易激综合征症状缓解

**证据仍存争议的场景**（需谨慎表达）
- 改善2型糖尿病血糖控制（需更多RCT）
- 治疗焦虑抑郁（机制研究多，临床证据弱）
- 减重效果（结果不一致）

**⚠️ 铁律级表达要求**

> CoachAgent在输出益生菌建议时，**必须**使用以下限定语：
> "目前研究提示……可能有帮助，但证据尚不充分，建议先从食物来源（酸奶、泡菜）入手，如需选择补充剂产品，建议咨询专业医生。"
> 
> **严禁**：直接推荐特定益生菌产品；向用户声称益生菌可以"治疗"某疾病。

### KI-GUT-006 | 衰老与肠道菌群的双向关系

```yaml
knowledge_id: KI-GUT-006
title: 老年人肠道菌群变化与健康衰老、认知保护的关联
life_dimension: 老年健康
applicable_stages: [S2, S3, S4]
evidence_tier: T3
tags: [衰老, 老年健康, 认知功能, 长寿, 菌群多样性]
retrieval_queries:
  - 老年人肠道菌群有什么特点
  - 为什么老人消化功能变差
  - 肠道健康和寿命有关系吗
  - 保护认知功能的方法
```

**核心发现**

- 随年龄增长，肠道菌群**多样性下降**是普遍规律
- 多样性降低与认知能力下降、虚弱综合征等老年问题相关
- 百岁老人研究显示：平衡的微生物组可能是长寿的保护因素

**平台目标人群关联**（本平台重点服务老年慢病人群）

| 老年问题 | 菌群角度 | 行为干预切入点 |
|---------|---------|--------------|
| 便秘 | 产气短杆菌和乳酸菌减少 | 增加膳食纤维+补水 |
| 肌少症 | 肠道蛋白质利用效率下降 | 足量优质蛋白+抗阻运动 |
| 认知下降 | 肠脑轴炎症信号增加 | 地中海饮食模式+睡眠管理 |
| 免疫力低下 | 肠道相关淋巴组织功能减弱 | 发酵食品+避免不必要抗生素 |

**傻瓜相机操作（老年人版）**

> "肠道越'年轻'，人就越长寿。让肠道保持年轻的方法其实很简单：每天多走走路、吃点酸奶、多吃菜少吃肉——这些老人们都做得到。"

## IT 干预模板（Intervention Templates）

### IT-GUT-001 | 肠道菌群重建21天处方

```yaml
template_id: IT-GUT-001
name: 肠道菌群重建行为处方（21天）
applicable_stages: [S3, S4]
intervention_type: 饮食行为重组
duration: 21天（习惯形成周期）
matching_dimensions:
  stage: [S3, S4]
  psychological_level: any
  bpt6_type: [action_oriented, knowledge_oriented]
  spi_min: 40
  domain: gut_health, metabolic_health
```

**三周递进方案**

**第1周（基础期）：减法优先**
- 停止1项伤害肠道的习惯（选1：戒含糖饮料 / 减少精加工食品 / 减少外卖）
- 傻瓜操作：用白开水替代饮料，冰箱里备一瓶水
- 打卡方式：每天截图1张喝水记录（平台内打卡）

**第2周（补充期）：加法开始**
- 每天增加1份发酵食品（低糖原味酸奶200ml 或 泡菜30g）
- 早餐增加1份粗粮（燕麦片 / 全麦面包 / 杂粮粥）
- 傻瓜操作：提前备好酸奶，放冰箱显眼处

**第3周（强化期）：行为链锚定**
- 建立"饭后走路"习惯：晚餐后15分钟出门步行20分钟
- 行为链：「放下碗筷 → 换鞋 → 出门」（3步无需思考）
- 反馈：关注大便状态改变（Bristol大便评分：3-4型为健康目标）

**效果感知指标**（用于增强动力）
- 便秘改善、腹胀减少
- 体重缓慢下降（预期1-2kg/3周）
- 精力和睡眠质量改善
