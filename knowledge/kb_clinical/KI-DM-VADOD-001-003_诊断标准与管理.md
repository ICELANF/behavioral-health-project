### 【知识层】诊断标准与血糖分类

#### T2D诊断标准（VA/DoD 2023）

| 状态 | 空腹血糖（FPG） | HbA1c | OGTT 2h血糖 |
|------|--------------|-------|------------|
| **正常** | < 5.5 mmol/L (100 mg/dL) | < 5.7% | — |
| **糖尿病前期** | 5.5-6.9 mmol/L (100-125 mg/dL) | 5.7-6.4% | 7.8-11.0 mmol/L |
| **糖尿病** | ≥ 7.0 mmol/L (126 mg/dL)，两次确认 | ≥ 6.5%（+FPG≥7.0确认）或≥7.0% | > 11.1 mmol/L |

**关键说明**：
- 空腹 = 至少8小时未摄入热量
- 无明确高血糖症状时，诊断需两次不同日检测确认
- HbA1c检测须使用标准实验室方法（不是床旁快速检测）

#### 糖尿病前期的临床意义

**为什么糖尿病前期比糖尿病更重要（平台视角）**：
- 全球约10亿人处于糖尿病前期，大多不知道
- 糖尿病前期→T2D：年转化率 5-10%，但**完全可逆**
- 生活方式干预可使糖尿病前期转化率降低 **58%**（DPP研究）
- 这是平台最高价值的用户群：S0-S1期干预收益最大

**高风险筛查人群**（任一条即应评估）：
- 超重/肥胖（BMI ≥ 24 kg/m²）
- 一级亲属有糖尿病
- 有多囊卵巢综合征史
- 有妊娠糖尿病史或生过≥4kg婴儿
- 久坐生活方式
- 高血压/血脂异常
- ≥ 45岁

### 【知识层】HbA1c目标：个体化而非一刀切

#### 核心原则

VA/DoD 2023明确：**不应为所有T2D患者设定统一HbA1c目标**。

最新循证建议（2023更新版）：
- **大多数T2D患者**：HbA1c目标范围 **7.0-8.5%**（若能安全达到）
- 个体化评估维度：
  1. 患者特征（年龄、预期寿命、并发症、共病）
  2. 风险收益比（强化控糖的微血管获益 vs 低血糖风险）
  3. 患者价值观和偏好（通过共同决策确定）

#### HbA1c与临床结局的关键数据

**微血管并发症（视网膜/肾脏/神经）**：
- 每降低1% HbA1c → 微血管并发症风险↓ 约37%
- 但HbA1c越低，继续降低的边际收益越小

**心血管结局（三大关键试验）**：
| 试验 | 结论 |
|------|------|
| UKPDS | 早期T2D强化控糖有微血管获益，无心血管获益 |
| ACCORD | HbA1c < 6% 组全因死亡率↑22%，提前终止 |
| ADVANCE | 心血管无获益；肾功能有长期保护 |
| VADT | 强化组10年后心血管事件稍少，但无生存获益 |

**平台重要结论**：
> 对于老年患者/有心血管疾病史/病程较长的T2D，**过于激进的降糖反而有害**。
> HbA1c 7-8.5%是大多数患者的"安全甜区"。

#### 特殊人群HbA1c参考范围

| 人群特征 | 建议HbA1c参考范围 | 原因 |
|---------|--------------|------|
| 新确诊/年轻/预期寿命长 | 6.5-7.5% | 早期控制可预防微血管病变 |
| 老年/多病共存/预期寿命短 | 7.5-8.5% | 避免低血糖风险 |
| 有严重低血糖发作史 | 7.5-8.5% | 低血糖对老年人威胁更大 |
| 已有微血管或大血管并发症 | 7.0-8.5% | 个体化，避免强化控糖的危害 |
| 孕妇（T2D） | < 6.5%（餐前），< 7.0%（餐后） | 胎儿保护 |

### 【应用层】Agent使用指南

**当用户问"我的血糖多少才算好？"**

```
CoachAgent话术（个体化框架）：
"这个问题没有一个对所有人都一样的答案。
我们平台会根据您的年龄、身体状况和您的目标，
帮您和您的医生一起商量一个适合您的目标范围。
一般来说，对大多数人，HbA1c控制在7-8.5%是安全有效的。
您现在的HbA1c是多少？我们来看看在哪个区间。"
```

**平台监测逻辑**：
- HbA1c > 8.5%：触发强化干预提醒（营养+运动+药物评估）
- HbA1c < 6.5% 且无药物：评估是否进入"缓解"状态
- HbA1c 变化 > 1% 在3个月内：触发MetabolicAgent深度分析
- 低血糖事件（< 3.9 mmol/L）：记录并评估是否需要调整药物（提示医疗咨询）

---

### 铁律

- ❌ **平台不得提供具体药物调整建议**（如"减少二甲双胍剂量"）——必须建议就医
- ❌ **禁止使用"您的血糖控制得很好"等简单评价**——应说"我们来看看这个数字意味着什么"
- ✅ **HbA1c下降时，明确肯定是用户行为改变的结果**（增强自我效能感）

---

### 叠加情境提示

- **与肾功能异常叠加**：HbA1c可能不准确（血红蛋白代谢改变），需结合空腹血糖和CGM
- **与贫血叠加**：HbA1c偏低假象，需配合空腹血糖诊断
- **与老年痴呆叠加**：放宽HbA1c目标，低血糖风险更危险

### 【知识层】DSMES：糖尿病自我管理教育与支持

**VA/DoD 强推荐（Strong for）**：所有T2D成人均应接受DSMES

#### DSMES八大核心主题

| 主题 | 内容 | 平台教育模块 |
|------|------|------------|
| 1. 糖尿病生理机制 | 什么是T2D，为什么会得 | 健康知识模块 |
| 2. 血糖监测 | 如何使用血糖仪/CGM，何时测 | 设备指导模块 |
| 3. 健康饮食 | 食物与血糖的关系，WFPB方案 | 营养处方模块 |
| 4. 积极活动 | 运动与血糖，FITT框架 | 运动处方模块 |
| 5. 规律服药 | 药物机制，不良反应，何时就医 | 药物管理模块（仅教育，不开药） |
| 6. 问题解决 | 应对血糖波动，低/高血糖处理 | 决策支持模块 |
| 7. 降低风险 | 预防并发症，足部/眼部/肾脏护理 | 预防教育模块 |
| 8. 健康应对 | 糖尿病心理压力，糖尿病痛苦 | 心理支持模块 |

#### 证据支持

DSMES对以下结局有明确改善证据：
- HbA1c 控制改善
- 血压管理改善
- 糖尿病知识提升
- 自我管理行为改善
- 患者报告的生活质量提升

**个性化教育效果更强**：
- 配合健康教练的DSMES项目：HbA1c改善更显著 + 更大体重减轻
- 在低收入人群中同样有效（经济障碍不是教育无效的理由）

### 【知识层】糖尿病痛苦（Diabetes Distress）

**VA/DoD发现**：目前证据不足以推荐特定筛查工具，但识别和关注仍重要

**糖尿病痛苦的定义**：
- 因患有糖尿病而产生的多种负面情绪（内疚、愤怒、悲伤、无助感）
- 与临床抑郁症不同，是疾病管理带来的特定情感负担

**临床表现信号**（CoachAgent识别提示）：
- "我厌倦了每天测血糖"
- "这辈子都要这样了，有什么意义"
- "我知道应该这样做，但就是做不到，我太差劲了"
- "我的家人不理解，没人支持我"

**CoachAgent干预原则**：
1. 不要立即跳到"解决方案"
2. 先反映并正常化情绪："面对糖尿病这么长时间，有这种感受是完全正常的"
3. 探索：是哪个方面最难？是自我管理的负担，还是担心并发症，还是家庭支持？
4. 评估抑郁风险：若有"活着没意思"等表述 → 触发CrisisAgent

---

### 【知识层】远程健康干预的有效性

**VA/DoD 弱推荐（Weak for）**：为T2D患者提供远程医疗方式改善结局

**证据摘要**：
- 远程监测+HCP互动：HbA1c平均下降1.3%（优于常规护理）
- 基于网络的血糖追踪系统：HbA1c有统计学显著改善
- 患者焦点组反馈：高度重视远程支持的便利性

**平台意义**：
- BehaviorOS的微信小程序模式直接对应这一循证基础
- 数字健康教练（CoachAgent）提供的即时支持，有强证据支持
- 关键是"HCP互动"——纯数据记录无效，需要CoachAgent的主动反馈

### 【应用层】Agent话术脚本

**情境：S2用户表达糖尿病痛苦**

```
CoachAgent话术（先接住情绪）：
"听到您说'每天测血糖真的烦死了'，
我完全理解——每天手指都要扎，看到数字还可能不好看，
这个负担是真实存在的，很多人都有这种感受，不是您太脆弱。
您现在最难熬的是哪一部分？是测血糖的过程，
还是看到数字后的那种感觉？"

教练行动指令：
1. 不要立即给解决方案
2. 先充分倾听+反映，至少2-3个来回
3. 评估情绪状态：是常规痛苦还是抑郁信号
4. 若确认是正常痛苦：引导重新锚定动机（您最初为什么开始管血糖？）
5. 引入"小胜利"设计：今天只要做一件事——是什么？
```

**情境：S3用户问"我用手机记录血糖有用吗？"**

```
CoachAgent话术（科学背书+个人价值）：
"有用！研究真的发现，把血糖数据和医生/教练分享、
得到反馈的人，血糖平均比自己管的人低了一截。
最关键的是：您记录的不只是数字，而是您的身体在告诉您——
哪顿饭之后血糖平稳，哪次运动之后降得好。
这就是您自己在做的小实验，没有人比您更了解自己的身体。"
```

### 【傻瓜相机检验】

> 问：一位50岁、确诊T2D 2年、每次血糖不好就自责的用户，今天教练能帮他做什么？

**今日可执行**：
1. 用手机记录今天三餐时间和大致内容（不需要称重，文字描述即可）
2. 饭后30分钟测一次血糖，记录数字
3. 在记录旁边写一句话：今天吃了什么感觉比较满意（不论血糖好坏）

目的：从"监控/评判"模式 → "观察/探索"模式

### 【知识层】血糖变异性：HbA1c之外的预警指标

**VA/DoD 弱推荐**：使用高血糖变异性作为低血糖、并发症和死亡率的预后指标

**血糖变异性的定义**：
- 血糖或HbA1c随时间的波动幅度（用变异系数CV或标准差SD表示）
- 与平均血糖水平相独立的风险因素

**高血糖变异性的临床后果**：
- 低血糖风险增加 50-300%
- 与全因死亡率、心血管事件独立相关
- 预示性比单一HbA1c值更敏感

**血糖变异性升高的常见原因**：
1. 胰岛素或磺脲类药物用量不稳定
2. 餐食不规律（漏餐/暴饮暴食）
3. 运动强度不稳定
4. 感染/压力等应激
5. 服药依从性差

---

### 【知识层】低血糖的识别与处理

**低血糖定义**：血糖 < 3.9 mmol/L (70 mg/dL)

**症状识别**：
- 轻度：出汗、心跳加速、手抖、饥饿感、焦虑
- 中度：头晕、注意力不集中、说话含糊
- 重度：意识模糊、抽搐、失去意识

**紧急处理（15-15法则）**：
1. 立即摄入 **15g** 快速吸收碳水（4-5块葡萄糖片 / 半杯果汁 / 3-4块饼干）
2. 等待 **15分钟**
3. 复测血糖
4. 若仍 < 3.9 → 重复上述步骤
5. 若意识不清 → 立即拨打急救电话 → 触发CrisisAgent

**平台代谢安全红线**（铁律）：
```
血糖 < 2.8 mmol/L → 立即触发CrisisAgent，停止所有行为干预指令
血糖 < 3.9 mmol/L → 禁止开始运动，要求先处理低血糖
血糖 > 16.7 mmol/L → 触发CrisisAgent，建议立即就医
```

---

### 【傻瓜相机检验】

> 问：一位用胰岛素的T2D用户，早上运动前头晕心跳快，今天能做什么？

**立即可执行**：
1. 停止准备运动的一切
2. 找家里最方便的甜食：3-4块饼干 / 半杯橙汁 / 几粒糖
3. 吃下去，坐着等15分钟
4. 用血糖仪测血糖，若超过4.5才考虑轻度活动
5. 今天把这件事记录下来，告诉教练/医生

✅ 通过检验——不需要任何专业知识，步骤清晰可立即执行

---

### 叠加情境提示

- **与老年人叠加**：低血糖症状可能不典型（可能只有意识混乱，无心跳加速）
- **与使用β阻断剂叠加**：低血糖的心跳加速信号被掩盖，更难察觉
- **与肾功能异常叠加**：肾脏清除慢，药物蓄积风险更大，更容易低血糖
