## 核心概念

### CGM vs 传统自我血糖监测（SMBG）的区别

```
传统SMBG（手指血糖）：
  → 单一时间点
  → 无法反映全天血糖波动
  → 无法发现低血糖发作时间和频率
  → 无法评估血糖变异性

CGM（动态血糖监测）：
  → 连续/频繁测量（每1-5分钟一次）
  → 完整记录全天血糖曲线
  → 可以发现隐性高血糖和低血糖
  → 评估血糖变异性（血糖波动幅度）
  → 与生活方式（饮食/运动/睡眠/压力）实时关联
  
→ 对生活方式干预的意义：
  用户可以立即看到"吃了这顿饭"或"运动了30分钟"
  对血糖的具体影响，形成强有力的行为反馈循环
```

## CGM两大类型

### KI-CGM-001 | 专业CGM系统（回顾性）

```yaml
knowledge_id: KI-CGM-001
title: 专业CGM系统（回顾性）：适应证与使用规范
tags: [CGM, 回顾性CGM, 专业CGM, 适应证, 临床应用, 盲测]
retrieval_queries:
  - 专业CGM和个人CGM有什么区别
  - 哪些患者适合用专业CGM
  - 回顾性CGM怎么用
```

**专业CGM（回顾性系统）的特点**：
- 患者和医生在监测期间**无法实时查看**数据
- 监测结束后下载数据，回顾分析血糖模式
- 一般需持续佩戴**3-5天**
- 操作简单，医院购买使用为主

**专业CGM适应证**（AACE推荐）：
- 未达标糖尿病患者，正在调整治疗方案
- 有夜间低血糖或黎明现象的1型糖尿病患者
- 无症状低血糖患者
- SMBG指导下血糖仍控制不佳者
- 血糖波动大、频发低血糖患者
- 孕妇（胰岛素强化治疗期间）

**中国指南适应证（CDS，2009）**：
- 1型糖尿病患者
- 胰岛素强化治疗的2型糖尿病
- SMBG指导下血糖控制仍不佳（血糖波动大、频发低血糖、无症状低血糖、黎明现象）
- 妊娠期糖尿病或糖尿病合并妊娠
- CGM自主意愿强烈的糖尿病患者

**特别说明**：
回顾性CGM可每日检测**94-390分钟**的隐性高血糖（对孕妇数据）；不能实时预警，监测期间医患均看不到数据。

### KI-CGM-002 | 个人CGM系统（实时）

```yaml
knowledge_id: KI-CGM-002
title: 个人CGM系统（实时）：特点、适应证与行为干预应用
tags: [实时CGM, 个人CGM, 血糖预警, 实时反馈, 生活方式监测, 行为改变]
retrieval_queries:
  - 实时CGM有什么好处
  - 哪些人需要用个人CGM
  - CGM怎么帮助改变生活方式
  - 血糖预警功能怎么用
  - CGM和行为处方怎么结合
```

**实时CGM的特点**：
- **实时提供血糖信息**
- 配备**血糖预警功能**（高血糖/低血糖报警）
- 患者可即时调整血糖
- 在调整治疗方案前，仍需自测血糖进一步确认
- 患者自费购买，规范使用以改善血糖控制

**实时CGM适应证**（AACE推荐）：
- HbA1c > 7%的1型糖尿病患者（无症状低血糖/频发低血糖）
- HbA1c < 7%的1型糖尿病（降低低血糖风险）
- 有妊娠计划或正处于妊娠期的患者
- 自我管理能力强的儿童/青少年（HbA1c < 7%）
- HbA1c ≥ 7.0%且有能力每天使用监测仪的青年1型患者
- 推荐2-4周试用期后决定是否正式使用的患者

**使用要求**：
- 需要患者有规律随访和医生指导
- 安装和初步培训约需60-90分钟
- 实时CGM使用期间需记录饮食、运动、用药日记
- 下载数据时，需将血糖数据与生活方式数据结合分析

## CGM关键临床证据

### KI-CGM-003 | CGM在1型/2型糖尿病中的循证依据

```yaml
knowledge_id: KI-CGM-003
title: CGM临床应用的核心循证证据（HbA1c改善、低血糖减少）
tags: [CGM证据, STAR研究, JDRF研究, HbA1c改善, 低血糖, 循证医学]
retrieval_queries:
  - CGM有没有临床证据
  - 用CGM真的能降HbA1c吗
  - CGM能减少低血糖发作吗
  - 有哪些CGM的临床研究
```

**主要循证证据**（表1整理）：

| 研究名称 | 人群 | 主要结局 | 主要结果 |
|---------|------|---------|---------|
| **STAR研究** | 12-72岁胰岛素泵治疗1型 | HbA1c下降，低血糖发生率 | CGM组HbA1c 7.77% vs SMBG组7.84%；探头使用时间>60%者血糖控制最佳 |
| **JDRF研究** | 15-24岁患者 | HbA1c下降值，低血糖发生率 | CGM组HbA1c较对照组低0.08%（15-24岁）或0.53%（≥25岁）；严重低血糖发生率相似 |
| **DirecNet** | 儿童患者 | HbA1c下降值 | 13周时，CGM患者HbA1c从7.1%降至6.8%（P<0.05）；低血糖发生率下降 |
| **妊娠研究** | 1型+2型妊娠妇女 | 隐性高血糖，新生儿结局 | 每日可发现94-390分钟隐性高血糖；CGM组孕期血糖控制和新生儿出生体重优于对照组 |

**CGM探头使用频率与HbA1c的关系**：
- 每周使用6-7天者：HbA1c水平明显改善
- 25岁以上患者探头使用频率最高
- → 结论：CGM效果与持续使用时间正相关

### KI-CGM-004 | CGM数据解读：血糖变异性与质量评估

```yaml
knowledge_id: KI-CGM-004
title: CGM数据解读：血糖变异性指标与临床意义
tags: [血糖变异性, CGM数据, 血糖曲线, 血糖波动, TIR目标范围时间, 数据解读]
retrieval_queries:
  - CGM数据怎么看
  - 血糖变异性是什么意思
  - 什么是目标范围内时间TIR
  - 血糖波动和HbA1c有什么不同
  - 如何用CGM评估生活方式干预效果
```

**CGM数据的核心价值**：
> CGM使我们可以便捷得到相当多的血糖数据及其衍生的血糖参数，以至于让大家无所适从。

→ 需要系统性方法分析和解读CGM数据。

**关键CGM指标**：

| 指标 | 含义 | 临床意义 | 平台应用 |
|------|------|---------|---------|
| **TIR（目标范围内时间）** | 血糖在3.9-10.0 mmol/L的时间占比 | 推荐目标：>70% | 生活方式干预效果的直观指标 |
| **TBR（低于目标范围时间）** | 血糖<3.9 mmol/L时间 | 越低越好，警戒：<4% | 低血糖风险监测 |
| **TAR（高于目标范围时间）** | 血糖>10.0 mmol/L时间 | 越低越好 | 餐后血糖控制评估 |
| **血糖变异系数（CV）** | 血糖变异程度 | <36%为稳定 | 饮食处方效果评估 |
| **HbA1c估算值（eA1c）** | 由平均血糖估算 | 与实测HbA1c对比 | 监测干预效果 |

**血糖波动与并发症**：
- 研究表明血糖波动可能与糖尿病并发症的发生发展有关
- 血糖变异性不能只靠HbA1c反映（HbA1c只是平均值）
- → 需要建立简单且有临床意义的血糖波动评估参数

**CGM数据解读的系统性方法**：
```
1. 查看TIR：是否>70%？
2. 查看TBR：是否有低血糖趋势？
3. 查看餐后血糖峰值：哪顿饭后血糖最高？
4. 查看夜间血糖：是否有夜间高血糖或低血糖？
5. 与生活记录对比：什么行为导致了波动？
6. 评估CV：血糖是否过于波动？
```

## CGM与生活方式干预的整合应用

### KI-CGM-005 | CGM作为行为改变反馈工具

```yaml
knowledge_id: KI-CGM-005
title: CGM在生活方式干预中的行为反馈应用
tags: [CGM反馈, 行为改变, 实时反馈, 生活方式干预, 运动血糖, 饮食血糖, 激励]
retrieval_queries:
  - CGM怎么帮助饮食改变
  - 运动后看CGM数据有什么意义
  - CGM作为行为改变工具怎么用
  - 用CGM来激励患者改变生活方式
  - ACLM指南怎么推荐CGM监测
```

**CGM作为行为改变工具的原理**：

```
生活方式干预 → 血糖变化（数小时内可见）→ CGM实时显示
                                              ↓
                                    用户看到直接结果
                                              ↓
                                    强化积极行为（正向反馈）
                                    或提示调整（纠正反馈）
                                              ↓
                                    加速行为改变的建立
```

**ACLM 2025指南推荐**（KAS 13）：
> "在可能的情况下，使用CGM装置追踪生活方式改变的**第一个月及以后**的血糖水平变化。"
> "CGM数据可作为个体的激励工具，以及增强安全性的手段。"

**具体行为反馈场景**：

| 行为干预 | CGM显示 | 平台响应 |
|---------|---------|---------|
| 餐后20分钟步行 | 血糖峰值降低15-20% | 正向强化："你刚才的步行让血糖峰值降低了！" |
| 白米饭换糙米 | 餐后血糖曲线更平稳 | 数据展示对比：换粮食前后的差异 |
| 不吃夜宵 | 夜间血糖稳定 | 显示睡眠期间血糖稳定曲线 |
| 压力事件 | 血糖应激性升高 | 提示：识别压力-血糖关联 |
| 睡眠不足 | 次日晨起血糖偏高 | 睡眠-血糖关联教育 |

**CGM数据的患者教育要点**：
- 血糖波动是**信息**，不是**评判**（避免用血糖数据制造羞耻感）
- 关注**趋势**（血糖正在上升还是下降）比单一数值更重要
- 同样的食物对不同人血糖影响不同 → 个性化干预依据

## CGM技术发展趋势

### KI-CGM-006 | CGM技术发展方向

```yaml
knowledge_id: KI-CGM-006
title: CGM技术发展趋势：从监测到闭环胰岛素输注系统
tags: [CGM发展, 闭环系统, 传感器准确性, 人工胰岛, 技术趋势]
retrieval_queries:
  - CGM技术有什么新进展
  - 闭环胰岛素系统是什么
  - CGM传感器准确性
  - 未来血糖监测技术
```

**三大发展方向**：

1. **CGM数据分析**：建立简单且有临床意义的血糖波动评估参数，解决数据过多但难以分析的困境

2. **临床试验扩展**：需要更长随访时间的研究验证CGM对长期并发症的影响（DCCT/UKPDS级别研究）

3. **闭环系统**：
   > "最终目标是建立CGM算法，针对血糖变化计算生理性胰岛素输注剂量，将成为近期CGM系统发展的另一热点。"
   → 即"人工胰岛"技术，代表CGM的终极应用方向

**准确性问题**：
- CGM测定的是**组织间液葡萄糖浓度**，与血糖值存在差异
- 实时CGM出现对传感器准确性提出更高要求
- 在使用CGM调整治疗前，仍需自测血糖校正确认

## 铁律声明

**CGM-铁律-01（CGM数据不能替代医学决策）**
CoachAgent在展示和讨论CGM数据时，不得用CGM数据直接给出药物调整建议。CGM数据用于生活方式反馈和行为激励，药物调整必须由医生根据CGM数据决定。

**CGM-铁律-02（避免CGM数据造成焦虑）**
CGM数据应作为积极的行为反馈工具，不应成为恐吓或评判用户的依据。CoachAgent处理血糖偏高数据时，应使用探索性语言（"这个时间段血糖偏高，我们来看看发生了什么"），而非评判性语言（"你又吃错了"）。

**CGM-铁律-03（用户培训不可省略）**
CGM的效果与用户的正确使用和数据理解密切相关。平台在引导用户使用CGM时，必须提供基本的使用培训和数据解读教育，不能只是数据传输而无指导。
