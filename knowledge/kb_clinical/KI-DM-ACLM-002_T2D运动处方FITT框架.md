### 【观念层】运动对T2D的三重代谢机制

```
运动 → 骨骼肌收缩
       ↓
  即时效应：肌肉直接摄取葡萄糖（非胰岛素依赖路径）→ 餐后血糖↓
       ↓
  短期效应：GLUT4转运体上调，胰岛素敏感性↑（持续24-72小时）
       ↓  
  长期效应：体脂↓ → 内脏脂肪↓ → 炎症因子↓ → 胰岛素抵抗↓ → HbA1c↓
```

**关键数据**：
- 达到运动指南推荐量的糖尿病前期人群，发展为T2D的风险**降低44%**（独立于体重变化）
- 有氧+力量训练联合，效果优于任一单独训练
- 美国仅**24.1%**的T2D成人达到推荐运动量——说明运动处方是最被忽视的临床工具

### 【知识层】FITT运动处方框架（T2D版）

#### A. 有氧运动处方

| FITT要素 | 最低标准 | 推荐目标 | T2D特别说明 |
|---------|---------|---------|------------|
| **F**（频率） | 每周3天 | 每周5-7天 | 避免连续2天不运动（胰岛素敏感性会降低） |
| **I**（强度） | 中等强度（说话不唱歌） | 中等+偶尔较强 | 初始：感觉有点喘但能说话即可 |
| **T**（时间） | 每次10分钟×3次 | 每次30-60分钟 | 碎片式"运动零食"同样有效 |
| **T**（类型） | 快走 | 快走+游泳+骑车+舞蹈 | 低冲击优先（保护关节） |

**总量目标**：每周 **150-300分钟** 中等强度有氧运动

**强度判断（傻瓜版）**：
- 🟡 中等强度：走路时能说完整句子，但不能唱歌
- 🔴 较强强度：只能说几个字，很喘
- 🟢 低强度：轻松说话，心率几乎不变

#### B. 力量训练处方

| FITT要素 | 推荐 |
|---------|------|
| **F** | 每周至少2次，非连续天 |
| **I** | 中等到较重（最后2个动作感到吃力） |
| **T** | 每次15-20分钟，覆盖主要肌群 |
| **T** | 哑铃/阻力带/器械/俯卧撑/深蹲 |

**为什么力量训练对T2D重要**：
- 肌肉是最大的葡萄糖储存库
- 肌肉量↑ → 基础代谢率↑ → 长期血糖控制↑
- 改善胰岛素敏感性的效果持续更长

#### C. 减少久坐处方（KAS5）

**久坐的代谢危害**：
- 久坐 > 8小时/天 → 血糖控制恶化（独立于是否完成运动目标）
- 看电视时间 → T2D发病率正相关（通过BMI↑和甘油三酯↑介导）

**中断久坐的操作策略**：
- 每**30分钟**站起来活动1-2分钟（站立、走廊走几步、做深蹲）
- **"运动零食"（Exercise Snacks）**：3-5分钟短暂爆发式活动，散布全天
- 餐后尽快开始轻度活动（餐后20分钟快走可显著降低餐后血糖）
- **饭前运动无此效果**，须在饭后进行

### 【应用层】Agent话术脚本

**情境：S2用户说"我不喜欢运动，运动太累了"**

```
CoachAgent话术（低阻力入口）：
"完全理解——其实您不需要'运动'，您只需要'动'。
我想问您：您平时走路去哪里最多？买菜？接孩子？
如果您每次多走5分钟，一天就有额外10-15分钟的快走。
您的血糖在饭后1小时最容易升高，这时候出门走一走，
身体会直接用掉那些糖——不用靠药物。
试一周，您自己来判断血糖有没有变化？"

教练行动指令：
1. 用运动生命体征（PAVS）工具记录用户当前运动量（≤2分钟）
2. 设定第一个运动SMART目标：餐后10分钟×3次/周×2周
3. 记入行为处方，2周后评估并递进
4. 若有CGM，引导用户看餐后运动对血糖曲线的影响（最直接正强化）
```

**情境：S3用户已开始运动但动力下降**

```
CoachAgent话术（强化意义感）：
"您上个月走了多少步？我查了一下——约合多走了X公里。
这X公里，研究说能让您的肌肉多储存了X克糖分，
相当于少给了胰腺X次压力。
不是每次都有感觉，但身体一直在记账。
这周我们把目标稍微提高一格，好吗——
原来10分钟，这周试试15分钟，就这一点点。"
```

### 【运动安全铁律】

| 情境 | 操作 |
|------|------|
| 血糖 < 3.9 mmol/L | ❌ 禁止开始运动 → 先进食15g碳水（约3-4块饼干）并等待15分钟 |
| 血糖 > 14.0 mmol/L（无酮症） | ⚠️ 轻度活动可以，禁止高强度运动 |
| 血糖 > 16.7 mmol/L | ❌ 停止运动 → 触发CrisisAgent |
| 有足部溃疡/严重神经病变 | 改为水中运动/上肢运动/椅上运动，禁止跑步 |
| 有增殖性视网膜病变 | 禁止高强度/憋气类运动（举重/倒立） |
| 运动中出现胸痛/头晕 | ❌ 立即停止 → 触发CrisisAgent |

### 【傻瓜相机检验】

> 问：一位62岁、膝盖不太好、刚确诊T2D的退休男性，今天下午能做什么？

**今日可执行的运动零食方案**：
1. 午饭后10分钟：穿鞋出门，在小区里慢慢走15分钟（不用快走，能说话即可）
2. 看电视时：每隔一集站起来走到厨房倒杯水再回来（约2分钟）
3. 今晚睡前：坐在椅子上做10个坐立起蹲（扶椅扶手，慢慢来）

✅ 通过傻瓜相机检验 — 不需要健身房、不需要运动服、膝盖不好也能完成

### 【监测重点】

| 指标 | 监测内容 | 目标 |
|------|---------|------|
| 每周运动分钟数 | 有氧运动累计时长 | ≥ 150分钟/周 |
| 久坐中断频率 | 每30分钟站立次数 | 工作日 ≥ 6次/日 |
| 餐后血糖响应 | 运动前后血糖对比（CGM用户） | 餐后峰值↓≥1.5 mmol/L |
| 运动相关低血糖事件 | 运动后2小时内血糖 | > 3.9 mmol/L 为安全 |

### 叠加情境提示

- **与肥胖叠加**：力量训练优先（保留肌肉量，减重同时维持代谢率）
- **与高血压叠加**：避免高强度憋气型力量训练，推荐中等强度有氧
- **与周围神经病变叠加**：推荐游泳/骑固定自行车，保护足部
- **与抑郁叠加**：团体运动（群组步行、太极拳）优于独自运动，提供社会支持
