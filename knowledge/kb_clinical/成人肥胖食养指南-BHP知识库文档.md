### KI-OBE-001 成人肥胖判定标准与分型

- **knowledge_id**: `KI-OBE-001`
- **title**: 成人肥胖判定标准与分型
- **category**: `CORE_CONCEPT`（核心概念）
- **life_dimension**: `RISK_FACTOR`
- **behavior_types**: `["assessment", "tracking"]`
- **applicable_stages**: `[S0_UNAWARE, S1_HESITANT, S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人肥胖食养指南（2024年版）》；《成人体重判定》(WS/T 428-2013)
- **tags**: `["肥胖", "BMI", "腰围", "中心型肥胖", "超重", "判定标准"]`

**content**:

肥胖是人体脂肪积聚过多达到危害健康程度的一种慢性代谢性疾病，根本原因是能量摄入大于能量消耗。

**BMI判定标准（中国成人）**：
- 体重正常：18.5 ≤ BMI < 24.0 kg/m²
- 超重：24.0 ≤ BMI < 28.0 kg/m²
- 肥胖：BMI ≥ 28.0 kg/m²
- 体重过低：BMI < 18.5 kg/m²

**中心型肥胖（腰围判定）**：
- 中心型肥胖前期：男性 85cm ≤ WC < 90cm；女性 80cm ≤ WC < 85cm
- 中心型肥胖：男性 WC ≥ 90cm；女性 WC ≥ 85cm

**BMI计算公式**：BMI = 体重(kg) / [身高(m)]²

**理想体重计算**：理想体重(kg) = 身高(cm) - 105

**肥胖分类**：
- 单纯性肥胖：无明显内分泌、代谢病病因可寻
- 继发性肥胖：继发于神经-内分泌-代谢紊乱

**流行病学**：
- 我国18岁及以上居民超重率34.3%、肥胖率16.4%
- 18~44岁肥胖率16.4%、45~59岁18.3%、60岁及以上13.6%
- 2019年全国11.98%心血管疾病死亡归因于高BMI
- 预测2030年成年人超重肥胖率可达65.3%

### KI-OBE-002 肥胖的中医辨证分型

- **knowledge_id**: `KI-OBE-002`
- **title**: 肥胖的中医辨证分型
- **category**: `CORE_CONCEPT`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["assessment"]`
- **applicable_stages**: `[S0_UNAWARE, S1_HESITANT, S2_INTENDING, S3_PREPARING]`
- **evidence_level**: `EXPERT_CONSENSUS`
- **source**: 国家卫生健康委《成人肥胖食养指南（2024年版）》；《黄帝内经》
- **tags**: `["中医", "辨证分型", "体质", "痰湿", "气郁", "脾虚", "胃热"]`

**content**:

中医学将肥胖归属于"脂人""膏人""肥人"等范畴，病因多与年龄、体质、饮食、情志、劳逸因素有关。肥胖属本虚标实证，病位多在脾胃，与肾气虚关系密切。

**五大辨证分型**：

**1. 胃热火郁证**
- 表现：肥胖多食，消谷善饥，大便不爽甚或干结，尿黄，口干口苦，喜饮水
- 舌脉：舌质红，苔黄，脉数
- 食药物质：铁皮石斛、麦芽、鲜芦根、荷叶

**2. 痰湿内盛证**
- 表现：形体肥胖，身体沉重，肢体困倦，脘痞胸满，头晕，口干不欲饮，大便黏滞不爽，嗜食肥甘醇酒，喜卧懒动
- 舌脉：舌质淡胖或大，苔白腻或白滑，脉滑
- 食药物质：砂仁、芡实、党参、白扁豆、茯苓、山药、莲子、薏苡仁

**3. 气郁血瘀证**
- 表现：肥胖懒动，喜太息，胸闷胁满，面晦唇暗，肢端色泽不鲜甚或青紫，便干，失眠，男性性欲下降甚至阳痿，女性月经不调
- 舌脉：舌质暗或有瘀斑瘀点，舌苔薄，脉弦或涩
- 食药物质：当归（仅做香辛料）、桃仁、佛手、砂仁、山楂

**4. 脾虚不运证**
- 表现：肥胖臃肿，神疲乏力，身体困重，脘腹痞闷，四肢轻度浮肿晨轻暮重，饮食如常或偏少，既往多有暴饮暴食史
- 舌脉：舌质淡胖，边有齿印，苔薄白或白腻，脉濡细
- 食药物质：山药、莲子、白扁豆、黄芪、麦芽、砂仁、茯苓、木瓜、大枣

**5. 脾肾阳虚证**
- 表现：形体肥胖，易于疲劳，四肢不温甚或厥冷，喜食热饮，小便清长
- 舌脉：舌淡胖，舌苔薄白，脉沉细
- 食药物质：山药、黄芪、肉桂、干姜、小茴香

### KI-OBE-003 肥胖能量控制核心原则

- **knowledge_id**: `KI-OBE-003`
- **title**: 肥胖能量控制核心原则
- **category**: `PRACTICE_GUIDE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["tracking", "planning"]`
- **applicable_stages**: `[S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人肥胖食养指南（2024年版）》；《中国居民膳食营养素参考摄入量2023版》
- **tags**: `["能量控制", "热量", "限能量膳食", "宏量营养素", "减重"]`

**content**:

控制总能量摄入和保持合理膳食是体重管理的关键。

**能量控制方法（四选一）**：

方法一：在每日能量需要量基础上降低30%~50%或降低500~1000kcal
方法二：推荐每日能量摄入——男性1200~1500kcal、女性1000~1200kcal
方法三：根据基础代谢率和身体活动的实际能量需要量，超重个体给予85%摄入标准，肥胖个体给予80%
方法四：理想体重(kg) = 身高(cm)-105，再乘以能量系数（卧床15kcal/kg、轻活动20~25kcal/kg、中活动30kcal/kg、重活动35kcal/kg）

**中国居民成人膳食能量需要量（kcal/d）**：

| 人群 | 低强度 | 中等强度 | 高强度 |
|------|--------|----------|--------|
| 成年男性 | 1950~2150 | 2400~2550 | 2800~3000 |
| 成年女性 | 1600~1700 | 1950~2100 | 2300~2450 |

**宏量营养素供能比**：
- 脂肪：20%~30%
- 蛋白质：15%~20%
- 碳水化合物：50%~60%

**三餐供能比**：早:中:晚 = 3:4:3

**主食要求**：全谷物至少占谷物一半，适当增加粗粮，减少精白米面

### KI-OBE-004 减重食物选择指南

- **knowledge_id**: `KI-OBE-004`
- **title**: 减重食物选择指南
- **category**: `PRACTICE_GUIDE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["substitution", "planning"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人肥胖食养指南（2024年版）》
- **tags**: `["食物选择", "优选食物", "限量食物", "禁忌食物", "减重饮食"]`

**content**:

**谷薯类（每日150~300g）**：
- 优选：蒸煮烹饪、粗细搭配的杂米饭、杂粮面
- 全谷物和杂豆每日50~150g（黑米、玉米等）
- 薯类每天50~100g
- 不宜：油条、炸薯条、方便面、奶油蛋糕、黄油面包

**蔬菜类（每日300~500g生重甚至更多）**：
- 深色蔬菜占1/2以上（深绿色：菠菜、油菜；橘红色：胡萝卜、西红柿；紫红色：紫甘蓝）
- 每天至少3~5种
- 不宜：炸藕夹、油焖茄子、油炸果蔬脆

**水果类（每日约200g）**：
- 优选：柚子、蓝莓、草莓、苹果、樱桃
- 限量：冬枣、榴莲、香蕉、荔枝、甘蔗、龙眼、芒果
- 不宜饮用果汁

**肉蛋水产类**：
- 水产品每周280~525g（每天40~75g），优先选择鱼虾蟹贝
- 畜禽肉每周不超过500g（每天不超过70g），选纯瘦肉
- 蛋类每周280~350g（≤7个鸡蛋）
- 不宜：肥肉、五花肉、蹄膀、油炸腌制水产

**奶豆类**：
- 每天低脂或脱脂牛奶300~500mL
- 大豆15~25g/天，选豆腐、无糖豆浆
- 不宜：油炸类、含盐较高的豆制品

**坚果类**：
- 每周50~70g（每天约10g），首选原味
- 能量计入一日总能量

**烹饪要求**：
- 每天食盐不超过5g
- 烹调油不超过20~25g
- 添加糖控制25g以下
- 多选蒸、煮、熘及水滑等方式

### KI-OBE-005 科学进餐行为规范

- **knowledge_id**: `KI-OBE-005`
- **title**: 科学进餐行为规范
- **category**: `BEHAVIOR_TECHNIQUE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["habit_formation", "substitution"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人肥胖食养指南（2024年版）》
- **tags**: `["进餐行为", "定时定量", "细嚼慢咽", "进餐顺序", "饮食习惯"]`

**content**:

**定时定量原则**：
- 一日三餐时间相对固定
- 重视早餐，不漏餐
- 晚餐建议17:00~19:00进食
- 晚餐后不宜再进食（可饮水；饥饿难忍者可少许低能量高膳食纤维食物）

**进食行为规范**：
- 不暴饮暴食
- 控制随意进食零食、饮料
- 避免夜宵
- 进餐宜细嚼慢咽——同样食物摄入量更少
- 减缓进餐速度增加饱腹感、降低饥饿感

**进餐顺序（简单有效的减重方法）**：
蔬菜 → 肉类 → 主食

**标准化配餐**：
- 根据生理条件和身体活动量
- 合理计划每日餐次和能量分配
- 力求饮食有节制、科学搭配

### KI-OBE-006 肥胖患者运动处方

- **knowledge_id**: `KI-OBE-006`
- **title**: 肥胖患者运动处方
- **category**: `PRACTICE_GUIDE`
- **life_dimension**: `EXERCISE`
- **behavior_types**: `["habit_formation", "tracking"]`
- **applicable_stages**: `[S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人肥胖食养指南（2024年版）》；《健康成年人身体活动能量消耗参考值》(T/CSSS 002-2023)
- **tags**: `["运动", "有氧运动", "抗阻运动", "能量消耗", "MET", "减重运动"]`

**content**:

**运动原则**：中低强度有氧运动为主，抗阻运动为辅

**有氧运动**：
- 每周150~300分钟中等强度
- 每周5~7天，至少隔天1次
- 每周通过运动消耗能量2000kcal或以上

**抗阻运动**：
- 每周2~3天，隔天1次
- 每次10~20分钟

**静坐控制**：
- 每天静坐和被动视屏时间控制在2~4小时以内
- 长期静坐或伏案工作者每小时起来活动3~5分钟

**运动原则**：
- 培养兴趣，把运动变为习惯
- 循序渐进，逐渐增进运动量
- 选择自己喜欢的运动方式，持之以恒
- 天天运动融入日常生活

**常见活动MET值参考**：

| 活动 | MET | 强度 |
|------|-----|------|
| 安静坐着 | 1.3 | 静态行为 |
| 步行4km/h | 3.3 | 中等 |
| 步行6km/h | 5.2 | 中等 |
| 跑步6km/h | 6.5 | 高 |
| 跑步8km/h | 8.2 | 高 |
| 太极拳24式 | 3.7 | 中等 |
| 八段锦 | 3.2 | 中等 |
| 瑜伽 | 3.1 | 中等 |
| 广场舞 | 5.5 | 中等 |
| 羽毛球 | 9.8 | 高 |
| 跳绳100次/min | 10.2 | 高 |

**强度分类**：静态行为≤1.5MET；低强度1.6~2.9MET；中等强度3.0~5.9MET；高强度≥6.0MET

**能量消耗计算**：MET值(kcal/(kg·h)) × 体重(kg) × 活动时长(h)

**基础代谢率预测公式**：
- 女性：655 + 9.5×体重(kg) + 1.8×身高(cm) - 4.7×年龄(岁)
- 男性：66 + 13.7×体重(kg) + 5.0×身高(cm) - 6.8×年龄(岁)

### KI-OBE-007 睡眠与作息对肥胖的影响

- **knowledge_id**: `KI-OBE-007`
- **title**: 睡眠与作息对肥胖的影响
- **category**: `CORE_CONCEPT`
- **life_dimension**: `SLEEP`
- **behavior_types**: `["habit_formation"]`
- **applicable_stages**: `[S1_HESITANT, S2_INTENDING, S3_PREPARING, S4_ACTING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人肥胖食养指南（2024年版）》
- **tags**: `["睡眠", "作息", "内分泌", "过劳肥", "生物节律"]`

**content**:

经常熬夜、睡眠不足、作息无规律可引起内分泌紊乱，脂肪代谢异常，增加肥胖风险，导致"过劳肥"。

**睡眠要求**：
- 按昼夜生物节律作息
- 保证每日7小时左右睡眠时间
- 建议夜里11点之前上床睡觉

### KI-OBE-008 安全减重目标与自我监测

- **knowledge_id**: `KI-OBE-008`
- **title**: 安全减重目标与自我监测
- **category**: `PRACTICE_GUIDE`
- **life_dimension**: `RISK_FACTOR`
- **behavior_types**: `["tracking", "planning"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人肥胖食养指南（2024年版）》
- **tags**: `["减重目标", "安全减重", "自我监测", "平台期", "体脂率"]`

**content**:

**减重目标**：
- 理想目标：6个月内减少当前体重的5%~10%
- 合理速度：每月减2~4kg
- 初始建议：每周减轻约0.5kg

**三种减重模式**：
1. 体重平稳下降：每周减0.5~1kg
2. 延迟下降型：初期1~2个月体重无明显变化，之后开始下降且速度较快
3. 快降-平台型：最初每周下降1~2kg，随后进入数周甚至数月平台期，突破后继续下降

**安全警示**：
- 过快减重易对机体器官组织造成损伤甚至危及生命
- 短期快速减重主要丢失水分而非脂肪，恢复饮食后快速反弹
- 孕妇、乳母、老年人及慢性代谢性疾病人群应在专业人员指导下减重

**自我监测内容**：
- 体重变化
- 食物摄入量
- 身体活动情况
- 体脂率变化
- 肌肉量变化

**核心原则**：减少肌肉流失，维持机体肌肉量和基础代谢率
