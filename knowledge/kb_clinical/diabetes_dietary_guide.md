## 第一部分：疾病基础与流行病学

### 1.1 核心概念

| 术语 | 定义 |
|------|------|
| 糖尿病（Diabetes Mellitus） | 遗传因素和环境因素长期共同作用导致的慢性、全身性及代谢性疾病，以高血糖为主要特点 |
| 典型症状 | "三多一少"：多饮、多食、多尿和体重减少 |
| 血糖生成指数（GI） | 衡量食物对血糖影响的相对指标，以50g葡萄糖GI=100为基准 |
| 食药物质 | 按照传统既是食品又是中药材的物质，融入膳食辅助预防和改善疾病 |
| 食养（Dietary Therapy） | 以中医辨证施膳为核心的非药物调养方法，强调根据体质、病因、证候给予不同食养方案 |
| 食物交换份 | 将食物按类别和营养特点分组，同组食物可等量互换的营养计量方法 |

### 1.2 流行病学数据（中国）

| 指标 | 数据 | 来源 |
|------|------|------|
| 18岁及以上成人糖尿病患病率 | 11.9% | 中国居民营养与慢性病状况报告（2020年） |
| 糖尿病前期检出率 | 35.2% | 同上 |
| 主要类型 | 2型糖尿病为主 | 50岁以上成年人患病率更高 |
| 发病趋势 | 逐年上升，低龄化 | 长病程、并发症多、医疗支出高 |

### 1.3 危险因素与并发症

- **膳食相关**：长期高糖、高脂肪、高能量膳食
- **体重因素**：超重肥胖是2型糖尿病的重要危险因素
- **身体活动不足**：导致体重过度增加，增加发病风险
- **并发症**：长期慢性高血糖可导致眼、神经、肾脏和心血管等组织器官损害

### 1.4 膳食干预的价值

- **糖尿病前期/短病程T2DM**：膳食干预+生活方式改善可帮助实现血糖良好控制
- **长病程/使用降糖药的T2DM及T1DM**：有助于血糖达标，预防和延缓并发症
- **核心地位**：膳食管理和治疗是糖尿病患者血糖控制的核心

## 第二部分：疾病分类与中医辨证分型

### 2.1 西医分类（WHO 2019更新）

| 类型 | 说明 |
|------|------|
| 1型糖尿病 | 胰岛β细胞破坏，胰岛素绝对缺乏 |
| 2型糖尿病 | 胰岛素抵抗和/或胰岛素分泌缺陷，最常见类型 |
| 混合型糖尿病 | 兼具1型和2型特征 |
| 其他特殊类型糖尿病 | 包括基因缺陷、胰腺疾病等 |
| 未分类糖尿病 | 暂不能明确分类者 |
| 妊娠期糖尿病 | 妊娠期间首次发现的糖代谢异常 |

### 2.2 中医辨证分型

#### 2.2.1 阴虚热盛证

| 项目 | 内容 |
|------|------|
| 临床表现 | 烦渴多饮，咽干舌燥，多食善饥，溲赤便秘 |
| 舌脉 | 舌红少津苔黄，脉滑数或弦数 |
| 食养要点 | 养阴清热，选用寒凉性、润燥滋阴食物，忌辛辣燥热食物 |
| 推荐食药物质 | 桑叶、决明子、莲子、百合、玉竹、金银花、菊花、铁皮石斛† |

#### 2.2.2 气阴两虚证

| 项目 | 内容 |
|------|------|
| 临床表现 | 倦怠乏力、心慌气短、盗汗自汗，口干舌燥，多饮多尿，五心烦热，大便秘结，腰酸膝软 |
| 舌脉 | 舌淡或舌红暗，舌边有齿痕，苔薄白少津或少苔，脉细弱 |
| 食养要点 | 益气养阴，选用补气滋阴食物，避免过于寒凉或温燥 |
| 推荐食药物质 | 黄芪†、桑椹、枸杞子、葛根、山药、茯苓、鸡内金、麦芽、薏苡仁 |

#### 2.2.3 阴阳两虚证

| 项目 | 内容 |
|------|------|
| 临床表现 | 乏力自汗，形寒肢冷，腰酸膝软，耳轮焦干，多饮多尿，或浮肿少尿，或五更泻，阳痿早泄 |
| 舌脉 | 舌淡苔白，脉沉细无力 |
| 食养要点 | 温阳滋阴并补，选用温性滋补食物，避免过于寒凉 |
| 推荐食药物质 | 山茱萸†、肉苁蓉†、山药、茯苓、肉桂、紫苏籽、干姜、黑胡椒、花椒 |

> **注意**：标†为试点物质，非试点地区限执业医师使用。食药物质需在限定使用范围和剂量内使用，食用方法请咨询医师、营养指导人员等专业人员。

### 2.3 糖尿病中医分型与病程对应关系

```
早期/初诊 ——→ 阴虚热盛证为主（消渴上消、中消）
中期/病程较长 ——→ 气阴两虚证为主（气阴双亏）
晚期/久病 ——→ 阴阳两虚证为主（阴阳俱损、并发症多）
```

> **平台逻辑**：tcm Agent根据用户病程和症状表现匹配中医证型，推荐对应食药物质。若用户已有中医辨证结论则优先使用。

## 第三部分：八大食养原则与建议

### 原则一：食物多样，养成和建立合理膳食习惯

**核心规则**：
- 膳食应由五大类食物组成：谷薯类、蔬菜和水果、动物性食物、大豆类和坚果、烹调油和盐
- 保持食物多样、膳食丰富多彩，保证营养素摄入全面充足
- 少油少盐限糖限酒
- 餐餐有蔬菜（每天≥500g，深色蔬菜占一半以上）
- 天天有奶类和大豆，常吃鱼、禽，适量蛋和畜肉
- 减少肥肉摄入，少吃烟熏、烘烤、腌制等加工肉类制品

### 原则二：能量适宜，控制超重肥胖和预防消瘦

**核心规则**：
- 膳食能量是体重管理和血糖控制的核心
- 宏量营养素占总能量比：蛋白质15%~20%、碳水化合物45%~60%、脂肪20%~35%
- 健康BMI：成人18.5~23.9 kg/m²，65岁以上老年人可适当增加
- 超重肥胖T2DM患者减重3%~5%即可产生有临床意义的健康获益
- 建议减重速度：每月减少1~2kg，3~6个月减少体重5%~10%
- 消瘦/营养不良患者应增加膳食能量和蛋白质，结合抗阻运动增加体重
- 老龄患者特别注意预防肌肉衰减并保持健康体重

### 原则三：主食定量，优选全谷物和低GI食物

**核心规则**：
- 主食定量，不宜过多，全谷物和杂豆类占主食的1/3以上
- 碳水化合物供能比45%~60%（略低于一般健康人）
- 成年人（1800~2000kcal）相当于一天碳水化合物总量200~300g
- 选择低GI食物有利于餐后血糖控制
- 调整进餐顺序：先吃菜，最后吃主食
- 对零食中的碳水化合物也应查看营养成分表，计入全天摄入量
- 记录膳食、运动和血糖水平，提高自我管理水平

**GI分类标准**：
| GI分级 | GI值 | 对血糖影响 |
|--------|------|-----------|
| 低GI | ≤55 | 血糖释放缓慢，餐后波动小 |
| 中GI | 55<GI≤70 | 中等影响 |
| 高GI | >70 | 血糖快速升高，波动大 |

### 原则四：积极运动，改善体质和胰岛素敏感性

**核心规则**：
- 可在餐后运动，每周至少5天，每次30~45分钟
- 中等强度运动占50%以上：快走、骑车、乒乓球、羽毛球、慢跑、游泳
- 如无禁忌，每周2次抗阻运动：哑铃、俯卧撑、器械类
- 运动可增加GLUT-4数量，增加骨骼肌对葡萄糖的摄取
- 运动前后加强血糖监测，避免低血糖
- 将运动列入每天的时间安排，培养运动意识和习惯

### 原则五：清淡饮食，限制饮酒，预防和延缓并发症

**核心规则**：
- 每日烹调油≤25g，食盐≤5g
- 少吃动物脂肪，适当控制富含胆固醇的食物
- 注意限制酱油、鸡精、味精、咸菜等含盐量较高的调味品
- 足量饮用白水，可适量饮用淡茶或咖啡，不喝含糖饮料
- **饮酒风险**：扰乱正常膳食和用药，导致血糖波动；服用胰岛素或促泌剂时可能增加低血糖风险；过量酒精损害胰腺、引起肝损伤，也是痛风、癌症和心血管疾病的危险因素

### 原则六：食养有道，合理选择应用食药物质

**核心规则**：
- 坚持辨证施膳原则，因人、因时、因地制宜
- 中医食物属性："四气"（寒热温凉）、"五味"（酸苦甘辛咸）、"归经"、"升降沉浮"
- 寒凉食物可清热但易伤阳，温热食物可祛寒但易伤阴，强调阴阳平衡
- 酸味入肝，苦味入心，甘味入脾，辛味入肺，咸味入肾，食养时五味调和
- 按中医辨证论治原则选择食药物质

### 原则七：规律进餐，合理加餐，促进餐后血糖稳定

**核心规则**：
- 一日三餐定时定量，时间相对固定
- 不暴饮暴食，不随意进食零食饮料，减少聚餐
- 进行标准化、定量的营养配餐，合理计划餐次和能量分配
- 吃饭宜细嚼慢咽
- 是否加餐、何时加餐、选择何种零食，根据血糖波动特点决定
- 血糖控制不佳、注射胰岛素的患者可适当加餐，预防低血糖
- 消瘦患者及妊娠期糖尿病患者也可安排加餐

### 原则八：自我管理，定期营养咨询，提高血糖控制能力

**核心规则**：
- 有效血糖管理很大程度取决于患者自我管理能力
- 需学习：膳食调理、规律运动、监测血糖、遵医嘱用药、胰岛素注射技术、低血糖预防和处理
- 将营养配餐、合理烹饪、运动管理和血糖监测作为基本技能
- 了解食物碳水化合物含量和GI值，学习食物交换份的使用
- 建立与营养师、医师团队的定期咨询和随访关系
- 初诊、年度检查和/或未达治疗目标时应及时就诊或咨询

## 第五部分：每日膳食摄入推荐量

### 5.1 宏量营养素推荐

| 营养素 | 推荐占比 | 说明 |
|--------|---------|------|
| 蛋白质 | 总能量的15%~20% | 良好来源：鱼、禽、蛋、奶、大豆 |
| 碳水化合物 | 总能量的45%~60% | 以全谷物、杂豆、蔬菜为主要来源，水果限量 |
| 脂肪 | 总能量的20%~35% | 控制烹调油和肥肉，减少饱和脂肪 |

### 5.2 每日食物摄入参考

| 食物类别 | 推荐量 | 注意事项 |
|---------|--------|---------|
| 谷薯类 | 根据能量需求定量 | 全谷物和杂豆≥1/3，碳水200~300g（1800~2000kcal） |
| 蔬菜 | ≥500g | 深色蔬菜≥1/2 |
| 水果 | 限量 | 选择低GI品种，计入全天碳水 |
| 奶类和大豆 | 天天有 | 蛋白质良好来源 |
| 鱼禽蛋肉 | 常吃鱼、禽，适量蛋和畜肉 | 减少肥肉，少吃加工肉 |
| 食盐 | ≤5g | 包括酱油、调味品中的盐 |
| 烹调油 | ≤25g | 品种多样，保障脂肪酸平衡 |
| 饮水 | 足量白水 | 不喝含糖饮料 |

### 5.3 能量估算方法

- 具体可查询DRI中国居民膳食营养素需要量表
- 简易估算：根据体重计算，如60kg轻体力活动成年女性约1500~1800kcal/天
- 体重过重或过轻可按25~30kcal/(kg·d)适当增减食材用量

## 第八部分：地区与季节膳食调适

### 8.1 七大地区膳食特点与修正方向

| 地区 | 膳食特点 | 修正方向 |
|------|---------|---------|
| 东北地区 | 口味偏咸多油能量高，冬春果蔬较少 | 增加水产比例，搭配多样化，增加蔬菜水果 |
| 西北地区 | 喜面食，蔬菜以根茎类为主，绿叶蔬菜少 | 增加绿叶蔬菜和水产，控制油盐 |
| 华北地区 | 厚重咸口，蔬菜水产不多 | 增加蔬菜及水产品，注重应季食物 |
| 华东地区 | 饮食偏甜，以大米制品为主 | 口味修正，平衡膳食，兼顾季节特点 |
| 华中地区 | 辣、重油重盐重口味 | 减油减盐，合理搭配 |
| 西南地区 | 以猪肉为主，鱼虾奶较少，辛辣麻味为主 | 增加鱼虾奶摄入，控制辛辣 |
| 华南地区 | 喜喝老火汤，以米制品为主 | 注意汤类能量控制，合理使用食药物质 |

### 8.2 季节性食养重点

| 季节 | 膳食重点 | 食药物质应用 |
|------|---------|-------------|
| 春季 | 养肝升发，清淡为主 | 山药、枸杞子、生姜、百合、紫苏 |
| 夏季 | 祛暑清热，清淡营养易消化 | 赤小豆、薏苡仁、百合、莲子、藿香 |
| 秋季 | 养阴润肺，防秋燥 | 百合、山药、莲子、银耳、枸杞子 |
| 冬季 | 温补御寒，增强体质 | 黄芪、生姜、枸杞子、肉桂、大枣 |

### 8.3 食谱设计规范

- 适合轻体力活动成人糖尿病患者，能量1600~2000kcal/天
- 宏量营养素占比：蛋白质15%~20%，碳水化合物45%~60%，脂肪20%~35%
- 全天限量烹调油25g、盐5g
- 食物重量为可食部生重
- 避免淀粉含量高的谷薯类过度烹调（升高GI）
- 避免沾粉或勾芡的烹调方式
- 食用禽类时去皮以减少脂肪摄入
- 烹调油品种多样，保障脂肪酸平衡

## 第十部分：与其他知识模块的集成逻辑

### 10.1 与血糖监测知识库（Glucose Monitoring Base）

- 空腹血糖≥7.0mmol/L或HbA1c≥6.5% → 触发本指南膳食方案推送
- 糖尿病前期（空腹血糖6.1~6.9mmol/L） → 推送膳食干预+生活方式改善方案
- 餐后血糖监测数据 → 作为GI食物选择和碳水化合物调整的客观依据
- HbA1c动态趋势持续改善 → 膳食干预有效性的验证指标

### 10.2 与ACSM运动处方知识库

- 糖尿病患者运动推荐联动ACSM第五部分（代谢疾病FITT）
- 餐后运动，每周至少5天，每次30~45分钟，中等强度为主
- 每周2次抗阻运动，提高肌肉力量和胰岛素敏感性
- 运动前后加强血糖监测，避免低血糖
- 超重肥胖T2DM患者联动ACSM肥胖FITT方案

### 10.3 与大五人格知识库（BFP Base）

- 高尽责性（BFP-C↑）：适合食物交换份精确计量+血糖-膳食日志+GI记录
- 低尽责性（BFP-C↓）：简化为"GI红绿灯食物卡"（红-高GI避免/黄-中GI限量/绿-低GI优选），降低执行门槛
- 高神经质（BFP-N↑）：避免过度强调"禁食"压力，转为正向引导"可以吃什么"，注意低血糖焦虑管理
- 高外向性（BFP-E↑）：注意聚餐场景（宴请、饮酒、甜点）的风险提示和替代策略
- 低开放性（BFP-O↓）：推荐简单易执行的标准化食谱模板，避免复杂食药物质搭配
- 高宜人性（BFP-A↑）：注意社交压力下的妥协进食（别人劝食），强化自我边界设定

### 10.4 与BPT6/TTM行为改变模型

- **前意向期用户**：推送糖尿病流行病学数据+并发症危害+膳食与血糖的关系证据（第一部分）
- **意向期用户**：提供GI分类食物表+简化膳食对照卡（第四部分）
- **准备期用户**：制定具体食谱计划+SMARTS目标（如"本周每餐主食≤2份谷薯类交换份"）
- **行动期用户**：每周追踪血糖-膳食记录匹配度+碳水化合物控制达标率+体重变化
- **维持期用户**：推送季节性食养方案调整+社交场景应对策略+并发症预防监测

### 10.5 与中医体质知识库（TCM Base）

| 体质类型 | 糖尿病相关性 | 膳食策略优先级 |
|---------|-------------|-------------|
| 阴虚体质 | 高风险，糖尿病最常见体质 | 阴虚热盛证食养方案→养阴清热为主 |
| 气虚体质 | 中高风险，病程较长者 | 气阴两虚证方案→益气养阴为主 |
| 痰湿体质 | 高风险，常合并肥胖/代谢综合征 | 健脾化湿+控制碳水+低GI食物 |
| 湿热体质 | 中等风险 | 清热利湿+清淡膳食 |
| 阳虚体质 | 久病后多见 | 阴阳两虚证方案→温阳滋阴 |
| 平和体质 | 低风险 | 标准糖尿病膳食+预防性食养 |

### 10.6 与痛风/高尿酸知识库的交叉管理

- 糖尿病+高尿酸血症（双标签）→ 取两个指南的膳食交集
- 共同原则：低盐、控制总能量、充分饮水、限酒、多蔬菜
- 差异注意：糖尿病强调GI控制，痛风强调嘌呤控制；部分低GI食物可能中高嘌呤（如豆类），需综合考量
- 共病用户优先推荐：蛋奶类、低GI且低嘌呤蔬菜、全谷物（适量）

### 10.7 与危机干预规则

| 触发条件 | 响应级别 | 处理方式 |
|---------|---------|---------|
| 血糖>16.7mmol/L伴酮症症状（恶心呕吐、呼吸深快、意识模糊） | P0（Safety Trigger） | 立即停止膳食指导→紧急就医提示→糖尿病酮症酸中毒需急诊 |
| 严重低血糖（血糖<3.9mmol/L伴意识障碍） | P0 | 立即推送急救指导→紧急就医→停止常规膳食建议 |
| 用户报告反复低血糖 | P1（24h内人工教练介入） | 评估膳食-药物-运动匹配→建议就医调整用药→推送加餐策略 |
| 用户初诊糖尿病 | P2 | 推送全面膳食评估+转介营养师+八大原则概要 |
| 用户HbA1c持续>9%且膳食干预3个月无改善 | P1 | 建议就医评估用药方案→膳食方案可能需要个性化调整 |

### 11.4 特殊注意事项速查

| 场景 | 注意事项 |
|------|---------|
| 使用胰岛素/促泌剂 | 运动和膳食调整时须防低血糖，可适当加餐 |
| 合并肥胖 | 减重3%~5%即有临床获益，按25~30kcal/(kg·d)调整能量 |
| 合并消瘦 | 增加膳食能量和蛋白质，结合抗阻运动增加体重 |
| 老龄患者 | 预防肌肉衰减，全谷物选择注意消化道负担 |
| 妊娠期糖尿病 | 可参考本指南但需专业个性化指导 |
| 饮酒 | 扰乱膳食和用药，增加低血糖风险，不建议饮酒 |
| 进餐顺序 | 先吃菜，最后吃主食，有利于控制血糖 |
