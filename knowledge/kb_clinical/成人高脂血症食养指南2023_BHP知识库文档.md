### KI-HLP-001 高脂血症定义与流行病学

- **knowledge_id**: `KI-HLP-001`
- **title**: 高脂血症定义、分型与流行病学
- **category**: `CORE_CONCEPT`
- **life_dimension**: `RISK_FACTOR`
- **behavior_types**: `["assessment", "awareness"]`
- **applicable_stages**: `[S0_UNAWARE, S1_HESITANT, S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["高脂血症", "血脂异常", "胆固醇", "甘油三酯", "动脉粥样硬化", "流行病学"]`

**content**:

高脂血症是指由于脂肪代谢或运转异常使血浆中一种或多种脂质高于正常水平的疾病，也称血脂异常。

**血脂异常分型**：

按临床分型：
1. 高胆固醇血症：TC≥6.2 mmol/L
2. 高甘油三酯血症：TG≥2.3 mmol/L
3. 混合型高脂血症：TC≥6.2 mmol/L 且 TG≥2.3 mmol/L
4. 低高密度脂蛋白胆固醇血症：HDL-C<1.0 mmol/L

按病因分型：
1. 原发性高脂血症：遗传因素和/或环境因素（不良生活方式）引起
2. 继发性高脂血症：由其他疾病引起（甲减、肾病综合征、糖尿病、肝病等）

**流行病学数据**：我国成人血脂异常总体患病率高达35.6%，其中高TC血症患病率为6.9%，高TG血症患病率为15.9%，低HDL-C血症患病率为20.4%。血脂异常患病率随年龄增长而升高。男性在50岁前高于女性，50岁后女性追赶并超过男性（与绝经相关）。

**危害**：LDL-C是动脉粥样硬化性心血管疾病（ASCVD）的核心致病因素。LDL-C每升高1 mmol/L，ASCVD风险增加约20%。高TG是急性胰腺炎的独立危险因素，TG≥5.6 mmol/L时风险显著增加。

**主要危险因素**：高饱和脂肪酸和反式脂肪酸膳食、高胆固醇膳食、超重/肥胖、缺乏体力活动、过量饮酒（升TG）、吸烟（降HDL-C）、遗传因素。

### KI-HLP-002 高脂血症食养核心原则

- **knowledge_id**: `KI-HLP-002`
- **title**: 高脂血症食养核心原则
- **category**: `DIETARY_PRINCIPLE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["dietary_pattern", "food_selection", "meal_planning"]`
- **applicable_stages**: `[S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["膳食原则", "降脂饮食", "脂肪酸", "膳食纤维", "植物固醇"]`

**content**:

高脂血症食养遵循以下核心原则：

**原则一：调控脂肪，少油限胆固醇**。每日烹调油用量20-25g。减少饱和脂肪酸摄入至总能量的7%-10%以下（高胆固醇血症者<7%）。完全避免反式脂肪酸（人造奶油、氢化植物油、起酥油）。增加不饱和脂肪酸比例，n-6:n-3脂肪酸比值控制在4-6:1。膳食胆固醇<300mg/d（高胆固醇血症者<200mg/d）。优先选择富含单不饱和脂肪酸的油脂（橄榄油、茶油、菜籽油）。

**原则二：食物多样，营养均衡**。每日食物种类≥12种，每周≥25种。增加全谷物和杂豆（占谷物总量1/3以上）。每日蔬菜≥500g（深色蔬菜占50%以上）、水果200-350g。适量优质蛋白（鱼、禽、蛋、瘦肉、大豆）。每日奶类及其制品300-500ml。

**原则三：增加膳食纤维摄入**。每日膳食纤维25-30g，其中可溶性膳食纤维10-25g。可溶性膳食纤维来源：燕麦β-葡聚糖、果胶（苹果、柑橘）、豆类、车前子壳。每增加5-10g可溶性膳食纤维/天，LDL-C可降低约5%。

**原则四：科学食养，辨证施食**。根据不同血脂异常类型选择食养策略。高TC者重点控制饱和脂肪和胆固醇；高TG者重点控制碳水化合物（尤其精制糖）和酒精；混合型需综合控制。结合中医辨证进行食养调理。

**原则五：吃动平衡，健康体重**。超重/肥胖者减重3%-5%即可改善血脂。有氧运动为主，每周≥150分钟中等强度。避免久坐，每小时起身活动。

### KI-HLP-003 降脂关键营养素与功能食物

- **knowledge_id**: `KI-HLP-003`
- **title**: 降脂关键营养素与功能食物
- **category**: `FUNCTIONAL_NUTRITION`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["food_selection", "supplement"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["功能营养素", "植物固醇", "omega-3", "可溶性纤维", "大豆蛋白", "降脂食物"]`

**content**:

**一、植物固醇/甾醇**

作用机制：竞争性抑制肠道胆固醇吸收。每日摄入2-3g植物固醇可降低LDL-C约6%-15%。

食物来源：植物油（玉米油、菜籽油富含）、坚果、豆类、全谷物。强化食品（添加植物固醇的酸奶、牛奶、食用油）。

**二、可溶性膳食纤维**

作用机制：结合胆酸，促进胆固醇经粪便排出，减少肝脏胆固醇合成。

核心来源与降脂效果：
| 来源 | 推荐量/天 | 降LDL-C效果 |
|------|----------|-------------|
| 燕麦β-葡聚糖 | ≥3g（约80g燕麦片） | 降5%-10% |
| 车前子壳（洋车前子） | 10-12g | 降5%-15% |
| 果胶（苹果、柑橘） | 10-15g | 降5%-10% |
| 豆类（黄豆、绿豆、芸豆） | 每日25-50g | 降5%-8% |

**三、Omega-3多不饱和脂肪酸**

作用机制：降低TG（抑制肝脏VLDL合成），抗炎，改善血管内皮功能。

推荐：每周食用鱼类2-3次（每次100-150g），优先深海鱼（三文鱼、鲭鱼、沙丁鱼、秋刀鱼）。高TG者可补充高纯度EPA/DHA（2-4g/d，需在医生指导下）。植物来源：亚麻籽、核桃、奇亚籽（含ALA，转化率低但仍有益）。

**四、大豆蛋白**

作用机制：促进LDL受体表达，增加LDL-C清除。每日25g大豆蛋白（约75g大豆或300g豆腐）可降低LDL-C约4%-6%。同时大豆异黄酮有抗氧化和血管保护作用。

**五、其他降脂活性成分**

茶多酚（绿茶）：抑制胆固醇吸收，降TC和LDL-C约5%。每日3-5杯绿茶。大蒜素：抑制胆固醇合成酶活性，降TC约5%-15%。每日1-2瓣生大蒜或等效。红曲：含天然洛伐他汀类似物，有降脂作用，但需注意与他汀类药物的叠加效应。姜黄素：抗氧化、抗炎，辅助改善血脂。

### KI-HLP-004 高脂血症食物选择指南

- **knowledge_id**: `KI-HLP-004`
- **title**: 高脂血症食物选择红绿灯
- **category**: `FOOD_GUIDE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["food_selection", "grocery_shopping", "food_preparation"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["食物选择", "降脂食物", "高胆固醇食物", "饱和脂肪", "反式脂肪"]`

**content**:

**绿灯食物（推荐多吃）**：

| 类别 | 代表食物 | 降脂机制 |
|------|---------|---------|
| 全谷物 | 燕麦、荞麦、糙米、全麦、藜麦 | β-葡聚糖降TC/LDL-C |
| 豆类 | 黄豆、黑豆、绿豆、红豆、芸豆 | 大豆蛋白+可溶性纤维 |
| 深色蔬菜 | 菠菜、西兰花、海带、木耳、香菇 | 膳食纤维+植物固醇 |
| 水果 | 苹果、柑橘、猕猴桃、蓝莓、山楂 | 果胶+黄酮类抗氧化 |
| 深海鱼 | 三文鱼、鲭鱼、沙丁鱼、秋刀鱼 | EPA/DHA降TG |
| 坚果 | 核桃、杏仁、开心果（原味无盐） | 不饱和脂肪酸+植物固醇 |
| 好油 | 橄榄油、茶油、亚麻籽油 | 单不饱和/多不饱和脂肪酸 |
| 茶类 | 绿茶、普洱茶 | 茶多酚抑制胆固醇吸收 |

**黄灯食物（限量摄入）**：

| 类别 | 代表食物 | 限量标准 | 原因 |
|------|---------|---------|------|
| 蛋黄 | 鸡蛋 | ≤1个/天（高TC者3-4个/周） | 胆固醇约186mg/个 |
| 瘦红肉 | 猪瘦肉、牛肉、羊肉 | ≤50g/天 | 含饱和脂肪酸 |
| 虾类 | 对虾、基围虾 | 适量（≤100g/次） | 胆固醇含量中等 |
| 精白米面 | 白米饭、白面条 | 与全谷物1:2搭配 | 升TG |
| 酒精 | — | 男≤25g/d，女≤15g/d，高TG者禁酒 | 升TG |

**红灯食物（避免或极少摄入）**：

| 类别 | 代表食物 | 原因 |
|------|---------|------|
| 动物脂肪 | 猪油、牛油、奶油、肥肉、鸡皮、鸭皮 | 高饱和脂肪酸 |
| 动物内脏 | 猪肝、猪脑、猪肾、鹅肝 | 极高胆固醇（猪脑约2571mg/100g） |
| 反式脂肪食品 | 人造奶油、起酥油、植脂末、油炸食品 | 升LDL-C降HDL-C |
| 加工肉制品 | 培根、香肠、火腿、午餐肉 | 高饱和脂肪+高钠 |
| 高糖食品 | 含糖饮料、糕点、糖果、甜点 | 升TG |
| 椰子油/棕榈油 | 多用于加工食品 | 高饱和脂肪酸 |

高TC者：最需控制的是饱和脂肪酸（<7%总能量）和膳食胆固醇（<200mg/d），绿灯区增加燕麦和大豆制品。

混合型：饱和脂肪+胆固醇+精制糖三重控制，食物种类选择最严格。

低HDL-C者：增加有氧运动是升高HDL-C最有效手段，戒烟，适量坚果和深海鱼。

### KI-HLP-005 高脂血症烹饪与进餐指南

- **knowledge_id**: `KI-HLP-005`
- **title**: 高脂血症烹饪方法与进餐行为
- **category**: `PRACTICAL_GUIDE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["food_preparation", "eating_behavior", "meal_timing"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["烹饪方法", "控油技巧", "进食顺序", "外食策略", "营养标签"]`

**content**:

**低脂烹饪法则**：

推荐烹饪方式（油脂用量由低到高）：蒸 > 煮 > 炖 > 凉拌 > 焖 > 烤 > 快炒。避免油炸、煎炸、干烧、红烧（重油重糖）。

控油技巧：使用控油壶定量倒油（每人每餐约7-8g）。不粘锅减少用油量。肉类去皮去脂后烹饪。炖煮后撇去浮油。用喷雾瓶喷油烹饪。

选油策略：日常烹饪首选菜籽油、花生油或玉米油（中高温适用）。凉拌首选橄榄油、亚麻籽油。避免动物油脂和椰子油/棕榈油。不同油脂轮换使用。

**进餐行为优化**：

进餐顺序：蔬菜→蛋白质→主食（与糖尿病食养一致，有助于控制TG）。细嚼慢咽（每口咀嚼20次以上），用餐时间≥20分钟。饮食规律，三餐按时定量，能量分配建议3:4:3。不要边看手机/电视边吃饭（易不自觉多食）。

**外食降脂策略**：选择清蒸/白灼/凉拌菜品。自行要求少油少盐。汤类撇去上层油脂。涮火锅优先选择菌菇、蔬菜、豆腐、鱼片。避免酱料过量（麻酱、花生酱高脂肪）。

**看标签识脂肪**：关注"脂肪"总量和"饱和脂肪"含量。"反式脂肪酸"必须标注为0g才安全。配料表中"氢化""部分氢化""植脂末""起酥油""代可可脂"均意味含反式脂肪。

### KI-HLP-006 高脂血症中医辨证食养

- **knowledge_id**: `KI-HLP-006`
- **title**: 高脂血症中医辨证食养方案
- **category**: `TCM_DIETARY_THERAPY`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["tcm_dietary", "herbal_food"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["中医食养", "辨证施食", "食药物质", "痰浊阻遏", "气滞血瘀", "脾虚湿盛", "肝肾阴虚"]`

**content**:

高脂血症在中医属"血浊""痰湿""膏脂"范畴。中医认为脂质代谢失常与脾失健运、肝失疏泄、肾精不足有关。

**证型一：痰浊阻遏型**

主要表现：形体肥胖，头重如裹，胸闷脘痞，肢体沉重，口黏腻，大便黏滞不爽，舌胖苔白腻，脉滑。此型最常见，多见于肥胖合并高TG者。

食养原则：化痰降浊，健脾利湿。

推荐食药物质与食养方：

荷叶（化浊降脂）：荷叶茶——干荷叶15g切碎，沸水冲泡代茶，每日1-2次。陈皮（理气化痰）：陈皮普洱茶——陈皮3g、普洱茶5g冲泡。薏苡仁（健脾利湿）：薏苡仁红豆粥——薏苡仁30g、赤小豆20g、粳米50g同煮。山楂（消食化积降脂）：山楂荷叶饮——山楂15g、荷叶半张，煎水代茶。

饮食禁忌：忌肥甘厚味、油炸食品、甜食、冷饮。

**证型二：气滞血瘀型**

主要表现：胸闷胁痛，烦躁易怒，肢体麻木，面色晦暗，口唇紫暗，舌暗有瘀点瘀斑，脉弦涩。此型多见于长期高脂血症合并动脉硬化者。

食养原则：行气活血，化瘀降脂。

推荐食药物质与食养方：

山楂（活血化瘀消积）：山楂丹参茶——山楂15g、丹参（需辨别是否为食药物质，建议在医指导下）代茶饮。玫瑰花（疏肝理气活血）：玫瑰花茶——玫瑰花5g沸水冲泡。黑木耳（活血化瘀）：凉拌黑木耳——黑木耳泡发后焯水，加醋和少许麻油拌食。洋葱（活血化瘀降脂）：洋葱炒木耳。

饮食禁忌：忌寒凉收涩、过量饮酒。

**证型三：脾虚湿盛型**

主要表现：食欲不振，腹胀便溏，面色萎黄，疲乏无力，肢体浮肿，舌淡胖有齿痕，苔白腻，脉沉弱。此型多见于体质虚弱或老年人。

食养原则：健脾益气，利湿化浊。

推荐食药物质与食养方：

山药（健脾益气）：山药薏苡仁粥——山药30g、薏苡仁20g、粳米50g同煮。茯苓（健脾利湿）：茯苓饼（自制）或茯苓粥——茯苓粉15g入粥。白扁豆（健脾化湿）：白扁豆莲子粥——白扁豆15g、莲子10g、粳米50g同煮。黄芪（补气健脾）：黄芪党参鸡汤——黄芪15g、党参10g炖鸡（去皮）。

饮食禁忌：忌生冷瓜果、油腻难消化食物。

**证型四：肝肾阴虚型**

主要表现：头晕耳鸣，腰膝酸软，五心烦热，口干咽燥，视物模糊，失眠多梦，舌红少苔，脉细数。此型多见于老年或合并高血压者。

食养原则：滋补肝肾，养阴降脂。

推荐食药物质与食养方：

枸杞子（滋补肝肾）：枸杞菊花茶——枸杞15g、菊花5g沸水冲泡。桑椹（滋阴补血降脂）：桑椹酸奶——桑椹30g拌入无糖酸奶。黑芝麻（补肝肾润燥）：黑芝麻核桃粉——黑芝麻20g、核桃15g研粉冲服。首乌（需在医指导下）有降脂养血作用，但须注意肝毒性风险，不建议自行使用。

饮食禁忌：忌辛燥温热、煎炸、浓茶咖啡过量。

**四型共同原则**：均需配合现代营养学膳食调整（控脂、增纤维、DASH或地中海饮食模式）；食养为辅助手段，不能替代降脂药物；证型可能兼夹（如痰浊+血瘀），食养方案需综合。

### KI-HLP-007 高脂血症运动与生活方式

- **knowledge_id**: `KI-HLP-007`
- **title**: 高脂血症运动处方与生活方式管理
- **category**: `EXERCISE_LIFESTYLE`
- **life_dimension**: `EXERCISE`
- **behavior_types**: `["aerobic_exercise", "resistance_training", "lifestyle_modification"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["运动处方", "降脂运动", "有氧运动", "戒烟限酒", "体重管理"]`

**content**:

**运动对血脂的影响**：

| 指标 | 运动效果 | 所需运动量 |
|------|---------|-----------|
| TG | ↓ 15%-30% | 中等强度有氧≥150分钟/周 |
| HDL-C | ↑ 5%-10% | 持续有氧运动≥12周 |
| LDL-C | ↓ 5%-10% | 有氧+饮食联合干预 |
| 体重 | ↓ 减重3-5%即改善血脂 | 能量负平衡+运动 |

**有氧运动处方**（降TG、升HDL-C最有效）：

频率：每周5-7天。强度：中等强度（RPE 12-13分，稍有呼吸急促但可对话）。时间：每次30-60分钟（可分段累计，每段≥10分钟）。类型：快走、慢跑、游泳、骑自行车、跳绳、健身操。

**抗阻运动**（辅助改善胰岛素敏感性和身体成分）：

频率：每周2-3天。强度：60%-70% 1RM。方案：大肌群训练，每个肌群2-3组×8-12次。

**生活方式其他要点**：

戒烟：吸烟降低HDL-C约5%，戒烟后HDL-C可回升。限酒：酒精直接升高TG。高TG者（≥2.3 mmol/L）应完全戒酒。正常血脂者限量：男≤25g/d酒精，女≤15g/d。体重管理：BMI≥24者减重目标3%-5%即可显著改善血脂谱。保持规律作息，充足睡眠（7-9小时/晚）。减少长时间久坐（每30-60分钟起身活动）。

### KI-HLP-008 高脂血症地域与季节食养

- **knowledge_id**: `KI-HLP-008`
- **title**: 高脂血症地域与季节食养调适
- **category**: `REGIONAL_SEASONAL`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["seasonal_diet", "regional_adaptation"]`
- **applicable_stages**: `[S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["地域食养", "季节食养", "南北差异", "四季调养", "高脂血症"]`

**content**:

**四季食养**：

春季：宜疏肝理气、清淡降脂。推荐芹菜、荠菜、菊花茶、绿茶。减少冬季滋补惯性，从厚味向清淡过渡。食养方：菊花山楂茶。

夏季：宜清热利湿、化浊降脂。推荐冬瓜、苦瓜、绿豆、荷叶。出汗多注意补水但控制含糖饮料。食养方：荷叶冬瓜汤、绿豆薏仁粥。

秋季：宜润燥养阴、平补降脂。推荐山药、百合、梨、银耳、黑芝麻。注意秋燥伤阴，阴虚型高脂血症者食养加强。食养方：银耳百合羹、黑芝麻核桃糊。

冬季：宜温补不燥、控制热量。冬季进补易致血脂升高，需特别注意控制肉类和油脂摄入。推荐核桃、黑豆、山药、木耳。避免过度"贴秋膘"。食养方：山药枸杞粥、木耳炖鸡（去皮）。

**地域调适**：

北方地区：高脂肪饮食习惯较重，炖肉、酱肉常见。重点减少动物脂肪，增加蔬果和全谷物，减少腌制品。

南方地区：煲汤文化需注意撇去汤面浮油，减少猪骨浓汤频率。增加蒸鱼、清炒蔬菜比例。利用本地水果资源增加膳食纤维。

西部地区：牛羊肉为主食，饮食偏高脂。推荐增加全谷杂粮（青稞、荞麦），少油烹饪，增加蔬菜摄入。

沿海地区：海产品丰富有利于omega-3摄入。注意海鲜胆固醇：虾蟹类适量，贝壳类胆固醇较高控量。推荐清蒸鱼代替油炸鱼。

### KI-HLP-009 不同类型高脂血症个性化食养策略

- **knowledge_id**: `KI-HLP-009`
- **title**: 不同类型高脂血症个性化食养策略
- **category**: `PERSONALIZED_STRATEGY`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["dietary_pattern", "food_selection"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["高胆固醇血症", "高甘油三酯血症", "混合型", "低HDL-C", "个性化策略"]`

**content**:

**高胆固醇血症（TC↑ 和/或 LDL-C↑）重点策略**：

第一优先级——减少饱和脂肪酸：<7%总能量（约15-16g/天按2000kcal计算）。具体措施：红肉≤50g/天（选瘦肉）、去皮禽肉、低脂奶制品、避免动物油脂烹饪。

第二优先级——限制膳食胆固醇：<200mg/天。具体措施：蛋黄每周3-4个、避免动物内脏、减少虾蟹摄入频率。

第三优先级——增加降TC功能食物：燕麦（β-葡聚糖≥3g/天）、大豆蛋白（25g/天）、植物固醇（2g/天）。

组合效果：以上三项联合可降低LDL-C约20%-30%，部分患者可达到或替代低剂量他汀效果。

**高甘油三酯血症（TG↑）重点策略**：

第一优先级——控制碳水化合物质量：减少精制谷物和添加糖（白米饭/白面条减量，添加糖<25g/天）。增加全谷物比例至50%以上。

第二优先级——严格限酒或戒酒：TG≥2.3 mmol/L建议完全戒酒。酒精直接刺激肝脏TG合成。

第三优先级——增加omega-3摄入：每周深海鱼3次以上。严重高TG（≥5.6 mmol/L）可在医生指导下补充高纯度EPA/DHA 2-4g/天。

第四优先级——控制总能量和减重：超重者减重3-5%可降TG约15-30%。

**混合型高脂血症（TC↑ + TG↑）策略**：

综合以上两种策略，优先级排序：控制饱和脂肪≥控酒≥减精制碳水≥增膳食纤维≥增omega-3。此型饮食控制最严格，可能需要药物辅助。

**低HDL-C血症策略**：

药物和饮食升高HDL-C效果均有限。最有效措施：有氧运动（可升5%-10%）、戒烟（可升5%）、减重（可升2%-4%/每减重1kg）。饮食辅助：适量坚果、深海鱼、橄榄油；避免反式脂肪酸（最强降HDL-C因素）。

### KI-HLP-010 高脂血症血脂监测与自我管理

- **knowledge_id**: `KI-HLP-010`
- **title**: 血脂监测规范与自我管理
- **category**: `SELF_MONITORING`
- **life_dimension**: `RISK_FACTOR`
- **behavior_types**: `["self_monitoring", "data_tracking", "medication_adherence"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高脂血症食养指南（2023年版）》
- **tags**: `["血脂监测", "检查规范", "达标标准", "自我管理", "服药依从性"]`

**content**:

**血脂检测规范**：

空腹采血（禁食12小时以上，可少量饮水）。检测前3天避免高脂饮食。检测前24小时避免饮酒和剧烈运动。注意：non-HDL-C可在非空腹状态下评估，逐渐成为更便捷的评估指标。

**血脂监测频率**：

| 阶段 | 频率 |
|------|------|
| 初次确诊/调整方案 | 每4-6周复查 |
| 生活方式干预初期 | 每3个月复查 |
| 药物治疗初期 | 每1-3个月复查直到达标 |
| 达标稳定期 | 每6-12个月复查 |
| ≥40岁常规筛查 | 每年至少1次 |

**血脂管理"三色预警"**（CoachAgent决策参考）：

绿灯（维持管理）：TC<5.2，TG<1.7，LDL-C<3.4，HDL-C≥1.0。维持当前生活方式，定期复查。

黄灯（强化干预）：TC 5.2-6.2，TG 1.7-5.6，LDL-C 3.4-4.1。强化生活方式干预3-6个月，未达标则考虑药物治疗。

红灯（紧急关注）：TC≥6.2，TG≥5.6（急性胰腺炎风险），LDL-C≥4.1，或已有ASCVD。立即就医评估，药物治疗+强化生活方式。

**自我管理要点**：建立血脂记录档案（含日期、检测值、当时用药和饮食情况）。他汀类药物服药依从性极为关键，不可自行停药。记录膳食日记（重点记录油脂、肉类、甜食摄入）。每周记录运动量和体重变化。

## 五、CoachAgent决策支持要点

### 5.1 高脂血症用户初次评估流程

1. 确认血脂类型（TC↑/TG↑/混合/HDL-C↓）及具体数值
2. 确认是否正在服用降脂药（他汀/贝特/依折麦布等）
3. 评估当前膳食模式（油脂摄入、饮酒量、甜食/精白主食摄入）
4. 评估BMI和运动习惯
5. 中医辨证分型初筛
6. 评估行为改变阶段
7. 根据血脂类型生成个性化方案（KI-HLP-009）

### 5.2 优先级排序（基于降脂效果循证证据）

**降LDL-C优先级**：

| 干预措施 | 预期降LDL-C幅度 | 优先级 |
|---------|----------------|-------|
| 减饱和脂肪<7% | 8%-10% | ★★★★★ |
| 燕麦β-葡聚糖≥3g/d | 5%-10% | ★★★★ |
| 大豆蛋白25g/d | 4%-6% | ★★★★ |
| 植物固醇2g/d | 6%-15% | ★★★★ |
| 膳食胆固醇<200mg/d | 3%-5% | ★★★ |
| 减重5% | 5%-8% | ★★★ |
| 组合疗法（以上全部） | 20%-30% | ★★★★★ |

**降TG优先级**：

| 干预措施 | 预期降TG幅度 | 优先级 |
|---------|-------------|-------|
| 戒酒 | 15%-30% | ★★★★★ |
| 控制添加糖和精制碳水 | 10%-20% | ★★★★★ |
| 有氧运动≥150分/周 | 15%-30% | ★★★★ |
| 减重3%-5% | 15%-30% | ★★★★ |
| 增加omega-3 | 10%-25% | ★★★ |

### 5.3 对话策略提示

**初次接触血脂异常用户**：
- 先确定血脂类型，不同类型干预策略完全不同
- 高TC用户从"控脂换油"切入，高TG用户从"戒酒控糖"切入
- 用"血管里的坏脂肪堵塞管道"等类比解释ASCVD风险

**常见抵触应对**：
- "我吃素也高" → 部分源于遗传，另外素食也可能高碳水高糖，关注整体膳食结构
- "吃药就行不用改饮食" → 药物+饮食联合效果远优于单独药物，且可能减少药量和副作用
- "坚果不是高脂肪吗" → 原味坚果含不饱和脂肪酸，适量（10g/天）有助降脂

**⚠️ 红线提醒**：
- 不能建议停用/减用降脂药物，药物调整须由医生进行
- TG≥5.6 mmol/L是急性胰腺炎高风险，必须提醒立即就医
- 红曲制品+他汀药物可能叠加副作用（肌痛/横纹肌溶解风险），必须提醒
- 何首乌（制首乌）有肝毒性报道，不建议自行使用
