## 第一部分：疾病基础与流行病学

### 1.1 核心概念

| 术语 | 定义 |
|------|------|
| 尿酸（Uric Acid） | 嘌呤化合物经肝脏代谢的终产物，通过肾脏（约2/3）和消化道（约1/3）排泄 |
| 嘌呤（Purine） | 核酸组成成分，内源性（体内合成+核酸降解，约80%）与外源性（膳食摄入，约20%）两个来源 |
| 高尿酸血症（HUA） | 正常膳食状态下，非同日2次检测空腹血尿酸水平 >420 μmol/L |
| 痛风（Gout） | 单钠尿酸盐晶体沉积引起的关节炎性疾病，高尿酸血症是其病理基础 |
| 食养（Dietary Therapy） | 以中医辨证施膳为指导的非药物调养方法，根据病因、证候、体质给予不同食养方案 |
| 食药物质 | 按照传统既是食品又是中药材的物质，可融入合理膳食中辅助预防和改善疾病 |

### 1.2 流行病学数据（中国）

| 指标 | 数据 | 来源 |
|------|------|------|
| 成人HUA患病率 | 14% | 2018-2019中国慢性病及危险因素监测 |
| 痛风患病率 | 0.86%~2.20% | 同上 |
| 性别差异 | 男性 > 女性 | 男性发病率约为女性3~4倍 |
| 地域差异 | 城市 > 农村，沿海 > 内陆 | 与膳食结构和生活方式相关 |
| 发病趋势 | 逐年上升，发病年龄趋于年轻化 | 国家流行病学调查 |

### 1.3 危险因素

- **膳食因素**：长期摄入高嘌呤食物、高能量食品、大量酒精和/或高果糖饮料
- **肥胖**：高尿酸血症与痛风发生的独立危险因素
- **遗传因素**：嘌呤代谢酶和尿酸转运蛋白基因异常
- **其他**：久坐、高血压、慢性肾病、代谢综合征、某些药物（利尿剂等）

## 第二部分：西医疾病分期与中医辨证分型

### 2.1 西医疾病分期

| 分期 | 临床特征 | 膳食管理重点 |
|------|---------|------------|
| **无症状高尿酸血症期** | 血尿酸 >420 μmol/L，无关节炎症状 | 低嘌呤膳食+控制体重+充分饮水 |
| **急性痛风性关节炎期** | 关节红肿热痛，多发于第一跖趾关节 | 严格限制嘌呤，充分饮水，忌酒，清淡饮食 |
| **痛风间歇期** | 急性发作缓解后的无症状间期 | 坚持低嘌呤膳食，避免诱发因素 |
| **慢性痛风性关节炎期** | 反复发作，痛风石形成，关节畸形 | 长期严格低嘌呤膳食+碱化尿液+综合管理 |

### 2.2 中医辨证分型

中医认为痛风属"痹证""历节病"范畴。先天脾胃虚弱或后天饮食失养、损伤脾胃是痛风发病的关键因素，脾胃失调、湿浊内生是痛风反复不愈的症结所在。

#### 2.2.1 湿浊证（常见于无症状HUA期和痛风间歇期）

| 项目 | 内容 |
|------|------|
| 体质类型 | 痰湿体质多见 |
| 临床表现 | 肢体困乏沉重，形体肥胖，嗜食肥甘，口中粘腻不渴，大便黏 |
| 舌脉 | 舌淡胖，或有齿痕，苔白腻，脉滑 |
| 食养要点 | 避免寒冷诱发，严格控制肥甘厚味，限制嘌呤和蛋白质总量，低盐膳食，尤忌甜、油腻、酸、涩食物 |
| 推荐食药物质 | 橘皮、生姜、赤小豆、郁李仁、茯苓、葛根、菊苣、百合、砂仁、木瓜、黄芪 |

#### 2.2.2 湿热证（常见于急性痛风性关节炎期）

| 项目 | 内容 |
|------|------|
| 体质类型 | 湿热体质多见 |
| 临床表现 | 关节红肿热痛，拒按，触之灼热，发热口渴，心烦不安，小便短黄 |
| 舌脉 | 舌红，苔黄腻，脉滑数 |
| 食养要点 | 清热利湿，以清淡、寒凉性食物为主，避免辛辣、燥热、油腻食物 |
| 推荐食药物质 | 薏苡仁、赤小豆、菊苣、茯苓、葛根、百合、山药、马齿苋、栀子、鱼腥草 |

#### 2.2.3 痰瘀证（常见于痛风间歇期至慢性期过渡阶段）

| 项目 | 内容 |
|------|------|
| 体质类型 | 痰湿/血瘀兼见 |
| 临床表现 | 关节肿胀刺痛，僵硬变形，局部皮下结节（痛风石），肢体麻木 |
| 舌脉 | 舌暗紫或有瘀斑，苔白腻，脉弦涩 |
| 食养要点 | 化痰祛瘀、通络止痛，食用活血化瘀类食药物质 |
| 推荐食药物质 | 山楂、桃仁、薏苡仁、橘皮、生姜、茯苓、木瓜、菊苣 |

#### 2.2.4 脾肾亏虚证（常见于慢性痛风性关节炎期）

| 项目 | 内容 |
|------|------|
| 体质类型 | 气虚/阳虚体质多见 |
| 临床表现 | 关节隐痛反复发作，劳累后加重，头晕耳鸣，腰膝酸软，畏寒肢冷 |
| 舌脉 | 舌淡胖，苔薄白，脉沉弱或沉细 |
| 食养要点 | 健脾补肾、益气扶正，食用温补性食物，避免寒凉食物 |
| 推荐食药物质 | 黄芪、山药、茯苓、枸杞子、百合、杜仲叶、菟丝子、桑椹 |

### 2.3 西医分期与中医证型对应关系

```
无症状HUA期 ──→ 湿浊证为主
急性痛风发作期 ──→ 湿热证为主
痛风间歇期 ──→ 湿浊证/痰瘀证
慢性痛风关节炎期 ──→ 痰瘀证/脾肾亏虚证
```

> **平台逻辑**：tcm Agent根据用户疾病分期自动匹配中医证型，推荐对应食药物质。若用户已有中医辨证结论则优先使用。

## 第三部分：七大食养原则与建议

### 原则一：食物多样，限制嘌呤

**核心规则**：
- 每天食物品种 ≥12种，每周 ≥25种
- 每天保证谷薯类、蔬菜和水果、畜禽鱼蛋奶、大豆和坚果摄入
- 以低嘌呤膳食为主，严格控制膳食中嘌呤含量
- 有痛风发作病史者需避免既往诱发发作的食物

**关于豆制品**：大豆嘌呤含量略高于瘦肉和鱼类，但植物性食物中嘌呤人体利用率低。豆腐、豆干等加工后嘌呤含量降低，可适量食用。

### 原则二：蔬奶充足，限制果糖

**核心规则**：
- 每天新鲜蔬菜 ≥500g，深色蔬菜占一半以上
- 每天奶及奶制品 ≥300mL（乳蛋白可促进尿酸排泄）
- 限制高果糖食品：含糖饮料、鲜榨果汁、果葡糖浆、果脯蜜饯
- 水果每天200~350g（水果中维C、黄酮、多酚、钾、膳食纤维可抵消果糖影响）

### 原则三：足量饮水，限制饮酒

**核心规则**：
- 心肾功能正常者每天饮水 2000~3000mL，维持尿量 >2000mL
- 优先白水，可选柠檬水、淡茶、无糖咖啡、苏打水
- 避免浓茶、浓咖啡、生冷饮品
- **限制饮酒**：急性痛风发作、药物控制不佳或慢性痛风性关节炎者应完全禁酒
- 酒类嘌呤排序：黄酒 > 啤酒 > 白酒（但白酒酒精度高→乳酸堆积→抑制尿酸排泄）

### 原则四：科学烹饪，少食生冷

**核心规则**：
- 每天食盐 ≤5g，烹调油 ≤25~30g
- 减少油炸、煎制、卤制等烹饪方式
- **肉类汆煮后食用，尽量不喝汤**（肉汤嘌呤含量高）
- 禁食腊制、腌制、熏制肉类（嘌呤+盐分双高）
- 少吃生冷食品（冰激凌、生冷海鲜）→损伤脾胃+尿酸盐结晶析出↑→诱发痛风

### 原则五：吃动平衡，健康体重

**核心规则**：
- 健康BMI：18~64岁成人 18.5~23.9 kg/m²，≥65岁老年人 20.0~26.9 kg/m²
- 超重肥胖→痛风风险↑，减重可显著降低血尿酸水平
- 选择适合自身的运动项目，循序渐进
- 提升心肺功能和肌肉耐力、力量、柔韧性，增强代谢与免疫功能
- **注意**：剧烈运动可能短暂升高血尿酸，以中低强度有氧运动为宜

### 原则六：辨证辨体，因人施膳

**核心规则**：
- 准确辨证是选择食药物质的前提
- 根据"因人、因时、因地"三因原则选择食药物质
- 食药物质提供的能量与宏量营养素需纳入全天饮食总量
- 对食药物质过敏者、正在服药可能有禁忌时，需专业人员指导

### 原则七：因地因时，择膳相宜

**核心规则**：

| 季节 | 食养重点 | 生活起居 |
|------|---------|---------|
| 春季 | 养肝，膳食清淡爽口 | 夜卧早起，适寒温 |
| 夏季 | 祛暑，清淡营养易消化 | 注意空调温度不过低 |
| 秋季 | 养阴润肺，燥气当道 | 早卧早起 |
| 冬季 | 温补，注意御寒保暖 | 早卧晚起，居处保暖 |

> **特别提醒**：痛风性关节炎遇冷易诱发，秋冬、夏秋季节交替时特别注意保暖。

## 第四部分：食物嘌呤含量分级

### 4.1 食物嘌呤分类表（mg/100g可食部分）

#### 第一类：高嘌呤食物（>150mg/100g）—— 痛风发作期禁食，缓解期严格限制

| 分类 | 食物 |
|------|------|
| 动物内脏 | 猪肝（275）、鸡肝（317）、猪肾（132~195）、鸭肝（301）、鹅肝（408） |
| 部分海鲜 | 凤尾鱼（363）、沙丁鱼（295）、白带鱼（391）、秋刀鱼（355） |
| 浓肉汤/浓鱼汤 | 火锅汤底（含嘌呤溶出物极高） |
| 其他 | 酵母粉（559）、牡蛎干（239） |

#### 第二类：中高嘌呤食物（75~150mg/100g）—— 缓解期限量摄入

| 分类 | 食物 |
|------|------|
| 畜肉 | 猪瘦肉（90~122）、牛肉（84~114）、羊肉（76~111） |
| 禽肉 | 鸡肉（84~138）、鸭肉（80~138） |
| 部分鱼虾 | 草鱼（112）、鲤鱼（89）、鲈鱼（82）、虾（78~131） |
| 豆类 | 黄豆（116~190）、黑豆（137）、绿豆（76） |

#### 第三类：中低嘌呤食物（30~75mg/100g）—— 可适量食用

| 分类 | 食物 |
|------|------|
| 豆制品 | 豆腐（56~68）、豆干（63~68）、豆浆（28~44） |
| 部分蔬菜 | 菠菜（57）、蘑菇（35~65）、花椰菜（47）、芦笋（53） |
| 谷物 | 糙米（37）、燕麦（42）、全麦面包（35） |

#### 第四类：低嘌呤食物（<30mg/100g）—— 自由食用

| 分类 | 食物 |
|------|------|
| 奶蛋类 | 牛奶（1.4）、鸡蛋（3.7）、奶酪（5.6） |
| 主食 | 大米（18）、馒头（19）、面条（17）、玉米（10） |
| 蔬菜 | 白菜（11）、黄瓜（8）、番茄（4.3）、茄子（7.3）、冬瓜（3） |
| 水果 | 苹果（1.3）、梨（1.1）、西瓜（1.0）、葡萄（0.9） |
| 薯类 | 土豆（6.5）、红薯（4.8） |

> **平台规则**：nutrition Agent在生成膳食建议时，必须查阅此嘌呤分级表，优先推荐第四类食物，限制第一类食物。

## 第五部分：每日膳食摄入推荐量

### 5.1 宏量营养素推荐

| 营养素 | 推荐占比 | 说明 |
|--------|---------|------|
| 碳水化合物 | 总能量的50%~60% | 选择全谷物、薯类等复合碳水 |
| 蛋白质 | 总能量的10%~15% | 优质蛋白（乳制品、蛋类）优先，肉类汆煮去嘌呤后食用 |
| 脂肪 | 总能量的25%~30% | 减少饱和脂肪酸摄入，增加不饱和脂肪酸 |

### 5.2 每日摄入量参考

| 食物类别 | 推荐量 | 注意事项 |
|---------|--------|---------|
| 谷薯类 | 250~400g | 全谷物和杂豆占1/3以上 |
| 蔬菜 | ≥500g | 深色蔬菜占1/2以上 |
| 水果 | 200~350g | 限制高果糖品种 |
| 奶及奶制品 | ≥300mL | 低脂或脱脂奶优先 |
| 畜禽肉 | 40~75g | 汆煮后食用，不喝肉汤 |
| 蛋类 | 1个/天 | 低嘌呤优质蛋白来源 |
| 食盐 | ≤5g | 包括酱油、调味品中的盐 |
| 烹调油 | 25~30g | 减少煎炸 |
| 饮水 | 2000~3000mL | 优先白水、柠檬水、苏打水 |

## 第六部分：特殊时期膳食管理

### 6.1 急性痛风发作期膳食

**核心原则**：严格低嘌呤 + 充分饮水 + 完全禁酒 + 清淡饮食

| 项目 | 要求 |
|------|------|
| 嘌呤摄入 | 每日膳食嘌呤 <150mg，仅选用第四类低嘌呤食物 |
| 蛋白来源 | 以鸡蛋、牛奶为主要蛋白质来源 |
| 碳水来源 | 精制谷物为主（嘌呤更低） |
| 蔬菜 | 大量低嘌呤蔬菜，避免食用菠菜、蘑菇等中嘌呤蔬菜 |
| 禁食 | 所有动物内脏、海鲜、肉汤、酒类 |
| 饮水 | ≥3000mL/天，促进尿酸排泄 |
| 烹饪方式 | 蒸、煮、炖为主，忌煎炸、卤制 |

### 6.2 缓解期/间歇期膳食

**核心原则**：低嘌呤为主 + 适量优质蛋白 + 控制总能量

| 项目 | 要求 |
|------|------|
| 嘌呤摄入 | 每日膳食嘌呤 <300mg |
| 蛋白来源 | 可适量食用畜禽肉（汆煮去嘌呤后），40~75g/天 |
| 豆制品 | 豆腐、豆浆可适量食用 |
| 海鲜 | 可少量选用低嘌呤鱼类，避免贝类和沙丁鱼等高嘌呤品种 |
| 饮酒 | 限量或不饮，完全避免黄酒和啤酒 |

### 6.3 合并代谢综合征时的膳食调整

| 合并症 | 额外膳食要点 |
|--------|------------|
| 肥胖 | 控制总能量，减少精制碳水，增加膳食纤维 |
| 高血压 | 食盐≤5g/天，增加钾摄入（蔬菜水果），限酒 |
| 糖尿病 | 控制碳水总量和升糖指数，均衡三餐分配 |
| 高脂血症 | 限制饱和脂肪和反式脂肪，增加不饱和脂肪酸 |

> **平台逻辑**：当用户同时存在HUA+其他代谢疾病标签时，nutrition Agent需综合多个知识库（本指南+糖尿病膳食+高血压膳食）给出交集最优方案。

- **膳食建议输出**：以"可食/限食/禁食"三级分类呈现，≤300字，附具体食物举例
- **嘌呤查询输出**：查阅第四部分食物嘌呤表，给出具体数值+分级+食用建议
- **证型匹配输出**：先确认证型→匹配食药物质→推荐1~2个茶饮方，≤200字
- **急性期警告输出**：标红显示禁食清单，强调饮水量≥3000mL，附Safety Trigger联动

## 第九部分：与其他知识模块的集成逻辑

### 9.1 与血尿酸监测知识库（Urate Monitoring Base）

- 血尿酸 >420 μmol/L → 触发本指南食养方案推送
- 血尿酸 >540 μmol/L且无症状 → 建议药物干预+食养方案双管齐下
- 血尿酸动态趋势持续下降 → 作为食养干预有效性的客观验证指标

### 9.2 与ACSM运动处方知识库

- HUA/痛风患者运动推荐联动ACSM第五部分（代谢疾病FITT）
- 中低强度有氧运动为宜（40~59% VO₂R/HRR），避免剧烈运动导致乳酸堆积
- 急性痛风发作期暂停运动，待关节炎症完全消退后逐步恢复
- 超重肥胖HUA患者联动ACSM肥胖FITT方案，目标250~300 min/周

### 9.3 与大五人格知识库（BFP Base）

- 高尽责性（BFP-C↑）：适合严格嘌呤记录+食物称重+每日饮水打卡
- 低尽责性（BFP-C↓）：简化为"红绿灯食物卡"（红-禁食/黄-限食/绿-自由），降低执行门槛
- 高神经质（BFP-N↑）：避免过度强调"禁食"压力，转为正向引导"可以吃什么"
- 高外向性（BFP-E↑）：注意社交聚餐场景（火锅、烧烤、饮酒）的风险提示和替代策略
- 低开放性（BFP-O↓）：推荐简单易执行的食谱模板，避免复杂食药物质搭配

### 9.4 与BPT6/TTM行为改变模型

- **前意向期用户**：推送HUA/痛风流行病学数据+膳食与尿酸的关系证据（第一部分）
- **意向期用户**：提供嘌呤分级表+简化膳食对照卡（第四部分）
- **准备期用户**：制定具体食谱计划+SMARTS目标（如"本周每天饮水≥2000mL"）
- **行动期用户**：每周追踪嘌呤摄入合规率+饮水达标率+血尿酸变化趋势
- **维持期用户**：推送季节性食养方案调整+社交场景应对策略+复发预防

### 9.5 与中医体质知识库（TCM Base）

| 体质类型 | HUA/痛风相关性 | 膳食策略优先级 |
|---------|---------------|-------------|
| 痰湿体质 | 高风险，HUA最常见体质 | 湿浊证食养方案为主→严格控制肥甘厚味 |
| 湿热体质 | 高风险，易急性发作 | 湿热证食养方案为主→清热利湿食物优先 |
| 血瘀体质 | 中等风险，慢性期多见 | 痰瘀证食养方案→活血化瘀食药物质 |
| 气虚体质 | 中等风险，长期病程后 | 脾肾亏虚证方案→健脾补气为主 |
| 阳虚体质 | 注意寒冷诱发 | 温补方案+保暖→避免寒凉食物 |
| 平和体质 | 低风险 | 标准低嘌呤膳食+预防性食养 |

### 9.6 与危机干预规则

| 触发条件 | 响应级别 | 处理方式 |
|---------|---------|---------|
| 血尿酸 >600 μmol/L | P1（24h内人工教练介入） | 转介医生评估，同时启动严格食养方案 |
| 急性痛风发作中 | P1 | 推送急性期膳食方案+建议就医+禁止推荐任何可能升高尿酸的食物 |
| 用户报告多关节同时发作或持续发热 | P0（Safety Trigger） | 立即停止膳食指导→紧急提示就医→可能为感染性关节炎需排除 |
| 用户肾功能异常（eGFR<60或肌酐升高） | P0 | 超出本指南适用范围→立即转介肾病专科→暂停本指南食养推送 |
