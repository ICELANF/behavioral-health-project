## 第一部分：疾病认知与流行病学基础

### 1.1 中国糖尿病流行现状

- 中国≥18岁人群糖尿病患病率从1980年的0.67%升至2018—2019年的12.4%（ADA 2010标准），居全球首位。
- 糖尿病人群中2型糖尿病（T2DM）占90%以上，男性患病率（13.3%）高于女性（11.5%）。
- 糖尿病知晓率36.7%、治疗率32.9%、控制率50.1%，仍处于较低水平。
- 未诊断糖尿病比例居高不下（约54%~63%）。

### 1.2 肥胖与T2DM的关联

- 中国肥胖/超重人群糖尿病患病率为36.8%，58%的糖尿病患者超重或肥胖。
- BMI≥30 kg/m²人群糖尿病患病率约20.1%，远高于正常体重人群（8.8%）。
- 肥胖是T2DM的首要病因，增加体重或腰围可加剧胰岛素抵抗，使血糖控制更加困难。

### 1.3 糖尿病逆转的科学依据

- 2016年WHO承认糖尿病可通过减重和热量限制实现逆转。
- β细胞去分化而非凋亡是T2DM的核心机制，通过生活方式干预可促进β细胞再分化、恢复胰岛功能。
- 肝脏脂肪沉积是胰岛素抵抗的主要来源，减重可减少肝脏脂肪、恢复肝脏胰岛素敏感性。
- 大庆研究证实：生活方式干预不仅延缓T2DM发生，还能降低心脑血管事件、微血管并发症及全因死亡风险，延长预期寿命。

## 第二部分：诊断标准与评估分级体系

### 2.1 糖尿病诊断标准（WHO 1999）

| 诊断标准 | 静脉血浆葡萄糖水平 |
|---------|----------------|
| 典型症状+随机血糖 | ≥11.1 mmol/L |
| 空腹血糖 | ≥7.0 mmol/L |
| OGTT 2h血糖 | ≥11.1 mmol/L |
| HbA1c（ADA 2010） | ≥6.5% |

### 2.2 肥胖诊断标准（中国成人标准）

| 指标 | 标准 |
|------|------|
| 超重（BMI） | ≥24 kg/m² |
| 肥胖（BMI） | ≥28 kg/m² |
| 腹型肥胖-腰围（男） | ≥90 cm |
| 腹型肥胖-腰围（女） | ≥85 cm |

### 2.3 患者评估与分级（ABCD框架）

**评估维度**：A = 年龄/HbA1c; B = BMI/血压; C = 胆固醇/吸烟; D = 饮食/病程

**疾病分级矩阵**（BMI × HbA1c）：

| 肥胖分类（BMI） | 糖尿病早期 | 1级(≤7.5%) | 2级(7.5-9%) | 3级(>9%) |
|---------------|---------|-----------|------------|---------|
| 超重(24-27.9) | 1级 | 2级 | 2级 | 3级 |
| 肥胖1度(28-32.9) | 2级 | 2级 | 3级 | 3级 |
| 肥胖2度(33-37.9) | 2级 | 3级 | 3级 | 4级 |
| 肥胖3度(≥38) | 3级 | 3级 | 4级 | 4级 |

### 2.4 生活方式评估维度

- 饮食习惯（结构、频次、热量）
- 运动习惯（类型、频率、强度）
- 运动意愿与可行性（体能评估 + 环境评估）
- 吸烟饮酒状况
- 心理评估（生活质量、睡眠状况、焦虑/抑郁筛查）

## 第三部分：糖尿病逆转定义与分期目标

### 3.1 糖尿病缓解/逆转定义

| 类别 | ADA 2009标准 | 国内实践标准 |
|------|-------------|----------|
| 部分缓解 | 无药物干预下HbA1c<6.5%，FPG 5.6-6.9 mmol/L，持续≥1年 | — |
| 完全缓解 | 无药物干预下HbA1c正常，FPG<5.6 mmol/L，持续≥1年 | — |
| 长期缓解 | 完全缓解持续≥5年 | — |
| ADA 2021新标准 | 停药≥3个月后HbA1c<6.5% | — |
| 国内标准 | — | 停药后FPG<7.0 mmol/L，餐后2h血糖<10 mmol/L，维持>1年 |

### 3.2 疾病分期与逆转目标

| 分期 | 特征描述 | 逆转目标 |
|------|--------|--------|
| **高危期** | 肥胖+胰岛素抵抗阶段 | 恢复正常体重，消除胰岛素抵抗 |
| **前期** | IGT/IFG阶段 | 逆转至正常糖耐量和正常空腹血糖 |
| **早期** | 新发糖尿病3年内，胰岛功能较好 | 90%可达正常血糖，无需用药 |
| **中期** | 病程>3年，胰岛功能尚可，无明显并发症 | 50%可停药，血糖正常 |
| **晚期** | 病程>10年，胰岛功能差，有明显并发症 | 血糖稳定，减药，并发症缓解 |

### 3.3 综合控制目标

| 指标 | 早期目标 | 中晚期目标 |
|------|--------|---------|
| HbA1c | <6.5-7.0% | 7.0-7.5% |
| 空腹血糖 | 完全正常 | 4.4-7.0 mmol/L |
| 餐后血糖 | 完全正常 | <10.0 mmol/L |
| 血压 | <140/90 mmHg | <140/90 mmHg |
| BMI | <24 kg/m² | <24 kg/m² |
| 腰围 | 男<85cm/女<80cm | 男<85cm/女<80cm |
| TG | <1.7 mmol/L | <1.7 mmol/L |
| LDL-C（无冠心病） | <2.6 mmol/L | <2.6 mmol/L |
| LDL-C（有冠心病） | <1.8 mmol/L | <1.8 mmol/L |

## 第四部分：生活方式干预六大支柱

### 4.1 营养干预（精准营养治疗）

#### 4.1.1 能量管理

- 按每日25-30 kcal/kg标准体重计算基础能量。
- 超重/肥胖患者首先通过饮食+运动+行为调整减轻体重≥5%。
- 不推荐长期极低能量饮食（<800 kcal/d）。

#### 4.1.2 宏量营养素分配

| 营养素 | 推荐供能比 | 要点 |
|-------|---------|------|
| 碳水化合物 | 45-60% | 选择低GI/GL食物，全谷物≥总谷物50%；严控精制糖 |
| 蛋白质 | 15-20% | 优质蛋白为主，避免长期高蛋白饮食 |
| 脂肪 | 20-30%（优质脂肪可至35%） | 限制饱和脂肪<12%，反式脂肪<2%，胆固醇<300mg/d |
| 膳食纤维 | 25-36 g/d | 可溶性纤维10-20 g/d，有助于血糖控制 |

#### 4.1.3 推荐膳食模式

- 地中海膳食、低碳水化合物膳食、间歇性禁食、江南膳食等在专业指导下短期有助于减重和代谢改善。
- 生酮饮食可作为特定阶段干预手段，需严格监测。
- 进餐定时定量，合理分配用餐时间以减少胰腺刺激，细嚼慢咽。
- 注意肠道健康，适当补充益生菌/益生元。

#### 4.1.4 药物-营养配合

- 使用GLP-1RA类药物时需保证充足优质蛋白（供能比≥15-20%），补充复合维生素。
- 长期服用二甲双胍者注意维生素B12缺乏。
- 注射胰岛素者应保持碳水化合物摄入与胰岛素剂量匹配。

### 4.2 运动治疗

#### 4.2.1 运动处方五要素

频率、强度、时间、类型、运动量

#### 4.2.2 核心运动建议

| 运动类型 | 推荐方案 |
|---------|--------|
| 有氧运动 | 每周150-300 min中等强度，每周3-7天，每次30-60 min |
| 抗阻训练 | 每周≥2次 |
| 柔韧性练习 | 每周2-3次，每个部位≥60秒 |
| 平衡训练 | 每周2-3次，降低跌倒风险 |
| 日常活动 | 每日步行5000-10000步，每30min久坐后活动1-5 min |

#### 4.2.3 运动安全管理

- 运动前血糖应在5.0-13.9 mmol/L之间。
- 禁止运动条件：血糖≥16.7 mmol/L、血糖<3.9 mmol/L、血糖剧烈波动、DKA、急性感染、严重并发症。
- 运动遵循"由轻到重、由少到多、由慢到快"的渐进原则。
- 根据CGM血糖监测数据及时调整运动方案。

### 4.3 压力管理与心理干预

- 肥胖和糖尿病增加患者心理负担，需加强专业心理干预。
- 心理评估维度：生活质量、睡眠质量、焦虑/抑郁状态。
- 血糖水平在情绪变化时也会波动，CGM可实时显示情绪诱导的葡萄糖反应。
- 通过放松练习、正念减压等行为干预改善血糖控制。
- 减重本身可克服心理障碍、增强自信，形成正向循环。

### 4.4 睡眠管理

- 评估并识别睡眠障碍（包括阻塞性睡眠呼吸暂停）。
- 阻塞性睡眠呼吸暂停与高血糖密切相关。
- 使用CGM期间如睡眠质量不佳，可调整血糖目标范围设定或取消报警功能。

### 4.5 社会连接与支持网络

- 增强患者社会支持网络，促进同伴支持。
- 通过线上平台实现远程随访和患者社群管理。
- 家庭参与在行为改变中至关重要。

### 4.6 避免高风险物质

- 戒烟是糖尿病综合管理的重要组成部分。
- 限制饮酒，警惕酒精诱发低血糖（尤其使用胰岛素/磺脲类药物者）。
- 针对烟草、酒精、娱乐性药物提供专业咨询。

## 第五部分：血糖监测与技术标准

### 5.1 持续葡萄糖监测（CGM）分类

| 类型 | 特点 | 适用场景 |
|------|------|--------|
| 回顾性CGM | 监测结束后上传分析，客观反映日常血糖 | 临床研究、基线评估 |
| 实时CGM | 即时血糖+高低报警+趋势箭头预警 | 血糖波动大、低血糖高风险 |
| 扫描式CGM（FGM） | 主动扫描获取数据，兼具两者特点 | 门诊/居家日常管理 |

### 5.2 CGM适用人群

| 证据等级 | 适用人群 |
|---------|--------|
| A | 所有1型糖尿病患者 |
| A | 接受胰岛素强化治疗或基础胰岛素治疗的T2DM患者 |
| B | 非胰岛素治疗但血糖未达标的T2DM患者 |
| A | 频发低血糖或低血糖高风险的T2DM患者 |
| B | 妊娠糖尿病、围手术期患者 |

### 5.3 CGM核心评价指标（10项标准化参数）

| 参数 | 说明/目标 |
|------|---------|
| 佩戴天数 | 推荐14天 |
| 使用时间占比 | ≥70% |
| 平均血糖（MG） | <8.5 mmol/L |
| 葡萄糖管理指标（GMI） | 预估HbA1c变化趋势 |
| 血糖变异系数（CV） | ≤36% |
| TIR（3.9-10.0 mmol/L） | >70%（一般T1DM/T2DM） |
| TAR-1级高血糖（>10.0） | <25% |
| TAR-2级高血糖（>13.9） | <5% |
| TBR-1级低血糖（3.0-3.8） | <4% |
| TBR-2级低血糖（<3.0） | <1% |

### 5.4 AGP图谱解读三步法

1. **第一步 — 看核心指标**：检查数据充足性（14天、≥70%覆盖），评估TIR/TAR/TBR和CV是否达标。达标则维持方案，不达标进入下一步。
2. **第二步 — 看AGP图谱**：重点关注IQR（25-75百分位，深蓝区域，反映固有因素/药物效果），根据超出目标范围的幅度和时段调整药物和生活方式。
3. **第三步 — 与患者沟通**：结合每日葡萄糖图谱，分析具体血糖异常的原因，制定个性化调整方案。

### 5.5 基于CGM的治疗方案调整规则

#### 夜间血糖曲线调整

| 图谱特征 | 判断 | 调整策略 |
|---------|------|--------|
| 夜间曲线持续上升 | 基础胰岛素/基础率不足 | 提前2h增加基础率约10%，或增加基础胰岛素剂量约10% |
| 夜间曲线持续下降但在目标范围内 | 基础胰岛素/基础率偏高 | 提前2h降低基础率约10%，或减少基础胰岛素剂量约10% |
| 下半夜持续低血糖 | 下半夜基础量过高 | 减少基础率/剂量10-20% |
| 凌晨4时后持续升高（黎明现象） | 下半夜基础量不足 | 提前2h增加基础率10% |

#### 餐后血糖曲线调整

| 图谱特征 | 判断 | 调整策略 |
|---------|------|--------|
| 餐后升高、4h回到正常 | 剂量适宜 | 无须调整 |
| 餐后升高、4h未回到正常 | 餐时胰岛素不足 | 增加餐前剂量或添加餐后降糖药 |
| 餐后3-4h出现低血糖 | 餐时胰岛素过量 | 减少餐前剂量 |

#### 基于TIR/TBR的门诊方案调整

| 分类 | 行动 | 随访 |
|------|------|------|
| TIR>70% + TBR<3% | 继续当前方案 | 3-4个月 |
| TIR>70% + TBR≥3% | 优先处理低血糖（停磺脲类、减基础胰岛素10-20%） | 2-4周 |
| TIR≤70% + TBR<3% | 处理高血糖（起始/加量GLP-1RA或SGLT2i） | 2-4周 |
| TIR≤70% + TBR≥3% | 先处理低血糖，再处理高血糖 | 2-4周 |

## 第六部分：逆转治疗模式与临床管理流程

### 6.1 2+N逆转模式

- **"2"**：中西医结合 + 药物与非药物结合
- **"N"**：精准营养、科学运动、不良生活习惯矫正、心理调适等综合措施

### 6.2 住院治疗适应证

- 新发糖尿病
- 血糖不稳定（FPG>10 mmol/L，随机血糖>15 mmol/L）
- 严重并发症

### 6.3 院外综合管理适应证

- 短期住院后病情稳定者
- 严重并发症已控制者
- 血糖达标者

### 6.4 多学科团队构成

糖尿病专科医生（团队负责人）、营养师、健康管理师、心理咨询师、减重外科医生、运动医学医生、康复理疗师、糖尿病专科护士、网络管理员

### 6.5 临床管理全流程

1. **筛查高危人群**：BMI/腰围测量、血糖筛查、OGTT、病因鉴别、并发症评估
2. **确诊患者**：同时满足T2DM和肥胖/超重诊断标准
3. **完善辅助检查**：体脂含量、血脂/血糖/HbA1c/胰岛素/C肽、甲状腺功能、肾上腺功能、周围神经检查、眼底检查、心电图、腹部超声
4. **评估分级**：疾病评估(ABCD)、生活方式评估、心理评估 → 确定1-4级
5. **制定干预方案**：个体化逆转干预计划（理想体重/血糖/并发症控制目标 + 生活方式/药物/手术方案）
6. **实施干预**：药物 + 营养 + 运动 + 心理 + 中医 + 习惯矫正
7. **监测随访**：每周至少1次随访；每次检测FPG和餐后血糖；定期监测HbA1c/血脂/体重；每年OGTT和胰岛功能检测
8. **逆转后疗效评价**：3-6个月评估血糖稳定性，前后对比所有指标，评价满意度和医疗费用

## 第七部分：药物治疗规则

### 7.1 降糖药物选择原则

- 优先选择改善胰岛素抵抗、保护β细胞功能、具有综合代谢获益的药物。
- 超重/肥胖患者优先选择有减重效果的药物，其次选择对体重无影响的药物，慎用增加体重的药物。

### 7.2 有减重效果的降糖药

| 药物类别 | 代表药物 | 核心优势 |
|---------|--------|--------|
| 二甲双胍 | 二甲双胍 | 降低HbA1c、低血糖风险低、轻度减重、心血管保护 |
| SGLT-2抑制剂 | 达格列净、恩格列净 | 减重、降压、心血管和肾脏获益 |
| GLP-1受体激动剂 | 利拉鲁肽、艾塞那肽、司美格鲁肽 | 显著减重、低血糖风险低、降低餐后血糖波动 |

### 7.3 对体重影响较小的降糖药

| 药物类别 | 代表药物 |
|---------|--------|
| DPP-4抑制剂 | 西格列汀、沙格列汀、利格列汀 |
| α-糖苷酶抑制剂 | 阿卡波糖、伏格列波糖 |

### 7.4 血糖控制分层目标

| 控制等级 | HbA1c | 适用人群 |
|---------|-------|--------|
| 严格 | <6.5% | 新诊断、<65岁、无并发症、无低血糖不良反应、备孕 |
| 标准 | 6.5-7.0% | <65岁胰岛素治疗者、>65岁但器官功能好/预期寿命>15年 |
| 宽松 | 7.0-9.0% | CVD或高CVD风险、≥65岁、低血糖高危、执行能力受限 |

## 第八部分：中医药辅助治疗规则

### 8.1 中医辨证思路

"肥胖/超重 → 脾瘟证 → 消渴 → 消渴并发症"为肥胖型糖尿病的自然发展过程。

### 8.2 中药方剂参考

- 开郁清热法：适用于早中期糖尿病
- 大黄黄连泻心汤、葛根芩连汤、大柴胡汤：可适当降低血糖、减少并发症风险因素

### 8.3 针灸治疗

- **主穴**：胰俞、脾俞、三阴交
- **脾胃湿热型配穴**：曲池、内庭
- **作用机制**：疏通经络、平衡阴阳、调节脏腑、抑制食欲、促进脂肪分解、调节胰岛素和生长激素分泌

## 第九部分：数字化平台功能规则

### 9.1 数据采集标准

| 数据类型 | 采集频率 | 采集方式 |
|---------|--------|--------|
| 血糖（CGM） | 连续14天/次 | 传感器自动采集 |
| 体重/BMI | 每日 | 智能秤 |
| 腰围/体脂 | 每周 | 手动/体脂秤 |
| 饮食记录 | 每餐 | 拍照+AI识别/手动记录 |
| 运动数据 | 实时 | 穿戴设备 |
| 睡眠数据 | 每日 | 穿戴设备 |
| 心理评估 | 每月 | 量表推送 |
| HbA1c/血脂 | 每3-6月 | 实验室检测 |

### 9.2 智能预警规则

| 预警类型 | 触发条件 | 处理流程 |
|---------|--------|--------|
| 低血糖预警 | CGM <3.9 mmol/L 或趋势箭头↓ | 指血确认 → 补充15g速效碳水 → 15-30min复测 |
| 高血糖预警 | CGM >13.9 mmol/L 或趋势箭头↑ | 调整餐前胰岛素 → 1h复测 |
| 严重低血糖 | CGM <3.0 mmol/L | 立即停用胰岛素 → 静脉注射葡萄糖15-20g |
| 运动安全预警 | 运动前血糖<5.0或>13.9 mmol/L | 暂停运动计划，先处理血糖 |
| 体重反弹预警 | 连续7天体重上升>1kg | 推送营养师随访 |

### 9.3 远程随访标准

- 逆转期内：每周至少1次面对面或电话随访
- 稳定期：可结合CGM数据远程管理，间隔3个月使用1次CGM
- 随访内容：饮食/运动执行情况、并发症监测、血糖控制、症状改善
- 记录体检数据并动态对比，据此调整诊疗方案

### 9.4 疗效评价标准

逆转后3-6个月进行全面评估：

- 血糖稳定性（TIR、HbA1c变化）
- 体重/BMI/腰围变化
- 并发症改善情况
- 用药减量/停药情况
- 患者满意度
- 医疗费用对比

## 第十部分：14条关键行动建议（生活方式干预指南）

基于《生活方式干预治疗2型糖尿病指南》的循证关键行动建议：

1. **倡导生活方式干预作为治疗核心**：生活方式干预应成为T2DM预防、治疗和缓解的前线管理手段。
2. **评估个体当前生活方式行为**：全面评估营养、运动、睡眠、压力、社会连接和高风险物质使用情况。
3. **设定行为改变优先事项**：基于评估结果确定最具影响力的干预切入点。
4. **处方有氧运动**：每周150-300 min中等强度有氧运动。
5. **处方力量训练**：每周≥2次抗阻训练。
6. **减少久坐时间**：每30 min进行1-5 min活动。
7. **识别和处理睡眠障碍**：筛查阻塞性睡眠呼吸暂停等。
8. **提供个体化饮食干预方案**：基于代谢指标选择适合的膳食模式。
9. **增强社会支持网络**：促进家庭参与和同伴支持。
10. **戒烟咨询**：所有吸烟的糖尿病患者应接受戒烟干预。
11. **限酒咨询**：评估和管理饮酒行为。
12. **药物与非药物评估**：评估娱乐性药物使用情况。
13. **心理干预**：评估并管理糖尿病相关心理压力和精神健康。
14. **制定连续护理计划**：确保从急性干预到长期维持的无缝衔接。
