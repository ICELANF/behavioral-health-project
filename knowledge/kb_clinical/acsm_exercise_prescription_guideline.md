## 第一部分：核心术语与强度分级体系

### 1.1 基础术语定义

| 术语 | 英文 | 定义 |
|------|------|------|
| 体力活动（PA） | Physical Activity | 任何可引起骨骼肌收缩并在静息能量消耗基础上引起能量消耗增加的活动 |
| 运动（Exercise） | Exercise | 有计划、有组织、可重复的体力活动，旨在促进或维持一个或多种体适能组成 |
| 体适能 | Physical Fitness | 个体能够精力充沛且机敏地进行日常工作，不过度疲劳，能享受闲暇并应对紧急情况的能力 |
| 运动处方（Ex Rx） | Exercise Prescription | 基于FITT-VP原则制定的个性化运动方案 |
| 代谢当量（MET） | Metabolic Equivalent | 运动代谢率与安静代谢率之比；1 MET = 3.5 mL O₂/kg/min |

### 1.2 健康相关体适能五要素

1. **心肺耐力（CRF）**：持续体力活动中循环和呼吸系统供氧的能力
2. **身体成分**：肌肉、脂肪、骨骼及身体其他重要组分的相对含量
3. **肌肉力量**：肌肉最大用力能力
4. **肌肉耐力**：肌肉在无疲劳状态下连续运动的能力
5. **柔韧性**：关节运动的有效范围

### 1.3 运动强度MET分级标准

| 强度分级 | MET值范围 | 活动举例 |
|---------|----------|---------|
| 极低/低强度 | <3.0 METs | 漫步（2.0）、站立轻度工作（2.0~2.5）、钓鱼-坐（2.5） |
| 中等强度 | 3.0~5.9 METs | 步行3 mph（3.0）、快速健步走4 mph（5.0）、乒乓球（4.0）、网球双打（5.0） |
| 较大强度 | ≥6.0 METs | 慢跑5 mph（8.0）、慢跑6 mph（10.0）、游泳-中/强（8~11）、网球单打（8.0） |

## 第二部分：体力活动的健康益处与风险

### 2.1 规律体力活动的健康益处

**心血管和呼吸功能改善**：
- 通过中枢和外周适应增加最大摄氧量（VO₂max）
- 同等绝对次大强度活动时每分通气量降低、心肌耗氧量下降、心率和血压降低
- 骨骼肌毛细血管密度增加
- 运动时血乳酸阈值增加

**心血管疾病危险因素减少**：
- 静息收缩压/舒张压降低
- 血清HDL-C水平增加和甘油三酯水平下降
- 总脂肪量减少，腹腔内脂肪减少
- 胰岛素需要量减少，葡萄糖耐量改善
- 血小板黏附和凝集下降，炎症水平下降

**发病率和死亡率降低**：
- 较高的活动/体适能水平与降低CAD死亡率相关
- 降低CVD、CAD、中风、T2DM、代谢综合征、骨质疏松性骨折、结肠癌、乳腺癌发生率
- MI后心脏康复运动可降低心血管疾病和全因死亡率

**其他收益**：减轻焦虑和抑郁、改善认知功能、增强老年人体质和独立生活能力、增加幸福感、减少跌倒风险

### 2.2 体力活动与健康结局的量效关系

| 变量 | 量效关系 | 证据级别 |
|------|---------|---------|
| 全因死亡率 | 负相关 | 强 |
| 心肺健康 | 负相关 | 强 |
| 代谢相关健康 | 负相关 | 中 |
| 减轻体重 | 负相关 | 强 |
| 腹型肥胖 | 负相关 | 中 |
| 骨骼健康 | 负相关 | 中 |
| 关节健康 | 负相关 | 强 |
| 肌肉健康 | 负相关 | 强 |
| 功能性健康 | 负相关 | 中 |
| 结肠和乳腺癌 | 负相关 | 中 |
| 抑郁和痛苦 | 负相关 | 中 |

### 2.3 运动风险与安全

- 健康个体中等强度PA发生SCD和AMI的风险很低
- 较大强度运动时SCD风险增加5倍，AMI风险增加3.5倍（与安静状态比）
- 绝对风险极低：男性较大强度运动150万次发生1次SCD
- 静坐少动者偶尔进行较大强度运动的AMI风险是规律运动者的50倍
- **关键原则**：规律运动可降低急性运动风险；循序渐进、低起始强度是安全保障

### 2.4 公众健康体力活动推荐

**ACSM-AHA成年人体力活动建议**：
1. 18~65岁健康成年人：每天30 min中等强度有氧运动×每周5天，或每天20 min较大强度运动×每周3天
2. 建议中等和较大强度相结合
3. 30 min中等强度运动可分多次完成，每次至少10 min
4. 每周至少2天进行维持/增加肌肉力量和耐力的运动
5. 量效关系：超过最低限度的PA建议可获得更多益处

## 第三部分：运动前健康筛查

### 3.1 ACSM运动前筛查流程（第十版更新要点）

新版核心变化：从基于CVD危险因素数量的三级分层（低/中/高危），转向更简洁实用的筛查逻辑：

**筛查决策三要素**：
1. 当前运动习惯（是否规律运动）
2. 已知CVD、代谢或肾脏疾病（是否已确诊）
3. 症状和/或体征（是否存在提示性症状）

**筛查结论**：
- 规律运动 + 无已知疾病 + 无症状 → 可进行中等至较大强度运动，无需医学检查
- 目前不运动 + 无已知疾病 + 无症状 → 可开始中等强度运动，较大强度前建议医学检查
- 已知疾病 → 低至中等强度可在获得医学许可后进行，较大强度需医学检查和运动测试

### 3.2 CVD危险因素（用于风险评估参考）

| 危险因素 | 阈值标准 |
|---------|---------|
| 年龄 | 男性≥45岁；女性≥55岁 |
| 家族史 | 父亲或一级男性亲属<55岁、母亲或一级女性亲属<65岁发生MI/猝死/冠脉血运重建 |
| 吸烟 | 当前吸烟或过去6个月内戒烟 |
| 静坐少动 | 未达到每周3天×30 min中等强度PA（至少3个月） |
| 肥胖 | BMI≥30 kg/m² 或腰围：男性>102 cm、女性>88 cm |
| 高血压 | SBP≥140 mmHg 和/或 DBP≥90 mmHg（或正在服药） |
| 血脂异常 | LDL-C≥130 mg/dL 或 HDL-C<40 mg/dL 或正在服药；TC≥200 mg/dL |
| 糖尿病前期 | FPG 100~125 mg/dL 或 OGTT 2h 140~199 mg/dL 或 HbA1c 5.7%~6.4% |
| **负性因素** | HDL-C≥60 mg/dL（减去一个危险因素） |

## 第四部分：运动处方FITT-VP原则（核心章节）

### 4.1 一次运动训练课的基本组成

| 阶段 | 时间 | 内容 |
|------|------|------|
| 热身 | ≥5~10 min | 低到中等强度心肺和肌肉耐力运动 |
| 训练内容 | ≥20~60 min | 有氧、抗阻、神经肌肉练习和/或体育运动 |
| 整理活动 | ≥5~10 min | 低到中等强度心肺耐力和肌肉耐力运动 |
| 拉伸 | ≥10 min | 在热身或整理活动之后进行 |

### 4.2 有氧运动FITT-VP推荐（健康成年人）

| FITT-VP | 基于证据的推荐 |
|---------|---------------|
| **频率（F）** | 中等强度≥5天/周，或较大强度≥3天/周，或中等+较大强度3~5天/周 |
| **强度（I）** | 中等强度：40%~59% HRR或VO₂R；较大强度：60%~89% HRR或VO₂R |
| **时间（T）** | 中等强度30~60 min/天（≥150 min/周）；较大强度20~60 min/天（≥75 min/周） |
| **方式（T）** | 规律的、大肌肉群参与的持续性周期性运动 |
| **总量（V）** | ≥500~1000 MET-min/周 ≈ 1000 kcal/周 ≈ 每天≥7000步 |
| **进阶（P）** | 循序渐进：开始4~6周内每1~2周延长5~10 min/次；规律锻炼1个月后逐渐增加FIT |

### 4.3 运动强度计算方法

**HRR法（心率储备法，推荐首选）**：
```
靶心率(THR) = (HRmax - HRrest) × 期望强度% + HRrest
```

**VO₂R法（储备摄氧量法）**：
```
靶VO₂ = (VO₂max - VO₂rest) × 期望强度% + VO₂rest
```

**HRmax推测公式**：
| 公式 | 适用人群 |
|------|---------|
| HRmax = 220 - 年龄 | 传统公式，适用有限 |
| HRmax = 208 - 0.7 × 年龄 | 健康男性和女性（Tanaka等） |
| HRmax = 207 - 0.7 × 年龄 | 所有年龄段和体适能水平（Gelish等） |
| HRmax = 206 - 0.88 × 年龄 | 中年女性（Gulati等） |

**能量消耗代谢公式**（单位：mL/kg/min）：
| 活动 | 公式 | 精确范围 |
|------|------|---------|
| 走路 | VO₂ = 3.5 + (0.1×速度) + (1.8×速度×坡度%) | 速度1.9~3.7 mph |
| 跑步 | VO₂ = 3.5 + (0.2×速度) + (0.9×速度×坡度%) | 速度>5 mph |
| 下肢自行车 | VO₂ = 3.5 + 3.5 + (1.8×功率)/体重 | 功率300~1200 kg·m/min |

### 4.4 有氧运动方式分类

| 类别 | 运动类型 | 推荐人群 | 举例 |
|------|---------|---------|------|
| A类 | 需要最少技能的耐力活动 | 所有成年人 | 步行、休闲自行车、水中有氧、慢舞 |
| B类 | 需要最少技能的较大强度耐力运动 | 有规律锻炼者/中等体适能 | 慢跑、跑步、划船、有氧操、椭圆机 |
| C类 | 需要技能的耐力运动 | 有技能者/中等体适能 | 游泳、越野滑雪、滑冰 |
| D类 | 休闲运动 | 有规律锻炼计划者 | 网羽运动、篮球、足球、徒步旅行 |

### 4.5 抗阻运动FITT推荐

| 要素 | 推荐 |
|------|------|
| 频率 | 每个大肌群2~3天/周，同一肌群间隔≥48小时 |
| 强度 | 肌肉力量：60%~80% 1-RM，8~12次/组；肌肉耐力：<50% 1-RM，15~25次/组 |
| 组数 | 每个肌群2~4组（即使1组也可显著提高初学者力量） |
| 方式 | 多关节复合练习为主（卧推、深蹲、下拉等），辅以单关节和核心练习 |
| 组间休息 | 2~3 min |
| 要点 | 同时练习主动肌与拮抗肌，预防肌力失衡 |

### 4.6 柔韧性与神经肌肉练习推荐

**柔韧性**：≥2~3天/周（每天更有效），每个拉伸动作保持10~30秒，重复2~4次，拉伸至感觉紧张或轻度不适

**神经肌肉练习**：≥2~3天/周，包括平衡、灵活性、协调性和本体感觉训练，对老年人（≥65岁）尤为重要

## 第五部分：代谢性疾病人群运动处方

### 5.1 糖尿病运动处方

**诊断标准参考**：

| 指标 | 正常 | 糖尿病前期 | 糖尿病 |
|------|------|----------|--------|
| HbA1c | <5.7% | 5.7%~6.4% | ≥6.5% |
| FPG | <100 mg/dL (5.6 mmol/L) | 100~125 mg/dL (5.6~6.9 mmol/L) | ≥126 mg/dL (7.0 mmol/L) |
| OGTT 2h | <140 mg/dL (7.8 mmol/L) | 140~199 mg/dL (7.8~11.0 mmol/L) | ≥200 mg/dL (11.1 mmol/L) |

**糖尿病FITT推荐**：

| 要素 | 有氧 | 抗阻 | 柔韧性 |
|------|------|------|--------|
| 频率 | 3~7天/周 | ≥2~3天/周（不连续） | ≥2~3天/周 |
| 强度 | 中等(40%~59% VO₂R, RPE 11~12)至较大(60%~80% VO₂R, RPE 14~17) | 中等(50%~69% 1-RM)至较大(70%~85% 1-RM) | 拉伸至紧张或轻度不适 |
| 时间 | T1DM：中等150 min/周或较大75 min/周；T2DM：中~较大150 min/周 | 8~10种动作，10~15次/组，1~3组 | 静态拉伸10~30s，2~4次 |
| 方式 | 持续性大肌群运动（步行、骑车、游泳） | 器械或自由力量 | 静态/动态/PNF拉伸 |

**糖尿病运动安全规则（平台危机协议衔接）**：

| 场景 | 血糖阈值 | 处置规则 |
|------|---------|---------|
| 低血糖 | <70 mg/dL (3.9 mmol/L) | 运动相对禁忌，补充15g碳水化合物 |
| 运动前偏低 | ≤100 mg/dL 且使用胰岛素 | 运动前补充碳水化合物（不超过15g） |
| 高血糖无酮 | >300 mg/dL (16.7 mmol/L) | 感觉良好+酮体阴性→可中等强度运动，密切监测 |
| 高血糖有酮 | >250 mg/dL (13.9 mmol/L)+酮体阳性 | **暂缓运动** |
| 严重低血糖 | <54 mg/dL (3.0 mmol/L) | **触发危机协议**→立即停止运动→紧急处置 |

**关键注意事项**：
- T2DM患者不应间隔2天以上不进行有氧运动（否则胰岛素活性改善作用降低）
- 联合训练时，先抗阻后有氧可降低T1DM低血糖风险
- 有氧+抗阻联合比单一形式更好地改善血糖控制
- HIIT和连续较大强度有氧均为推荐运动方式
- 自主神经病变患者应通过RPE（非心率）监控运动强度
- 足部防护：保持干燥、硅胶/气垫鞋垫、涤纶或混纺袜、每日足部检查

### 5.2 高血压运动处方

**FITT推荐**：

| 要素 | 有氧 | 抗阻 | 柔韧性 |
|------|------|------|--------|
| 频率 | 5~7天/周 | 2~3天/周 | ≥2~3天/周 |
| 强度 | 中等(40%~59% VO₂R/HRR, RPE 12~13) | 60%~70% 1-RM，可渐增至80% | 拉伸至紧张或轻度不适 |
| 时间 | 累积或连续≥30 min/天 | 8~10种动作，8~10次×2~4组 | 静态拉伸10~30s，2~4次 |
| 方式 | 持续性大肌群运动 | 器械和/或自由力量 | 静态/动态/PNF拉伸 |

**降压效果**：规律有氧运动可使安静SBP和DBP各降低5~7 mmHg

**安全阈值**：运动时SBP≤220 mmHg和/或DBP≤105 mmHg；抗阻训练时避免Valsalva动作

### 5.3 脂代谢紊乱运动处方

- 有氧运动可使LDL-C降低3~6 mg/dL
- 抗阻训练可降低LDL-C和TG 6~9 mg/dL
- 运动处方基础：以增加能量消耗减轻体重为目的的有氧运动
- 运动时间目标：250~300 min/周
- 使用他汀类药物者注意肌痛：异常或持续肌肉疼痛时应咨询医生

### 5.4 代谢综合征运动处方

**诊断标准**（满足≥3项）：

| 组分 | NCEP/ATP III切点 |
|------|-----------------|
| 腰围 | 男性>102 cm，女性>88 cm |
| 甘油三酯 | ≥150 mg/dL 或正在药物治疗 |
| HDL-C | 男性<40 mg/dL，女性<50 mg/dL 或正在药物治疗 |
| 血压 | ≥130/85 mmHg 或正在药物治疗 |
| 空腹血糖 | ≥100 mg/dL 或正在药物治疗 |

**运动要点**：
- 初始强度从中等（40%~59% VO₂R/HRR）开始，逐步增至较大强度
- 运动量逐渐增至250~300 min/周或每天50~60 min
- 抗阻+有氧联合比单独有氧降低MetSyn患病率更显著
- 每个危险因素或健康状况需统筹考虑，以最保守方案起始

### 5.5 超重和肥胖运动处方

**FITT推荐**：

| 要素 | 有氧 | 抗阻 | 柔韧性 |
|------|------|------|--------|
| 频率 | ≥5天/周 | 2~3天/周 | ≥2~3天/周 |
| 强度 | 中等起始(40%~59% VO₂R/HRR)→渐增至较大(≥60%) | 60%~70% 1-RM | 拉伸至紧张或轻度不适 |
| 时间 | 30 min/天→渐增至60 min/天或≥250~300 min/周 | 每主要肌群2~4组×8~12次 | 静态拉伸10~30s，2~4次 |

**减重量效关系**：
- <150 min/周 PA → 轻度减重
- >150 min/周 PA → 适度减重2~3 kg
- >225~420 min/周 PA → 减重5~7.5 kg
- 维持减重：可能需要200~300 min/周 PA

**减重目标**：3~6个月内减少初始体重3%~10%；每日减少500~1000 kcal摄入→每周减重0.5~0.9 kg

## 第六部分：特殊人群运动注意事项

### 6.1 老年人（≥65岁）

- 有氧运动建议与成年人类似，强度应与个体CRF水平相关
- 特别强调柔韧性、神经肌肉练习和肌力训练
- 爆发力训练（快速肌肉收缩）可安全进行，对防跌倒尤为重要
- 应关注从低强度开始，循序渐进

### 6.2 儿童和青少年

- 每天≥60 min中等到较大强度PA
- 包括有氧、肌肉力量和骨骼强化活动
- 强调乐趣和内在动机培养
- 学校是理想干预场所（体育课、课间休息、课内活动）

### 6.3 妊娠期

- 无禁忌证的孕妇可进行中等强度运动
- 避免仰卧位运动（妊娠中晚期）
- 避免高跌倒/腹部创伤风险的运动
- 避免高海拔、高温环境运动

## 第七部分：行为理论与运动促进策略

### 7.1 核心行为理论框架

| 理论模型 | 核心概念 | 在平台中的应用 |
|---------|---------|---------------|
| 社会认知理论（SCT） | 自我效能感、结果预期、自我调节 | 用户信心评估→分级目标→自我监控反馈闭环 |
| 跨理论模型（TTM） | 5阶段：前意向→意向→准备→行动→保持 | BAPS评估阶段→阶段匹配干预策略推送 |
| 健康信念模型（HBM） | 主观觉察易感性/严重性/收益/障碍、行动线索、自我效能 | 健康风险评估→动机启动→障碍识别与克服 |
| 自我决定理论（SDT） | 自主需要、能力需要、关系需要；内部→外部动机连续体 | 支持用户选择权→建立胜任感→社区归属感 |
| 计划行为理论（TPB） | 行为意向=态度+主观规范+知觉行为控制 | 信念评估→社会支持设计→障碍感知干预 |
| 社会生态模型（SET） | 多层次影响：个人→人际→组织→环境→政策 | 多层次干预设计：个人目标+社群支持+环境适配 |

### 7.2 TTM阶段匹配干预策略（Agent输出指南）

**前意向→意向**：
- 提供定期体力活动益处信息
- 纠正关于障碍的误解（如"没时间→可短时间多次完成"）
- 引导想象运动后的短期益处（睡眠改善、压力减轻、精力增加）
- 探讨不运动对家人的影响

**意向→准备**：
- 探索消除阻碍因素的方案
- 评估并建立自我效能感
- 强调即使前进一小步也重要
- 鼓励自我认同为"健康的、活跃的人"

**准备→行动**：
- 帮助制订具体PA计划，使用SMARTS目标设置
- 使用强化/奖励机制
- 教授自我监控技术（时间、距离追踪）
- 继续讨论障碍克服策略
- 鼓励创造有利于PA的环境

**行动→保持**：
- 为目标进度提供积极反馈
- 探索不同运动方式防止倦怠
- 讨论预防复发策略（预见高危情境+制定应对预案）
- 鼓励"要么全有要么全无"思维的替代（过失≠复发）

### 7.3 提高自我效能感的四类策略

| 信息来源 | 策略 |
|---------|------|
| 成功经验（最有效） | 设置可实现的现实目标；循序渐进；提供指导和监督；使用PA日志跟踪进度 |
| 替代性经验 | 提供相似背景的榜样人物；使用视频供模仿学习；讨论同伴"成功"经历 |
| 言语说服 | 经常给予鼓励、赞扬和反馈；表达对个人能力的信心 |
| 生理反馈 | 提供适当的指导和保证；讨论运动感受；针对不适提供教育；鼓励利用音乐使运动愉快 |

### 7.4 SMARTS目标设置原则

- **S**pecific 具体性：目标清晰具体
- **M**easurable 可测性：目标可量化
- **A**ction-oriented 行动性：指明需要做什么
- **R**ealistic 现实性：可实现的
- **T**imely 限时性：有具体时间框架
- **S**elf-determined 自主性：主要由客户/患者制定

### 7.5 动机性访谈5A模型

1. **Assess（评估）**：评估PA行为、信念、知识和改变愿望
2. **Advise（告知）**：告知PA益处和不活动的健康风险
3. **Agree（同意）**：基于兴趣、信心、能力合作制定PA目标
4. **Assist（协助）**：利用问题解决技术和社会/环境支持克服障碍
5. **Arrange（安排）**：包含随访反馈、评估和支持的计划

### 8.2 输出格式建议

- **运动处方输出**：使用FITT-VP结构化格式，每项≤2句话，总计不超过400字
- **强度计算输出**：显示计算过程+结果范围，提供RPE作为辅助监控
- **行为阶段干预输出**：根据TTM阶段匹配策略，1~3条具体建议，≤200字
- **安全预警输出**：糖尿病用户每次运动前提醒血糖检测，显示安全阈值表

### 8.3 禁止误用场景

- ❌ 不得对未经筛查的高危人群直接推荐较大强度运动
- ❌ 不得忽略用药对运动中HR/BP反应的影响（如β-受体阻滞剂）
- ❌ 不得对伴严重增殖性糖尿病视网膜病变者推荐较大强度有氧/抗阻
- ❌ 不得对血糖>250 mg/dL+酮体阳性者继续运动指导
- ❌ 不得用统一的FITT参数套用所有人群（必须个体化调整）
- ❌ 不得将柔韧性训练作为血糖控制或体重管理的独立干预手段

## 第九部分：与其他知识模块的集成逻辑

### 与血糖监测知识库（BGM Base）
- ACSM的糖尿病运动安全阈值（3.9/16.7 mmol/L）与BGM Base的预警规则直接衔接
- 运动前/中/后血糖监测频率建议应结合CGM/SMBG数据
- TIR作为运动干预效果的客观验证指标

### 与大五人格知识库（BFP Base）
- **高尽责性（BFP-C↑）**：适合结构化FITT运动处方+自我监控日志
- **低尽责性（BFP-C↓）**：简化监控要求，采用"10分钟也算"策略降低启动门槛
- **高神经质（BFP-N↑）**：强调运动减压益处，避免过度关注运动强度数值
- **高外向性（BFP-E↑）**：推荐D类休闲运动和团体运动，利用社会支持
- **低开放性（BFP-O↓）**：推荐A类低技能运动（步行、骑车），渐进引入新运动形式

### 与BPT6/TTM行为改变模型
- TTM阶段评估结果直接映射到第七部分的阶段匹配干预策略
- 前意向期用户：推送PA益处信息（第二部分量效关系数据）作为动机启动材料
- 行动/保持期用户：推送FITT-VP结构化运动处方+进阶方案
- 过失/复发期用户：触发预防复发策略（7.2行动→保持部分）

### 与中医体质知识库（TCM Base）
- 痰湿/湿热体质（代谢风险高）：优先推荐中等强度有氧+MetSyn运动方案
- 气虚/阳虚体质（能量不足）：从A类低强度运动起始，强调循序渐进
- 血瘀体质（微循环不良）：有氧运动促进毛细血管密度增加的科学依据可作为TCM运动处方的现代医学验证

### 与危机干预规则
- 运动中血糖<3.0 mmol/L → 触发平台P0级Safety Trigger，立即停止教练对话，指导紧急处置
- 运动中SBP>220 mmHg 或 DBP>105 mmHg → 立即停止运动，评估是否需要急救
- 运动中出现胸痛/呼吸困难/头晕等症状 → 停止运动，评估是否需要拨打急救电话

## 第十部分：关键运动处方速查表

### 10.1 各类慢性病FITT速查

| 疾病 | 有氧频率 | 有氧强度 | 有氧时间 | 抗阻 | 特殊注意 |
|------|---------|---------|---------|------|---------|
| T2DM | 3~7天/周 | 40%~80% VO₂R | 150 min/周 | ≥2天/周 | 不间隔>2天；监测血糖 |
| T1DM | 3~7天/周 | 40%~80% VO₂R | 150 min/周 | ≥2天/周 | 先抗阻后有氧；胰岛素调整 |
| 高血压 | 5~7天/周 | 40%~59% VO₂R | ≥30 min/天 | 2~3天/周 | 避免Valsalva；运动后低血压注意 |
| 脂代谢紊乱 | ≥5天/周 | 40%~75% VO₂R | 30~60 min/天 | 2~3天/周 | 强调减重；他汀肌痛监测 |
| 代谢综合征 | ≥5天/周 | 40%~59%→渐增 | 150→250~300 min/周 | ≥2天/周 | 多危险因素统筹；最保守起始 |
| 超重/肥胖 | ≥5天/周 | 40%~59%→渐增 | 30→60 min/天 | 2~3天/周 | 250~300 min/周维持减重 |

### 10.2 运动量速算参考

| 指标 | 数值 | 说明 |
|------|------|------|
| 推荐PA量 | 500~1000 MET-min/周 | 相当于≈1000 kcal/周 |
| 步行推荐 | ≥7000步/天 | 5400~7900步已满足推荐量 |
| 步行换算 | 100步/分≈中等强度 | 1英里≈2000步 |
| 能量换算 | kcal/min = METs×3.5×体重(kg)÷1000×5 | |
| MET-min换算 | METs×运动分钟数 | 如7 METs×30 min×3天=630 MET-min/周 |
