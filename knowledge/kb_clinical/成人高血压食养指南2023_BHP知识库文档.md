### KI-HTN-001 高血压定义与流行病学

- **knowledge_id**: `KI-HTN-001`
- **title**: 高血压定义、分级与流行病学
- **category**: `CORE_CONCEPT`
- **life_dimension**: `RISK_FACTOR`
- **behavior_types**: `["assessment", "awareness"]`
- **applicable_stages**: `[S0_UNAWARE, S1_HESITANT, S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》
- **tags**: `["高血压", "血压分级", "流行病学", "患病率", "危险因素"]`

**content**:

高血压定义：在未使用降压药物的情况下，非同日3次测量诊室血压，收缩压（SBP）≥140 mmHg和/或舒张压（DBP）≥90 mmHg。患者既往有高血压史，目前正在使用降压药物，即使血压低于140/90 mmHg，仍应诊断为高血压。

**血压水平分级**：

| 分级 | 收缩压(mmHg) | 舒张压(mmHg) |
|------|-------------|-------------|
| 正常血压 | <120 | 且 <80 |
| 正常高值 | 120-139 | 和/或 80-89 |
| 1级高血压（轻度） | 140-159 | 和/或 90-99 |
| 2级高血压（中度） | 160-179 | 和/或 100-109 |
| 3级高血压（重度） | ≥180 | 和/或 ≥110 |

当收缩压和舒张压分属不同级别时，以较高分级为准。

**流行病学数据**：我国约有2.45亿高血压患者，18岁及以上居民高血压患病率为27.5%。高血压是心脑血管疾病最重要的危险因素，是我国心脑血管疾病死亡的首位原因。高血压患病率随年龄增长而升高，且城乡差距逐步缩小。

**主要危险因素**：高钠低钾膳食、超重和肥胖、过量饮酒、长期精神紧张、缺乏体力活动、遗传因素、年龄增长。其中高钠低钾膳食是我国人群最重要的高血压发病危险因素。我国居民人均每日烹调用盐约10.5g，远超推荐量。

### KI-HTN-002 高血压食养核心原则

- **knowledge_id**: `KI-HTN-002`
- **title**: 高血压食养核心原则
- **category**: `DIETARY_PRINCIPLE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["dietary_pattern", "food_selection", "meal_planning"]`
- **applicable_stages**: `[S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》
- **tags**: `["DASH饮食", "食养原则", "限钠增钾", "膳食结构", "高血压饮食"]`

**content**:

高血压患者食养遵循四大核心原则：

**原则一：减钠增钾，饮食清淡**。每日食盐摄入量逐步降至5g以下（约1啤酒盖平齐）。增加富钾食物摄入（新鲜蔬果、豆类、坚果）。减少腌制食品、酱菜、加工食品摄入。使用低钠盐替代普通食盐（肾功能不全者除外）。学习看食品营养标签，识别高钠食品。增钾目标≥3510 mg/d。

**原则二：合理膳食，科学食养**。遵循DASH饮食模式（终止高血压膳食方案）。增加全谷物、蔬菜、水果、低脂乳制品、鱼类、豆类和坚果的摄入。减少饱和脂肪酸和胆固醇的摄入。保证充足的膳食纤维（≥25g/d）。适量优质蛋白质。控制添加糖摄入。

**原则三：吃动平衡，健康体重**。超重或肥胖高血压患者应以减轻体重为首要目标。减重5%-10%即可显著降低血压。每周至少150分钟中等强度有氧运动或75分钟高强度有氧运动。避免久坐，每小时起身活动。

**原则四：戒烟限酒，心理平衡**。完全戒烟。不饮酒或限制饮酒量（男性酒精量<25g/d，女性<15g/d）。保持良好心理状态，学习减压技术。充足睡眠（7-9小时/晚）。

### KI-HTN-003 限钠增钾操作指南

- **knowledge_id**: `KI-HTN-003`
- **title**: 限钠增钾操作指南
- **category**: `PRACTICAL_GUIDE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["food_preparation", "food_selection", "seasoning"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》
- **tags**: `["限盐", "低钠", "增钾", "调味替代", "隐性钠", "营养标签"]`

**content**:

**限钠实操策略**：

1. 使用限盐勺或限盐罐定量用盐。每人每餐约1g盐（约1/5啤酒盖）。全天总量不超过5g（包含酱油、味精等所有调味品中的钠）。

2. 减少高钠调味品：酱油（约15%含钠）、蚝油、豆瓣酱、味精（约含13%钠）、鸡精等。1汤匙酱油约含3g盐当量。

3. 用天然调味替代：姜、蒜、葱、花椒、八角、醋、柠檬汁、香菜、辣椒（适量）。酸味可弥补咸味不足。

4. 减少高钠食品摄入：腌制品（腌菜、腊肉、咸鱼）、加工食品（火腿肠、方便面、罐头）、快餐食品、零食（薯片、话梅、肉干）。

5. 学会看营养标签：NRV%（营养素参考值百分比）>30%属于高钠食品。钠含量×2.54=食盐含量（mg换算）。

6. 烹饪技巧：出锅前再放盐（减少钠溶解入菜）；蒸、煮、炖为主要烹饪方式；利用食物本身鲜味减少用盐。

**增钾实操策略**：

富钾食物推荐（钾含量mg/100g）：

| 食物类别 | 代表食物 | 钾含量(mg/100g) |
|---------|---------|----------------|
| 豆类 | 黄豆1503、绿豆787、红豆860 | 极高 |
| 薯类 | 红薯394、土豆342、山药213 | 高 |
| 蔬菜 | 菠菜311、苋菜340、毛豆478、竹笋389 | 高 |
| 菌藻 | 口蘑3106（干）、紫菜1796（干）、海带1503（干） | 极高 |
| 水果 | 香蕉256、橙199、猕猴桃198、牛油果485 | 中-高 |
| 坚果 | 开心果970、杏仁728、花生587 | 高 |

钠/钾比：目标≤1:1（当前我国居民平均约2.7:1）。每日蔬菜摄入≥500g、水果200-350g，可有效增加钾摄入。

**⚠️ 注意事项**：慢性肾脏病（eGFR<60 ml/min/1.73m²）或服用保钾利尿剂/ACEI/ARB患者，增钾需在医生指导下进行，防止高钾血症。

### KI-HTN-004 DASH饮食模式

- **knowledge_id**: `KI-HTN-004`
- **title**: DASH饮食模式（终止高血压膳食方案）
- **category**: `DIETARY_PATTERN`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["dietary_pattern", "meal_planning", "food_selection"]`
- **applicable_stages**: `[S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》；DASH原始研究
- **tags**: `["DASH饮食", "降压饮食", "膳食模式", "循证饮食"]`

**content**:

DASH饮食是目前循证证据最充分的降压膳食模式。严格执行DASH饮食可降低收缩压8-14 mmHg。

**DASH饮食每日推荐摄入量（以2000kcal为基准）**：

| 食物类别 | 每日份数 | 每份标准量 | 主要营养素贡献 |
|---------|---------|-----------|-------------|
| 谷物和谷物制品 | 6-8份 | 1片面包/半碗米饭/半碗面条 | 能量、膳食纤维 |
| 蔬菜 | 4-5份 | 1碗生菜/半碗熟菜 | 钾、镁、膳食纤维 |
| 水果 | 4-5份 | 1个中等大小水果/半碗果汁 | 钾、镁、膳食纤维 |
| 低脂乳制品 | 2-3份 | 1杯牛奶(240ml)/1杯酸奶 | 钙、蛋白质 |
| 瘦肉/禽/鱼 | ≤6份 | 28g熟肉 | 蛋白质、镁 |
| 坚果/豆类/种子 | 4-5份/周 | 1/3杯坚果/半碗豆 | 镁、钾、蛋白质、膳食纤维 |
| 油脂 | 2-3份 | 1茶匙植物油 | 脂肪（以不饱和脂肪为主） |
| 甜食 | ≤5份/周 | 1汤匙糖/果冻 | 控制在最低限度 |

**DASH饮食中国化调整建议**：用豆浆、酸奶替代牛奶（乳糖不耐受者）；增加大豆制品（豆腐、豆干）；以茶代替果汁减少糖摄入；增加深色蔬菜比例至50%以上。

**DASH饮食降压机制**：高钾促进尿钠排泄；高钙调节血管平滑肌张力；高镁改善血管内皮功能；高膳食纤维改善胰岛素敏感性；低饱和脂肪减少血管炎症。

### KI-HTN-005 高血压食物选择与禁忌

- **knowledge_id**: `KI-HTN-005`
- **title**: 高血压食物选择与禁忌
- **category**: `FOOD_GUIDE`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["food_selection", "grocery_shopping"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》
- **tags**: `["食物选择", "降压食物", "禁忌食物", "钾钙镁", "膳食纤维"]`

**content**:

**推荐食物（绿灯区）**：

谷薯类：全麦面包、糙米、燕麦、红薯、玉米、荞麦。优先全谷物，占谷物总量1/3以上。

蔬菜类（每日≥500g，深色蔬菜≥50%）：芹菜（含芹菜素，有辅助降压作用）、菠菜（高钾高镁）、西兰花、番茄、胡萝卜、洋葱、大蒜（含蒜素，可改善血管弹性）、木耳、海带、紫菜。

水果类（每日200-350g）：香蕉（高钾）、橙子/柑橘（高钾高维C）、猕猴桃、苹果、山楂（活血化瘀）、柚子。注意：服用CCB类降压药（如硝苯地平）者慎食柚子（影响药物代谢）。

蛋白质类：鱼类（深海鱼富含omega-3脂肪酸，每周2-3次）、禽肉（去皮）、大豆及豆制品（每日25g大豆或等量豆制品）、蛋类（每日1个）、低脂奶类（每日300-500ml）。

油脂类：橄榄油、菜籽油、亚麻籽油为首选。坚果每日10g（约一小把核桃仁或杏仁）。

饮品：淡茶（绿茶含茶多酚，有降压作用）、白开水、柠檬水。

**限制食物（黄灯区）**：精白米面（适量，与全谷物搭配）；猪肉、牛肉、羊肉（瘦肉适量，每日畜肉≤50g）；蛋黄（每日不超过1个）。

**避免食物（红灯区）**：腌制食品（咸菜、咸鱼、腊肉、酱肉）；加工肉制品（火腿肠、培根、午餐肉）；高钠调味品过量使用；含糖饮料和高糖食品；反式脂肪酸食品（人造奶油、氢化植物油食品）；浓茶、咖啡过量（≤3杯/日）；酒精。

### KI-HTN-006 高血压中医辨证食养

- **knowledge_id**: `KI-HTN-006`
- **title**: 高血压中医辨证食养方案
- **category**: `TCM_DIETARY_THERAPY`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["tcm_dietary", "herbal_food"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》
- **tags**: `["中医食养", "辨证施食", "食药物质", "高血压证型", "肝阳上亢", "痰湿壅盛", "阴虚阳亢", "阴阳两虚"]`

**content**:

高血压中医分为四大证型，各型食养策略不同：

**证型一：肝阳上亢型**

主要表现：头痛头胀，面红目赤，急躁易怒，口干口苦，便秘溲赤，舌红苔黄，脉弦有力。此型多见于1-2级高血压初中期，常因情绪激动血压波动明显。

食养原则：平肝潜阳，清肝泻火。

推荐食药物质：菊花（平肝明目）、决明子（清肝明目通便）、夏枯草（清肝散结）、芹菜（平肝清热）、苦瓜（清热泻火）。

食养方：①菊花决明子茶——菊花5g、决明子10g，沸水冲泡代茶饮，每日1-2次。②芹菜粥——芹菜连根120g切碎，粳米100g同煮粥。③天麻鱼头汤——天麻10g、鳙鱼头1个，炖汤。

饮食禁忌：忌辛辣燥热（辣椒、胡椒、白酒）、煎炸油腻、温补类（羊肉、鹿茸、桂圆）。

**证型二：痰湿壅盛型**

主要表现：头重如裹，胸闷脘痞，体胖痰多，肢体困重，食少腹胀，舌淡胖有齿痕，苔白腻，脉滑。此型多见于肥胖型高血压患者。

食养原则：化痰祛湿，健脾和胃。

推荐食药物质：薏苡仁（健脾利湿）、赤小豆（利水消肿）、山楂（消食化积活血）、荷叶（清热利湿降脂）、陈皮（理气化痰）。

食养方：①荷叶薏苡仁粥——干荷叶15g煎汤取汁，薏苡仁30g、粳米100g同煮粥。②山楂决明子茶——山楂15g、决明子10g，沸水冲泡。③陈皮茯苓茶——陈皮6g、茯苓10g，煎水代茶。

饮食禁忌：忌肥甘厚味、生冷寒凉、甜食及黏滞难消化食物。

**证型三：阴虚阳亢型**

主要表现：头晕耳鸣，头痛目涩，腰膝酸软，五心烦热，心悸失眠，舌红少苔，脉弦细数。此型多见于老年高血压或长期高血压患者。

食养原则：滋阴潜阳，补肾养肝。

推荐食药物质：枸杞子（滋补肝肾）、桑椹（滋阴补血）、黑芝麻（补肝肾益精血）、女贞子（补肝肾阴）、墨旱莲（补肝肾阴凉血止血）。

食养方：①枸杞菊花茶——枸杞子15g、菊花5g，沸水冲泡。②桑椹枸杞粥——桑椹15g、枸杞子10g、粳米100g同煮粥。③黑芝麻核桃糊——黑芝麻30g、核桃仁30g，研磨冲服。

饮食禁忌：忌温燥辛辣、煎炸、浓茶咖啡、忌过量补阳之品。

**证型四：阴阳两虚型**

主要表现：头晕目眩，面色无华，腰膝酸冷，畏寒肢冷，心悸气短，夜尿多，舌淡胖，脉沉细。此型多见于高龄、久病高血压患者。

食养原则：滋阴助阳，补益肝肾。

推荐食药物质：山药（补脾益肾）、黄精（滋肾润肺健脾）、杜仲（补肝肾强筋骨）、核桃（补肾益脑）、黑豆（补肾益精）。

食养方：①杜仲核桃猪腰汤——杜仲10g、核桃仁30g、猪腰1个，炖汤。②山药黄精粥——山药30g、黄精15g、粳米100g同煮粥。③枸杞桑椹饮——枸杞子15g、桑椹15g，煎水代茶。

饮食禁忌：忌大寒大热之品，忌过食生冷和辛辣，宜温和平补。

**四型共同食养原则**：均需限钠增钾；均需控制总能量；均需避免暴饮暴食和过度饮酒；均需结合药物治疗，食养为辅。

### KI-HTN-007 高血压运动处方

- **knowledge_id**: `KI-HTN-007`
- **title**: 高血压患者运动处方
- **category**: `EXERCISE_PRESCRIPTION`
- **life_dimension**: `EXERCISE`
- **behavior_types**: `["aerobic_exercise", "resistance_training", "flexibility"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》；结合ACSM运动处方指南
- **tags**: `["运动处方", "有氧运动", "降压运动", "运动禁忌", "运动安全"]`

**content**:

规律运动可降低收缩压约5-8 mmHg。

**有氧运动处方**（FITT原则）：

| 要素 | 推荐 |
|------|------|
| 频率(F) | 每周5-7天 |
| 强度(I) | 中等强度（40%-59%储备心率，RPE 12-13分）|
| 时间(T) | 每次30-60分钟（可分段累计，每段≥10分钟）|
| 类型(T) | 快走、慢跑、游泳、骑自行车、太极拳、广场舞 |

**抗阻运动处方**：

| 要素 | 推荐 |
|------|------|
| 频率 | 每周2-3天，非连续日 |
| 强度 | 60%-70% 1RM（可对话但有些吃力）|
| 时间 | 每个肌群2-3组，每组8-12次 |
| 类型 | 弹力带、哑铃、自重训练 |

**柔韧性运动**：每次运动前后进行，每个拉伸动作保持15-30秒，重复2-4次。太极拳、八段锦、瑜伽兼具柔韧性和降压效果。

**运动安全注意事项**：

血压>180/110 mmHg时暂停运动。运动前做5-10分钟热身，运动后5-10分钟整理。避免屏气用力（Valsalva动作），尤其是抗阻运动时。运动中如出现头晕、胸闷、心悸、气促等症状立即停止。服用β受体阻滞剂者不能单纯依赖心率判断运动强度，建议使用RPE。清晨血压高峰期（6:00-10:00）不建议进行高强度运动。

**运动降压的特殊人群调整**：

合并冠心病：需在医生评估后运动，强度降低至30%-39%储备心率起步。合并糖尿病：注意防低血糖，运动前后监测血糖。合并肥胖：优先中低强度长时间有氧运动，保护关节。老年人（≥65岁）：增加平衡训练，防跌倒，从低强度开始。

### KI-HTN-008 高血压心理与睡眠管理

- **knowledge_id**: `KI-HTN-008`
- **title**: 高血压患者心理调适与睡眠管理
- **category**: `PSYCHOLOGY_AND_SLEEP`
- **life_dimension**: `PSYCHOLOGY`
- **behavior_types**: `["stress_management", "sleep_hygiene", "emotion_regulation"]`
- **applicable_stages**: `[S2_INTENDING, S3_PREPARING, S4_ACTING, S5_MAINTAINING]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》
- **tags**: `["心理调适", "压力管理", "睡眠管理", "情绪血压", "焦虑", "失眠"]`

**content**:

长期精神紧张是高血压重要危险因素。交感神经持续兴奋使血管收缩、心率加快、血压升高。焦虑和愤怒情绪可导致急性血压升高20-30 mmHg。

**心理管理策略**：

1. 情绪觉察：识别引起血压波动的情绪触发因素（工作压力、人际冲突、经济焦虑等）。建立"情绪-血压日记"，记录情绪事件与血压变化的关联。

2. 减压技术：每日10-20分钟正念冥想或腹式呼吸。渐进性肌肉放松（PMR）训练。太极拳、八段锦、瑜伽等身心运动。

3. 认知调整：识别"必须完美""不能犯错"等不合理信念。学习ABC情绪管理法（事件-信念-后果）。培养"接纳"态度，减少对不可控事件的过度反应。

4. 社会支持：参加高血压患者互助小组。家庭成员共同参与饮食和运动改变。减少独居和社会孤立。

**睡眠管理策略**：

睡眠不足（<6小时/晚）或质量差是高血压独立危险因素。睡眠呼吸暂停（OSAHS）是继发性高血压重要病因。

目标：每晚7-9小时高质量睡眠。固定就寝和起床时间（即使周末）。睡前1小时避免电子屏幕。卧室环境：安静、暗、凉爽（18-22°C）。睡前避免：咖啡因（午后）、酒精（虽然助入睡但破坏睡眠结构）、大量进食。

打鼾严重或白天过度嗜睡→筛查OSAHS→转诊睡眠科。OSAHS确诊后持续气道正压通气（CPAP）治疗可显著降低血压。

### KI-HTN-009 高血压地域与季节食养调适

- **knowledge_id**: `KI-HTN-009`
- **title**: 高血压地域与季节食养调适
- **category**: `REGIONAL_SEASONAL`
- **life_dimension**: `NUTRITION`
- **behavior_types**: `["seasonal_diet", "regional_adaptation"]`
- **applicable_stages**: `[S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》
- **tags**: `["地域食养", "季节食养", "南北差异", "四季调养"]`

**content**:

**四季食养调适**：

春季（肝阳易亢）：宜清淡平肝，多食芹菜、菊花茶、绿叶蔬菜。注意防春困导致的不规律饮食。食养方推荐：菊花决明子茶、芹菜苹果汁。

夏季（暑热伤阴）：宜清热利湿，多食苦瓜、冬瓜、绿豆、西瓜（适量）。注意出汗多补充钾但需控制钠。食养方推荐：荷叶绿豆汤、冬瓜薏仁汤。注意夏季血压偏低，降压药需遵医嘱调整。

秋季（燥邪伤肺）：宜滋阴润燥，多食梨、百合、银耳、莲藕。注意秋冬血压升高趋势，加强监测。食养方推荐：百合银耳羹、枸杞雪梨水。

冬季（寒邪收引）：宜温补而不燥，多食羊肉（适量，痰湿型慎用）、核桃、山药、黑芝麻。注意冬季血压偏高，寒冷可使血压升高5-10mmHg。食养方推荐：山药枸杞粥、杜仲核桃汤。避免高盐腌肉腌菜。

**地域食养调适**：

北方地区：高血压患病率高于南方，与高盐饮食习惯密切相关。重点减盐，目标从日均12-15g逐步减至5g。增加新鲜蔬菜水果摄入，减少腌制品。

南方地区：湿热气候，痰湿型高血压较常见。利湿化痰为重点，薏苡仁、赤小豆、冬瓜常用。注意梅雨季节血压波动。

西北地区：饮食偏咸偏油腻，牛羊肉摄入多。控盐控脂为首要任务，增加蔬菜水果和杂粮。

沿海地区：海产品丰富，有利于omega-3脂肪酸摄入。注意海产品加工品（虾皮、咸鱼、鱼干）高钠问题。

高原地区：海拔高，血压易升高。增加抗氧化食物（深色蔬果、茶叶），充足饮水，避免缺氧加重血压问题。

### KI-HTN-010 高血压家庭血压监测

- **knowledge_id**: `KI-HTN-010`
- **title**: 家庭血压监测与自我管理
- **category**: `SELF_MONITORING`
- **life_dimension**: `RISK_FACTOR`
- **behavior_types**: `["self_monitoring", "data_tracking", "medication_adherence"]`
- **applicable_stages**: `[S3_PREPARING, S4_ACTING, S5_MAINTAINING, S6_AUTONOMOUS]`
- **evidence_level**: `GUIDELINE`
- **source**: 国家卫生健康委《成人高血压食养指南（2023年版）》
- **tags**: `["家庭血压监测", "自我管理", "血压计使用", "服药依从性"]`

**content**:

**家庭血压监测规范**：

设备选择：推荐经过国际标准认证的上臂式电子血压计。不推荐手腕式或手指式血压计。每年至少校验一次。

测量方法：测量前安静休息5分钟，30分钟内避免吸烟、饮咖啡、运动。取坐位，双脚平放地面，上臂与心脏平齐。袖带下缘距肘窝2-3cm，松紧以能插入1-2指为宜。每次测量2-3次，间隔1-2分钟，取平均值。

测量频率：初诊或调整药物期间，连续7天每日早晚各测1次。血压稳定后，每周至少测量1天。推荐时间：早起后1小时内（排尿后、服药前、早餐前），晚睡前1小时。

记录内容：日期、时间、收缩压、舒张压、心率、特殊事件（情绪波动、饮食、运动、用药变化）。

**家庭血压判断标准**：家庭血压≥135/85 mmHg即为高血压（低于诊室标准5mmHg）。

**血压管理"三色预警"**（CoachAgent决策参考）：

绿灯（自我管理）：<135/85 mmHg，维持当前方案，每周监测1天。
黄灯（加强关注）：135-159/85-99 mmHg，加强生活方式干预，增加监测频率，1个月内复查。
红灯（立即就医）：≥160/100 mmHg持续，或出现头痛剧烈、胸闷、视物模糊、鼻出血等症状，立即就医。

## 五、CoachAgent决策支持要点

### 5.1 高血压用户初次评估流程

当用户被识别为高血压或高血压高危人群时：
1. 确认血压分级（1/2/3级）和用药情况
2. 评估当前膳食模式（估算每日食盐摄入）
3. 评估运动习惯（每周运动分钟数）
4. 评估BMI和腰围
5. 中医辨证分型（四大证型初筛）
6. 评估行为改变阶段
7. 生成个性化干预方案

### 5.2 优先级排序（基于降压效果循证证据）

| 干预措施 | 预期降压幅度(SBP) | 优先级 |
|---------|-----------------|-------|
| DASH饮食 | 8-14 mmHg | ★★★★★ |
| 限钠<5g/天 | 5-6 mmHg | ★★★★★ |
| 减重5%-10% | 5-8 mmHg per 5kg | ★★★★★ |
| 规律有氧运动 | 5-8 mmHg | ★★★★ |
| 限酒/戒酒 | 2-4 mmHg | ★★★★ |
| 增钾≥3510mg/d | 2-4 mmHg | ★★★ |
| 正念/减压 | 2-5 mmHg | ★★★ |
| 中医食养 | 辅助协同 | ★★ |

### 5.3 对话策略提示

**初次接触高血压用户**：
- 不要一次性给出所有改变建议，从限盐这个最高优先级切入
- 评估当前盐摄入水平，设定渐进目标（第1周减1/3，第2周再减1/3）
- 使用"1啤酒盖=5g盐"等具象化表达

**遭遇抵触时**：
- "口淡没味道" → 介绍天然替代调味法，强调味觉适应期约2-4周
- "我血压不高不用管" → 分享正常高值发展为高血压的风险数据
- "吃药就行不用改饮食" → 解释药物+生活方式双管齐下效果最佳，且可能减少药量

**⚠️ 红线提醒**：
- 不能建议停药或减药，药物调整须由医生进行
- 血压≥180/110 mmHg或伴随头痛剧烈/胸闷/视物模糊/鼻出血→立即就医
- 合并肾病患者增钾需慎重
- 服用特定降压药物时的食物禁忌（如CCB+柚子）需提醒
