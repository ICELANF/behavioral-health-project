## 核心定义

### 重要术语

```yaml
T2D缓解（Remission）：
  通过生活方式干预，血糖降至正常范围，
  减少或停用降糖药物，持续3个月以上。
  这不是"治愈"，而是通过持续生活方式改变维持的状态。

糖尿病前期（Prediabetes）：
  HbA1c 5.7%-6.4%（ADA标准）
  未达诊断标准但已增加T2D和心血管病风险
  → 平台干预最高价值窗口（可逆转）

全食物植物为主饮食（WFPB）：
  以全谷物、蔬菜、豆类、水果、坚果、种子为主
  最小化或避免动物性食品、精制食品、超加工食品
  
超加工食品（UPF）：
  根据NOVA分类系统，第4类：
  高糖/高盐/高添加剂/高精制淀粉，配方设计激活大脑奖赏系统
  促进过量饮食，纤维/微量元素含量最低

标准美式饮食（SAD/西方饮食）：
  高钠、高热量（肉类/加工食品/精制碳水）
  水果/蔬菜/全谷物缺乏
  → 慢性病的直接饮食原因
```

## 6大生活方式干预支柱

```yaml
生活方式医学6大支柱（Lifestyle Medicine 6 Pillars）：
  1. 营养（Nutrition）
  2. 体力活动（Physical Activity）
  3. 压力管理（Stress Management）
  4. 睡眠（Sleep）
  5. 社会连接（Social Connectedness）
  6. 危险物质回避（Avoidance of Risky Substances）
```

**6大支柱与平台干预维度对照**：

| 支柱 | 临床证据 | 平台对应模块 | 核心行为处方 |
|------|---------|-----------|------------|
| 营养 | 全食物植物为主饮食可实现T2D缓解（多项RCT） | NutritionAgent | WFPB饮食模式，减少UPF |
| 体力活动 | 每周150-300分钟中等强度有氧运动显著降HbA1c | ExerciseAgent | FITT框架个性化运动处方 |
| 压力管理 | 正念/CBT/瑜伽显著改善HbA1c（多项系统综述） | StressAgent | 正念、冥想、CBT |
| 睡眠 | 睡眠不足与HbA1c升高、胰岛素抵抗密切关联 | SleepAgent | 7-9小时，筛查OSA |
| 社会连接 | 孤独/孤立增加T2D管理难度；同伴支持改善结局 | CommunityAgent | 建立支持网络，同伴支持 |
| 戒毒物 | 烟草/酒精/毒品损害血糖控制，影响药物疗效 | CoachAgent | 戒烟、限酒、戒毒 |

**KAS 1：倡导生活方式干预**（强推荐）

> 临床医生或HCP应该成为生活方式干预作为糖尿病前期和T2D**一线管理**的倡导者，包括：
> 恢复性睡眠、压力管理、充足体力活动、积极社会连接、全食物植物为主饮食、避免危险物质。

**关键意义**：这是历史性转变——将生活方式干预从"补充疗法"提升为"一线管理"。
平台定位的最高级别科学依据。

评估内容：

- 识别改善机会并教育个体
- 评估**改变准备度**（readiness to change）→ 直接对应TTM阶段评估

**KAS 3：确立生活方式改变优先级**（强推荐）

通过**共享决策**（SDM）确立6大支柱的改变优先级：
- 用户**自主选择**优先改变的领域（而非医生指定）
- 讨论**SMART目标**在支持积极生活方式改变中的作用
- 避免"多吃蔬菜""多运动"等非具体建议 → 直接支持平台行为处方设计原则

**SMART目标框架（在T2D管理中的应用）**：

```
S（Specific-具体）：
  "每天步行30分钟" 而非 "多运动"

M（Measurable-可衡量）：
  "每周5天" 而非 "经常"

A（Achievable-可实现）：
  考虑个人能力、时间、资源

R（Relevant-有相关性）：
  连接到个人价值观和优先级

T（Time-Bound-有时限）：
  "本周开始，持续4周"
```


**KAS 4：有氧+肌力运动处方**（强推荐）

**FITT框架用于T2D运动处方**：

| 维度 | 推荐 | T2D特殊考量 |
|------|------|-----------|
| **F**requency（频率） | 每周≥5天 | 有氧隔天一次可；肌力2-3次/周 |
| **I**ntensity（强度） | 中等-高强度 | 从轻度开始，逐渐增加 |
| **T**ime（时间） | 150-300分钟/周（中强度） | 可分解为10-15分钟短段 |
| **T**ype（类型） | 有氧+肌力 | 两者结合比单独更有效 |

**核心证据**：
- 3项系统综述+2项大型RCT支持（Grade A）
- 参加推荐运动量的糖尿病前期人群：T2D风险降低**44%**
- 达标运动量的T2D患者：心脏病、肾病、截肢并发症风险显著降低
- 美国T2D患者中，仅**24.1%**达到推荐的150分钟/周运动量

**运动时机与血糖管理**：
- 餐后20分钟步行 → 有效降低餐后高血糖
- 餐前运动 → 不能降低餐后高血糖
- → 平台行为处方：餐后运动优先推荐

---

**KAS 5：减少久坐时间**（强推荐）

**久坐的定义**：能量消耗<1.5 METs的清醒状态（坐、斜倚、卧）

**久坐危害证据**：
- 久坐与T2D发病率独立相关（即便达到运动量）
- 美国成年人平均每天久坐**8.2小时**（范围4.9-11.9小时）

**减少久坐的策略**：
- 运动零食（Exercise Snacks）：每隔30分钟起立活动2-3分钟
  - 半蹲/小腿抬高/快走 = 显著降低餐后血糖+改善胰岛素反应
- ADA建议：至少每30分钟打断一次长时间久坐
- 计步器：平均每天额外增加1800步+40分钟步行

### KI-ACLM-003 | KAS 6：睡眠障碍筛查

```yaml
knowledge_id: KI-ACLM-003
title: KAS 6：T2D患者的睡眠障碍筛查与干预（OSA/睡眠剥夺）

tags: [睡眠障碍, 阻塞性睡眠呼吸暂停, OSA, CPAP, 睡眠与血糖, 失眠]
retrieval_queries:
  - 睡眠和血糖有什么关系
  - 糖尿病患者需要查睡眠吗
  - 睡眠呼吸暂停和糖尿病有关系吗
  - 改善睡眠能降血糖吗
  - 多少睡眠时间最健康
```

**睡眠-代谢关联证据**：

```
睡眠质量/时长 → HbA1c和代谢指标
（多项RCT系统综述：持续关联，高度一致）

最佳睡眠：7-9小时/夜
<6小时 OR >9小时 → 死亡率增加

睡眠不足→
  前额叶功能下降（冲动控制减弱）→行为失控
  饥饿素升高、瘦素降低→食欲增加
  日间热量消耗减少
  胰岛素敏感性下降→血糖控制恶化
```

**T2D相关睡眠障碍**：
- 阻塞性睡眠呼吸暂停（OSA）**：与糖尿病前期、T2D的发生呈剂量依赖关系
- CPAP治疗OSA → 改善HbA1c
- 轮班工作睡眠障碍
- 慢性失眠

**STOP-Bang问卷**（OSA筛查）：
> Snoring（打鼾）/ Tired（疲劳）/ Observed apnea（目击窒息）/
> Pressure（血压高）/ BMI>35 / Age>50 / Neck>40cm / Gender（男性）
> ≥3项阳性 → 中高风险，建议转诊睡眠专科

**平台睡眠干预策略**：
- 首先筛查OSA（STOP-Bang）
- 睡眠卫生教育（CBT-I）
- 不推荐长期安眠药作为首选（可影响血糖）
- 与KI-NEURO-BRAIN-BODY-AMEN的12种睡眠改善方法结合

**KAS 7：糖尿病前期的营养处方**（强推荐）

针对糖尿病前期或GDM史成年人，处方**全食物植物为主（WFPB）饮食**：
- 卡路里摄入
- 营养素需求
- WFPB框架的食物选择

**膳食纤维的证据**：
- 高纤维饮食 → T2D风险降低**15-20%**
- 每增加10g/天纤维 → 风险再降9%
- 机制：延缓葡萄糖吸收、改善胰岛素敏感性、调节肠道菌群

---

**KAS 8：T2D治疗的营养处方**（强推荐）

**可实现T2D缓解的饮食方案（证据等级对比）**：

| 饮食方案 | T2D缓解证据 | 注意事项 |
|---------|-----------|---------|
| **全食物植物为主（WFPB）** | 强（多项RCT） | 最推荐，需要补充B12 |
| 低碳/极低碳水饮食 | 中（部分RCT） | 短期有效，但长期依从性差；注意低血糖风险 |
| 地中海饮食 | 中（观察性研究为主） | 对心血管风险较好 |
| 限卡路里饮食 | 有效（体重驱动） | 通过减重间接改善血糖 |

**超加工食品（UPF）的危害**：
- UPF能激活大脑奖赏系统，促进过量饮食（成瘾机制）
- 与KI-ADDIC-PSYCHOLOGY-SHEN-2024（饮食成瘾）和KI-NEURO-BRAIN-BODY-AMEN-2024（冲动型过量饮食）直接关联
- NOVA分类第4类：应最小化或避免

**饮食改变障碍与解决策略**：

| 障碍 | 策略 |
|------|------|
| 不会烹饪植物性食品 | 提供食谱手册，推荐烹饪课程 |
| 担心健康饮食费用高 | 重点推荐豆类/全谷物/冷冻蔬菜等平价选项 |
| 家人饮食方式不同 | 帮助制定沟通策略，讨论可能的妥协方案 |
| 食物获取困难 | 优先可行方案，不评判；探索送餐选项 |
| 文化饮食习惯差异 | 在传统饮食文化基础上寻找植物性替代 |

**低碳饮食的特别警示**：
- 不推荐长期使用低碳饮食（尤其是肉类为主的）
- 如使用，持续时间有限，然后逐步重新引入营养丰富的碳水（水果/蔬菜/豆类/全谷物）
- 结合体力活动和总卡路里管理

**KAS 10：心理干预**（推荐）

> 识别焦虑/抑郁/糖尿病压力，处方正念/CBT干预以改善糖尿病结局。

**证据**：
- T2D患者抑郁风险是普通人群的2-3倍
- 焦虑风险比普通人群高约20%
- 正念（8周MBSR）、瑜伽、太极、气功 → 显著改善HbA1c，同时降低抑郁/焦虑症状和感知压力

**糖尿病特异性心理压力测量**：
- 问题领域量表（PAID）（20项）
- 糖尿病压力量表（DDS）（17项）

**KAS 11：烟草/酒精/毒品**（强推荐）

评估T2D患者的烟草/酒精/娱乐性毒品使用，并提供戒除咨询：
- 吸烟 → 加重代谢功能障碍，增加胰岛素抵抗
- 酒精/毒品（包括大麻）→ 损害血糖控制，干扰药物疗效，增加心血管风险
- 强调戒除是T2D整体管理的核心部分（不只是"建议减少"）

---

**KAS 12：实现以人为本的持续正向行为改变**（强推荐）

> 通过循证方法实现持续、以人为中心的正向行为改变，包括但不限于：
> 健康教练（HWC）、动机访谈（MI）、认知行为疗法（CBT）。

**健康教练（HWC）循证依据**：
- 6项系统综述、8项RCT、2项准实验研究
- **≥12次健康教练会话，≥6个月** → 显现有意义的健康改善
- HbA1c降低：39% vs 20%（标准护理）
- 自我管理支持中以文化调适的干预最为成功（少数族裔）

**三种核心行为改变干预方式对比**：

| 干预 | 核心特征 | 技术 | 剂量/时机 | 转介建议 |
|------|---------|------|---------|---------|
| **动机访谈（MI）** | 以人为中心的咨询风格 | OARS（开放性问题/确认/反映/总结） | 初次诊断、周期性随访 | 无进步时转健康教练 |
| **健康教练（HWC）** | 通过以人为中心的过程建立伙伴关系 | SMART目标、自我探索、问责策略 | ≥12次/6个月+ | 目标未达成时强化或转专科 |
| **认知行为疗法（CBT）** | 修正功能失调的思维和行为 | 行为激活、认知重构 | 持续有害行为模式/高心理负担 | 有心理障碍时转心理专业 |

**动机访谈核心工具（OARS）**：
- **O**pen-ended questions（开放性问题）
- **A**ffirmations（确认）
- **R**eflections（反映）
- **S**ummaries（总结）

**KAS 13：建立连续性护理计划**（强推荐）

连续性护理的关键组成部分：

| 组成部分 | 循证建议 |
|---------|---------|
| **随访频率** | 首次生活方式讨论后密集随访，然后尽可能频繁 |
| **血糖监测** | 优先使用**CGM**追踪生活方式改变期间的血糖变化；无法用CGM时，每天2次指尖血糖（尤其是饮食干预第一周） |
| **药物管理** | 定期随访监测药物反应，防止不良事件（低血糖/药物相互作用） |
| **技术利用** | 远程医疗/自动邮件随访/支持性社交媒体账号 → 增强连续性 |

**CGM在生活方式干预中的核心价值**（与KI-CGM-AACE模块直接关联）：
- 实时显示食物、运动、睡眠对血糖的影响 → 增强行为改变动力
- 指南明确建议：**在生活方式干预第一个月及之后使用CGM**
- CGM数据可作为行为处方效果的即时反馈（不只是每3个月查HbA1c）

---

**KAS 14：调整药物治疗**（推荐）

> 根据生活方式干预对药物需求的影响，调整药物的种类和剂量。

**药物减量（Deprescribing）原则**：

```
风险：低血糖（尤其在以下情况）
  - 体重减轻时
  - 低碳水饮食期间
  - 能量限制性饮食期间
  - 全食物植物为主饮食（低脂植物性）期间

减药触发时机：
  - 血糖开始改善（FBG持续低于目标值下限）
  - CGM显示持续低血糖趋势
  - 患者报告头晕/出汗等低血糖症状

减药原则：
  不是等血糖"正常了再减药"，而是：
  当生活方式改变开始生效时，主动与医生沟通调整药物
  避免"药物继续原剂量+生活方式干预"导致的低血糖
```

## 铁律声明

**ACLM-铁律-01（药物减量须由医生决定）**
CoachAgent在引用KAS 14时，必须强调：药物减量必须由处方医生决定，平台不建议用户自行停药或减药。平台的角色是：当用户血糖改善时，主动建议用户与医生沟通药物调整，而非自行操作。

**ACLM-铁律-02（WFPB饮食的中国化适配）**
指南基于美国人群数据，"全食物植物为主饮食"的具体食物选择需要中国文化适配。CoachAgent应结合藏气法时论和中国饮食文化，将WFPB原则转化为符合用户习惯的具体食物建议（如全谷物→小米/糙米/燕麦；豆类→各类豆腐/豆制品）。

**ACLM-铁律-03（12次教练会话的剂量概念）**
证据显示≥12次/6个月的健康教练会话才能产生有意义的改变。平台设计应体现这一剂量概念，而非追求单次"完美方案"。CoachAgent的价值在于持续陪伴，而非一次性给出完整处方。
