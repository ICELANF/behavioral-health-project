## 向量文件技术规范

**文件**：`bhp_vector_chunks_v1.2.json`

**块结构**：
```json
{
  "chunk_id": "KI-BL011-01-01",
  "content": "...(≤800字符，100字符重叠)...",
  "char_count": 756,
  "knowledge_id": "BL011",
  "domain": "clinical",
  "evidence_tier": "T1",
  "evidence_grade": "A",
  "source_file": "clinical_BL011_心血管预防生活方式干预.md",
  "file_hash": "sha256前16位",
  "batch": 3,
  "dimension": "KI",
  "section_id": "KI-BL011-01",
  "chunk_index": 1,
  "total_chunks_in_section": 2
}
```

**Qdrant集合配置**：
```python
collection_name = "bhp_knowledge_v12"
vector_size = 768  # 或1024，取决于embedding模型
distance = Distance.COSINE
```

**嵌入推荐**：
- 优先：`bge-large-zh`（768维，中文优化）
- 备选：`text-embedding-3-large`（1024维）
