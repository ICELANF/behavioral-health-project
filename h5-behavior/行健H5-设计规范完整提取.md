## 一、设计哲学与定位

### 产品定位
- **行为健康评估导流H5** — 公众号/朋友圈入口，3分钟完成评估，导向注册
- **目标用户**: 30-50岁职场人，对慢性疲劳/亚健康有模糊焦虑但尚未就医
- **核心体验**: 不是问卷，是"被理解的对话"；不是诊断，是"发现自己站在哪里"

### 情绪曲线（10页流程）
```
Landing(好奇) -> Scene(共鸣) -> Belief(确认) -> Question x12(沉浸)
-> Analyzing(期待) -> Factormap(认知冲击) -> Result(画像揭示)
-> Expectation(目标锚定) -> Prescription(行动方案) -> Register(转化)
-> Companion(留存入口)
```

### 基调
- **温暖但不幼稚** — 面向成年人，有质感，不卡通
- **轻量但不简陋** — 微妙阴影、渐变、动画营造品质感
- **科学但不冰冷** — 中西医结合标签，用人话解释机制

### 色彩使用规则
1. **按钮色 = 当前页面的语义色** — Scene 页用门色、Question 页用维度色、CTA 统一用 teal

3. **背景氛围色**: 语义色 + 8%-15% 透明度做卡片/面板底色
4. **渐变条**: `linear-gradient(90deg, 语义色, 语义色aa)` — 尾部带 67% 透明度
5. **按钮阴影**: `0 4px 14px 语义色40`（25% 透明度）

## 九、页面特殊设计模式

### Landing 三扇门
- 每扇门: 横向 flex = 色条(4px) + 图标块(50px圆角方) + 文字区 + 箭头
- 图标块: 门色 + 9%透明度背景, 24px emoji
- 标题高亮: "悄悄" 用 `linear-gradient(135deg, amber, rose)` 渐变裁剪为文字色
- 底部统计: "已有 12,847 人完成评估 - 匿名无需注册"（数字 teal 加粗）

### Factormap 双栏对比
- 左栏（用户认知）: amber 系 — `rgba(amber,.06)` 背景 + `rgba(amber,.15)` 边框
- 右栏（AI识别）: teal 系 — `rgba(teal,.05)` 背景 + `rgba(teal,.15)` 边框
- 条形图: 90px 宽度区域内渐变填充
- 标签: "checkmark 有效" / "checkmark 你提到了" -> sage 系; "NEW 你没提到" -> teal 系

### Prescription 7天日历
- 7 格网格 `grid-template-columns: repeat(7, 1fr)`

## 十、交互模式

### 选择反馈（Multi-select 通用模式）
1. 点击 -> `toggleXxx(index)` 切换选中数组
2. 选中视觉:
   - 边框变色（语义色）
   - 3px 外发光环（语义色 9-12% 透明度）
   - checkbox 填充色 + 白色 checkmark
   - 文字加粗 + 变深
3. 按钮文字更新: `已选 N 项，继续 ->`
4. 按钮启用: 背景从 `rgba(0,0,0,.06)` 变语义色, 阴影出现

### 滚动与过渡
- 页面切换: Vue Router（无特殊过渡动画，靠 `.fu` 入场动画覆盖）
- 进度条: `transition: width .5s cubic-bezier(.4,0,.2,1)`
- 维度条: `transition: width 1.2s .2s cubic-bezier(.4,0,.2,1)`

### Analyzing 步骤推进
- 4 步，每步间隔 900ms
- 每步状态: pending(muted dot) -> active(amber dot + blink) -> done(sage dot + sage border + checkmark)
- 总等待 3.5s 后自动跳转

## 十三、技术实现备注（供重设计参考）

- **框架**: Vue 3 + Vite 5 + Pinia + vue-router (hash mode)
- **样式方案**: CSS custom properties (tokens.css) + scoped styles (每页)
- **无 UI 库依赖**: 所有组件手写，无 Vant/Element
- **无图标库依赖**: 全部使用系统 emoji
- **内联样式**: 动态色彩通过 `:style` 绑定（如按钮色、选中阴影）
- **CSS 绑定**: `v-bind('doorData.color')` 在 scoped 中使用 JS 变量
- **Token 双轨**: CSS 变量 (tokens.css) + JS 对象 (tokens.ts) 同步维护
- **移动优先**: max-width 390px，所有触控目标 >= 44px
