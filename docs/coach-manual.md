## 3. 学员管理

### 3.1 查看学员列表

点击底部导航「学员」，进入学员列表页。

`[截图: 学员列表]`

**按行为阶段筛选（TTM 阶段）**：

```
全部 | 前意向 | 意向 | 准备 | 行动 | 维持
```

每个学员卡片显示：
- 姓名 + 当前阶段标签
- 风险等级（R1 绿色 → R5 红色）
- 近 7 天微行动完成数
- 最后联系时间

---

### 3.2 学员档案详情

点击学员卡片，进入该学员的档案详情页。

`[截图: 学员档案页]`

档案页分为 **4 个标签页**：

#### 📋 概览

| 信息项 | 说明 |
|--------|------|
| 基本信息 | 姓名、年龄、加入时间 |
| 风险等级 | R1（低）～ R5（危急） |
| 当前阶段 | TTM 行为改变阶段 |
| 近期指标 | 血糖、体重、步数等最新值 |
| 今日微行动 | 打卡完成情况 |

#### 📈 行为记录

展示该学员的历史行为数据，按类型筛选（饮食/运动/情绪/睡眠）。

#### 📝 督导记录

教练与学员的互动历史，包括：
- 教练笔记
- 已开出的处方记录

#### 🩺 健康数据

血糖 / 体重 / 运动 / 睡眠趋势图（近 7 天）。

---

### 3.3 四大快捷操作

在学员档案页顶部，有 4 个快捷按钮：

```
┌──────────┬──────────┬──────────┬──────────┐

#### 💊 开处方

```
① 选择处方类型（行为 / 运动 / 饮食 / 情绪）

`[截图: 开处方弹窗]`

### 4.3 AI 评估建议

在分配弹窗中，点击「获取 AI 建议」：

- AI 根据学员当前阶段、历史数据、行为域自动推荐量表组合
- 每项推荐附带**推荐理由**（例如：「该学员处于行动期，BIG5 可评估执行力特质」）

`[截图: AI评估建议]`

---

### 4.4 审核评估结果

学员完成评估后，状态变为「待审」，教练需要：

**操作流程：**
```
① 在评估列表点击「待审」卡片
        ↓
② 选择一条评估记录 → 点击「查看详情」
        ↓
③ 查看各量表得分 + AI 解读报告
        ↓
④ 选择操作：
   [通过并推送]  → 自动生成处方草稿发至推送队列
   [修改后推送]  → 手动编辑处方内容后推送
   [退回]        → 要求学员重新填写（附说明）
```

`[截图: 评估审核详情]`

---

### 4.5 推送处方

审核通过后，处方进入推送队列（待审处方）。

进入推送队列 → 找到该处方 → 点击「审核推送」：

```
① 查看 AI 起草的处方内容
        ↓
② 可选：手动修改文字
        ↓
③ 点击「批准推送」 → 学员收到处方通知
   或「拒绝」 → 该处方作废
```

## 5. AI 跟进飞轮

AI 跟进飞轮是教练**最强大的工具**，用于为学员生成个性化的行为改变跟进计划。

`[截图: AI跟进飞轮页]`

### 5.1 页面结构

页面分为两个标签：

| 标签 | 内容 |
|------|------|
| ✨ 生成计划 | 选学员 → 运行 AI → 查看结果 |
| ⏳ 待审计划 | 已生成、等待教练审核推送的计划列表 |

---

### 5.2 生成跟进计划

**完整操作流程：**

```
第一步：选择目标学员
┌─────────────────────────────────┐
│  从列表中选择一名学员           │
│  系统显示：阶段 / 风险 / 近况  │
└─────────────────────────────────┘
            ↓
第二步：点击「运行 AI 分析」
┌─────────────────────────────────┐
│  ⏳ AI 正在分析学员数据...      │
│  预计 10-30 秒                  │
└─────────────────────────────────┘
            ↓
第三步：查看生成结果（底部弹出）
┌─────────────────────────────────┐
│  📋 跟进计划草稿                │
│                                 │
│  [完整计划内容显示于此]         │
│                                 │
│  ✅ 已保存至待审队列 #fw_xxx   │
│                                 │
│  [前往待审]      [继续生成]    │
└─────────────────────────────────┘
```

> AI 分析基于：学员当前阶段 + 历史行为数据 + 近期健康指标 + 既往处方记录。

---

### 5.3 审核并推送计划

计划生成后自动保存至「待审计划」。

**操作流程：**
```
「待审计划」标签 → 找到对应计划 → 点击查看
        ↓
阅读完整计划内容
        ↓
[批准推送] → 学员收到跟进计划通知
[修改后推送] → 编辑内容后推送
[拒绝] → 标记为无效，不推送
```

> ⚠️ **注意**：每份计划需要教练人工审核后才会推送给学员，平台不会自动发送未经审核的内容。

## 7. 风险管理

### 7.1 风险等级说明

平台对每位学员进行实时风险评分，分为 5 个等级：

| 等级 | 标识 | 含义 | 建议响应时间 |
|------|------|------|------------|
| R1 | 🟢 绿色 | 低风险，状态稳定 | 常规跟进 |
| R2 | 🔵 蓝色 | 轻度风险，需关注 | 本周内联系 |
| R3 | 🟡 黄色 | 中度风险，需介入 | 48 小时内联系 |
| R4 | 🟠 橙色 | 高度风险，紧急 | 24 小时内联系 |
| R5 | 🔴 红色 | 危急，需立即处理 | 立即联系 + 上报督导 |

> R3 及以上学员会自动出现在首页「今日待办」的重点学员区。

## 8. 健康数据审核

当学员上报的健康数据触发阈值时，进入**教练待审健康队列**。

`[截图: 健康审核页]`

### 8.1 需审核的数据类型

| 数据类型 | 触发条件示例 |
|----------|------------|
| 血糖 | 餐前 > 7.0 mmol/L 或 餐后 > 11.1 mmol/L |
| 体重 | 单周变化 > 2 kg |
| 运动 | 连续 5 天步数 < 3000 |
| 睡眠 | 连续 3 天睡眠 < 5 小时 |

### 8.2 审核操作

```
点击待审健康记录
        ↓
查看异常数据 + 趋势图
        ↓
选择处理方式：
  [确认知悉]    → 标记已处理，无需追加操作
  [发消息]      → 直接向学员发送提醒
  [开处方]      → 针对该异常开具干预处方
  [上报督导]    → 严重情况转交督导处理（R4/R5）
```

## 11. 附录：量表说明

### TTM7 — 行为改变阶段模型

> 评估学员当前处于行为改变的哪个阶段，是制定干预策略的基础。

| 阶段 | 代码 | 描述 |
|------|------|------|
| 前意向期 | S0 | 未认识到问题，无改变意愿 |
| 意向期 | S1 | 认识到问题，但犹豫不决 |
| 准备期 | S2 | 计划在近期（1个月内）采取行动 |
| 行动期早期 | S3 | 刚开始改变，不足6个月 |
| 行动期稳定 | S4 | 持续改变中 |
| 维持期 | S5 | 改变保持超过6个月 |
| 终止期 | S6 | 新行为完全内化，无复发风险 |

**建议使用时机**：每季度评估一次，跟踪阶段进展。

---

### BIG5 — 大五人格

> 评估学员的核心人格特质，帮助教练制定个性化沟通策略。

| 维度 | 全称 | 高分特征 | 低分特征 |
|------|------|---------|---------|
| O | 开放性 | 喜欢尝试新事物 | 倾向传统、务实 |
| C | 尽责性 | 有条理、自律 | 灵活、随性 |
| E | 外向性 | 善于社交、活跃 | 内向、独立 |
| A | 宜人性 | 合作、体贴 | 竞争、批判 |
| N | 神经质 | 易焦虑、情绪化 | 情绪稳定 |

**建议使用时机**：建档初期评估一次，作为长期参考。

---

### BPT6 — 行为类型标签

> 识别学员的行为偏好模式，用于匹配合适的微行动类型。

6 种行为类型：**规律运动型 / 健康饮食型 / 社交活跃型 / 学习成长型 / 情绪管理型 / 作息规律型**

**建议使用时机**：制定个性化微行动计划前。

---

### 能力评估（Capacity）

> 评估学员在健康行为改变方面的自我效能感和实际能力。

涵盖：知识水平 / 技能储备 / 社会支持 / 环境资源

**建议使用时机**：学员进入行动期时评估，评估执行障碍。

---

### SPI — 自我潜力指数

> 综合评估学员的改变潜力与内在动力。

**建议使用时机**：制定长期目标前，评估学员可达到的最优状态。

## 常见问题

**Q：分配评估后学员没有收到通知怎么办？**
A：检查学员是否已登录小程序。可在学员档案页点击「发消息」手动提醒。

**Q：AI 生成计划失败或超时怎么办？**
A：网络波动可能导致超时，等待 30 秒后重试。若持续失败请联系平台管理员。

**Q：评估审核后如何修改已推送的处方？**
A：已推送处方无法直接修改，需在学员档案「开处方」重新开具新处方，旧处方保留在记录中。

**Q：学员风险等级如何计算？**
