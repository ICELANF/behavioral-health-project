# 行为健康教练认证体系 · 工程级系统规范（可直接并入主动行为健康中枢）

版本：v1.0
定位：平台级「教练人才生产系统」+ 行为健康中枢能力放大器
适用对象：行为健康平台 / 医疗协作平台 / 保险健康管理 / 慢病干预项目

---

## 一、系统总体目标与设计原则

### 1.1 系统目标

本认证体系用于：

* 构建可持续输出高质量行为健康教练的人才流水线
* 与「主动行为健康中枢」共享同一评估、画像、干预、追踪能力
* 实现：

  * 免费学习 → 入门认证 → 专业认证 → 高级认证 → 专家 / 导师体系
  * 学习、实训、带教、真实案例、平台实战的一体化闭环

核心目标三点：

1. 保证教练“可上岗可实战”而非只会讲课
2. 让认证等级与平台 Agent 权限、客户类型、收入能力直接挂钩
3. 形成“内容—能力—认证—任务—收益—晋级”的正循环系统

---

### 1.2 设计原则

* 与行为健康中枢共享同一核心数据结构（UserProfile / BehaviorPattern / InterventionPlan）
* 认证不是考试系统，而是“能力验证系统”
* 所有晋级必须满足：

  * 知识合格
  * 技能过关
  * 实战案例达标
  * 平台行为数据验证

---

## 二、认证等级总体系结构（五级制）

```
L0 公众学习者（免费）
   ↓
L1 初级行为健康教练（助理级）
   ↓
L2 中级行为健康教练（独立上岗）
   ↓
L3 高级行为健康教练（慢病 / 专项）
   ↓
L4 行为健康督导 / 讲师 / 专家
```

每一级对应：

* 能服务的人群范围
* 可调用的平台 Agent 权限
* 可参与的项目等级
* 收费与收益分成比例

---

## 三、能力模型四大维度（贯穿全部等级）

所有等级统一采用「K-M-S-V 四维能力模型」：

### 3.1 Knowledge（知识体系）

* 行为科学基础
* 慢病与代谢医学基础
* 生活方式医学
* 心理与动机理论
* 数据与指标解读

### 3.2 Method（方法体系）

* 行为评估方法
* 阶段模型（TTM / COM-B / 自主性模型）
* 行为处方设计
* 干预路径拆解
* 复盘与调整方法

### 3.3 Skill（核心技能）

* 动机访谈（MI）
* 教练式对话
* 阻抗处理
* 目标拆解
* 陪伴与反馈
* 平台工具使用能力

### 3.4 Value / View（观念与职业心智）

* 主动健康观
* 行为改变伦理
* 边界与角色认知
* 长程陪伴理念

## 四、分级课程体系设计

### L0 ｜公众学习层（免费入口层）

目标：导流 + 建立行为健康基础认知 + 识别潜在教练苗子

内容模块：

* 行为健康入门（6 讲 × 15 分钟）
* 慢病与代谢基础认知
* 血糖 / 血压 / 体重基础解读
* 行为改变常见误区

形式：

* 短视频
* 微课
* 互动测评

晋级条件 → L1：

* 完成全部课程
* 通过入门知识测验（≥80%）
* 完成 1 份个人行为画像

---

### L1 ｜初级行为健康教练（助理级）

定位：能辅助服务、能带基础人群、能执行标准路径

#### 知识模块

* 行为健康基础理论
* 代谢与慢病风险入门
* 行为阶段模型
* 指标解读基础（CGM / 血压 / 体脂）

#### 方法模块

* 标准评估流程
* 标准 14 天控糖路径
* 行为处方模板

#### 技能模块

* 基础教练对话
* 目标设定练习
* 简单阻抗处理

#### 考核构成

* 理论机考（40%）
* 标准案例模拟（30%）
* 对话录音评估（30%）

晋级条件：

* 理论 ≥ 80 分
* 完成 ≥ 3 个模拟案例
* 平台评估通过

权限：

* 可调用基础评估 Agent
* 只能服务低风险人群

---

### L2 ｜中级行为健康教练（独立上岗级，核心商业层）

定位：平台主力教练层，可独立带慢病 / 代谢异常客户

#### 知识模块

* 代谢综合征系统课程
* CGM 深度解读
* 行为—生理联动模型
* 心理与压力调节

#### 方法模块

* 个性化干预路径设计
* 多指标联动处方
* 动态调整方法

#### 技能模块

* 高级动机访谈
* 阻抗与失败复盘
* 长程陪伴能力

#### 实战要求

* 真实客户 ≥ 5 人
* 完整 14 天路径 ≥ 3 例
* 提交完整干预报告

#### 考核构成

* 理论考试（30%）
* 实战案例评审（40%）
* 对话质控评分（30%）

晋级条件：

* 至少 1 例指标明显改善案例
* 平台评分 ≥ A-

权限：

* 可调用全部代谢触手 Agent
* 可带收费客户

---

### L3 ｜高级行为健康教练（专项 / 慢病专家级）

定位：平台慢病主力 + 项目骨干 + 医疗协作层

专项方向可选：

* 糖尿病逆转专项
* 高血压专项
* 体重管理专项
* 心理压力专项

要求：

* 专项系统课程
* ≥20 例真实案例
* ≥3 例长期随访（3 个月以上）

考核：

* 专项笔试
* 真实病例答辩
* 督导现场评估

权限：

* 可参与方案设计
* 可带教低级教练
* 可参与平台内容共创

---

### L4 ｜督导 / 讲师 / 专家体系

定位：平台方法论中枢 + 认证体系核心师资

要求：

* ≥100 例干预案例
* ≥10 名教练带教记录
* 参与系统模型共建

权限：

* 课程讲师
* 认证评委
* Agent 规则共建权限

## 八、商业与生态闭环设计

### 8.1 收费路径

* 免费学习（L0）
* 初级认证收费
* 中级认证高价值收费
* 高级专项高端收费

### 8.2 收益绑定

* 等级越高 → 客户单价越高
* 等级越高 → 分成比例越高
* 等级越高 → 平台项目优先派单
