# 行健行为健康促进平台 — 用户角色体系说明

---

## 一、体系总览

行健平台采用 **六级晋升 + 四同道者** 体系，覆盖从个人健康管理到专家服务的完整成长路径。每位用户从"成长者"起步，通过学习、实践、分享、带教逐步成长为行为健康领域的教练、促进师乃至大师。

```
L1 成长者 → L2 分享者 → L3 行为健康教练 → L4 行为健康促进师 → L5 行为健康促进大师
   🌱          💬            🎯                ⭐                  👑
```

**三维积分体系**：
- **成长积分 (Growth)**：个人学习、完成评估、行为尝试等获得
- **贡献积分 (Contribution)**：分享知识、内容投稿、辅助他人等获得
- **影响力积分 (Influence)**：带教成果、社区影响、知识传播等获得

**四同道者制度**：L3 及以上级别须完成 4 位下级学员的带教陪伴并帮助其成长毕业，体现"助人即助己"理念。

---

## 二、L1 成长者（Grower）🌱

### 定义
成长者是平台的核心用户群体，代表正在积极管理自身健康、学习行为改变知识的个人。从"观察"走向"行动"，开启系统化的自我健康管理之旅。

### 如何成为成长者
- 成长积分达到 **100 分**
- 完成首次健康评估

### 在平台享有的权利

| 类别 | 具体权利 |
|------|---------|
| **AI 健康对话** | 与 12 个专业 AI Agent 一对一对话，涵盖代谢、睡眠、情绪、运动、营养、中医等领域 |
| **个性化学习** | 访问学习中心全部内容（文章、视频、课程、案例），智能推荐学习路径 |
| **健康数据管理** | 绑定穿戴设备，记录血糖、血压、睡眠、运动、心率等健康数据 |
| **行为处方** | 获取个性化行为处方（Behavior Rx），含 TTM 分期评估和阶段性策略 |
| **每日微行动** | 系统每日推送适合自身阶段的微小行动任务 |
| **智能监测方案** | 报名参加 14 天血糖监测等标准化健康管理方案 |
| **评估与测验** | 参加健康知识测验，巩固学习成果 |
| **积分与激励** | 每日签到、完成学习、参与挑战均获积分；连续打卡触发里程碑奖励 |
| **食物识别** | 拍照识别食物营养成分，建立饮食日记 |
| **内容投稿** | 向平台投稿健康知识（审核后发布），贡献积分 +50 |
| **同道者关系** | 可接受导师邀请，加入带教陪伴关系 |

### 可获得的帮助
- AI Agent 24 小时在线解答健康疑问
- 被分配至专业教练名下，接受一对一教练指导
- 收到设备异常告警提醒（血糖/血压/心率异常自动通知教练）
- 获取基于个人数据的 AI 推送建议
- 参与问卷调查获取针对性反馈

### 对平台的贡献
- 通过参与学习和行为尝试积累平台真实用户数据
- 投稿健康知识丰富内容库
- 参与问卷和评估帮助优化平台服务
- 行为改变的成功案例成为激励他人的榜样

### 晋级路径
- 积累成长积分至 **500** + 贡献积分至 **50** → 晋升为 L2 分享者
- 预计周期：**约 3 个月**

## 三、L2 分享者（Sharer）💬

### 定义
分享者是已在自身行为改变方面取得初步成效，并开始向他人传递健康理念和经验的用户。从"自我管理"走向"知识分享"，在帮助他人的过程中深化自身理解。

### 如何成为分享者
- 成长积分达到 **500**
- 贡献积分达到 **50**
- 完成 L1 阶段全部学分要求（总计 200 学分，必修 120）
- 行为稳定天数 ≥ 90 天，至少 2 项健康指标改善

### 在平台享有的权利

| 类别 | 具体权利 |
|------|---------|
| **L1 全部权利** | 继承成长者的所有功能和服务 |
| **知识分享** | 在社区发布经验分享、心得文章，获得互动反馈 |
| **内容评论** | 对平台内容发表评论，参与讨论 |
| **学习排行榜** | 参与积分、学习时长、连续打卡排行榜竞争 |
| **晋级进度看板** | 查看晋级进度（五维雷达图：学分/积分/同道者/实践/考核） |
| **导师角色** | 可被邀请作为 L0 观察员的"同道者"（陪伴导师） |
| **培训时长** | 参加培训课程积累培训时长（+5/小时），为晋级做准备 |

### 可获得的帮助
- 继续接受专属教练指导
- 学习进阶内容和案例分析
- 获得同道者互助网络支持
- 晋级系统提供明确的晋升路径指引和差距分析

### 对平台的贡献
- 以自身经历激励新用户
- 陪伴 4 名 L0 观察员成长（四同道者制度入门）
- 贡献优质知识内容
- 参与社区讨论提升平台活跃度

### 晋级路径
- 成长积分 **800** + 贡献积分 **200** + 影响力积分 **50**
- 通过 L3 晋级考试
- 完成 4 位 L1 成长者的带教陪伴（质量评分 ≥ 3.5）
- 累计 800 学分（必修 380），含案例实操 ≥ 10 例、带教时长 ≥ 50 小时
- 预计周期：**约 10 个月**

## 四、L3 行为健康教练（Coach）🎯

### 定义
行为健康教练是经过系统培训和考核认证的专业角色，具备指导他人进行行为改变的能力。从"分享者"正式转变为"教育者"，承担起带教和辅导的责任。

### 如何成为教练
- 通过三维积分门槛（成长 800 / 贡献 200 / 影响力 50）
- 通过 L3 晋级考试
- 完成 4 位 L1 成长者的带教陪伴（质量评分 ≥ 3.5）
- 累计学分 800（必修 380，覆盖理论与实操各模块）
- 实操评分 ≥ 240（400 分制），伦理考核满分通过

### 在平台享有的权利

| 类别 | 具体权利 |
|------|---------|
| **教练工作台** | 专属 Coach Dashboard：学员概览、健康数据监控、任务管理 |
| **学员管理** | 查看名下学员列表、详细档案、行为画像、评估报告 |
| **健康数据监测** | 查看学员的血糖、睡眠、运动、体征等实时数据 |
| **消息指导** | 一对一消息系统与学员沟通，发送鼓励、建议、提醒 |
| **评估分配** | 向学员推送评估量表，查看提交结果并评审 |
| **挑战活动** | 创建、管理、推送健康挑战活动，审核学员提交 |
| **内容创作** | 创建和发布健康内容（文章、课程、视频） |
| **问卷管理** | 创建问卷、发布、推送给学员，查看统计和导出数据 |
| **AI Agent 调用** | 调用 12+ AI Agent 辅助分析学员健康状况和生成建议 |
| **异常告警** | 接收学员设备异常告警，进行处理和跟进 |
| **推送审批** | 审批 AI 生成的推送建议，确保内容质量 |
| **数据分析** | 查看教练分析面板（群组概览、阶段分布、风险矩阵、干预效果） |

| **Agent 反馈** | 对 AI Agent 的回答质量进行评价，推动 Agent 持续改进 |
| **工具箱** | 使用教练专属工具（消息模板、快捷提醒、评估工具等） |

- 系统自动分配学员，智能匹配教练专长领域
- AI 推荐引擎自动生成学员推送建议（教练审批后执行）
- V007 策略引擎辅助选择最佳干预策略
- 行为处方系统（Behavior Rx）提供循证行为干预方案
- 教练排行榜和绩效面板提供自我评估参照
- 可升级为专家工作室（Expert Tenant），获得独立品牌空间

### 对平台的贡献
- 直接指导学员进行行为改变，是平台服务交付的核心力量
- 创建优质健康内容丰富平台知识库
- 带教 4 名 L2 分享者，壮大教练队伍
- 审批 AI 推送确保内容安全和质量
- 提供 Agent 反馈帮助 AI 系统持续进化

- 通过 L4 晋级考试
- 完成 4 位 L2 分享者的带教陪伴（质量评分 ≥ 4.0）+ 累计带教 5 名 L3

## 五、L4 行为健康促进师 / 督导专家（Promoter / Supervisor）⭐

### 定义
促进师/督导是行为健康领域的资深从业者和教学管理者。不仅精通行为改变方法，更能设计项目、开发课程、指导教练团队，是平台的骨干力量和质量保障者。促进师侧重推广和项目管理，督导侧重教学质量监控和教练培训。

### 如何成为促进师/督导
- 通过三维积分门槛（成长 1500 / 贡献 600 / 影响力 200）
- 通过 L4 晋级考试
- 完成 4 位 L2 分享者的带教并帕其晋级（质量评分 ≥ 4.0），累计带教 5 名 L3
- 主持 ≥ 2 个健康管理项目、开发 ≥ 1 门课程、贡献 ≥ 2 套方案模板

### 在平台享有的权利

| 类别 | 具体权利 |
|------|---------|
| **L3 全部权利** | 继承教练的所有功能 |
| **督导权限** | 监督和指导教练团队的工作质量 |
| **项目管理** | 设计和管理大型健康促进项目 |
| **课程开发** | 开发标准化培训课程和教学模块 |
| **方案模板** | 创建可复用的健康管理方案模板（如 14 天血糖监测） |
| **Agent 创建** | 作为专家创建自定义 AI Agent（动态 LLM 模板） |

## 九、四同道者制度详解

"同道者"是行健平台最核心的人才培养机制。每个高级别用户都需要带教 4 位下级学员，帮助他们成长毕业。

| 导师级别 | 需带教的学员 | 质量要求 |
|---------|-------------|---------|
| L2 分享者 | 4 位 L0 观察员 | 温和邀请模式 |
| L3 教练 | 4 位 L1 成长者 | 质量评分 ≥ 3.5 |
| L4 促进师 | 4 位 L2 分享者 | 质量评分 ≥ 4.0 |
| L5 大师 | 4 位 L3 教练 | 质量评分 ≥ 4.5 |

带教关系状态流转：邀请 → 活跃（进行中）→ 毕业（达标结业）/ 中断（未完成）

带教质量从以下维度评估：
- 学员的积分增长速度
- 学员的健康指标改善
- 学员的行为阶段推进（TTM S0→S6）
- 互动频率和响应时效
- 学员满意度评分
