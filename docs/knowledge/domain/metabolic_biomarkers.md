## 一、适用范围

- **适用Agent**：GlucoseAgent、WeightAgent、CardiacRehabAgent、CoachCopilotAgent、MetabolicExpertAgent、NutritionAgent、ExerciseAgent
- **适用情境**：S01情绪进食、S04绩效焦虑、S05职场高压、S07社交应酬（代谢指标是干预效果的客观锚点）
- **适用人群**：有穿戴设备数据或体检报告的用户；教练审核处方时的参考基线
- **不适用**：无任何代谢数据的用户（此时用BPT6+TTM行为评估替代）、急性病发作期（转CrisisAgent）

## 二、核心代谢指标红绿灯速查表（Agent首要调用入口）

> **调用规则**：当设备数据或用户描述涉及以下指标时，按本表判定风险等级，决定干预力度和是否触发医疗转诊。

### 2.1 血糖指标

| 指标 | 绿灯（正常） | 黄灯（警戒） | 红灯（危险） | 黑灯（急症→CrisisAgent） |
|------|------------|------------|------------|------------------------|
| 空腹血糖 (FPG) | 3.9~6.1 mmol/L | 6.1~7.0 mmol/L | 7.0~16.7 mmol/L | <2.8 或 >16.7 mmol/L |
| 餐后2h血糖 (2hPG) | <7.8 mmol/L | 7.8~11.1 mmol/L | 11.1~16.7 mmol/L | >16.7 mmol/L |
| CGM即时值 | 3.9~10.0 mmol/L | 10.0~13.9 mmol/L | 13.9~16.7 mmol/L | <2.8 或 >16.7 mmol/L |
| 糖化血红蛋白 (HbA1c) | <5.7% | 5.7~6.5% | 6.5~9.0% | >9.0% |
| CGM TIR（范围内时间） | >70% | 50~70% | <50% | — |

**Agent决策规则**：
- 绿灯 → 维持当前方案，正向激励
- 黄灯 → 强化行为干预，增加监测频率，教练关注
- 红灯 → 行为干预+建议就医，教练预警，CoachPushQueue优先审批
- 黑灯 → **立即触发CrisisAgent**，停止一切常规干预

**血糖特殊场景**：
- 低血糖（<3.9 mmol/L）：立即补充15g速效碳水（3~4块葡萄糖片），15分钟后复测
- 黎明现象（晨起空腹偏高但其他正常）：不急于干预，建议记录3天确认规律
- 运动后低血糖（运动后2h内<3.9）：调整运动前加餐策略，非运动禁忌

### 2.2 血压指标

| 指标 | 绿灯（正常） | 黄灯（正常高值） | 红灯（高血压） | 黑灯（急症→CrisisAgent） |
|------|------------|---------------|------------|------------------------|
| 收缩压 (SBP) | <120 mmHg | 120~139 mmHg | 140~179 mmHg | ≥180 mmHg |
| 舒张压 (DBP) | <80 mmHg | 80~89 mmHg | 90~109 mmHg | ≥110 mmHg |

**血压测量规范（Agent提醒用户）**：
- 测量前安静休息5分钟，排空膀胱
- 同一手臂、同一时段测量，取3次平均值
- 单次偏高不等于高血压，需非同日3次确认

**血压分级干预**：
- 正常高值 → 生活方式干预（减盐<5g/天、运动、减重、限酒）
- I级高血压（140~159/90~99）→ 强化行为干预3个月，若无改善建议就医
- II级高血压（160~179/100~109）→ 行为干预+建议尽快就医
- ⚠️ 收缩压≥180或舒张压≥110 → **CrisisAgent**

### 2.3 体重与体脂指标

| 指标 | 绿灯 | 黄灯 | 红灯 | Agent策略关键词 |
|------|------|------|------|--------------|
| BMI | 18.5~23.9 | 24.0~27.9（超重） | ≥28.0（肥胖） | 「代谢负担」（禁用「胖」字） |
| 腰围（男） | <85 cm | 85~90 cm | ≥90 cm | 中心性肥胖=内脏脂肪=代谢风险 |
| 腰围（女） | <80 cm | 80~85 cm | ≥85 cm | 同上 |
| 体脂率（男） | 10~20% | 20~25% | >25% | 仅参考，非诊断标准 |
| 体脂率（女） | 18~28% | 28~33% | >33% | 仅参考，非诊断标准 |

**BMI+腰围联合判定**：
- BMI正常+腰围超标 = 隐性肥胖（代谢风险与BMI超重等同）
- BMI超重+腰围正常 = 可能为肌肉型，需体脂率辅助判断

**减重目标设定规则**：
- 初始目标：3~6个月减轻体重5~10%（非一步到BMI正常）
- 周减重速度：0.5~1.0 kg/周（超过1.5 kg/周为不安全）
- ⚠️ BMI<18.5 → 禁止任何减重建议，筛查营养不良/进食障碍

### 2.4 血脂指标

| 指标 | 绿灯 | 黄灯 | 红灯 | 临床意义 |
|------|------|------|------|---------|
| 总胆固醇 (TC) | <5.2 mmol/L | 5.2~6.2 mmol/L | >6.2 mmol/L | 综合心血管风险 |
| 甘油三酯 (TG) | <1.7 mmol/L | 1.7~2.3 mmol/L | >2.3 mmol/L | 与饮食/饮酒直接相关 |
| LDL-C | <3.4 mmol/L | 3.4~4.1 mmol/L | >4.1 mmol/L | "坏"胆固醇，动脉粥样硬化核心 |
| HDL-C | >1.0 mmol/L(男) / >1.3(女) | 0.9~1.0 / 1.0~1.3 | <0.9 / <1.0 | "好"胆固醇，低值=风险升高 |
| 非HDL-C | <4.1 mmol/L | 4.1~4.9 mmol/L | >4.9 mmol/L | 综合致动脉粥样硬化脂蛋白 |

**行为干预优先级**：
- TG偏高 → 最易通过行为干预改善（戒酒、减糖、减重、运动）
- LDL偏高 → 饮食调整（减少饱和脂肪<总热量7%）+ 运动，效果有限时需药物
- HDL偏低 → 有氧运动是最有效的非药物手段

### 2.5 尿酸指标

| 指标 | 绿灯 | 黄灯 | 红灯 | 急症信号 |
|------|------|------|------|---------|
| 血尿酸（男） | <420 μmol/L | 420~480 μmol/L | >480 μmol/L | 关节红肿热痛→就医 |
| 血尿酸（女） | <360 μmol/L | 360~420 μmol/L | >420 μmol/L | 同上 |

**行为干预**：
- 严格限制：动物内脏、浓汤、啤酒、贝类海鲜
- 充分饮水：每日>2000ml（尿量>2000ml/天有利于尿酸排泄）
- ⚠️ 痛风急性发作期禁止运动、禁止节食减重（会升高尿酸）

### 2.6 其他关键指标

| 指标 | 绿灯 | 黄灯 | 红灯 | 与行为健康的关联 |
|------|------|------|------|--------------|
| TSH（促甲状腺素） | 0.27~4.2 mIU/L | 4.2~10.0 mIU/L | >10.0 或 <0.1 | 阳虚质必查；甲减→乏力/怕冷/体重增加 |
| CRP（C反应蛋白） | <1.0 mg/L | 1.0~3.0 mg/L | >3.0 mg/L | 慢性炎症标志；湿热质关注；与压力/睡眠相关 |
| HRV-SDNN | >100 ms | 50~100 ms | <50 ms | 自主神经功能；压力/睡眠/运动核心监测指标 |
| HRV-RMSSD | >40 ms | 20~40 ms | <20 ms | 副交感神经活性；低值=恢复能力差 |

## 三、指标联动解读矩阵

> 单一指标异常可能偶发，多指标联动异常才需要重点关注。

### 3.1 代谢综合征判定（中华医学会标准，满足≥3项）

| 组分 | 阈值 |
|------|------|
| 腰围 | 男≥90cm / 女≥85cm |
| 甘油三酯 | ≥1.7 mmol/L |
| HDL-C | 男<1.04 / 女<1.30 mmol/L |
| 血压 | ≥130/85 mmHg 或已服药 |
| 空腹血糖 | ≥6.1 mmol/L 或已确诊糖尿病 |

**Agent决策**：满足≥3项 → 标记「代谢综合征高风险」→ 教练优先关注 → 多Agent协同（Glucose+Weight+Exercise+Nutrition）

### 3.2 常见联动模式

| 联动模式 | 指标组合 | 核心机制 | 干预优先级 |
|---------|---------|---------|-----------|
| 糖脂联动 | FPG↑ + TG↑ + 腰围↑ | 胰岛素抵抗→肝脏脂肪合成增加 | 减重>控糖>降脂 |
| 压力-代谢 | HRV↓ + FPG↑ + CRP↑ | 皮质醇升高→胰岛素抵抗+炎症 | 压力管理>代谢干预 |
| 炎症-代谢 | CRP↑ + TG↑ + 尿酸↑ | 慢性炎症→代谢紊乱 | 抗炎饮食+运动+减重 |
| 甲减-代谢 | TSH↑ + BMI↑ + TC↑ | 甲状腺功能减退→代谢率降低 | **先治甲减**，再行为干预 |
| 睡眠-代谢 | HRV↓ + FPG↑ + 腰围↑ | 睡眠障碍→交感亢进→胰岛素抵抗 | 睡眠改善>代谢干预 |

### 3.3 体质×代谢风险交叉参照（对接 tcm_constitution_base.md）

| 体质 | 最高风险指标 | 必查项 | 行为干预首选 |
|------|-----------|--------|-----------|
| 痰湿质(TCM-4) | 腰围 + TG + FPG | 代谢综合征五项 | 减重（目标：腰围先达标） |
| 湿热质(TCM-5) | 尿酸 + CRP + FPG | 尿酸+肝功能+血糖 | 抗炎饮食 + 限酒 |
| 血瘀质(TCM-6) | LDL-C + 血压 | 血脂四项+血压+心电图 | 有氧运动 + 避免久坐 |
| 阳虚质(TCM-2) | TSH | 甲状腺功能 | **先排除甲减** |
| 气郁质(TCM-7) | HRV + CRP | HRV监测+CRP | 压力管理 + 户外运动 |

## 四、Agent调用指引

### 调用时机
```
以下情况触发本文件：
  - 设备数据上报（CGM/血压计/体重秤/手环）
  - 用户提及体检报告数值
  - 教练审核处方时需要参考基线
  - BAPS评估完成后的代谢风险映射

→ 调用本文件，进行指标→风险等级→干预力度判定
```

### 输出格式建议
```
1. 指标状态：「您的[指标]目前在[绿/黄/红]灯范围」
2. 风险说明：「这意味着[简明解释]」
3. 行动方向：「最优先的调整是[具体行为]」
4. 监测建议：「建议[频率]监测[指标]」
5. 联动提醒：「同时关注[关联指标]」（仅多指标异常时）
```

### 重点段落（按Agent分配）
- **GlucoseAgent** → §2.1血糖 + §3.2糖脂联动
- **WeightAgent** → §2.3体重体脂 + §3.1代谢综合征

- **CoachCopilotAgent** → §3.1代谢综合征判定 + §3.3体质交叉

## 五、禁止误用场景

- ❌ 不可用单次测量值判定风险等级（「你今天血压150，你是高血压」是错的——需非同日3次确认）
- ❌ 不可在用户情绪低落时展示多项异常指标（会加重习得性无助，违反S03/S10情境原则）
- ❌ 不可将黄灯指标描述为「不正常」（应说「需要关注」或「有改善空间」）
- ❌ 不可替代医疗诊断（「你的HbA1c 6.8%说明你有糖尿病」是错的——应说「这个数值建议到内分泌科进一步确认」）
- ❌ 不可在痰湿质用户的体重讨论中使用「胖」字（用「代谢负担」替代）
- ❌ 不可忽略甲减排查——阳虚质用户BMI偏高时，先查TSH再谈减重
