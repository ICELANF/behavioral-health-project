## 二、心理筛查量表速查（Agent评估入口）

> **调用规则**：当用户描述涉及情绪关键词时，优先用PHQ-2/GAD-2快速筛查；阳性再展开完整量表。严禁未经筛查直接给心理诊断标签。

### 2.1 抑郁筛查：PHQ-9

| 条目 | 内容 | 评分(0-3) |
|------|------|-----------|
| 1 | 做事时提不起劲或没有兴趣 | 完全不会(0)/好几天(1)/一半以上(2)/几乎每天(3) |
| 2 | 感到心情低落、沮丧或绝望 | 同上 |
| 3 | 入睡困难、睡不安稳或睡眠过多 | 同上 |
| 4 | 感觉疲倦或没有活力 | 同上 |
| 5 | 食欲不振或吃太多 | 同上 |
| 6 | 觉得自己很糟糕——觉得自己是失败者，或让自己或家人失望 | 同上 |
| 7 | 注意力难以集中，例如阅读报纸或看电视 | 同上 |
| 8 | 动作或说话速度缓慢到别人注意得到，或正好相反 | 同上 |
| 9 | 有不如死掉或用某种方式伤害自己的念头 | 同上 |

**PHQ-9 分级与Agent行为**：

| 总分 | 等级 | Agent行为 |
|------|------|-----------|
| 0~4 | 无/极轻 | 正常流程，预防性心理教育 |
| 5~9 | 轻度 | 行为激活(BA)建议 + 情绪日记 + 正常干预流程 |
| 10~14 | 中度 | 建议专业心理咨询 + CBT自助技术 + 通知教练关注 |
| 15~19 | 中重度 | **强烈建议专业治疗** + 教练主动介入 + 降低行为目标 |
| 20~27 | 重度 | **立即建议就医** + 教练预警 + 暂停高强度干预 |

> **条目9阳性(≥1)** → 不论总分，立即触发 `base/crisis_protocol.md` 自杀风险评估流程

**PHQ-2快速筛查**（条目1+2，总分0-6）：
- ≥3分 → 展开完整PHQ-9
- <3分 → 排除当前抑郁发作，继续常规流程

### 2.2 焦虑筛查：GAD-7

| 条目 | 内容 | 评分(0-3) |
|------|------|-----------|
| 1 | 感到紧张、焦虑或急切 | 完全不会(0)/好几天(1)/一半以上(2)/几乎每天(3) |
| 2 | 不能停止或控制担忧 | 同上 |
| 3 | 对各种各样的事情担忧过多 | 同上 |
| 4 | 很难放松下来 | 同上 |
| 5 | 由于不安而无法静坐 | 同上 |
| 6 | 变得容易烦恼或急躁 | 同上 |
| 7 | 感到似乎将有可怕的事情发生 | 同上 |

**GAD-7 分级与Agent行为**：

| 总分 | 等级 | Agent行为 |
|------|------|-----------|
| 0~4 | 无/极轻 | 正常流程，压力管理教育 |
| 5~9 | 轻度 | 放松训练(4-7-8呼吸/渐进肌肉放松) + 正常干预 |
| 10~14 | 中度 | CBT焦虑管理 + 建议心理咨询 + 通知教练 |
| 15~21 | 重度 | **强烈建议专业治疗** + 教练预警 + 暂停可能加剧焦虑的干预 |

**GAD-2快速筛查**（条目1+2，总分0-6）：
- ≥3分 → 展开完整GAD-7
- <3分 → 排除当前广泛性焦虑，继续常规流程

### 2.3 压力感知量表：PSS-10

| 条目 | 内容 | 评分(0-4) |
|------|------|-----------|
| 1 | 因为意料之外的事情发生而感到心烦意乱 | 从不(0)/偶尔(1)/有时(2)/常常(3)/总是(4) |
| 2 | 感到无法控制生活中重要的事情 | 同上 |
| 3 | 感到紧张和有压力 | 同上 |
| 4 | *(反向)* 有信心处理个人的问题 | 同上(反向计分) |
| 5 | 感到事情正朝着对你有利的方向发展 | 同上(反向计分) |
| 6 | 发觉自己无法应付所有必须做的事情 | 同上 |
| 7 | *(反向)* 能够控制生活中恼人的事情 | 同上(反向计分) |
| 8 | *(反向)* 感到自己掌控着一切 | 同上(反向计分) |
| 9 | 因为事情超出了控制范围而感到愤怒 | 同上 |
| 10 | 感到困难不断积累，以至于无法克服 | 同上 |

**PSS-10 分级**：

| 总分 | 等级 | Agent行为 |
|------|------|-----------|
| 0~13 | 低压力 | 维持现有压力管理策略 |
| 14~26 | 中等压力 | 引入正念/呼吸练习 + 联动 `domain/stress_cortisol_metabolism.md` |
| 27~40 | 高压力 | 压力为首要干预靶点 + 降低行为目标 + 教练关注 |

## 三、情绪-代谢双向联动模型

> **核心机制**：情绪问题与代谢异常互为因果，Agent需双向评估，避免单维度干预。

### 3.1 抑郁→代谢影响路径

```
抑郁情绪 → 5-HT/去甲肾上腺素↓ → 皮质醇持续升高
                                      ↓
                    ┌─────────────────┼─────────────────┐
                    ↓                 ↓                 ↓
              食欲改变           运动动力↓          睡眠障碍
           (暴食或厌食)      (久坐增加)        (失眠或嗜睡)
                    ↓                 ↓                 ↓
              体重波动          肌肉流失           HRV下降
           (BMI异常)         (基础代谢↓)       (自主神经失调)
                    └─────────────────┼─────────────────┘
                                      ↓
                              代谢综合征风险↑
```

### 3.2 代谢→情绪影响路径

| 代谢异常 | 情绪影响机制 | Agent联动策略 |
|----------|-------------|--------------|
| 持续高血糖 | 炎症因子↑ → 神经炎症 → 情绪低落 | 血糖达标优先，同步情绪监测 |
| 低血糖反复 | 焦虑-恐惧循环 → 过度进食代偿 | 稳定血糖 + 焦虑脱敏（联动GAD-7） |
| 肥胖/体重增加 | 身体意象障碍 → 自我否定 → S03螺旋 | 去体重执念 + 行为指标替代 |
| 慢性疼痛 | 疼痛-抑郁-失眠恶性循环 | 疼痛管理 + 睡眠优化 + 情绪支持 |
| HRV持续偏低 | 自主神经失调 → 焦虑躯体化 | 联动 `stress_cortisol_metabolism.md` HRV干预 |

### 3.3 Agent联合判定规则

| 条件组合 | 优先干预 | 联动文件 |
|----------|----------|---------|
| PHQ-9≥10 + 血糖红灯 | 先稳血糖→同步情绪支持 | `metabolic_biomarkers.md` |
| GAD-7≥10 + HRV<50ms | 先HRV呼吸训练→同步焦虑管理 | `stress_cortisol_metabolism.md` |
| PHQ-9≥15 + BMI≥28 | 先情绪稳定→再谈体重（禁止减重压力） | `weight_management.md` |
| PSS≥27 + 失眠 | 先睡眠干预→压力管理→情绪恢复 | `sleep_science.md` |
| PHQ-9条目9≥1 | **立即CrisisAgent，终止一切** | `base/crisis_protocol.md` |

## 四、核心情绪干预技术（Agent可调用工具箱）

### 4.1 行为激活(BA) — 轻度抑郁首选

**原理**：打破"低动力→不活动→更低落"循环

| 步骤 | Agent话术 | 注意事项 |
|------|----------|---------|
| 1. 活动监测 | "这周能记录每天做了什么、感觉如何吗？不用改变，只是观察" | 不评判，只记录 |
| 2. 价值-活动清单 | "什么活动做完后会觉得有点成就感或开心？" | 关注掌控感和愉悦感 |
| 3. 活动安排 | "我们试试这周加一个你喜欢的活动？哪怕只有10分钟" | 从极小行动开始 |
| 4. 渐进增加 | "上周做到了！要不要这周试试多加一个？" | 成功体验优先于量 |

### 4.2 认知重构(CR) — 中度焦虑/抑郁

**5栏表法**（Agent引导版）：

| 栏目 | Agent提问 | 示例 |
|------|----------|------|
| 情境 | "当时发生了什么？" | 血糖又超标了 |
| 自动思维 | "脑海中第一个想法是什么？" | "我永远管不住自己" |
| 情绪评分 | "当时有多难受？0-100分" | 沮丧80分 |
| 替代思维 | "如果好朋友遇到同样的事，你会怎么跟TA说？" | "一次超标不代表失败，看看整体趋势" |
| 再评分 | "用这个角度想，难受程度变了吗？" | 沮丧45分 |

### 4.3 正念减压(MBSR) — 压力/焦虑通用

| 技术 | 时长 | Agent引导要点 | 适用BPT6 |
|------|------|-------------|----------|
| 身体扫描 | 15~20min | "从脚趾开始注意身体感觉，不评判" | 情绪型(E)、矛盾型(A) |
| 正念呼吸 | 5~10min | "注意呼吸的进出，走神了轻轻拉回来" | 全类型 |
| 行走冥想 | 10~15min | "每一步都慢慢感受脚底接触地面" | 行动型(D)、环境型(V) |
| 慈心冥想 | 10min | "对自己说：愿我平安、愿我快乐、愿我健康" | 情绪型(E)、关系型(R) |

### 4.4 情绪日记(ABC模型) — 日常情绪管理

| 维度 | 含义 | Agent提问模板 |
|------|------|-------------|
| A (Activating event) | 触发事件 | "发生了什么事让你有这个感觉？" |
| B (Belief) | 信念/想法 | "你当时是怎么理解这件事的？" |
| C (Consequence) | 情绪和行为后果 | "然后你做了什么？感觉怎么样？" |

## 五、BPT6 × 情绪干预适配

| BPT6类型 | 情绪风险特征 | 首选干预 | 禁忌 |
|----------|------------|---------|------|
| **行动型(D)** | 压抑情绪→躯体化、否认心理问题 | 运动释放→行为激活 | 不要过早要求情绪表达 |
| **知识型(K)** | 过度理性化情绪、回避感受 | 心理教育→认知重构 | 不要跳过认知解释 |
| **情绪型(E)** | 情绪波动大、容易陷入反刍 | 正念→情绪日记→认知重构 | 不要否认TA的感受 |
| **关系型(R)** | 因关系问题引发情绪困扰 | 社交支持→人际效能训练 | 不要建议"别在意别人" |
| **环境型(V)** | 对环境变化敏感、适应困难 | 环境调节→渐进暴露 | 不要要求"忍一忍" |
| **矛盾型(A)** | 认知失调→焦虑、决策困难 | 价值澄清→决策平衡单 | 不要催促做选择 |

## 六、TTM阶段 × 心理筛查策略

| TTM阶段 | 筛查时机 | 评估重点 | Agent态度 |
|---------|---------|---------|----------|
| S0 无知无觉 | 仅在建立信任后 | PSS压力感知（门槛最低） | 纯倾听，不建议 |
| S1 知而未行 | 用户主动谈情绪时 | PHQ-2 + GAD-2快速筛查 | 共情为主，轻度心理教育 |
| S2 准备期 | 常规评估环节 | 完整PHQ-9 + GAD-7 | 评估作为行动计划依据 |
| S3 行动期 | 遇到障碍/退步时 | 聚焦压力源(PSS) + 情绪波动 | 正常化挫折感 |
| S4 维持期 | 定期复评(每月) | PHQ-9趋势对比 + 复发预警 | 肯定成就 + 预防 |
| S5 内化期 | 年度回顾 | 生活满意度 + 整体幸福感 | 成长叙事 |

## 七、特殊人群注意事项

### 7.1 围产期情绪筛查
- 使用EPDS（爱丁堡产后抑郁量表）替代PHQ-9
- EPDS≥13分 → 建议产科/精神科联合评估
- Agent额外关注：母婴关系、哺乳压力、睡眠剥夺、身体形象

### 7.2 老年人情绪筛查
- 使用GDS-15（老年抑郁量表）替代PHQ-9
- 关注躯体化表达（"浑身疼""吃不下""没力气"可能是抑郁）
- Agent注意：避免过度数字化，语言要更简洁温暖

### 7.3 青少年情绪筛查
- PHQ-A（青少年版）适用12~17岁
- 额外关注：学业压力、同伴关系、网络成瘾、身体意象
- Agent注意：尊重隐私、不向家长"告密"（除非安全红线）

## 八、Agent调用指南

### 调用入口
当用户出现以下关键词/信号时调用本知识：
- **情绪词**：抑郁、焦虑、紧张、烦躁、低落、绝望、无助、崩溃、想哭、没意思
- **行为信号**：暴饮暴食、失眠加重、社交回避、无法集中注意力、反复自责
- **代谢联动**：血糖波动+情绪低落、体重增加+自我否定、HRV持续偏低+焦虑

### 调用路径
1. 快速筛查（PHQ-2 + GAD-2）→ 确定风险等级
2. 如阳性 → 完整量表 → 分级Agent行为
3. 查BPT6类型 → 匹配干预技术
4. 联动相关领域 → 双向干预
5. 定期复评 → 趋势监测

### 禁止行为
- ❌ 给用户下精神科诊断（"你有抑郁症"）
- ❌ 在PHQ-9条目9阳性时继续常规对话
- ❌ 建议用户自行减药/停药
- ❌ 用"想开点""振作起来"等无效安慰
- ❌ 在情绪激动时推送行为任务
- ❌ 将筛查结果直接告知非专业人士（需教练审核后沟通）
