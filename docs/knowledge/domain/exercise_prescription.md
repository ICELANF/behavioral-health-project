## 一、适用范围

- **适用Agent**：ExerciseAgent、WeightAgent、CardiacRehabAgent、VisionGuideAgent、CoachCopilotAgent、TCMWellnessAgent
- **适用情境**：S08运动恐惧（核心场景）、S05职场高压（运动嵌入）、S01情绪进食（运动替代）、S09价值矛盾（运动≠痛苦）
- **适用人群**：所有用户（按PAR-Q+筛查分级后进入不同方案）
- **不适用**：急性心肌梗死/不稳定心绞痛/急性感染发热/未控制的严重高血压(>180/110)→CrisisAgent

## 二、运动前安全筛查（Agent首要检查点）

### 2.1 PAR-Q+快速筛查（7题，任一"是"需医疗清除）

| 编号 | 问题 | 阳性处理 |
|------|------|---------|
| 1 | 医生是否曾说您有心脏问题，只能做医生推荐的运动？ | → 医疗清除 |
| 2 | 运动时是否感到胸痛？ | → 医疗清除 |
| 3 | 过去一个月是否在非运动时感到胸痛？ | → 医疗清除 |
| 4 | 是否曾因头晕失去平衡或失去意识？ | → 医疗清除 |
| 5 | 是否有因运动而可能恶化的骨骼或关节问题？ | → 医疗清除 |
| 6 | 医生是否正在给您开血压或心脏方面的药物？ | → 医疗清除 |
| 7 | 是否有其他不宜运动的原因？ | → 医疗清除 |

**Agent规则**：
- 全部"否" → 可直接开始低~中强度运动
- 任一"是" → 建议就医获取运动许可后再开始
- 年龄>50岁 + 久坐生活方式 + 拟开始中高强度运动 → 建议运动前医学评估

### 2.2 运动绝对禁忌（触发CrisisAgent）

- 急性心肌梗死（48h内）
- 不稳定型心绞痛
- 未控制的心律失常伴血流动力学障碍
- 急性心力衰竭
- 急性肺栓塞/深静脉血栓
- 急性心肌炎/心包炎
- 主动脉夹层
- 收缩压>180mmHg或舒张压>110mmHg

## 三、FITT-VP运动处方六要素

### 3.1 总览

| 要素 | 含义 | 核心原则 |
|------|------|---------|
| **F** Frequency | 频率 | 有氧3~5天/周；抗阻2~3天/周 |
| **I** Intensity | 强度 | 按心率/RPE/谈话测试分级 |
| **T** Time | 时间 | 有氧150~300min/周；抗阻每组8~12次 |
| **T** Type | 类型 | 有氧/抗阻/柔韧/神经运动 |
| **V** Volume | 总量 | 每周≥500~1000 MET·min |
| **P** Progression | 递进 | 每1~2周递增≤10% |

### 3.2 强度分级标准

| 强度级别 | %HRmax | %HRR(Karvonen) | RPE(0-10) | 谈话测试 | 典型活动 |
|---------|--------|----------------|-----------|---------|---------|
| 极轻 | <57% | <30% | 0~2 | 唱歌无碍 | 散步、轻度家务 |
| 轻度 | 57~63% | 30~39% | 2~3 | 说话无碍 | 快走(5km/h) |
| 中等 | 64~76% | 40~59% | 4~5 | 说话略喘 | 快走(6~7km/h)、骑行、游泳 |
| 较强 | 77~95% | 60~89% | 6~8 | 说话困难 | 慢跑、爬坡、HIIT |
| 极强 | ≥96% | ≥90% | 9~10 | 无法说话 | 冲刺、竞技 |

**Karvonen公式（目标心率计算）**：
```
目标心率 = 安静心率 + (最大心率 - 安静心率) × 目标强度%
最大心率 = 220 - 年龄（估算公式，±10bpm误差）
```

**示例**：35岁用户，安静心率70bpm，目标中等强度(40~59%HRR)
- HRmax = 220-35 = 185bpm
- 下限 = 70 + (185-70)×0.40 = 116bpm
- 上限 = 70 + (185-70)×0.59 = 138bpm
- → 中等强度运动心率范围：116~138bpm

### 3.3 WHO推荐运动量（成人）

| 类别 | 最低推荐 | 额外获益 |
|------|---------|---------|
| 有氧运动 | 150min中等 或 75min较强/周 | 300min中等 或 150min较强/周 |
| 抗阻运动 | ≥2天/周，涵盖所有大肌群 | — |
| 久坐限制 | 每30~60min起来活动3~5min | 减少久坐总时长 |
| 柔韧性 | ≥2~3天/周 | 每次≥10min |

## 四、按人群分层的运动方案

### 4.1 按TTM阶段（与运动恐惧S08联动）

| TTM阶段 | 运动策略 | 单次时长 | 强度 | Agent话术方向 |
|---------|---------|---------|------|-------------|
| S0 无意向 | **不谈运动**，仅植入觉察 | — | — | 「您现在不需要运动。我只是好奇，您平时走路多不多？」 |
| S1 意向 | NEAT微运动（非运动性热量消耗） | 每次1~5min | 极轻 | 「如果有一种不用换衣服、不出汗的运动方式……」 |
| S2 准备 | 5分钟规则 + 散步 | 5~15min | 轻度 | 「只做5分钟，做完再决定要不要继续」 |
| S3 行动(早) | 快走/骑行，建立固定时段 | 20~30min | 中等 | 「这周3次已经很棒了，保持这个节奏」 |
| S3 行动(中) | 加入抗阻训练，目标300min/周 | 30~45min | 中等 | 「试试在快走基础上加2天力量训练」 |
| S4 维持 | 多样化运动，防止厌倦 | 45~60min | 中~较强 | 「您已经是运动者了，想不想试试新挑战？」 |
| S5 复发 | **降级重启**，回到S1~S2策略 | 5~15min | 极轻~轻 | 「停下来不是失败。我们从散步重新开始。」 |

### 4.2 NEAT——非运动性热量消耗（S0~S2核心策略）

| NEAT行为 | 额外消耗 | 植入场景 | 心理门槛 |
|---------|---------|---------|---------|
| 站立办公（每天2h） | ~170 kcal/天 | 工作中 | 极低 |
| 走楼梯代替电梯 | ~30 kcal/层 | 通勤 | 低 |
| 步行开会/打电话 | ~100 kcal/30min | 工作中 | 低 |
| 饭后散步10min | ~40 kcal | 午饭/晚饭后 | 极低 |
| 做家务（拖地/擦窗） | ~150 kcal/30min | 家庭 | 低 |
| 远处停车/提前一站下 | ~50 kcal/天 | 通勤 | 低 |

> **Agent核心话术**：「NEAT不是运动，是把动作悄悄嵌入到您已有的生活里。不需要换运动服，不需要去健身房，不需要出汗。」

### 4.3 按体质（对接TCM底座）

| 体质 | 推荐运动 | 强度上限 | 时间建议 | 禁忌 |
|------|---------|---------|---------|------|
| 气虚质(TCM-1) | 散步→太极→八段锦 | ≤60% HRmax | 上午9~11时 | ❌剧烈运动 ❌过度出汗 ❌空腹运动 |
| 阳虚质(TCM-2) | 户外快走+晒太阳、太极 | ≤60% HRmax | 上午9~11时 | ❌夜间运动 ❌水中运动(失温) |
| 阴虚质(TCM-3) | **游泳**（最适合）、瑜伽 | ≤70% HRmax | 傍晚16~18时 | ❌高温运动 ❌桑拿 ❌烈日下运动 |
| 痰湿质(TCM-4) | 快走→慢跑→游泳，**必须出汗** | 70~85% HRmax | 每次45~60min | ❌久坐 ❌运动后立即吃东西 |
| 湿热质(TCM-5) | 游泳、快走、骑行 | ≤75% HRmax | 避开正午 | ❌高温运动(加重湿热) |
| 血瘀质(TCM-6) | **有氧为主**(快走/慢跑/游泳) | 60~75% HRmax | 避免久坐>1h | ❌突然剧烈运动 ❌寒冷环境运动 |
| 气郁质(TCM-7) | **户外有氧**优先，集体运动>独自 | 60~80% HRmax | 每周5次×30~60min | ❌独自在家运动 ❌高竞争性运动 |
| 特禀质(TCM-8) | 室内运动为主(花粉季) | 个案评估 | 携带应急药物 | ❌花粉季户外 ❌未测试新环境 |

### 4.4 按BPT6行为类型

| 行为类型 | 运动适配策略 | 启动方式 |
|---------|-----------|---------|
| 行动型(ACT) | 给挑战目标（如半马训练计划），设置阶段里程碑 | 直接给方案，不用太多铺垫 |
| 知识型(KNO) | 先解释运动科学（FITT/心率区/代谢机制），理解后自主设计 | 推荐运动追踪App/手环，用数据驱动 |
| 情绪型(EMO) | 选择愉悦感强的运动（舞蹈/团课/户外），不强调指标 | 「今天想动就动，不想动就休息」 |
| 关系型(REL) | 伙伴运动（约跑/团操/家庭运动日），建立社交约束 | 帮TA找运动伙伴 |
| 环境型(ENV) | 环境设计优先（运动服放门口/健身房选家附近/设手机闹钟） | 减少启动摩擦 |
| 矛盾型(AMB) | 5分钟规则微承诺 + ACT接纳（接受矛盾，仍然行动） | 「带着犹豫出门，走5分钟再决定」 |

## 五、运动安全红线

### 5.1 运动中危险信号（立即停止）

| 信号 | 处理 |
|------|------|
| 胸痛/胸闷/压迫感 | **立即停止** → 坐下休息 → 持续>5min→CrisisAgent+120 |
| 头晕/眼前发黑 | 立即停止 → 坐或躺下 → 补水 → 未缓解→就医 |
| 严重气短（无法说出短句） | 立即降低强度 → 如不缓解→停止 |
| 心悸/心律不齐 | 停止 → 记录心率 → 反复发作→就医 |
| 运动中低血糖(<3.9mmol/L) | 停止 → 补充15g速效碳水 → 15min后复测 |
| 关节/骨骼剧痛 | 停止该动作 → RICE处理 → 48h未缓解→骨科就诊 |

### 5.2 特殊人群运动注意

| 人群 | 关键注意 |
|------|---------|
| 糖尿病用户 | 运动前血糖<5.0→先吃点心；>13.9+酮体→禁止运动；携带糖果 |
| 高血压用户 | 避免憋气动作（Valsalva）；避免倒立/头低于心脏；药物稳定后运动 |
| 心脏康复用户 | 严格按METs分级递进(详见cardiac_rehab_protocol.md)；全程心率监测 |
| 骨质疏松 | 避免高冲击运动（跳跃/跑步）；推荐步行+抗阻+平衡训练 |
| 孕妇 | 避免仰卧位运动(孕中晚期)；避免接触性运动；心率<140bpm |

## 六、运动与代谢指标的关系（对接metabolic_biomarkers.md）

| 代谢指标 | 运动效果 | 最佳运动方式 | 所需时间 |
|---------|---------|------------|---------|
| 空腹血糖 | ↓0.5~1.0 mmol/L | 餐后30min快走 | 每次30min，2周见效 |
| HbA1c | ↓0.5~0.7% | 有氧+抗阻组合 | 8~12周 |
| 甘油三酯 | ↓15~25% | 有氧运动（中等强度） | 4~8周 |
| HDL-C | ↑5~10% | 有氧运动（每周≥150min） | 8~12周 |
| 血压 | 收缩压↓5~8mmHg | 有氧+抗阻 | 4~12周 |
| 腰围 | ↓2~5cm（配合饮食） | 有氧≥300min/周+抗阻 | 8~16周 |
| HRV | SDNN↑10~20ms | 有氧（中等强度）+ 瑜伽 | 4~8周 |

## 八、禁止误用场景

- ❌ 不可对S0/S1阶段用户推运动方案（先解决动机，NEAT微运动是上限）
- ❌ 不可对气虚质/阳虚质用户推高强度运动（「动则生气但不过劳」）
- ❌ 不可忽略PAR-Q筛查直接推中高强度运动（安全第一）
- ❌ 不可用"必须每天运动"给用户施压（WHO最低标准是150min/周，可分散安排）
- ❌ 不可对运动恐惧(S08)用户描述运动的"痛苦"或"坚持"（应描述"轻松"和"享受"）
- ❌ 不可在血糖<5.0mmol/L时推荐运动（先补充碳水→血糖恢复→再运动）
- ❌ 不可忽视运动中危险信号（胸痛/头晕→立即停止→CrisisAgent评估）
