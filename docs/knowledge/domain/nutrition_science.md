## 二、三大营养素基础（Agent必备知识）

### 2.1 三大营养素速查表

| 营养素 | 供能比 | 功能 | 1g产热 | 推荐摄入 |
|--------|--------|------|--------|---------|
| **碳水化合物** | 50~65% | 主要能源、大脑唯一葡萄糖源 | 4kcal | 250~400g/天 |
| **蛋白质** | 10~20% | 组织修复、酶/激素、免疫 | 4kcal | 0.8~1.2g/kg/天(普通) |
| **脂肪** | 20~30% | 储能、激素合成、脂溶性维生素载体 | 9kcal | 总热量20~30% |

### 2.2 碳水化合物分类（血糖管理核心）

| 类别 | GI范围 | 代表食物 | 血糖影响 | Agent建议 |
|------|--------|---------|---------|----------|
| 低GI | ≤55 | 全谷物、豆类、大部分蔬菜、苹果 | 缓慢平稳升高 | ✅ 优先选择 |
| 中GI | 56~69 | 糙米、红薯、玉米、香蕉 | 中等升高 | ⚠️ 适量搭配 |
| 高GI | ≥70 | 白米饭、面包、糖果、西瓜 | 快速升高 | ❌ 控量/替换 |

> **GL(血糖负荷) = GI × 碳水克数 / 100**，比GI更准确反映实际血糖影响。
> Agent应指导用户关注GL而非单纯GI（如西瓜GI高但GL低，适量可吃）。

### 2.3 蛋白质需求（场景化）

| 人群/场景 | 蛋白质需求 | 优质来源 | 注意 |
|-----------|-----------|---------|------|
| 普通成人 | 0.8~1.0g/kg/天 | 鸡蛋/鱼/豆腐/瘦肉 | 均衡分配三餐 |
| 减重期 | 1.2~1.6g/kg/天 | 鸡胸肉/鱼虾/低脂奶 | 防止肌肉流失 |
| 运动者 | 1.2~2.0g/kg/天 | 蛋/奶/肉/蛋白粉 | 运动后30min补充 |
| 老年人(≥65) | 1.0~1.2g/kg/天 | 软嫩蛋白(蛋羹/鱼) | 预防肌少症 |
| 肾功能不全 | 0.6~0.8g/kg/天 | 优质蛋白为主 | **需医师指导** |

### 2.4 脂肪分类与健康影响

| 脂肪类型 | 来源 | 健康影响 | Agent建议 |
|----------|------|---------|----------|
| 单不饱和 | 橄榄油、坚果、牛油果 | ✅ 降LDL、护心血管 | 优先选择 |
| 多不饱和(ω-3) | 深海鱼、亚麻籽、核桃 | ✅ 抗炎、护心、改善HRV | 每周鱼2~3次 |
| 多不饱和(ω-6) | 大豆油、玉米油 | ⚠️ 过多促炎 | 控制ω-6:ω-3 <4:1 |
| 饱和脂肪 | 红肉、黄油、椰子油 | ⚠️ 升LDL | 控制<总热量10% |
| 反式脂肪 | 氢化植物油、薯条、奶精 | ❌ 升LDL降HDL | **严格避免** |

## 三、膳食模式速查（Agent推荐依据）

### 3.1 主要膳食模式对比

| 膳食模式 | 核心原则 | 适用人群 | 代谢获益 | 可持续性 |
|----------|---------|---------|---------|---------|
| **中国居民膳食指南** | 谷薯为主+蔬菜丰富+鱼禽蛋奶+大豆坚果 | 普通中国人群(首选) | 全面营养 | ⭐⭐⭐⭐⭐ |
| **地中海饮食** | 全谷物+鱼+橄榄油+蔬果+适量红酒 | 心血管风险高/代谢综合征 | 降CVD 30%、降炎症 | ⭐⭐⭐⭐ |
| **DASH饮食** | 高钾低钠+蔬果+低脂乳+全谷+坚果 | 高血压/心脏病 | 降SBP 8~14mmHg | ⭐⭐⭐⭐ |
| **低碳水饮食** | 碳水<130g/天，高蛋白/高脂 | 胰岛素抵抗/T2DM | 降HbA1c 0.5~1.0% | ⭐⭐⭐ |
| **极低碳水/生酮** | 碳水<50g/天 | 严重肥胖(短期) | 快速减重 | ⭐⭐(难持续) |
| **间歇性断食(16:8)** | 8h进食窗口 | S3+自律型 | 降胰岛素、促自噬 | ⭐⭐⭐ |

> **Agent选择原则**：以中国膳食指南为基础，根据用户代谢状况叠加特化模式。**禁止向糖尿病用药者推荐间歇断食或极低碳水（低血糖风险）**。

### 3.2 每日摄入量参考（中国膳食指南2022）

| 食物类别 | 推荐量/天 | 实操建议 |
|----------|----------|---------|
| 谷薯类 | 200~300g(谷)+50~100g(薯) | ≥1/3全谷物 |
| 蔬菜 | 300~500g | 深色蔬菜≥一半 |
| 水果 | 200~350g | 完整水果，非果汁 |
| 畜禽肉 | 40~75g | 优先禽肉、少红肉 |
| 水产品 | 40~75g | 每周至少2次鱼 |
| 蛋类 | 40~50g(约1个) | 整蛋(含蛋黄) |
| 奶及奶制品 | 300~500ml | 低脂/脱脂奶 |
| 大豆及坚果 | 25~35g | 无盐坚果 |
| 油 | 25~30g | 植物油为主 |
| 盐 | <5g | 高血压者<3g |
| 水 | 1500~1700ml | 白开水/淡茶 |

## 四、代谢指标 × 饮食干预策略

### 4.1 血糖异常（联动 `metabolic_biomarkers.md`）

| 指标状态 | 饮食策略 | 关键技术 |
|----------|---------|---------|
| FPG 6.1~7.0(黄灯) | 低GI为主 + 餐序调整(菜→肉→饭) | 先菜后肉再主食，降餐后峰值20~30% |
| FPG≥7.0(红灯) | 严格碳水管理 + 分餐(少量多餐) | 每餐碳水<50g + 搭配蛋白 |
| HbA1c 5.7~6.5%(前糖尿病) | 全谷物替换精白米 + 增加纤维 | 每日膳食纤维≥25g |
| CGM TIR<50% | 个体化碳水匹配 + CGM反馈学习 | 用CGM数据找出个人"友好食物" |

### 4.2 血脂异常

| 指标 | 饮食策略 | 具体措施 |
|------|---------|---------|
| LDL-C偏高 | 减少饱和脂肪 + 增加植物甾醇 | 植物油替黄油、每日坚果一把 |
| TG偏高 | 限制精制碳水+酒精 | 忌含糖饮料、白米→杂粮、限酒 |
| HDL-C偏低 | 增加ω-3脂肪酸 + 运动 | 每周鱼3次 + 有氧运动 |

### 4.3 高血压（DASH饮食核心）

| 营养素 | DASH目标 | 来源 |
|--------|---------|------|
| 钾 | 4700mg/天 | 香蕉/土豆/菠菜/番茄 |
| 钙 | 1000~1250mg/天 | 低脂奶/豆腐/深绿色蔬菜 |
| 镁 | 500mg/天 | 坚果/全谷/深绿叶菜 |
| 钠 | <2300mg(理想<1500mg) | 控盐 + 少加工食品 |
| 膳食纤维 | 30g/天 | 全谷/豆类/蔬果 |

### 4.4 高尿酸

| 策略 | 措施 | 注意 |
|------|------|------|
| 限高嘌呤 | 忌动物内脏/浓汤/某些海鲜 | 豆制品适量可吃(新指南) |
| 充足饮水 | ≥2000ml/天 | 白开水为主，忌含糖饮料 |
| 限酒 | 啤酒>白酒>红酒(风险排序) | 急性期绝对禁酒 |
| 碱化尿液 | 蔬菜水果(碱性食物)为主 | 尿pH目标6.2~6.9 |

## 六、TTM阶段 × 饮食干预策略

| TTM阶段 | 饮食干预焦点 | Agent行为 | 话术示例 |
|---------|------------|----------|---------|
| S0 | 不谈"该怎么吃" | 记录当前饮食（如有数据） | "你平时最喜欢吃什么？"(探索非评判) |
| S1 | 健康饮食意识唤醒 | 分享"一个食物知识" | "你知道先吃菜再吃饭可以让血糖更平稳吗？" |
| S2 | 选择一个小改变 | 从最容易的改变开始 | "这周试试把白米饭换一半杂粮？" |
| S3 | 渐进式饮食调整 | 餐盘法+食物日记+CGM反馈 | "看你这周的数据，杂粮饭那几天血糖很好！" |
| S4 | 灵活饮食+特殊场景应对 | 80/20法则+外出就餐策略 | "偶尔吃顿火锅完全可以，关键是平时的习惯" |
| S5 | 直觉饮食+身份认同 | 内在饥饿信号+不依赖规则 | "你已经知道身体需要什么了" |

## 九、实用饮食工具

### 9.1 餐盘法（最简饮食指导工具）

```
┌─────────────────────────────────────┐
│              餐盘法                  │
│   ┌──────────┬──────────┐          │
│   │          │          │          │
│   │  蔬菜    │  蛋白质  │          │
│   │  1/2     │  1/4     │          │
│   │          │          │          │
│   ├──────────┼──────────┤          │
│   │          │          │          │
│   │  全谷物  │  水果    │          │
│   │  1/4     │  (加餐)  │          │
│   │          │          │          │
│   └──────────┴──────────┘          │
│   + 一杯水/无糖饮品                 │
└─────────────────────────────────────┘
```

### 9.2 餐序法（控糖核心技术）

```
第1步：蔬菜(5分钟) → 第2步：蛋白质(5分钟) → 第3步：主食
效果：降低餐后血糖峰值 20~30%
原理：纤维+蛋白先到→延缓胃排空→碳水吸收减慢
```

### 9.3 手掌法（简易分量估算）

| 食物 | 参照物 | 约等于 |
|------|--------|--------|
| 蛋白质(肉/鱼) | 手掌大小×厚度 | ~75~100g |
| 碳水(米饭) | 拳头大小 | ~150g(熟) |
| 蔬菜 | 双手捧满 | ~200g |
| 脂肪(油/坚果) | 拇指尖大小 | ~10g |
| 水果 | 拳头大小 | ~150g |

## 十、Agent调用指南

### 调用入口关键词
- **饮食词**：吃什么、饮食、营养、热量、卡路里、碳水、蛋白质、脂肪、GI、食谱
- **食物词**：米饭、面条、水果、蔬菜、肉、鱼、油、盐、糖、零食、奶茶
- **行为词**：暴饮暴食、节食、代餐、不吃早餐、吃夜宵、外卖
- **指标词**：血糖高了/怎么吃、血脂高/饮食、尿酸高/忌口

### 调用路径
1. 识别饮食需求类型(控糖/降脂/减重/降压/降尿酸)
2. 联动代谢指标 → 确定饮食约束
3. 查TTM阶段 → 匹配干预深度
4. 查BPT6 → 匹配沟通方式
5. 查体质 → 叠加体质饮食宜忌
6. 选择膳食模式 → 给出具体餐食建议
7. 提供实用工具(餐盘法/餐序法/手掌法)

### 禁止行为
- ❌ 推荐极低热量饮食(<1000kcal)而无医嘱
- ❌ 向糖尿病用药者推荐断食(低血糖风险)
- ❌ 推荐特定品牌保健品/代餐产品
- ❌ 对过敏/食物不耐受者推荐致敏食物
- ❌ 用"垃圾食品"等道德化标签评价食物
- ❌ 在S0/S1阶段给具体饮食方案(先唤醒意识)
