## 一、适用范围

- **适用Agent**：CoachCopilotAgent、BehaviorRxAgent、MentalHealthAgent、MetabolicExpertAgent
- **适用情境**：S01–S10全情境（人格是行为模式的底层驱动变量）；优先用于 S03自我否定、S05职场高压、S06情绪崩溃、S07人际冲突
- **适用人群**：完成大五人格测评（NEO-PI-R / BFI / TIPI 任一量表），或教练通过观察已初步识别主导维度的用户
- **不适用**：用户未完成任何评估、急性精神障碍、人格障碍确诊期（转介临床）

> **与中医体质的关系**：两套框架并行。人格解释"为何这样做"，体质解释"身体如何反应"。双框架叠加可实现最高精度的个性化干预。

## 二、大五维度速查表（Agent首要调用入口）

> **调用规则**：用户描述行为困境或教练录入测评结果时，按本表匹配主导风险维度，进入对应干预路径。

### 2.1 五维度基本特征

| 维度 | 代码 | 高分表现 | 低分风险 | 健康关联 |
|------|------|---------|---------|---------|
| 神经质 | BFP-N | 情绪敏感、焦虑、自我批评 | （低神经质为健康因子）| 高分 → 应激反应过强、皮质醇升高、炎症风险↑ |
| 外向性 | BFP-E | 社交活跃、乐观、能量充沛 | 孤独、退缩、动力低落 | 低分 → 社会支持不足、情绪缺乏出口 |
| 开放性 | BFP-O | 好奇、创造、思维灵活 | 认知僵化、抗拒变化 | 低分 → 健康新知接受困难、行为改变阻力大 |
| 宜人性 | BFP-A | 友善、共情、合作 | 防御性高、冲突频繁 | 低分 → 人际紧张、社会支持缺乏 |
| 尽责性 | BFP-C | 自律、计划性、执行力强 | 冲动、混乱、规律感差 | 低分 → 健康行为维持困难、作息饮食不规律 |

## 三、心理机制解析（Agent理解层）

> **用途**：教练理解用户"为什么这样做"，不直接对用户呈现，用于引导语和干预策略选择。

### BFP-01｜高神经质 + 低外向性

**心理机制**：
- 杏仁核高度活跃，对威胁和不确定性过度敏感
- 内向使情绪缺乏社交渠道宣泄，情绪体验持续内化
- 前额叶调控不足 → 情绪反应强但调节力弱

**典型心理路径**：
```
外部刺激 → 威胁感放大 → 内在紧张 → 自我否定 → 回避 → 孤立加深 → 焦虑循环
```

**干预关键点**：先安全后成长；训练"觉察—命名—表达"情绪链；渐进式社交暴露

---

### BFP-02｜高神经质 + 低尽责性

**心理机制**：
- 情绪波动大，冲动反应强，多巴胺驱动即时满足
- 理性自我调节弱 → 计划制定后难以执行
- 循环结构：想改变 → 失败 → 自责 → 逃避

**典型心理路径**：
```
压力/负情绪 → 即时补偿（吃/玩/逃避）→ 罪恶感 → 自我否定 → 重复失控
```

**干预关键点**：结构化外部约束；小步成功体验强化自效；替换即时奖赏源

---

### BFP-03｜低尽责性 + 高外向性

**心理机制**：
- 强社交刺激需求 + 即时反馈偏好 → 延迟满足能力差
- 群体影响力大 → 易受社交情境驱动冲动行为

**典型心理路径**：
```
群体/情境刺激 → 冲动行为 → 愉悦体验 → 警觉放松 → 自控力下降
```

**干预关键点**：社交正向反馈替代刺激性行为；游戏化自律系统；价值导向动机连接

---

### BFP-04｜低宜人性 + 高神经质

**心理机制**：
- 防御性高 + 敌意归因偏差（预期他人有敌意）
- 内心信念："只有我能掌控""别人不可信"
- 杏仁核对社会威胁信号过敏

**典型心理路径**：
```
人际冲突 → 情绪过度反应 → 控制欲增强 → 关系恶化 → 孤立与敌意加深
```

**干预关键点**：觉察防御触发点；非暴力沟通练习；正念降低杏仁核反应

---

### BFP-05｜低开放性 + 高尽责性

**心理机制**：
- 对秩序和确定性强烈依赖 + 完美主义倾向
- 内部信念："必须做得对""不能犯错"
- 面对模糊和变化触发焦虑和强迫性控制

**典型心理路径**：
```
遇到新情境 → 焦虑/排斥 → 强迫控制 → 紧张疲惫 → 自责循环
```

**干预关键点**：允许不完美；正念觉察"控制欲"模式；建立试错友好环境

---

### BFP-06｜高外向性 + 高神经质

**心理机制**：
- 外向带来高社交能量需求，神经质导致易疲惫与情绪波动
- "外在兴奋掩盖内在焦虑"的补偿性模式
- 典型信念："我必须保持热情""我不能让人看到我脆弱"

**典型心理路径**：
```
过度社交 → 情绪耗竭 → 焦虑/抑郁 → 维持外向面具 → 内外不一致
```

**干预关键点**：建立社交恢复时间；练习情绪真实性表达；区分"表现的我"与"真实的我"

---

### BFP-07｜低开放性 + 低外向性

**心理机制**：
- 对外部世界兴趣低 + 探索动机弱
- 固守旧有经验 → 难以从环境获得积极刺激
- 长期处于"低激活状态" → 神经系统反应迟钝

**典型心理路径**：
```
缺乏新体验 → 情绪平淡 → 动力下降 → 社交退缩 → 心理封闭
```

**干预关键点**：小而新的体验激活兴趣系统；身体运动唤醒活力；可预测而温和的社交接触

---

### BFP-08｜低宜人性 + 低外向性

**心理机制**：
- 内向使社交动机弱，低宜人性使人际适应差
- 以理智或怀疑防御自己，对情感表达高度回避
- 典型信念："别人不会理解我""情绪是弱点"

**典型心理路径**：
```
社交压力 → 防御与回避 → 孤立与冷漠 → 情绪压抑 → 身心紧张
```

**干预关键点**：强化自我觉察；训练同理与表达；营造安静信任的对话氛围

## 四、五型行为健康干预策略（核心调用层）

> **说明**：以下按五个主导维度分别列出营养/运动/认知/社会支持/睡眠五维干预策略，供 Agent 生成个性化方案时直接引用。

---

### 4.1 高神经质型（BFP-N↑）——情绪安全优先

| 维度 | 核心策略 | 具体建议 |
|------|---------|---------|
| 营养 | 稳定血糖，抗炎饮食 | 温热均衡饮食，限咖啡因；补镁、维B群、Omega-3；避免饿—暴食循环 |
| 运动 | 舒缓性运动降应激 | 瑜伽、散步、游泳（每次30分钟）；关注"身体舒缓感"而非消耗量 |
| 认知 | 情绪命名 + 认知重构 | 觉察—命名—表达三步法；识别灾难化思维；正念冥想每日5分钟 |
| 社会支持 | 建立安全社交圈 | 从熟人小圈层入手；练习说"我有点需要倾听"而非压抑 |
| 睡眠 | 固定节律 + 夜间情绪卸载 | 固定入睡时间；睡前腹式呼吸5分钟；避免夜间反刍思维 |

**禁忌**：
- ❌ 禁止用高强度挑战性目标作为激励（会强化失败焦虑）
- ❌ 禁止在未建立安全感前要求开放式社交

**教练引导语（通用）**：
> "当你感到紧张时，身体最先出现什么信号？……我们能不能先照顾这些身体信号，而不是马上去解决问题？"

---

### 4.2 低外向性型（BFP-E↓）——能量激活优先

| 维度 | 核心策略 | 具体建议 |
|------|---------|---------|
| 营养 | 提升色氨酸/B群，激活神经递质 | 增加深色蔬菜、坚果、优质蛋白；规律三餐维持能量节律 |
| 运动 | 晨光暴露，温和激活 | 晨间户外步行（阳光+空气）；从5分钟开始；可以独自进行 |
| 认知 | 自我表扬 + 微成就记录 | 每天写下1件做到的事；练习"我今天做得不错"；避免对比自责 |
| 社会支持 | 小圈层深度关系 | 无需广泛社交；每天与1位信任的人有意义地交流3分钟即可 |
| 睡眠 | 建立早起仪式 + 晨光锚定 | 固定起床时间；清晨10分钟户外光照；睡前避免过度刺激 |

**禁忌**：
- ❌ 禁止强迫参与团体活动（适得其反）
- ❌ 禁止与外向型标准对比

**教练引导语**：
> "有没有一种社交方式让你觉得既安全又舒服？……我们可以从那里开始，不需要一下子改变很多。"

---

### 4.3 低宜人性型（BFP-A↓）——防御降低优先

| 维度 | 核心策略 | 具体建议 |
|------|---------|---------|
| 营养 | 抗炎饮食 + 降低应激性进食 | 少加工食品，高纤维；减少咖啡因与刺激性食物；吃饭时不讨论冲突 |
| 运动 | 压力释放型运动 | 跑步、登山、拳击类（释放积累敌意）；每周3次40分钟以上 |
| 认知 | 共情训练 + 非暴力沟通 | 觉察"防御触发词"；练习NVC四要素（观察/感受/需要/请求）|
| 社会支持 | 从倾听开始，建立信任 | 每天倾听他人3分钟不打断；记录一次共情体验 |
| 睡眠 | 夜间降压 + 安静独处 | 睡前独处安静环境；音乐放松辅助入睡；减少睡前回想冲突场景 |

**禁忌**：
- ❌ 禁止直接对质或批评
- ❌ 禁止在关系信任建立前要求情感开放

**教练引导语**：
> "我注意到你在关系里很有力量感，同时也承受着很多……如果对方并非有意为之，这件事会有什么不同？"

---

### 4.4 低尽责性型（BFP-C↓）——结构建立优先

| 维度 | 核心策略 | 具体建议 |
|------|---------|---------|
| 营养 | 规律三餐，预制化管理 | 固定用餐时间；事先准备健康食材；外出就餐预设策略 |
| 运动 | 短时多次 + 游戏化 | 每次10分钟，每天2-3次；加入打卡群；设置完成奖励 |
| 认知 | 任务分解 + 即时反馈 | 每天列"今日三件事"；完成即打卡；不用担忧明天 |
| 社会支持 | 责任伙伴制 | 找一个执行伙伴；约定每日回报进度；利用群体激励 |
| 睡眠 | 固定节点 + 放松仪式 | 固定起床/入睡时间（±15分钟）；睡前"关机仪式"（放下手机/关灯）|

**禁忌**：
- ❌ 禁止设定复杂长期计划（容易导致放弃）
- ❌ 禁止道德化批评（"你怎么又没做到"）

**教练引导语**：
> "我们一起把这个目标拆得再小一点，小到你今天就能完成那种……完成后你准备怎么奖励自己？"

---

### 4.5 低开放性型（BFP-O↓）——认知灵活性优先

| 维度 | 核心策略 | 具体建议 |
|------|---------|---------|
| 营养 | 渐进式多样化，不强制改变 | 每周尝试1种新食材；保留熟悉主食；用"试试看"代替"必须改变" |
| 运动 | 固定计划，循序渐进 | 先建立固定运动时段；再尝试新运动类型；体验优先于成绩 |
| 认知 | 反思训练 + 心理灵活性 | 正念觉察"控制欲"和"完美化"模式；允许自己不完美 |
| 社会支持 | 同质性支持群体 | 选择价值观相近的社群；强调理解而非改变 |
| 睡眠 | 维持固定节律，减少对睡眠完美化 | 规律节律优先；接纳偶尔睡眠不完美；不强迫完美入睡 |

**禁忌**：
- ❌ 禁止突然推荐颠覆性改变
- ❌ 禁止贬低其熟悉的方式

**教练引导语**：
> "你愿意先试试看，试一次，不喜欢可以不继续……你最熟悉的方式里有哪些部分其实对你很有效？"

## 六、每日觉察练习库（学员直接使用）

### 6.1 高神经质型（BFP-N↑）— 觉察焦点：安全感与情绪识别

每日反思问题：
1. 今天什么时候我感到身体紧绷？那一刻我在担心什么？
2. 我今天有表达真实的感受吗？还是选择了压抑？
3. 有什么事情其实没那么糟，只是我放大了它？
4. 今天我允许自己"什么都不解决"的时间有几分钟？

**教练提示**：引导学员注意"身体—情绪—想法"的链条，帮助从恐惧思维回到身体安全感。

---

### 6.2 低外向性型（BFP-E↓）— 觉察焦点：能量与联结

每日反思问题：
1. 今天有没有一个时刻让我感到平静或有活力？
2. 我今天有主动联系任何人吗？感受如何？
3. 我身体的活力指数（1-10分）是多少？我能为它加1分吗？

**教练提示**：用"兴趣微光"唤醒能量，引导从低激活走向温和的动力恢复。

---

### 6.3 低宜人性型（BFP-A↓）— 觉察焦点：防御与信任

每日反思问题：
1. 今天我在哪个时刻感到被冒犯？我如何解释对方的意图？
2. 如果我假设"对方并非敌意"，会发生什么？
3. 我今天是否主动表达了需求，而不是用愤怒掩饰？
4. 我最信任的人是谁？我如何与他保持连接？

**教练提示**：帮助学员看到"愤怒后面隐藏的脆弱"，引导表达真实需求而非防御反应。

---

### 6.4 低尽责性型（BFP-C↓）— 觉察焦点：冲动与自我承诺

每日反思问题：
1. 今天我在哪个时刻最想"放弃"？我当时的情绪是什么？
2. 我做过哪些让我短暂舒服但后来后悔的事？
3. 今天我是否完成了一个小目标？这让我感觉如何？
4. 我有没有在冲动之前暂停3秒？发生了什么变化？

**教练提示**：强化"从情绪驱动到有意识选择"的体验，让学员看到"能控制一点点"的力量。

---

### 6.5 低开放性型（BFP-O↓）— 觉察焦点：灵活性与控制

每日反思问题：
1. 今天什么事情没有按计划进行？我当时的感受是？
2. 我今天是否允许自己犯一点小错？结果如何？
3. 有没有一件我尝试的新体验？从中学到了什么？
4. 我今天的"完美"标准是否让我更紧张？

**教练提示**：鼓励在可控范围内"尝试新"，用体验替代评判。

## 七、行为练习处方（五维×五型矩阵）

> **使用说明**：Agent根据用户主导人格类型，从对应行组提取当周练习处方（每次选1-2条，不超过3条）。

### 7.1 高神经质型行为练习

| 维度 | 小步行动 |
|------|---------|
| 营养 | 每天记录一次"进食与情绪"关系；早餐固定时间进食建立生理节律感 |
| 运动 | 每天至少5分钟缓慢步行+觉察呼吸；每周一次舒展型运动（瑜伽/伸展操）|
| 认知 | 出现"担心"念头时用一句"这是我在想象，不是事实"中断；每天写下1件自己做得好的小事 |
| 社会支持 | 每天与1位朋友/家人进行3分钟真实交流；遇压力时说出"我现在有点需要倾听" |
| 睡眠 | 睡前30分钟关灯关屏，呼吸放松5分钟；记录一周睡前焦虑念头出现的时段 |

### 7.2 低外向性型行为练习

| 维度 | 小步行动 |
|------|---------|
| 营养 | 吃前10秒觉察：身体是否真的饿了？；每周选2次"静心用餐"不看手机 |
| 运动 | 将运动变成觉察仪式：注意呼吸与节奏；每周一次独自运动，观察与安静的关系 |
| 认知 | 每天留出5分钟独处，问自己"我今天真正需要的是什么"；遇兴奋决策时练习"推迟10分钟再行动" |
| 社会支持 | 每周主动倾听一位朋友，不提供建议，只反映情绪；检查是否用"忙碌"掩盖内在空虚 |
| 睡眠 | 睡前1小时不社交不刷屏，进行慢节奏拉伸；建立固定上床时间 |

### 7.3 低宜人性型行为练习

| 维度 | 小步行动 |
|------|---------|
| 营养 | 每天问自己"这顿饭是因为我想吃，还是因为别人希望我吃"；每周独自用餐一次 |
| 运动 | 每天早晨3分钟伸展+深呼吸，为"自己"而动；学习在不想运动时对自己诚实 |
| 认知 | 想说"没关系"时停1秒问"其实我有没有关系"；记录每天一次"表达真实感受"的时刻 |
| 社会支持 | 每天与信任的人分享一次"我自己的需要"；练习区分"帮助他人"与"讨好他人" |
| 睡眠 | 睡前写下1句对自己的温柔话语；减少睡前担心别人看法的思考 |

### 7.4 低尽责性型行为练习

| 维度 | 小步行动 |
|------|---------|
| 营养 | 每天允许一顿"无计划餐"练习放松控制；观察是否用严格饮食获取安全感 |
| 运动 | 每天固定起床时间（±15分钟）；设置简单目标（如每天喝2杯水）坚持7天，完成后自我奖励 |
| 认知 | 每天结束时问"今天我已经足够好了吗"；用"完成80%就停下"练习放下完美主义 |
| 社会支持 | 每周与朋友进行一次"无议题聊天"；学习表达"我也需要帮助" |
| 睡眠 | 睡前写下"今天我允许未完成的事明天继续"；失眠时使用渐进性放松代替强迫入睡 |

### 7.5 低开放性型行为练习

| 维度 | 小步行动 |
|------|---------|
| 营养 | 每周尝试一种新食材或菜式；用颜色标记饮食规律性建立记录习惯 |
| 运动 | 尝试不同类型运动（舞蹈、气功等），每周保持一次固定运动建立生活锚点 |
| 认知 | 每天记录一个新想法+一个旧模式；练习"让思绪落地"——写下可执行行动步骤 |
| 社会支持 | 每周与不同类型的人交流一次；练习表达"我正在探索，不代表我否定你" |
| 睡眠 | 睡前10分钟静坐，放下创意与思考；建立"创意笔记本"，把灵感写下再去睡觉 |

## 八、教练引导语库（按信任等级分层）

> **信任等级说明**：2-4级=初识/低信任；5-6级=建立中；7-9级=高信任/深度陪伴。对应 CoachCopilotAgent 的 trust_level 参数。

### 8.1 通用带练结构

| 阶段 | 时长 | 内容 |
|------|------|------|
| 开场安全感 | 2分钟 | "今天我们不是要改变你，而是一起看清楚你正在做什么。" |
| 身体觉察 | 2分钟 | "请注意此刻你的呼吸、身体姿态、情绪能量在哪里。" |
| 主题提问 | 5分钟 | 使用上方觉察练习问题，鼓励自由分享 |
| 深层反思 | 5分钟 | "这个习惯背后，也许藏着什么样的需求？" |
| 行动承诺 | 2分钟 | "你愿意从今天开始做一个微小的尝试吗？哪怕只是1分钟的觉察。" |

### 8.2 分人格类型引导语（2-4级信任）

| 类型 | 引导语 |
|------|--------|
| 高神经质 | "你今天感觉怎么样？……我注意到你刚才说话时好像有些担心，你愿意多说一点吗？" |
| 低外向性 | "我们不用今天就做很多改变，你愿意先说说你现在最在意的是什么吗？" |
| 低宜人性 | "你在某些方面非常清楚自己要什么，这是一种力量……今天遇到了什么让你不舒服的事吗？" |
| 低尽责性 | "上周你提到想尝试……后来发生了什么？没关系，我们来看看是什么挡住了你。" |
| 低开放性 | "你一直在用的方式里，有哪些部分其实对你很有效？我们先从那里开始。" |

### 8.3 分人格类型引导语（5-6级信任）

| 类型 | 引导语 |
|------|--------|
| 高神经质 | "你可以允许自己暂时不完美吗？让身体告诉你现在的需要，而不是脑子里的担心。" |
| 低外向性 | "有没有一种社交方式让你觉得安全又自在？我们可以先尝试那一种。" |
| 低宜人性 | "如果你试着理解对方的立场，你看到了什么新东西？" |
| 低尽责性 | "把目标拆成今天能做的一步，会是什么？……完成后你打算怎么奖励自己？" |
| 低开放性 | "你愿意先试试看，再来讨论结果吗？一次就好。" |

### 8.4 分人格类型引导语（7-9级信任）

| 类型 | 引导语 |
|------|--------|
| 高神经质 | "当这种担心出现，你身体深处真正需要的是什么？不是那个答案，而是那个感受。" |
| 低外向性 | "我想问你一个可能有点难的问题……当你真的允许自己不社交、不表现、只是存在，你有什么感受？" |
| 低宜人性 | "我注意到你在很多事情上都表现得很强……那个强的背后，住着一个怎样的你？" |
| 低尽责性 | "这个循环你自己也看到了……你觉得是什么让你这次选择了不一样的方式？" |
| 低开放性 | "如果你未来的自己回头看今天，这个'必须做对'的要求，是在保护你，还是在限制你？" |

## 九、Agent调用规则

### 9.1 调用时机

| 触发信号 | 调用动作 |
|---------|---------|
| 用户提交大五人格测评结果 | 调用本文档对应维度章节，生成解读摘要 |
| 用户描述行为困境（如"我总是戒不掉""我和同事总是闹矛盾"）| 匹配最近似组合类型，调用对应机制+干预策略 |
| 教练请求制定本周行动计划 | 根据人格类型调用第七章对应维度练习 |
| 学员觉察练习回顾 | 根据人格类型调用第六章对应问题 |
| 制定教练对话脚本 | 根据 trust_level 参数调用第八章对应等级引导语 |

### 9.2 输出格式建议

- **解读摘要**：不超过200字，结构为"你的主要人格特征是…这解释了为什么…你的优势是…需要关注的是…"
- **行动处方**：不超过3条，具体到时间和频率
- **引导语推送**：每次对话结束前推送1条本类型引导语

### 9.3 禁止误用场景

- ❌ 不能用人格类型标签定性用户（"你是焦虑型人"）
- ❌ 不能在无测评数据时仅凭主观猜测调用人格干预策略
- ❌ 不能用人格解读替代临床诊断（焦虑障碍/抑郁障碍等需转介）
- ❌ 不能同时对用户输出超过3种人格分析维度（信息过载）
