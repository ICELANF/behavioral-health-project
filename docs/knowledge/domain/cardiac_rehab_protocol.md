## 一、适用范围

- **适用Agent**：CardiacRehabAgent（主）、CardiacExpertAgent、ExerciseAgent、CoachCopilotAgent、WeightAgent、StressAgent
- **适用情境**：S08运动恐惧（心脏事件后恐惧运动）、S03自我否定（"我是废人了"）、S10习得性无助（"心脏病没救了"）
- **适用人群**：冠心病(PCI/CABG术后)、心肌梗死恢复期、稳定性心绞痛、慢性心力衰竭(NYHA I-III)、心脏瓣膜手术后、先天性心脏病修复后
- **不适用**：急性心肌梗死(<48h)、不稳定心绞痛、失代偿心衰(NYHA IV)、未控制的严重心律失常、主动脉夹层、急性心肌炎/心包炎 → 立即转CrisisAgent

## 二、心脏康复三期模型（Agent核心决策框架）

### 2.1 三期概览

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   I期(急性)  │──→│  II期(恢复)  │──→│  III期(维持) │
│   住院期     │    │  门诊监督    │    │  社区/居家   │
│  1~7天      │    │  3~6个月    │    │  终生       │
│  低强度活动  │    │  渐进运动   │    │  自主管理   │
│  教育为主    │    │  多学科团队  │    │  Agent支持  │
└─────────────┘    └─────────────┘    └─────────────┘
```

### 2.2 各期详细参数

| 参数 | I期(住院) | II期(门诊) | III期(社区) |
|------|----------|-----------|------------|
| **时间** | 出院前1~7天 | 出院后1~12周(核心)→6月 | 6个月后→终生 |
| **运动强度** | 1~3 METs | 3~6 METs(渐进) | 个体化(通常4~8 METs) |
| **心率目标** | 静息+20bpm以内 | HRR 40~60% | HRR 50~80% |
| **RPE** | 9~11(极轻~轻) | 11~14(轻~有点累) | 12~16(中等~累) |
| **监测** | 持续心电+SpO2 | 运动中心电+BP | 自我监测(心率带) |
| **频率** | 每日2~3次 | 3~5次/周 | ≥3次/周 |
| **时长** | 5~15min/次 | 20~40min/次 | 30~60min/次 |
| **Agent角色** | 信息提供 | 运动指导+恐惧管理 | 自主支持+复发预防 |

## 三、心脏康复运动处方（联动 `exercise_prescription.md`）

### 3.1 运动前安全评估

| 评估项 | 合格标准 | 不合格→行动 |
|--------|---------|------------|
| 静息心率 | <100bpm | 暂停运动，联系医生 |
| 静息血压 | SBP<160, DBP<100 | 药物调整后再启动 |
| 静息SpO2 | ≥95% | 心衰患者≥92%可运动 |
| 胸痛 | 无 | 有→立即停止→CrisisAgent |
| 呼吸困难 | 静息无 | 静息喘→暂停→评估 |
| 心电图 | 无急性ST改变 | 异常→医师评估 |

### 3.2 运动中监测红线（立即停止运动）

| 信号 | 阈值 | 行动 |
|------|------|------|
| 胸痛/胸闷 | 任何程度 | 停止→坐下→含服硝酸甘油→CrisisAgent |
| 心率 | >目标HR+20bpm 或 >年龄预测最大HR×85% | 立即降速→走路→坐下 |
| 血压 | SBP>220 或 DBP>110 | 停止运动→坐下→监测 |
| 血压 | 运动中SBP下降>10mmHg | 立即停止→平卧→呼叫 |
| 头晕/晕厥先兆 | 任何程度 | 停止→坐/卧→呼叫 |
| 心悸/不规则心跳 | 持续>30秒 | 停止→记录→通知教练 |
| RPE | ≥17(非常累) | 降低强度 |
| SpO2 | <90% | 停止→吸氧→呼叫 |

### 3.3 心脏康复运动类型

| 类型 | II期方案 | III期方案 | 禁忌 |
|------|---------|---------|------|
| **有氧** | 步行→功率车→快走 | 快走/游泳/骑车 | 无监护下跑步(早期) |
| **力量** | 8周后引入，低负荷高次数 | 渐进抗阻(RPE<14) | Valsalva动作(憋气发力) |
| **柔韧** | 每次运动前后5min拉伸 | 瑜伽(温和)/太极 | 深度前屈(头低于心脏) |
| **呼吸** | 腹式呼吸训练(每日) | 呼吸操+正念呼吸 | 过度换气 |

### 3.4 Karvonen目标心率计算（心脏康复特化）

```
目标HR = 静息HR + (最大HR - 静息HR) × 目标强度%

心脏康复特化：
- 最大HR：优先使用运动负荷试验实测值
- 若无实测：β-受体阻滞剂用户不能用220-年龄公式
  → 改用运动中实际达到的最高HR × 85%作为上限
- II期目标：HRR 40~60%
- III期目标：HRR 50~80%
```

## 四、恐惧-回避循环破解（CardiacExpertAgent核心能力）

### 4.1 心脏事件后恐惧模型

```
心脏事件(心梗/支架) → 对运动/活动的恐惧
         ↓
    ┌────┴────┐
    ↓         ↓
  回避运动   过度监测
    ↓         ↓
  体力下降   焦虑增加
    ↓         ↓
  活动更难   更怕运动
    └────┬────┘
         ↓
    功能下降+复发风险↑
```

### 4.2 系统脱敏阶梯（4周方案）

| 周次 | 行为目标 | Agent支持 | 恐惧预期(0-10) |
|------|---------|----------|---------------|
| Week 1 | 原地踏步2min + 腹式呼吸 + 自测心率 | 安全感建立："你的心率在正常范围" | 8→6 |
| Week 2 | 室内走动5min + 心率带佩戴 + 感觉日记 | 数据确认："看，你的心脏很好地应对了" | 6→4 |
| Week 3 | 户外散步10min + 社交陪伴 | 正常化："散步后心率升高是健康反应" | 4→3 |
| Week 4 | 快走15min + 自主选择路线 | 赋能："你已经掌握了安全运动的方法" | 3→2 |

### 4.3 认知重构（常见恐惧信念）

| 恐惧信念 | Agent回应 |
|----------|----------|
| "运动会引发第二次心梗" | "适度运动反而降低复发风险，不运动的风险更大。医学研究显示心脏康复可降低26%的死亡率" |
| "心跳加快说明心脏要出事" | "运动时心率升高是正常的、健康的反应。你的目标心率范围是XX-XX，在这个范围内是安全的" |
| "我的心脏太脆弱了" | "你的心脏经过治疗后正在恢复。就像骨折愈合后需要活动，心脏也需要适度'锻炼'来变得更强壮" |
| "我再也不能像以前那样了" | "很多心脏康复患者最终的运动能力超过了发病前。关键是渐进式恢复，而不是一步到位" |

## 五、药物-运动交互注意事项

| 药物类别 | 运动影响 | Agent注意 |
|----------|---------|----------|
| β-受体阻滞剂 | 心率反应钝化(HR不能如实反映强度) | **不能用目标心率指导运动** → 改用RPE(11~14) |
| ACE-I/ARB | 运动后低血压风险 | 起身缓慢 + 充足补水 + 避免蹲→站突变 |
| 他汀类 | 肌肉疼痛/无力 | 区分药物副作用vs运动后酸痛 |
| 抗凝药(华法林) | 出血风险 | 避免接触性运动/碰撞性运动 |
| 硝酸酯类 | 低血压 | 运动前30min不用药 |
| 利尿剂 | 脱水/电解质紊乱 | 充足补水 + 注意头晕 |

> **铁律**：Agent不建议调整药物剂量或停药，任何药物问题建议咨询主治医师

## 八、BPT6 × 心脏康复适配

| BPT6类型 | 康复障碍 | 适配策略 |
|----------|---------|---------|
| **行动型(D)** | 急于恢复→过度运动 | 明确安全边界 + 用数据约束 |

| **矛盾型(A)** | "康复有用吗？"犹豫 | 决策平衡单 + 证据说服(26%死亡率降低) |

## 九、特殊心脏人群注意事项

### 9.1 心力衰竭(HFrEF)
- 运动强度：HRR 40~50%起步，不超过60%
- 呼吸训练是核心（呼吸肌无力是运动耐量限制因素）
- 体重每日监测：3天内增>2kg → 液体潴留 → 通知医生
- 避免等长收缩(Isometric)运动

### 9.2 PCI/支架术后
- 术后1~2周：散步+日常活动
- 术后2~6周：开始正规康复运动
- 穿刺部位：股动脉→1周避免下肢运动；桡动脉→3天避免用力握
- 双抗期间(通常12个月)：避免接触性运动

### 9.3 CABG(搭桥)术后
- 胸骨愈合期(6~8周)：严禁上肢负重>5kg
- 不提重物、不开车(6周)
- 先恢复下肢运动→8周后渐进引入上肢

## 十、Agent调用指南

### 调用入口关键词
- **疾病词**：心脏、心脏病、冠心病、心梗、支架、搭桥、心衰、心律失常、心绞痛
- **症状词**：胸闷、胸痛、心悸、气短、喘不上气
- **康复词**：心脏康复、术后运动、康复训练、什么运动能做
- **恐惧词**：不敢运动、害怕复发、心脏会不会出事

### 调用路径
1. 安全评估 → 排除CrisisAgent红线（§3.1）
2. 判定康复分期(I/II/III) → 运动参数匹配
3. 评估恐惧程度 → 系统脱敏阶梯
4. 查药物清单 → 运动-药物交互注意
5. TTM+BPT6适配 → 个性化话术
6. 多组分干预 → 联动营养/心理/危险因素管理

### 禁止行为
- ❌ 给出超过医嘱范围的运动强度建议
- ❌ 建议调整/停用心脏药物
- ❌ 忽视运动中停止信号（§3.2红线必须执行）
- ❌ 对运动恐惧说"没事的，别怕"（必须用系统脱敏）
- ❌ 在I期建议无监护运动
- ❌ 对NYHA IV级心衰建议运动
