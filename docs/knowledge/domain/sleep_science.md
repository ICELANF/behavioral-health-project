## 一、适用范围

- **适用Agent**：SleepAgent、StressAgent、WeightAgent、MentalHealthAgent、CoachCopilotAgent、ExerciseAgent
- **适用情境**：S01情绪进食（睡眠不足→食欲素失调→暴食）、S05职场高压（熬夜→恶性循环）、S08运动恐惧（疲劳→抗拒运动）
- **适用人群**：PSQI>5分用户、穿戴设备显示睡眠效率<85%用户、自述失眠/早醒/嗜睡用户

## 二、睡眠基础知识（Agent回答框架）

### 2.1 睡眠阶段与功能

| 阶段 | 占比 | 脑波 | 核心功能 | 缺乏后果 |
|------|------|------|---------|---------|
| N1 浅睡 | 5% | θ波 | 入睡过渡 | — |
| N2 轻睡 | 45~55% | 纺锤波+K复合波 | 记忆巩固、运动学习 | 学习效率下降 |
| N3 深睡(SWS) | 15~25% | δ波（慢波） | **代谢修复**：生长激素分泌、组织修复、免疫重建、废物清除（glymphatic） | 代谢紊乱、免疫下降、认知退化 |
| REM 快速眼动 | 20~25% | β波（类清醒） | **情绪处理**：情绪记忆整合、创伤消化、创造力 | 情绪失调、焦虑加重、PTSD风险 |

**Agent可用的科普话术**：
> 「深度睡眠是身体的'维修时间'——血糖调节、脂肪代谢、免疫修复都在这个阶段完成。快速眼动睡眠是大脑的'情绪洗衣机'——白天的压力和情绪在这个阶段被处理和整合。两者缺一不可。」

### 2.2 睡眠需求量（按年龄）

| 年龄段 | 推荐时长 | 可接受范围 | 低于此值需关注 |
|--------|---------|-----------|-------------|
| 18~25岁 | 7~9h | 6~11h | <6h |
| 26~64岁 | 7~9h | 6~10h | <6h |
| ≥65岁 | 7~8h | 5~9h | <5h |

**⚠️ "我只需要睡5小时"是危险信念**：真正的短睡眠者（DEC2基因突变）<1%人口，绝大多数自称"少睡就够"的人处于慢性睡眠剥夺而不自知。

### 2.3 睡眠质量评估标准

| 指标 | 良好 | 一般 | 差 | 数据来源 |
|------|------|------|---|---------|
| 入睡潜伏期 | <15min | 15~30min | >30min | 穿戴设备/自述 |
| 睡眠效率 | >85% | 75~85% | <75% | 卧床时间vs实际睡眠 |
| 夜间觉醒次数 | ≤1次 | 2~3次 | >3次 | 穿戴设备 |
| 深睡占比 | >20% | 15~20% | <15% | 穿戴设备（⚠️精度有限） |
| PSQI总分 | ≤5 | 6~10 | >10 | 匹兹堡睡眠质量指数 |

## 三、CBT-I——失眠的一线非药物治疗（循证A级）

> **核心定位**：CBT-I（认知行为治疗-失眠）是AASM和ACP推荐的成人慢性失眠**一线治疗**，优先于药物。有效率70~80%，效果持久。

### 3.1 四大组件

#### 组件1：刺激控制（Stimulus Control）

**原理**：重建"床=睡眠"的条件反射，打破"床=焦虑/清醒"的错误关联。

**规则**（Agent可直接告知用户）：
1. 只在困倦时才上床
2. 床只用于睡觉和性生活——禁止在床上看手机/看电视/工作/吃东西
3. 如果躺下20分钟仍未入睡，**起床**到另一个房间做放松活动（昏暗灯光下阅读纸质书），直到再次困倦才回床
4. 每天固定时间起床（包括周末），**不补觉**
5. 白天不午睡（或限制≤20分钟，且在14:00前）

#### 组件2：睡眠限制（Sleep Restriction）

**原理**：通过压缩卧床时间，增加睡眠驱动力，提高睡眠效率。

**操作步骤**：
1. 记录1周睡眠日记，计算平均实际睡眠时长（如5.5h）
2. 设定「睡眠窗口」= 平均睡眠时长（最少不低于5h）
3. 固定起床时间（如06:30），倒推上床时间（01:00）
4. 当连续5天睡眠效率>85%时，窗口前移15~30分钟（00:30上床）
5. 当睡眠效率<80%时，窗口后移15分钟（01:15上床）
6. 逐步调整直至达到理想睡眠时长

**⚠️ 安全限制**：
- 卧床时间最少不低于5小时
- 有癫痫/双相障碍/嗜睡症者禁用
- 需要高警觉性职业（司机/操作员）慎用——初期白天嗜睡可能加重

#### 组件3：认知重构（Cognitive Restructuring）

**常见睡眠认知扭曲 → Agent纠正话术**：

| 扭曲想法 | Agent回应 |
|---------|---------|
| 「今晚一定又睡不着」 | 「您注意到了吗？越是担心睡不着，身体越紧张，反而更难入睡。试试告诉自己：即使今晚睡得不好，明天也能正常度过。」 |
| 「我必须睡满8小时」 | 「每个人需要的睡眠量不同。真正重要的不是时数，而是第二天的精力和状态。」 |
| 「失眠会毁掉我的健康」 | 「偶尔几晚睡不好不会造成永久伤害。身体有很强的自我调节能力，长期改善才是关键。」 |
| 「我需要安眠药才能睡」 | 「安眠药能帮助短期应急，但长期依赖反而削弱自然入睡能力。我们正在建立的方法，效果会更持久。」 |

#### 组件4：睡眠卫生教育（Sleep Hygiene）

| 规则 | 具体操作 | 原理 |
|------|---------|------|
| 固定作息 | 每天同一时间起床（±30min），包括周末 | 校准昼夜节律 |
| 光线管理 | 晨起30min内接受强光；睡前90min减少蓝光 | 褪黑素分泌调控 |
| 温度管理 | 卧室16~20°C；睡前90min温水浴/泡脚 | 核心体温先升后降→触发入睡 |
| 咖啡因限时 | 14:00后不摄入咖啡/茶/可乐/巧克力 | 咖啡因半衰期5~7h |
| 酒精禁忌 | 睡前3h不饮酒 | 酒精抑制REM→上半夜似好实差→后半夜反跳觉醒 |
| 运动时机 | 有氧运动最佳在下午~傍晚；睡前3h避免剧烈运动 | 运动升高核心体温，需3h降温 |
| 环境优化 | 遮光窗帘、耳塞/白噪音、手机放卧室外 | 减少觉醒刺激 |

## 四、睡眠与代谢的双向联动

### 4.1 睡眠不足→代谢恶化

| 睡眠不足时长 | 代谢后果 | 机制 |
|------------|---------|------|
| 1晚<6h | 次日胰岛素敏感性↓20~25% | 交感亢进→皮质醇↑→糖异生↑ |
| 连续5晚<6h | 食欲素(Ghrelin)↑28%，瘦素(Leptin)↓18% | 食欲激素失调→渴望高热量食物 |
| 慢性<6h(≥4周) | 代谢综合征风险↑45% | HPA轴慢性激活+炎症+脂肪重分布 |
| 慢性<5h | 2型糖尿病风险↑2.5倍 | 胰岛β细胞功能受损 |

### 4.2 代谢异常→睡眠恶化

| 代谢状况 | 对睡眠的影响 |
|---------|-----------|
| 血糖波动大（CGM SD>2.0） | 夜间低血糖觉醒 + 高血糖渗透性利尿→夜尿 |
| 肥胖（BMI≥28） | OSA风险↑4倍，深睡占比↓ |
| 痰湿质(TCM-4) | 身重嗜睡但睡眠质量差，OSA高发 |
| 阴虚质(TCM-3) | 盗汗→觉醒→入睡困难，阴虚火旺 |

### 4.3 打鼾/OSA筛查（STOP-BANG）

| 问题 | 评分 |
|------|------|
| **S**noring: 打鼾声大到隔壁能听到？ | 是=1分 |
| **T**ired: 白天经常疲倦/嗜睡/打瞌睡？ | 是=1分 |
| **O**bserved: 有人观察到你睡觉时呼吸暂停？ | 是=1分 |
| **P**ressure: 有高血压或正在服降压药？ | 是=1分 |
| **B**MI: BMI≥35？ | 是=1分 |
| **A**ge: 年龄>50岁？ | 是=1分 |
| **N**eck: 颈围>40cm？ | 是=1分 |
| **G**ender: 男性？ | 是=1分 |

- ≥3分 → 中高风险，建议就医进行多导睡眠监测(PSG)
- ≥5分 → 高风险，**强烈建议**就医

**Agent处理**：发现打鼾+白天嗜睡+BMI≥28 → 主动进行STOP-BANG评估 → ≥3分提醒就医

## 五、特殊人群睡眠要点

### 5.1 按体质（对接TCM底座）

| 体质 | 睡眠问题特征 | 首选干预 | 禁忌 |
|------|-----------|---------|------|
| 气虚质(TCM-1) | 易疲但入睡难、多梦、醒后不解乏 | 22:00前入睡+午休≤30min+黄芪红枣茶 | 过度劳累后不允许休息 |
| 阳虚质(TCM-2) | 嗜睡但睡不深，夜尿多(≥3次) | 睡前足浴+腹部保暖+姜枣茶 | 空调直吹、冷饮 |
| 阴虚质(TCM-3) | 盗汗、烦热、浅眠、梦多 | 吸汗睡衣+室温偏凉+银耳百合 | 睡前辛辣/热食 |
| 痰湿质(TCM-4) | 嗜睡、身重、打鼾、OSA高风险 | 侧卧+控制睡眠≤8h+除湿+减重 | 过度睡眠(>9h) |
| 气郁质(TCM-7) | 入睡难（心烦意乱）、早醒 | CBT-I刺激控制+玫瑰花茶+3件好事 | 躺在床上反刍 |

### 5.2 按BPT6行为类型

| 行为类型 | 睡眠干预适配 |
|---------|-----------|
| 行动型(ACT) | 给明确规则（几点上床/几点起床），直接执行 |
| 知识型(KNO) | 先解释CBT-I原理和证据，理解后才会配合 |
| 情绪型(EMO) | 先做认知重构（解除焦虑），再谈睡眠规则 |
| 关系型(REL) | 引入伴侣/家人配合（如约定全家熄灯时间） |
| 环境型(ENV) | 重点做环境改造（遮光/温度/白噪音/手机隔离） |
| 矛盾型(AMB) | 微承诺（先只做1件：固定起床时间），不强推全套 |

## 七、禁止误用场景

- ❌ 不可在S0/S1阶段强推CBT-I全套方案（会引发抗拒，先从睡眠卫生1~2条开始）
- ❌ 不可建议痰湿质用户"多睡觉"（嗜睡≠需要更多睡眠，控制≤8h）
- ❌ 不可对打鼾+嗜睡用户仅做行为干预而忽略OSA筛查（可能漏诊，有猝死风险）
- ❌ 不可推荐安眠药（超出行为健康范畴，需医师处方）
- ❌ 不可用"你睡太少了"制造焦虑（焦虑本身加重失眠）
- ❌ 不可在用户报告打鼾时轻描淡写（应做STOP-BANG评估→建议就医）
