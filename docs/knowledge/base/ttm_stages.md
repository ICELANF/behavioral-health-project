# L1 底座知识 · TTM 跨理论模型（六阶段）判断逻辑

> **适用范围**：全部 Agent 的阶段识别与干预策略匹配。
> **核心逻辑**：不同 TTM 阶段的用户，需要完全不同的干预重心——对 S0 用户推行动方案等于揠苗助长。

---

## 1. 维度定义

TTM（Transtheoretical Model，跨理论模型）是行健平台评估用户行为改变动机的核心工具。
六个阶段不是线性递进，而是螺旋式循环——复发是正常的，不是失败。

---

## 2. 六阶段判断标准与 Agent 策略

| 阶段代码 | 阶段名称 | 关键特征（判断逻辑） | Agent 干预重心 |
| :--- | :--- | :--- | :--- |
| **S0** | **无意向期** | 用户不认为自己的行为有问题，甚至抵触改变。关键词：「没问题」「就这样吧」「随便」。 | 去道德化与觉察，绝不给行动建议。 |
| **S1** | **意向期** | 意识到问题，打算在未来 6 个月内改变，但未行动。关键词：「应该改」「有空再说」「知道不好」。 | 动机激发，强化「为什么要变」，不催方案。 |
| **S2** | **准备期** | 打算在未来 1 个月内采取行动，可能有小步骤尝试。关键词：「想试试」「怎么开始」「有没有简单的方法」。 | 方案设计，提供具体低门槛 M-Action。 |
| **S3** | **行动期** | 已经迈出实质性步伐，持续时间在 6 个月以内。关键词：「我这周做了」「今天坚持了」「但有一天没做到」。 | 即时反馈，监测代谢指标，强化微小胜利。 |
| **S4** | **维持期** | 行为改变已持续 6 个月以上，成为日常习惯。关键词：「已经习惯了」「很少再想着要刻意做」。 | 环境支持，防止复发，向更高级挑战过渡。 |
| **S5** | **复发/循环期** | 重新回到了旧的行为模式，常伴随羞耻感。关键词：「又回到老样子了」「没用的」「算了」。 | 情感承接优先，BFR 复盘，重新路由回 S1/S2。 |

---

## 3. 跨阶段流转触发规则

* **S0 → S1**：用户主动询问「这个行为对血糖真的有影响吗？」
* **S1 → S2**：用户说「我想这个月开始改变」或主动索要方案。
* **S2 → S3**：用户反馈「我今天已经尝试了……」
* **S3 → S4**：行为坚持连续达到 6 个月，无明显复发。
* **任意阶段 → S5**：用户描述放弃或倒退，伴随负面情绪。
* **S5 → S1**：情感承接完成后，用户表达「想重新开始」。

---

## 4. 全局干预铁律

* **禁止跨阶推销**：严禁对 S0 阶段用户直接给出复杂行动指令——这是 TTM 干预最常见的失败原因。
* **情感优先**：在 S5 复发期，必须先执行「情感承接」节点，再做任何行动建议。
* **不评判复发**：复发是改变过程的正常组成部分，禁止用「又失败了」框架描述。
