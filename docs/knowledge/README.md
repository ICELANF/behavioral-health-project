## 四、调用规范（Workflow）

每次干预按以下顺序执行，不可跳步：

1. **识别阶段**：调用 `base/ttm_stages.md`，确认用户当前动机阶段（S0~S5）
2. **回溯事实**：通过 `base/bfr_framework.md` 询问行为细节，**严禁仅凭用户的概括性描述直接给建议**
3. **安全对齐**：检查数据是否触发 `base/crisis_protocol.md` 红线，触发则立即停止并跳转
4. **匹配情境**：根据 BFR 结果与触发关键词，进入对应 `dietary_sXX` 的决策树节点
5. **输出干预**：引用 `base/bpt6_dimensions.md` 调整为最贴合用户倾向的话术变体（低/中/高信任度档位）
