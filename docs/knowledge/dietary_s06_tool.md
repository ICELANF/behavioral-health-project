# S06 · 工具依赖 — 被CGM绑架的「精密机器」

> **典型话语**：「CGM摘掉就觉得没人管我了，周末连续暴食两天」

---

## 适用识别条件

| 维度 | 判断标准 |
|------|---------|
| TTM阶段 | S3–S4（准备期→行动期）—— 外控依赖型 |
| 信任分 | established（50–75%） |
| BPT6特征 | 外控依赖 · 内感觉知缺失 · 自主决策能力未发展 |
| 触发关键词 | CGM/设备/摘掉/空虚/没人管/数据/报警/暴食/传感器/拐杖 |

**代谢风险**：工具撤除→行为真空→指标反弹；数字焦虑→皮质醇升高；丧失内感觉知→无法预判进食对身体的影响

---

## 干预核心逻辑

工具依赖的本质是「内化失败」——患者学会了和数字对话，但没学会和身体对话。
**目标：完成从「工具反馈」到「自体感知」的迁移。**
代谢核心：建立「行为→身体感受→代谢结果」的完整内部反馈回路。

---

## 决策树（6节点，按顺序执行）

### 节点1：依赖程度评估
**触发条件**：用户描述工具撤除后的崩溃感
```
Agent话术：
「当设备离线时，你的感觉是什么？
空虚？焦虑？还是觉得失控？
打个分吗（0–10分）？」
→ 区分工具使用 vs 工具依赖
```
**教练行动**：摘除设备后焦虑 ≥ 7分 → 标记「高依赖」，设计渐进式脱离计划（不可突然撤除）

---

### 节点2：工具地位重置
**触发条件**：用户对数字有强烈的崇拜或恐惧
```
Agent话术：
「CGM是翻译官——它把身体语言翻成数字。
但你是主人，它是助手，不是监工。
它离开后你的反弹，是因为你还没学会
离开翻译官直接跟身体对话。」
```
**教练行动**：记录用户「看到数据后的情绪反应」；若多为负面（恐惧/焦虑）→ 优先处理工具关系

---

### 节点3：预测练习（建立内连接）
**触发条件**：用户愿意尝试「不看数据」的练习
```
Agent话术：
「下次你戴着设备吃饭前，我们先不看屏幕。
先闭上眼预测一下：
'吃完这顿饭，我的身体会有什么感觉？
精力充沛，还是沉重想睡？'
饭后一小时，先写下主观感受，再去对数据。
我们要训练的，是你那双内在的眼睛。」
```
**教练行动**：每周统计「预测准确率」；≥ 60% → 内感觉知开始建立，可推进间歇性佩戴方案

---

### 节点4：间歇脱离计划
**触发条件**：预测准确率达标，准备减少依赖
```
Agent话术：
「我们不需要一辈子戴着它。
下周尝试'间歇性佩戴'：戴三天，摘四天。
在摘掉的那四天里，模拟戴着它时的选餐逻辑。
我们要看看，没有报警声的时候，
你的'行为惯性'能维持多久。」
```
**教练行动**：记录「脱敏期」的行为偏差率；偏差率 < 20% → 每周逐步延长脱离时间

---

### 节点5：行为锚点替代
**触发条件**：准备建立不依赖设备的行为规则
```
Agent话术：
「把『看到血糖10.0就去运动』
替换为『只要吃完午饭就自动散步10分钟』——
锚点行为是内生的生活习惯，
不依赖任何电子设备的电源或传感器。」
```
**教练行动**：建立3个「设备无关」的行为锚点，写入行为处方，在工具脱离期也保持执行

---

### 节点6：身体感受日志
**触发条件**：开始建立内感觉知体系
```
Agent话术：
「记录下吃完高碳水后的『脑雾感』或『困倦感』——
这种主观感受比冷冰冰的数字更能产生持久动力。
不用写很多，就一句话：『今天吃了什么，感觉怎么样』。」
```
**教练行动**：追踪身体感受日志的丰富度；从「没感觉」到「能描述具体感受」是内感觉知发展的里程碑

---

## M-Action（按代谢改善优先级）

| 优先级 | 具体动作 | 代谢机制 |
|--------|---------|---------|
| P1 | 饭前「盲测」预测（不看血糖先写感受预测） | 训练内脏感受器和大脑预测系统的协同，是内感觉知的核心练习 |
| P2 | 身体感受日志（进食后20分钟，一句话） | 建立食物-感受映射的内部数据库，替代外部数字依赖 |
| P3 | 建立「饭后散步」锚点（设备无关） | 无论是否佩戴CGM都执行，从「数字驱动」转为「习惯驱动」 |
| P4 | 间歇性佩戴（每周脱离4天） | 渐进脱敏，每次脱离后记录自主行为表现，建立「我能独立做到」的证据 |
| P5 | 腹部呼吸感知练习（3分钟，进食前） | 激活内感觉知系统，训练肠-脑轴的信号感知，是工具依赖的根本解药 |

---

## 代谢监测重点

| 指标 | 风险阈值 | 行为健康干预 |
|------|---------|------------|
| 血糖变异性CV | CV > 36% 为高风险 | 即使佩戴CGM，若行为未内化，摘除后CV会迅速上升；内化成功则CV稳定 |
| 自主神经功能 | HRV作为代理指标 | 工具脱离期HRV稳定，说明自主神经调节改善，内感觉知发展良好 |
| 进食节律 | 监测进食时间规律性 | CGM在位时进食规律，摘除后是否保持？规律性维持是内化成功的指标 |
| 代谢灵活性 | 空腹与餐后血糖差值 < 4 mmol/L | 真正的代谢健康是「不依赖工具也能维持的稳定」，是干预的终极目标 |

---

## 铁律边界 ⚠️

| 禁令 | 原因 |
|------|------|
| ❌ 禁止突然撤除工具 | 高依赖者骤然摘除CGM会引发焦虑崩溃，必须渐进式脱敏，不可强制撤除 |
| ❌ 禁止强化数字恐惧 | 「看，血糖10.0，你今天又失败了」——强化工具依赖，而非推动内化 |
| ❌ 禁止单纯依赖工具随访 | 只看数据不看行为，是很多医疗随访失败的原因——工具是辅助，行为才是核心 |
| ⚕️ 工具撤除时间点 | 内感觉知准确率 ≥ 60% 后，才启动正式脱离计划；切勿提前撤除 |

---

## 叠加情境提示

- **S06 + S04**（工具依赖 + 绩效焦虑）：数字控制型患者高发组合；先处理S04降低频率反馈，再处理S06建立内感觉知

## 8. 话术变体库（扩展层）

* **低信任度（2~4分）**："你现在非常依赖CGM给你安全感，这完全合理——当身体还没建立信任时，数字是唯一的锚点。我们不着急摘掉它，先来练一个小游戏：在看数字之前，先猜猜你现在的血糖大概是多少？"
* **中信任度（5~6分）**："你的CGM是教练，不是监控。我们现在要做的，是让你的身体开始学会自己汇报——不是替代CGM，而是让你有两个信息来源：一个机器的，一个身体的。"
* **高信任度（7~9分）**："你的内感觉知准确率这周达到了60%以上，说明身体已经开始'说话'了。我们来做一个阶段性实验：周末两天不戴CGM，记录你的预测和实际感受差距，看看差多少？"
