# S07 · 社交应酬 — 身不由己的「餐桌交际者」

> **典型话语**：「每周三次应酬，客户劝酒老板盯着，血糖血压每次都爆表」

---

## 适用识别条件

| 维度 | 判断标准 |
|------|---------|
| TTM阶段 | S2–S3（意向→准备期） |
| 信任分 | established（50–70%） |
| BPT6特征 | 环境控制能力低 · 社交压力应对策略匮乏 · 冲动控制弱（社交情境下）|
| 触发关键词 | 应酬/客户/劝酒/老板/饭局/不给面子/大鱼大肉/没法控制/生意 |

**代谢风险**：高频应酬→长期高钠高脂→血压升高+内脏脂肪累积；酒精→肝糖原释放→血糖骤升；习得性无助→放弃代谢管理

---

## 干预核心逻辑

应酬是「高压情境」，患者不是在与食物对抗，而是在与社交权力结构对抗。
**策略：建立「防御性选择架构」——从被动受害者到主动剧作家。**
代谢核心：在应酬中建立「护城河」，最小化代谢损害。

---

## 决策树（6节点，按顺序执行）

### 节点1：应酬频率评估
**触发条件**：用户提到应酬困境
```
Agent话术：
「你每周大概有几次？主要是客户应酬还是内部聚餐？
你在席间通常处于什么角色——主请、被请，还是同级？」
→ 绘制「应酬权力地图」
```
**教练行动**：应酬频率 ≥ 4次/周 → 代谢损害风险极高，需要医疗评估+行为双线干预

---

### 节点2：面子-行为解构
**触发条件**：用户认为不吃不喝就是不给面子
```
Agent话术：
「生意成不成，核心在专业和资源，
不在你多吃的那口油腻或多喝的那杯酒。
我们可以表现出极高的社交热情，
但在'入口'的动作上，我们可以做一个聪明的剧作家。」
```
**教练行动**：帮助用户建立「社交热情」与「入口选择」分离的认知框架——这是应酬干预的核心前提

---

### 节点3：餐前预防出击
**触发条件**：下次应酬前的准备
```
Agent话术：
「进包间前先吃一个苹果或喝一杯高纤维饮料——
带着7分饱的肚子进场，大脑会更理智，
不会被那些高脂高盐的菜品瞬间俘虏。」
→ 饱和攻击策略
```
**教练行动**：追踪「餐前预防执行率」；执行时vs不执行时的血糖峰值对比，作为持续动力来源

---

### 节点4：点菜权争夺
**触发条件**：用户有机会参与点菜
```
Agent话术：
「主动拿菜单，建议加两道白灼蔬菜或白灼虾——
只要餐桌上有你可吃的东西，你就有了行为安全区。
加菜时可以说：'今天我来做东道主精神，再加两个清淡的'」
```
**教练行动**：提供「应酬加菜脚本」——加菜时的社交话术（不暴露健康目的）

---

### 节点5：进食顺序护城河
**触发条件**：已经坐下，菜开始上桌
```
Agent话术：
「记住这个顺序：水→菜→肉→粮。
先喝两杯温水，大口吃绿叶菜垫底——
膳食纤维像滤网一样，
减缓随后摄入的油脂和糖分的吸收速度。」
```
**教练行动**：应酬后30分钟血糖数据追踪；实施进食顺序后血糖峰值下降是持续执行的激励

---

### 节点6：酒精降级策略
**触发条件**：无法避免饮酒的场合
```
Agent话术：
「永远在酒杯旁放一杯大白水。
每喝一口酒，必喝三口水。
加速酒精代谢，同时增加拿杯子的动作频率，
减少被倒酒的机会。
或者以茶代酒：'最近在调理一个指标，
指标达标前以茶代酒，敬大家的兴致。'」
```
**教练行动**：建立「应酬饮酒量-次日血糖」的个体化数据库，让客观数据替代主观感受成为约束力

## M-Action（按代谢改善优先级）

| 优先级 | 具体动作 | 代谢机制 |
|--------|---------|---------|
| P1 | 餐前30分钟吃苹果或喝蔬菜汁 | 果胶+纤维素降低胃排空速率，进入餐厅时已有60–70%饱腹感 |
| P2 | 进食顺序：水→蔬菜→蛋白质→主食 | 可使餐后血糖峰值降低20–30%，是应酬场景中代谢保护最强的单一干预 |
| P3 | 涮水技术（每口菜涮白开水/清茶） | 减少油盐摄入约35–40%，配合清茶不影响社交形象 |
| P4 | 酒：喝一口水喝三口（1:3策略） | 加速酒精代谢，降低血液酒精峰值，减少对肝脏代谢的冲击 |
| P5 | 应酬后次日：晨走20分钟+轻断食早餐 | 帮助肝脏代谢过剩的肝糖原，加速恢复代谢基线 |

## 代谢监测重点

| 指标 | 风险阈值 | 行为健康干预 |
|------|---------|------------|
| 餐后2h血糖 | ≥ 10 mmol/L 为显著升高 | 应酬高碳水+酒精→血糖双重打击；餐前预防+进食顺序是必要干预 |
| 血压（应酬后） | ≥ 140/90 需立即关注 | 高钠菜肴+酒精→血压波动；应酬次日需监测 |
| ALT/AST | > 40 U/L 提示肝功能负担 | 高频饮酒+高脂应酬→肝脂肪变性，需纳入季度代谢评估 |
| 甘油三酯 | ≥ 1.7 mmol/L 为边界 | 应酬人群甘油三酯普遍偏高，是代谢综合征早期信号 |

---

## 铁律边界 ⚠️

| 禁令 | 原因 |
|------|------|
| ❌ 禁止孤立建议 | 「你就说你生病了不能喝」——忽略职场生态，导致社交受挫，产生更强补偿性进食需求 |
| ❌ 禁止过度补偿 | 「应酬完第二天断食」——暴食-断食循环破坏代谢弹性，加剧代谢综合征风险 |
| ❌ 禁止忽视社交结构 | 不建立策略脚本就要求「保持健康」，是无效干预 |
| ⚕️ 代谢监测节点 | 应酬频率 ≥ 3次/周 → 需每月检测空腹血糖+甘油三酯+血压，医疗随访不可缺失 |

## 8. 话术变体库（扩展层）

* **低信任度（2~4分）**："应酬这件事，我完全理解你没得选——那是你的工作，不是你的错。我们不是要让你拒绝饭局，而是找一套只有你自己知道的'餐桌护具'，在这个局里把自己保护起来。"
* **中信任度（5~6分）**："你每次应酬后的血糖数字，其实不只是食物的问题——是酒精、高钠、还有那种'被看着必须吃'的紧张感叠加在一起的。我们先找到哪个因素对你影响最大，从那里下手。"
* **高信任度（7~9分）**："你已经掌握了进食顺序和以水换酒的策略，上次应酬你的峰值比之前低了不少。这次饭局之前，我们来做一个'预防性设计'——菜单出来后发给我，我们一起选你的'护城河菜'。"
