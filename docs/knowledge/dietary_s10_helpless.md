# S10 · 习得性无助 — 从「权威依赖」到「自主回归」

> **典型话语**：「医生每次都警告，每次指标都红，我可能天生就没救了」

---

## 适用识别条件

| 维度 | 判断标准 |
|------|---------|
| TTM阶段 | S0（前意向期）—— 慢性失败状态 |
| 信任分 | not_established（< 25%） |
| BPT6特征 | 自我效能归零 · 外控依赖 · 习得性无助 · 行动路径缺失 |
| 触发关键词 | 没救/天生/没那根筋/反正/做不到/指标都红/医生说/干什么都不行 |

**代谢风险**：长期放弃→HbA1c/血糖/体重持续恶化；「天生没救」叙事→永久性行为停滞；对医疗权威恐惧→回避随访→失去代谢监测窗口

---

## 干预核心逻辑

习得性无助是「无数次失败」积累的神经学结果，不是人格问题。
**目标：不从「应该做什么」出发，而从「你能做什么」出发——把掌控感从权威手里要回来。**
代谢核心：建立第一个「我能做到」的成功体验，激活自我效能的神经通路。

---

## 决策树（6节点，按顺序执行）

### 节点1：权威负荷评估
**触发条件**：用户描述对医疗系统的无助感
```
Agent话术：
「听到'医生的警告'这四个字，
你脑子里出现的第一个感觉是什么？」
→ 测量「医疗权威」的心理负担值
```
**教练行动**：若患者对医疗系统有强烈恐惧/敌意 → 在干预中**不引用**医疗数据，避免激活防御反应

---

### 节点2：终点 vs 地图分离
**触发条件**：用户把「达不到医嘱目标」等同于「失败」
```
Agent话术：
「医生给的是终点，但没给你地图。
你做不到，不代表你没救，
而代表那个目标对你现在不兼容。
我们把医生的命令先放进抽屉5分钟——
不谈'你应该'做什么，我们谈'你能'做什么。」
```
**教练行动**：记录患者说「你应该」vs「我能」的频率比；随干预进展，「我能」频率提升是核心进展指标

---

### 节点3：自主权归还
**触发条件**：用户准备探索「自己能做的事」
```
Agent话术：
「完全自主选择权下，在不考虑任何指标的前提下，
你觉得生活中有哪个微小的、关于健康的小动作，
是你即使闭着眼也能轻松做到的？
咱们从那个'我能做到'的动作开始。」
→ 不提供选项，让用户自己说出来
```
**教练行动**：哪怕用户说的是「每天多喝一杯水」，接受并从这里开始。这是重建自我效能的起点，不能替代。

---

### 节点4：最小成功制造
**触发条件**：用户说出了「闭着眼也能做到」的动作
```
Agent话术：
「好。这周只做这一件事。做到了就是100分。」
→ 明确、具体、可追踪，执行1次即为成功
```
**教练行动**：第一次成功后**立即**正向反馈。这个「第一个成功」是打破习得性无助的神经学拐点，不能跳过、不能推迟。

---

### 节点5：连胜建立（3–7天）
**触发条件**：完成了第一个成功
```
Agent话术：
「这周只做这一件事，做到了就是100分。
不要加新任务。就这一件事，连续7天。」
→ 设计连续成功体验，累计「我能做到」的神经通路
```
**教练行动**：连续3天成功 → 发放「习得性有效」徽章（内部激励）；连续7天成功 → 评估是否准备加第二个动作

---

### 节点6：代谢意义连接
**触发条件**：用户已完成3–5次连续成功后
```
Agent话术：
「你知道你坚持喝水这7天，
对你的血液黏稠度/肾脏代谢意味着什么吗？」
→ 用真实的代谢改善强化动机，不泛泛说「有益健康」
```
**教练行动**：基于已完成动作，提供对应的具体代谢改善证据——具体到「这个动作对你的哪个指标有什么影响」

---

## M-Action（按代谢改善优先级）

| 优先级 | 具体动作 | 代谢机制 |
|--------|---------|---------|
| P1 | **用户自定义第一个动作**（必须由用户自己说出） | 自主选择是重建内控感的核心——任何由教练指派的动作都会延续「外控依赖」 |
| P2 | 喝够2升水（若用户选择） | 最低门槛、最高代谢收益的单一行为；血液流动性改善可在3天内感受到 |
| P3 | 饭前等待5分钟（若用户选择） | 训练延迟满足的最简形式，同时降低进食量约10–15%，不改变任何饮食内容 |
| P4 | 每日精力评分（仅打一个数字，1–10） | 最低负担的自我监测方式，建立「我在追踪我的身体」的主体感 |
| P5 | 连续3天打卡单一动作 | 「连胜体验」是重建自我效能最有效的神经学干预，3天即可开始重塑信念 |

---

## 代谢监测重点

| 指标 | 风险阈值 | 行为健康干预 |
|------|---------|------------|
| HbA1c | ≥ 6.5%（长期无管理） | 习得性无助者HbA1c通常持续上升；第一个目标不是达标，而是「停止上升」 |
| BMI/体重 | 长期缓慢上升趋势 | 不以体重为短期目标——先建立行为基线，再谈代谢改善方向 |
| 血压 | 若持续 ≥ 140/90 须医疗优先 | 习得性无助者常逃避随访；若有代谢危险信号，建立医疗信任关系优先于行为干预 |
| 肾功能（eGFR） | 若已是2型糖尿病需监测 | 长期高血糖+回避随访→肾功能损害可能被忽视，纳入代谢风险排查 |

## 铁律边界 ⚠️

| 禁令 | 原因 |
|------|------|
| ❌ 禁止强化权威 | 「医生也是为了你好，你得听」——进一步加剧「犯错者羞耻感」，强化无助状态 |
| ❌ 禁止廉价鼓励 | 「你一定行，慢慢来」——无事实支撑的鼓励对慢性失败者显得虚伪，立即失去信任 |
| ❌ 禁止目标设定（第一阶段） | 习得性无助者不能承受「再一次设定目标后失败」——第一阶段只做行为，不设目标 |
| ⚕️ CrisisAgent优先 | 「活着没有意义」「反正都一样」的表达 → 立即触发CrisisAgent，心理健康评估优先于代谢管理 |

---

## 叠加情境提示

- **S10 + S02**（习得性无助 + 家庭冲突）：S10优先重建自我效能，S02作为支撑系统干预并行推进
- **S10 + S03**（习得性无助 + 自我否定螺旋）：最严重组合，须教练主动介入，AI仅做辅助陪伴

## 8. 话术变体库（扩展层）

* **低信任度（2~4分）**："你说'可能天生就没救了'——我不打算反驳你，因为你试过很多次，每次都很认真，也每次都很受伤。我今天只问你一件事：这一周里，有没有哪一天、哪一刻，你的身体感觉比其他时候稍微好一点点？就一点点也算。"
* **中信任度（5~6分）**："你已经连续三天做到了你自己选的那一件事——不管那件事有多小，这个'三天'是你的，不是我说的，不是医生要求的，是你自己选的。这就是区别。下一步也是你来定：你觉得第四天想做什么？"
* **高信任度（7~9分）**："你已经建立了一个属于你自己的小系统，不依赖任何人告诉你做什么——这是你三个月前完全不敢想的事。现在我们来做一件事：把你这套系统写下来，用你自己的话，不用任何医学术语。因为下次当你又觉得'没救了'的时候，你需要一个证明——那个证明必须是你自己的语言写的。"
