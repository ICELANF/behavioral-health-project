# S05 · 职场高压 — 身不由己的「职场精英」

> **典型话语**：「早8晚10，三餐靠外卖，你让我去健身房完全不现实」

---

## 适用识别条件

| 维度 | 判断标准 |
|------|---------|
| TTM阶段 | S1–S2（前意向→意向期） |
| 信任分 | building（30–50%） |
| BPT6特征 | 时间管理资源耗尽 · 环境控制受限 · 行为启动成本高 |
| 触发关键词 | 太忙/没时间/早8晚10/外卖/应酬/抽不出时间/拼命换钱/不现实 |

**代谢风险**：长期外卖→高钠高油→血压升高；不规律进餐→血糖波动→胰岛素抵抗；久坐→肌肉量下降→基础代谢率降低

---

## 干预核心逻辑

职场人群的核心问题不是「意愿不足」，而是「行为启动成本过高」。
**策略：微量嵌入与环境借力——不「插入」新任务，而是「置换」旧动作。**
代谢核心：在不占用额外时间的条件下，改善能量代谢效率。

---

## 决策树（6节点，按顺序执行）

### 节点1：现实认可（同频）
**触发条件**：用户强调时间限制
```
Agent话术：
「你的时间表已经被塞满了，
要求你额外拿出1小时管健康确实强人所难。
我们不玩那种'挤时间'的游戏，那太累了。
我们换个思路——不插入新任务，而是置换旧动作。」
```
**教练行动**：不布置「额外任务」；审核行为处方时删除所有需要 > 15分钟额外时间的干预建议

---

### 节点2：行为缝隙扫描
**触发条件**：用户接受「不额外占用时间」的前提
```
Agent话术：
「你每天在公司接水、上厕所、等电梯——
我们从这些缝隙里找动作。
你平均一天用几次洗手间？接几次水？」
→ 绘制职场时间行为地图
```
**教练行动**：协助找到3个「锚点行为」（必然发生的日常动作），作为行为堆叠的基础

---

### 节点3：行为堆叠设计
**触发条件**：确定了锚点行为
```
Agent话术（示例）：
「等电梯时踮脚10下；
电脑开机时喝200ml水（趁着等待的30秒）；
接电话时自动站立行走。
这些动作不需要任何额外时间，
但你的代谢曲线已经悄悄在变化了。」
```
**教练行动**：追踪3个堆叠行为执行率；连续7天 ≥ 70% → 推进下一个堆叠层

---

### 节点4：外卖降级策略
**触发条件**：用户无法避免外卖
```
Agent话术：
「不强求你吃沙拉。下次点餐时只做一个微调：
备注'少油少盐'，并加一份纯素菜。
饭来先吃那份素菜。
你还是在吃外卖，但你的升糖曲线已经因为这个小动作发生了改变。」
```
**教练行动**：提供外卖平台「健康改造」清单：优先选择的品类+降级改造技巧

---

### 节点5：能量管理重构
**触发条件**：用户表现出能量透支状态
```
Agent话术：
「太累时绝对不要强迫去健身。
能量不足时的强制运动会引发身体的炎症反应和情绪上的反弹。
今晚的目标改成高质量睡眠——能量是一切改变的基础。」
```
**教练行动**：当用户报告连续3天能量 ≤ 4分（1–10分）→ 暂停所有运动建议，优先睡眠干预

---

### 节点6：环境工程设计
**触发条件**：用户准备做一次「办公室改造」
```
Agent话术：
「把透明水壶放在视线正前方；
把抽屉零食换成小包装坚果或者干脆清空；
桌上备一根小型阻力带。
改变物理环境，让健康选择成为默认选项。」
```
**教练行动**：提供「办公室代谢友好改造清单」，用户确认可执行项目后追踪执行情况

## M-Action（按代谢改善优先级）

| 优先级 | 具体动作 | 代谢机制 |
|--------|---------|---------|
| P1 | 电脑开机时喝200ml水（堆叠在开机动作上） | 提高日均饮水量，促进代谢，降低血液黏稠度，启动成本接近零 |
| P2 | 接电话时站立行走（堆叠在接电话动作上） | 减少久坐，激活下肢肌肉，改善胰岛素敏感性，每次3–15分钟 |
| P3 | 外卖备注少油少盐+加素菜 | 减少钠摄入，增加膳食纤维，降低餐后血糖峰值15–20% |
| P4 | 午餐先吃蔬菜再吃主食（进食顺序） | 延缓胃排空，降低GI效应，午后血糖曲线更平稳 |
| P5 | 回程提前一站下车步行 | 增加NEAT（非运动性产热），等效于额外消耗约100–150 kcal/天 |

## 代谢监测重点

| 指标 | 风险阈值 | 行为健康干预 |
|------|---------|------------|
| 空腹血糖 | ≥ 5.6 mmol/L 需关注 | 不规律进餐+外卖高GI食物→空腹血糖逐渐上移，是代谢综合征早期信号 |
| 血压 | ≥ 130/80 mmHg | 高钠外卖+久坐+工作应激→血压上升；少油少盐是一线干预 |
| 腰围 | 男>90cm / 女>85cm | 久坐+高热量外卖→内脏脂肪优先堆积；步行堆叠可显著改善 |
| 静息心率 | > 80次/分 需关注 | 长期高应激→交感激活→静息心率升高；步行+睡眠改善是优先干预 |

## 铁律边界 ⚠️

| 禁令 | 原因 |
|------|------|
| ❌ 禁止额外增量 | 「再忙也要挤出时间运动」——在能量透支状态下任何增量都是压榨系统，必然反弹 |
| ❌ 禁止理想化方案 | 要求职场人「回家自己做减脂餐」是脱离现实的干预，会立即导致依从性归零 |
| ❌ 禁止恐吓式教育 | 「不花时间看病，以后花时间住院」——焦虑增加皮质醇，皮质醇升高血糖，适得其反 |
| ⚕️ 能量优先原则 | 能量评分 ≤ 3分时，暂停所有运动干预，先恢复睡眠和基础能量 |

## 8. 话术变体库（扩展层）

* **低信任度（2~4分）**："你每天的工作量听起来真的很重。我不打算让你再加一件事——我们来想想，有没有一个动作，可以嵌入你已经在做的事情里，不需要多花一分钟？"
* **中信任度（5~6分）**："午饭那15分钟是你今天唯一能稍微缓一缓的时间，我们不浪费它——就一个动作：先吃菜，再吃饭。你今天能试一次吗？"
* **高信任度（7~9分）**："你上周的能量评分回升了，说明'步行堆叠'策略有在发挥作用。这周我们加一个动作——午餐后站着消化10分钟，就站着，不用走。你觉得在你的办公室能做到吗？"
