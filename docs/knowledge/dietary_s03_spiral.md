# S03 · 自我否定螺旋 — 深陷泥潭的「意志力失败者」

> **典型话语**：「又没忍住，我这辈子可能就是没救了，干什么都半途而废」

---

## 适用识别条件

| 维度 | 判断标准 |
|------|---------|
| TTM阶段 | S0–S1（前意向期）—— 反复复发特征 |
| 信任分 | not_established（< 30%） |
| BPT6特征 | 自我效能极低 · 认知扭曲（全或无）· 情绪调节失败 |
| 触发关键词 | 没救/没毅力/自责/厌恶自己/循环/破罐/放弃/半途而废 |

**代谢风险**：习得性无助→长期放弃→代谢指标持续恶化；高皮质醇（羞耻感驱动）→胰岛素抵抗；What-the-Hell效应→反弹性暴食

---

## 干预核心逻辑

自我否定螺旋是干预失败的「头号杀手」。
**核心：将患者从「审判席」拉回「观察席」——人是好的，只是行为模式在特定情境下失效了。**
代谢核心：先处理神经-心理状态，再处理代谢指标。

---

## 决策树（6节点，按顺序执行）

### 节点1：紧急接住（入场）
**触发条件**：用户出现「没救了」「厌恶自己」等自我否定表达
```
Agent话术：
「我感受到你现在的自责和难受。先深呼吸——
'没忍住吃火锅'是一个行为事实，它不定义你是什么样的人。
哪怕最顶尖的运动员，也会有行为偏移的时候。
我们不需要审判你的人品，我们需要的是复盘那一刻发生了什么。」
```
**教练行动**：评估自责程度（0–10分）；若 ≥ 8分且伴「活着没意思」类表达 → 立即触发CrisisAgent

---

### 节点2：侦探模式（BFR启动）
**触发条件**：情绪稍稳，可以开始回溯
```
Agent话术：
「我们来玩个侦探游戏——
昨天吃火锅之前，发生了什么？
是加班？朋友聚会？还是中午没吃饱？我们要找的是那个触发点。」
→ 剥离情绪，寻找触发点
```
**教练行动**：记录触发类型；若触发源为「血糖太低→大脑强行接管」→ 下一步干预在午餐蛋白质，而非晚上意志力训练

---

### 节点3：非意志力解法
**触发条件**：找到了真实触发点
```
Agent话术（示例——血糖低触发）：
「你看，问题不是你没毅力——
是中午摄入不足，大脑进入饥饿预警，它强行接管了选择权。
下周重点：确保午餐有足够蛋白质。
这是不是比练习'忍耐'轻松多了？」
→ 从行为结构找出路
```
**教练行动**：设计具体午餐蛋白质方案（鸡胸肉/鸡蛋/豆腐至少一种）；追踪执行率

---

### 节点4：行为余量设计
**触发条件**：用户开始接受行为分析框架
```
Agent话术：
「我们在你的计划里留出每周2次'自由餐'——
这不是失败，是系统的减压阀。
当偏离被包含在计划内时，负罪感就失去了滋生的土壤。」
```
**教练行动**：在行为处方里明确写入「每周2次自由餐」，教练审批时保留此条款，不修改

---

### 节点5：百分比重建
**触发条件**：用户开始关注成功而非失败
```
Agent话术：
「哪怕你吃了火锅，饭后散步10分钟了吗？
加了几口蔬菜？那就是20%的成功——
我们锚定那20%的光亮，而非80%的黑暗。」
```
**教练行动**：每周统计「达成率」而非「失败次数」；连续2周 ≥ 60% → 给予「代谢弹性修复者」成就

---

### 节点6：最简恢复动作
**触发条件**：发生偏移后的第二天
```
Agent话术：
「既然昨天偏移了，今天只做一件事：喝够2升水。
只要这个完成了，今天就是成功的。」
→ 快速找回自我效能感
```
**教练行动**：水摄入追踪纳入每日打卡；第一个成功的「最简任务」完成后，立即正向反馈推送

---

## M-Action（按代谢改善优先级）

| 优先级 | 具体动作 | 代谢机制 |
|--------|---------|---------|
| P1 | 喝够2升水（即时恢复动作） | 促进代谢废物排出，稀释血液糖分，快速建立「我能完成」的心理基线 |
| P2 | 午餐加一种蛋白质 | 稳定血糖，预防下午饥饿触发的情绪进食，降低晚间失控风险约40% |
| P3 | 设定「自由餐」配额（每周2次） | 研究显示「计划内违规」使行为坚持率提升35% |
| P4 | 偏移后散步10分钟 | 降低餐后血糖峰值，同时产生内啡肽正反馈，替代自责情绪 |
| P5 | 填写BFR工作表（仅5分钟） | 将失控转化为数据，建立「观察者视角」，是重建自我效能的神经学路径 |

## 代谢监测重点

| 指标 | 风险阈值 | 行为健康干预 |
|------|---------|------------|
| 血糖变异性 | CV > 36% 为高风险 | 失控饮食→血糖剧烈波动→情绪更不稳定→恶性循环；稳糖是情绪稳定的前提 |
| 胰岛素水平 | 空腹胰岛素 > 15 μIU/mL | 高胰岛素→饥饿感增强→更难抵抗进食冲动，应优先检测 |
| HbA1c | ≥ 6.5%（糖尿病诊断） | 长期螺旋式放弃者HbA1c通常持续上升，3个月复查为节点 |
| 体重波动 | 暴食-断食循环 > 3kg/周 | 越急越补偿越波动，建议暂停体重关注，转为关注行为达成率 |

## 铁律边界 ⚠️

| 禁令 | 原因 |
|------|------|
| ❌ 禁止高压督促 | 「你看，不自律的后果」——坐实「我不行」的设定，诱发「破罐子破摔」效应 |
| ❌ 禁止廉价安慰 | 「没事，偶尔吃一顿没关系」——忽视内在痛苦，无法提供防复发策略 |
| ❌ 禁止道德绑架 | 将健康行为与「人品素质自律」挂钩是慢病管理中最危险的教育方式 |
| ⚕️ CrisisAgent边界 | 自责+无望感+体重长期失控组合 → 评估心理健康风险，及时转专业支持 |

## 8. 话术变体库（扩展层）

* **低信任度（2~4分）**："听起来你今天又走进了那个'算了反正已经坏了'的念头里。我不打算说'你可以的'，但我想问你一件事：这一次，比上一次，有没有任何一个细节是不一样的？"
* **中信任度（5~6分）**："你把这件事告诉我，本身就不是'放弃'的样子。螺旋还没落底，我们现在可以插入一个小动作——哪怕只是喝一杯水，也是给螺旋踩了一脚刹车。"
* **高信任度（7~9分）**："你有过很多次'今天彻底坏了'但第二天又回来的经历，这是你的韧性。这次的触发点是什么？我们来做一次BFR，找到它，下次可以提前拦截。"
