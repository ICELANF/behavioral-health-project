# 行健行为健康促进平台 — 用户角色权益与服务手册

> 版本: v1.0 | 2026-02-13
> 适用对象: 平台所有注册用户
> 性质: 面向用户的服务说明底稿

---

## 前言

行健行为健康促进平台采用"六级四同道者"成长体系，每位用户从注册开始，通过学习、实践、分享、带教逐步成长。本手册详细说明每个角色层级在平台可以享受的服务内容与形式、拥有的权利、承担的责任与义务，以及平台对不同角色发挥的作用。

**核心理念**: 权利与责任对等 —— 每提升一个层级，您获得更丰富的服务与工具，同时也承担更多回馈平台和帮助他人的义务。

---

## 角色层级总览

| 层级 | 角色名称 | 定位 | 核心关键词 |
|------|---------|------|-----------|
| L0 | 观察员 | 新入门的体验者 | 观察、了解、试探 |
| L1 | 成长者 | 积极改变的实践者 | 实践、记录、蜕变 |
| L2 | 分享者 | 乐于助人的传播者 | 分享、启发、传递 |
| L3 | 健康教练 | 专业服务的指导者 | 指导、陪伴、赋能 |
| L4 | 促进师/督导 | 体系建设的推动者 | 设计、培训、治理 |
| L5 | 行为健康促进大师 | 领域标杆的引领者 | 创新、标准、传承 |
| L99 | 系统管理员 | 平台运营的守护者 | 运维、审核、保障 |

# 第一章 L1 成长者（Grower）

> **一句话定位**: 正在积极改变自身行为习惯、用平台工具管理健康的实践者。

## 一、成为成长者的条件

| 条件 | 要求 |
|------|------|
| 前置角色 | 观察员 (L0) |
| 成长积分 | ≥ 100 分 |
| 其他条件 | 无额外要求 |

**如何积累 100 成长分？**
- 每日签到: +5 分/天 (连续签到有加成)
- 完成一节课程: +20 分/次
- 完成评估问卷: +50 分/次
- 使用平台工具: +10 分/次
- 行为尝试记录: +10 分/次 (每日最多 3 次)

> 预计时间: 活跃用户约 7-14 天可达到

## 二、可享受的服务

### 2.1 AI 智能健康助手

| 服务内容 | 具体形式 | 说明 |
|---------|---------|------|
| 12 个专业 AI Agent | 一对一文字对话 | 血糖管理、睡眠健康、压力管理、营养指导、运动健身、心理健康、中医养生、动力激发等 |
| 危机干预 Agent | 即时响应 | 识别情绪危机关键词，自动提供专业支持资源 |
| 食物识别 | 拍照识别 | 上传食物照片，AI 识别食物名称、估算热量、分析营养成分、给出饮食建议 |

## 三、成长者的权利

1. **健康数据所有权**: 您上传的所有健康数据归您所有，可随时查看和导出
2. **隐私保护**: 您的健康数据仅您本人和分配的教练可见，受平台隐私政策保护
3. **AI 服务使用权**: 无限次使用 12 个 AI Agent 进行健康咨询
4. **学习资源访问**: 访问 T1+T2 层级的全部学习资源
5. **评估服务**: 免费使用全部行为评估工具
6. **晋级申请权**: 满足条件后有权申请晋升为分享者
7. **教练服务**: 获得分配教练的一对一指导
8. **激励奖励**: 享受签到、徽章、里程碑等激励体系
9. **问卷匿名权**: 填写问卷时可选择匿名提交
10. **退出自由**: 随时退出任何方案、挑战、学习计划

## 四、成长者的责任与义务

1. **真实记录**: 如实记录健康数据和行为日志，不虚报不造假
2. **积极参与**: 按方案要求完成每日推送的交互问答
3. **持续学习**: 定期参加平台学习内容，提升健康素养
4. **遵守规范**: 遵守社区公约，评论内容文明健康
5. **配合教练**: 积极回应教练的指导建议和沟通
6. **安全告知**: 遇到健康危机时如实向平台和教练反馈
7. **尊重隐私**: 不泄露他人健康数据和个人信息
8. **合理使用**: 不滥用 AI 服务，不测试安全边界

## 五、平台对成长者的作用

| 平台角色 | 具体体现 |
|---------|---------|
| **健康管家** | 7×24 小时 AI 助手随时解答健康疑问 |
| **数据中心** | 统一管理血糖/血压/心率/睡眠/运动等多维健康数据 |

# 第二章 L2 分享者（Sharer）

> **一句话定位**: 有成功经验、乐于帮助他人的健康知识传播者。

## 一、成为分享者的条件

| 条件 | 要求 |
|------|------|
| 前置角色 | 成长者 (L1) |
| 成长积分 | ≥ 500 分 |
| 贡献积分 | ≥ 50 分 |
| 同道者 | 4 名已毕业的 L0 观察员同道者 |

**贡献分获取途径**:
- 投稿内容: +10 分/次 (每日最多 3 次)
- 整理数据: +5 分/次 (每日最多 5 次)
- 发布经验分享: +30 分/次 (每日最多 3 次)

> 预计时间: 持续活跃约 1-3 个月

## 二、在成长者基础上新增的服务

### 2.1 内容创作与分享 (新增)

| 服务内容 | 具体形式 | 说明 |
|---------|---------|------|
| 内容投稿 | 提交文章/经验分享 | 将自己的健康改变经验写成文章投稿 |
| 投稿管理 | 我的投稿列表 | 查看投稿状态 (待审/通过/退回) |
| 投稿修改 | 根据审核反馈修改 | 收到修改建议后调整重新提交 |
| 社区发言 | 评论回复 | 在社区中更积极地回复和帮助新用户 |

### 2.2 升级的学习资源 (扩展)

| 服务内容 | 具体形式 | 说明 |
|---------|---------|------|
| T3 层级内容 | 文章/视频/课程 | 在 T1+T2 基础上新增 T3(成长) 层级学习资源 |
| 深入课程 | 体系化课程 | 开放更深入的行为改变方法论课程 |

**学分体系**: 分享者需累积 800 学分
- M1 (60学分): 行为类型识别、结构化表达、转介信号识别
- M2 (60学分): 五维分享方法论、节律差异认知
- M3 (40学分): 成长故事结构、共情vs投射辨析
- M4 (60学分): 同伴支持技术 (40 学时)、伦理边界、ICF 入门
- 选修 (420学分)

## 三、分享者的权利

**继承成长者全部权利，另新增：**

1. **内容创作权**: 可向平台投稿文章、经验分享
2. **T3 资源访问**: 访问 T1+T2+T3 三级学习资源
3. **同道者带教权**: 可邀请和指导 L0 观察员
4. **社区回复权**: 在社区中可回复其他用户的问题 (每日最多 10 次，+10 贡献分/次)
5. **知识贡献权**: 向平台知识库提交整理后的健康知识
6. **内容传播分**: 分享内容被阅读可获影响力分 (+5/次，每日最多 5 次)

## 四、分享者的责任与义务

**继承成长者全部责任，另新增：**

1. **内容质量**: 投稿内容应真实、准确、有价值，不传播未经证实的健康信息
2. **同道者责任**: 对邀请的同道者承担引导义务，帮助其完成 L0 阶段学习
3. **示范作用**: 以身作则保持良好的健康行为记录
4. **知识传递**: 将自己学到的有效方法分享给更多人
5. **社区贡献**: 积极在社区中回答新用户的疑问
6. **规范传播**: 分享内容不含医疗诊断或处方建议

## 五、平台对分享者的作用

| 平台角色 | 具体体现 |
|---------|---------|
| **创作平台** | 提供内容发布渠道，让经验得到更大传播 |
| **影响力放大器** | 通过推荐算法让好内容触达更多用户 |
| **社交连接** | 建立同道者关系，形成互助成长网络 |
| **进阶引导** | 提供从"学习者"到"传播者"的过渡支持 |

# 第八章 服务层级 (Service Tier) 说明

除角色层级外，平台还根据用户来源和服务套餐提供差异化服务:

## 用户来源

| 来源 | 说明 |
|------|------|
| 自然注册 (organic) | C 端公众自主注册 |
| 教练引荐 (coach_referred) | 由教练推荐注册 |
| 机构服务 (institution) | 医疗机构合作用户 |
| 企业客户 (enterprise) | 企业健康管理用户 |

## 服务套餐

| 套餐 | 包含功能 |
|------|---------|
| **免费版** | 内容浏览、社区阅读、需求调查 |
| **基础版** | + 自助评估、基础学习、工具库、进度追踪 |
| **高级版** | + 完整学习、专家咨询、小组会议、社区互动、AI 伙伴 |
| **VIP 版** | + 180 天课程、私人教练、危机支持、家庭支持 |

> 服务套餐与角色层级互相独立：一位 L1 成长者可能是 VIP 版用户，享受完整功能；一位 L3 教练在角色功能之外还可能拥有企业级服务。
