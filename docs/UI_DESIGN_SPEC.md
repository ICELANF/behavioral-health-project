**版本**: v1.0

## 一、平台概述

### 1.1 产品定位
行为健康管理平台，聚焦糖尿病等慢病患者的行为干预与健康管理。

### 1.2 用户角色

| 角色 | 说明 | 端类型 |
|------|------|--------|
| 患者 | 慢病患者，日常健康数据录入和管理 | H5/小程序 |
| 教练 | 健康管理师，负责患者干预和跟进 | Web管理台 |
| 专家 | 医生/营养师/心理咨询师，提供专业咨询 | Web + Dify |
| 学员 | 教练培训体系中的受训者 | H5/Web |
| 管理员 | 平台运营和系统管理 | Web管理台 |

### 2.2 核心页面详细设计

### 5.2 核心功能设计

#### 5.2.1 案例练习流程

```
1. 选择案例 → 2. 查看患者信息 → 3. 分析数据
      ↓
4. 撰写分析报告 → 5. 制定干预方案 → 6. AI评分
      ↓
7. 查看参考答案 → 8. 导师点评 → 9. 记录档案
```
