## 二、观察员独立通道

观察员（L0）通过单独注册或邀请进入平台，底层数据完整保留。
- **不参与**三维积分计算
- **不参与**同道者匹配（不作为 mentor 或 mentee 出现在普通成长者轨迹里）
- 有独立「转为成长者」申请入口
- H5 ObserverHome 提供健康数据浏览、AI 健康向导体验（3 次免费）

## 三、双轨道晋级设计

晋级由**两条独立轨道**共同决定，必须同时满足才可申请：

### 轨道一：积分轨道（三维积分）

| 积分类型 | 定义 | L1→L2 | L2→L3 | L3→L4 |
|---------|------|-------|-------|-------|
| 成长积分 | 学习课程、完成评估、参与挑战 | ≥200 | ≥800  | ≥2000 |
| 贡献积分 | 投稿被收录、帮助他人 | ≥50  | ≥200  | ≥500  |
| 影响力积分 | 同道者毕业、内容被引用 | ≥20  | ≥50   | ≥150  |

### 轨道二：成长轨道（实践与认证）

| 维度 | L1→L2 | L2→L3 | L3→L4 |
|------|-------|-------|-------|
| 学分 | ≥100 学分 | ≥800 学分 | ≥2000 学分 |
| 同道者 | — | ≥4 名 L1 同道者毕业 | ≥4 名 L2 同道者毕业 |
| 带教质量 | — | 平均评分 ≥3.5 | 平均评分 ≥4.0 |
| 认证考试 | — | 通过 L3 认证考试 | 通过 L4 认证考试 |

## 四、同道者（四同道者版）

每个在轨用户最多同时维系 **4 位同道者**。

### 关系状态机

```
mentor 发起邀请 → PENDING
  ↓ mentee 接受     → ACTIVE（正常带教）
  ↓ mentee 拒绝     → DROPPED（dissolve_reason: rejected_by_mentee）
ACTIVE
  ↓ 自然结束/毕业    → GRADUATED（贡献影响力积分给 mentor）
  ↓ 中断             → DISSOLVED
```

### 关键接口（后端已实现）

| 接口 | 说明 |
|------|------|
| `POST /api/v1/companions/invite` | 发起邀请，创建 PENDING 状态记录 |
| `GET /api/v1/companions/pending-invitations` | 被邀请方查询待处理邀请 |
| `POST /api/v1/companions/invitations/{id}/accept` | 接受邀请 → ACTIVE，mentor +3积分 |
| `POST /api/v1/companions/invitations/{id}/reject` | 拒绝邀请 → DROPPED |
| `GET /api/v1/companions/stats` | 返回毕业数、质量分 |

## 五、晋级申请审核流程（三级）

```
分享者提交晋级申请
      ↓
  [阶段 1] 行为健康教练（L3）审核
      ↓ approve
  [阶段 2] 行为健康促进师（L4）复核
      ↓ approve
  [阶段 3] ≥ 2 名行为健康大师（L5）联合确认
      ↓ 全部 approve
  晋升为行为健康教练（L3） ✅
```

> **当前实现状态**：教练单级审核页面已实现（`pages/coach/promotion/index.vue`）。
> 三级流程需后端 `promotion_applications` 表增加 `review_stage` 字段及前端促进师/大师审核页面（**待实现 P2**）。

**前端页面**：`/pages/trajectory/index`（角色感知，sharer 看到「教练之道」视图）

| 维度 | 检查项 | 未满足时提示 |

| 积分轨道 | 成长≥800 / 贡献≥200 / 影响力≥50 | 「提升成长积分：完成更多课程」等 |
| 学分 | 总学分≥800 | 「继续完成课程学习积累学分」 |
| 同道者 | 已毕业≥4 / 质量≥3.5 | 「需要更多同道者毕业」 |

申请晋级按钮：`POST /api/v1/journey/promotion/apply`
